# <a name="verifying-backup-prerequisites-in-lync-server-2013"></a>Vérification des conditions préalables à la sauvegarde dans Lync Server 2013

_**Dernière modification de la rubrique :** 2013-02-17_

Avant de commencer la sauvegarde de Lync Server, vérifiez que vous êtes préparé comme suit :

  - Outils de sauvegarde. Pour plus d’informations, reportez-vous à la rubrique [Configuration requise pour la sauvegarde et la restauration dans Lync Server 2013 : outils et autorisations](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md).

  - Autorisations. Pour plus d’informations, reportez-vous à la rubrique [Configuration requise pour la sauvegarde et la restauration dans Lync Server 2013 : outils et autorisations](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md).

  - Emplacement pour le stockage des sauvegardes. Pour plus d’informations, reportez-vous à [la rubrique Configuration d’un emplacement de sauvegarde pour Lync Server 2013](lync-server-2013-setting-up-a-backup-location.md).

  - Média pour les sauvegardes. Pour plus d’informations, reportez-vous à [la rubrique Configuration d’un emplacement de sauvegarde pour Lync Server 2013](lync-server-2013-setting-up-a-backup-location.md).
