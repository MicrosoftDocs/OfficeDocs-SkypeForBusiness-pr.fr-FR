# Vérification des conditions préalables de sauvegarde

 

_**Dernière rubrique modifiée :** 2013-02-17_

Avant de commencer la sauvegarde de Lync Server, vérifiez que vous avez préparé les éléments suivants :

  - Outils de sauvegarde. Pour plus d’informations, voir [Spécifications de sauvegarde et de restauration : outils et autorisations](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md).

  - Autorisations. Pour plus d’informations, voir [Spécifications de sauvegarde et de restauration : outils et autorisations](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md).

  - Emplacement pour le stockage des sauvegardes. Pour plus d’informations, voir [Configuration d’un emplacement de sauvegarde](lync-server-2013-setting-up-a-backup-location.md).

  - Média pour les sauvegardes. Pour plus d’informations, voir [Configuration d’un emplacement de sauvegarde](lync-server-2013-setting-up-a-backup-location.md).
