# Application des mises à jour d’Office Communications Server 2007 R2

 

_**Dernière rubrique modifiée :** 2016-12-16_

Avant de procéder à la migration vers Lync Server 2013, vous devez appliquer plusieurs mises à jour à votre environnement Office Communications Server 2007 R2. Pour obtenir les dernières informations sur Office Communications Server 2007 R2, reportez-vous à **Mises à jour pourCommunications Server 2007 R2** à l’adresse [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802).

Pour installer des mises à jour pour Office Communications Server 2007 R2, nous vous recommandons de suivre la procédure **Méthode 1 : Programme d’installation de mise à jour serveur cumulative** décrite dans l’article 968802 de la base de connaissances Microsoft, « Mises à jour pour Communications Server 2007 R2 » à l’adresse [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802).
