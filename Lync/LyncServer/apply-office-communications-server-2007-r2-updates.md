_**Dernière modification de la rubrique :** 2012-10-19_

Avant de procéder à la migration vers Lync Server 2013, les mises à jour doivent être appliquées à votre environnement Office Communications Server 2007 R2. Pour obtenir les dernières informations sur Office Communications Server 2007 R2, consultez la rubrique **mises à jour pour** **Communications Server 2007 R2** à l’adresse [https://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](https://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802) .

Pour installer des mises à jour pour Office Communications Server 2007 R2, nous vous recommandons de suivre la procédure décrite dans l’article 968802 de la base de connaissances pour la **mise à** jour du serveur, décrite dans l’article de la base de connaissances Microsoft, « mises à jour pour Communications Server 2007 R2 », à l’adresse [https://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](https://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802) .
