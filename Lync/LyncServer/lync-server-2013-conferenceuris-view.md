# Vue ConferenceUris

 

_**Dernière rubrique modifiée :** 2015-03-09_

L’affichage ConfernceUris stocke des informations sur les URI qui ont participé à des sessions de conférence. Cette vue est une nouveauté de Microsoft Lync Server 2013.
