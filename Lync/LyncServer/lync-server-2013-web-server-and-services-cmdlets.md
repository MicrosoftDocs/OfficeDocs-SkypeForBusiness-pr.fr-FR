# <a name="web-server-and-services-cmdlets-in-lync-server-2013"></a>Applets de commande de serveur Web et de services dans Lync Server 2013

_**Dernière modification de la rubrique :** 2013-02-18_

De nombreux composants Microsoft Lync Server 2013 sont basés sur le Web : ces composants utilisent des services Web ou des pages Web pour exécuter leurs tâches. Les applets de commande de serveur et de service web vous permettent d’effectuer des tâches, comme configurer des paramètres de serveur web et gérer des URL simples. Les URL simples permettent aux utilisateurs de rejoindre plus facilement des réunions et des conférences, et permettent aux administrateurs de se connecter plus facilement au panneau de configuration Lync Server 2013.
