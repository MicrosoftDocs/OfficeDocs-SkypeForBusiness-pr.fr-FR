# <a name="windows-update-for-lync-server-2013"></a>Mise à jour Windows pour Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2013-12-05_

Recherchez et appliquez fréquemment les mises à jour et les mises à jour de sécurité à l’aide de Windows Update Services. Cela permet d’éviter les vulnérabilités dans d’autres composants système susceptibles de conduire des agresseurs à accéder aux serveurs exécutant Microsoft Lync Server 2013 avec des droits d’administrateur et donc de compromettre Lync Server 2013.

Les mises à jour pour Microsoft SQL Server 2008 Express (64 bits Edition) s’exécutent sur chaque serveur Lync Server 2013 Standard Edition (pour la base de données principale) et sur tous les autres rôles serveur Lync Server 2013 (pour le magasin de configuration local), sauf si vous avez mis à niveau ces bases de données vers SQL Server 2008 R2 Express. Vous devriez considérer ces bases de données dans le cadre de l’activité de maintenance régulière de mise à jour de sécurité, à l’instar de SQL Server sur la base de données principale d’un pool de serveurs frontaux, la base de données de surveillance et la base de données d’archivage.

<div>

## <a name="best-practice"></a>Meilleures pratiques

  - Mettez à jour vos applications à l’aide de Windows Update.
