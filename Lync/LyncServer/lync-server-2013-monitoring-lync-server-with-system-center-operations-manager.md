# <a name="monitoring-lync-server-2013-with-system-center-operations-manager"></a>Surveiller Lync Server 2013 avec System Center Operations Manager

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique:** 2015-05-06_

Le pack d’administration de Lync Server vous permet de surveiller n’importe quel déploiement de Lync Server.

Le panneau de contrôle implémente le journal des événements traditionnel et l’instrumentation basée sur les compteurs de performance, ainsi que l’instrumentation qui vient d’être disponible dans Lync Server, comme les événements de jumelage (échec/réussite) pour plusieurs indicateurs d’intégrité clés, et implémente entièrement le nouveau Transactions synthétiques (applets\* de contrôle Windows PowerShell de test-CS).

Vous pouvez trouver le pack d’administration 2013 de Lync Server et sa documentation [http://go.microsoft.com/fwlink/p/?LinkId=400468](http://go.microsoft.com/fwlink/p/?linkid=400468)associée sur. C’est recommandé si vous exécutez System Center Operations Manager 2012.
