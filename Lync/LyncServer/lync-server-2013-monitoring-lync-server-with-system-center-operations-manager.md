# <a name="monitoring-lync-server-2013-with-system-center-operations-manager"></a>Surveillance de Lync Server 2013 avec System Center Operations Manager

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2015-05-06_

Le pack d’administration Lync Server (MP) est la solution de surveillance de votre choix pour surveiller tout déploiement Lync Server.

Le panneau de contrôle implémente le journal des événements traditionnels et l’instrumentation basée sur les compteurs de performance et active l’instrumentation nouvellement disponible dans Lync Server, comme les événements de paires (échec/réussite) pour plusieurs indicateurs d’intégrité clés et implémente entièrement\* les nouvelles transactions synthétiques (cmdlets test-CS Windows PowerShell).

Vous trouverez le pack d’administration Lync Server 2013 et sa documentation correspondante à [https://go.microsoft.com/fwlink/p/?LinkId=400468](https://go.microsoft.com/fwlink/p/?linkid=400468)l’adresse. Cette option est recommandée si vous exécutez System Center Operations Manager 2012.
