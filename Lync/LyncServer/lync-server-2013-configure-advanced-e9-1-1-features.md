# <a name="configure-advanced-e9-1-1-features-in-lync-server-2013"></a>Configurer les fonctionnalités avancées d’E9-1-1 dans Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-06-06_

Lync Server 2013 inclut les fonctionnalités suivantes que vous pouvez utiliser pour personnaliser votre déploiement de E9-1-1 :

  - Une interface de service web pour connecter le serveur de localisation (LIS, Location Information Server) à une application SNMP.

  - Une interface de service web pour la connexion à une base de données SLS (Secondary Location Source).
