# Configurer les fonctionnalités E9-1-1 avancées dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2012-06-06_

Lync Server 2013 intègre les fonctionnalités suivantes que vous pouvez utiliser pour personnaliser votre déploiement E9-1-1 :

  - Une interface de service web pour connecter le serveur de localisation (LIS, Location Information Server) à une application SNMP.

  - Une interface de service web pour la connexion à une base de données SLS (Secondary Location Source).
