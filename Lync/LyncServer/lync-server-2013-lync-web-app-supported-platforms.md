## <a name="hardware-requirements"></a>Configuration matérielle requise

La configuration matérielle requise de l’ordinateur est déterminée par le système d’exploitation et le navigateur. Les fonctionnalités vocales et de téléphonie nécessitent un micro et des haut-parleurs, un casque avec microphone ou un appareil équivalent compatible avec l’ordinateur. Les fonctionnalités vidéo nécessitent un périphérique vidéo compatible avec l’ordinateur. Pour obtenir des informations détaillées sur la prise en charge du matériel vidéo et la qualité vidéo attendue, voir [Lync client Video Requirements for Lync Server 2013](lync-server-2013-lync-client-video-requirements.md).
