# <a name="migrating-group-chat-servers"></a>Migration des serveurs de conversation de groupe

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-09-29_

Si vous avez déployé le serveur de conversation de groupe dans votre environnement Lync Server 2010 hérité, vous devez déployer le serveur de conversation permanente Lync Server 2013. Le serveur de conversation de groupe et le serveur de conversation permanente peuvent coexister, mais le contenu (par exemple, les salles de conversation, etc.) n’est pas partagé entre ces serveurs. Pour accéder au contenu du serveur de conversation de groupe hérité à partir du serveur de conversation permanente, vous devez migrer le serveur de conversation de groupe vers le serveur de conversation permanente. Pour plus d’informations sur la migration vers le serveur de conversation permanente, voir [migration from Lync server 2010, Group chat ou Office Communications server 2007 R2 Group chat to Lync Server 2013, persistent Chat Server](migration-from-lync-server-2010-group-chat-or-office-communications-server-2007-r2-group-chat-to-lync-server-2013-persistent-chat-server.md) dans la documentation de migration.
