_**Dernière modification de la rubrique :** 2012-10-08_

Les cmdlets de journalisation centralisée permettent aux administrateurs de gérer et de configurer les fonctionnalités de journalisation centralisée introduites dans Microsoft Lync Server 2013. La journalisation centralisée permet aux administrateurs d’activer ou de désactiver simultanément le suivi des événements sur plusieurs ordinateurs.
