# <a name="stop-lync-server-2010-services"></a>Arrêter les services Lync Server 2010

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-10-04_

Vous pouvez utiliser le panneau de configuration Lync Server pour démarrer ou arrêter tous les services Lync Server 2010 en cours d’exécution sur un ordinateur spécifique ou pour démarrer ou arrêter un service Lync Server 2010 spécifique.

<div>

## <a name="to-start-or-stop-all-lync-server-services-on-a-computer"></a>Pour démarrer ou arrêter tous les services Lync Server sur un ordinateur

1.  Ouvrez le Panneau de configuration Lync Server.

2.  Dans la barre de navigation de gauche, cliquez sur **Topologie**, puis sur **État**.

3.  Dans la page **État**, recherchez dans la liste l’ordinateur qui exécute les services que vous souhaitez démarrer ou arrêter, ou triez cette liste, puis cliquez sur cet ordinateur.

4.  Cliquez sur **Action**.

5.  Cliquez sur **Démarrer tous les services** ou **Arrêter tous les services**.

</div>

<div>

## <a name="to-start-or-stop-a-specific-service"></a>Pour démarrer ou arrêter un service spécifique

1.  Ouvrez le Panneau de configuration Lync Server.

2.  Dans la barre de navigation de gauche, cliquez sur **Topologie**, puis sur **État**.

3.  Dans la page **État**, recherchez dans la liste l’ordinateur qui exécute le service que vous souhaitez démarrer ou arrêter, ou triez cette liste, puis cliquez sur cet ordinateur.

4.  Cliquez sur **Propriétés**.

5.  Triez la liste de services, si nécessaire et cliquez sur le service que vous souhaitez démarrer ou arrêter.

6.  Cliquez sur **Action**.

7.  Cliquez sur **Démarrer le service** ou **Arrêter le service**.

8.  Cliquez sur **Fermer**.

</div>
