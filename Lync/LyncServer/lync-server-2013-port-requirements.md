# Configuration requise pour les ports

 

_**Dernière rubrique modifiée :** 2013-03-27_

Lync Server nécessite l’ouverture de ports spécifiques sur le pare-feu. De plus, si la sécurité IPsec (Internet Protocol security) est déployée dans votre organisation, elle doit être désactivée sur la plage de ports utilisée pour l’acheminement des flux audio, vidéo et de vidéo panoramique.
