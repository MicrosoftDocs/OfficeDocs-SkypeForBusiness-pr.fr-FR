# <a name="port-requirements-for-lync-server-2013"></a>Configuration requise pour les ports pour Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2013-03-27_

Lync Server nécessite l’ouverture de ports spécifiques sur le pare-feu. En outre, si la sécurité IPsec (Internet Protocol Security) est déployée dans votre organisation, IPsec doit être désactivé sur la plage de ports utilisée pour la fourniture de l’audio, de la vidéo et de la vidéo panoramique.
