# <a name="deploy-the-online-meeting-add-in-for-lync-2013"></a>Déploiement du complément de réunion en ligne pour Lync 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-09-30_

Déployez le complément de réunion en ligne pour Lync 2013 afin que les utilisateurs puissent planifier des conférences qui prennent en charge la Conférence rendez-vous.

Le complément de réunion en ligne pour Lync 2013 est installé automatiquement lorsque vous installez Lync 2013. Pour plus d’informations sur la personnalisation du complément de réunion en ligne pour Lync 2013 pour Lync, reportez-vous à [la rubrique Configuration de l’invitation à une réunion dans Lync Server 2013](lync-server-2013-configuring-the-meeting-invitation.md).
