# Nouvelles fonctionnalités de surveillance dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2012-11-08_

Lync Server 2013 n’a pas de rôle serveur de surveillance distinct. La surveillance est une fonctionnalité facultative disponible sur tous les serveurs frontaux dans un déploiement Enterprise Edition, ainsi que sur les serveurs Standard Edition, qui peut être mise en œuvre et configurée pour un pool ou un site. Dans Lync Server 2013, vous pouvez désormais activer la mise en miroir de bases de données SQL Server pour votre base de données de surveillance.
