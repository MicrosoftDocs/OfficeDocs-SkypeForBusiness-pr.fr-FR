# <a name="choosing-an-e9-1-1-service-provider-for-lync-server-2013"></a>Choix d’un fournisseur de services E9-1-1 pour Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-06-06_

Le fournisseur de services E9-1-1 achemine les appels d’urgence émanant de Lync Server vers le point de réponse publique (PSAPI) public approprié en fonction des informations d’emplacement contenues dans l’appel.

Pour prendre en charge E9-1-1 dans le cadre d’un déploiement Lync Server, vous devez obtenir un service de routage E9-1-1 à partir d’un fournisseur de services E9-1-1 de Lync Open Interoperability Program. Choisissez le fournisseur qui répond le mieux aux besoins de votre organisation.
