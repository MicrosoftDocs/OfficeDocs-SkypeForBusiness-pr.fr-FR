# Enregistrements DNS requis pour les serveurs Edge et fonctionnalités

 

_**Dernière rubrique modifiée :** 2014-04-08_

Les Lync Server 2013serveurs Edge, les pools de serveurs Edge et les proxys inverses ont des exigences spécifiques pour les enregistrements DNS. Lorsque vous utilisez IPv4 et IPv6 dans Lync Server 2013, vous devez prévoir les enregistrements d’hôte A et AAAA.
