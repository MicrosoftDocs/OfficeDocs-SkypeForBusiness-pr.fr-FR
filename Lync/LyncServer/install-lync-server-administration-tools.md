# <a name="install-lync-server-administration-tools"></a>Installer les outils d’administration Lync Server

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-10-02_

Cette rubrique décrit la procédure d’installation des outils d’administration nécessaires au déploiement et à la gestion de Lync Server 2013. Vous pouvez également installer ces outils sur d’autres ordinateurs, notamment sur des consoles d’administration dédiées.

<div>

## <a name="to-install-the-lync-server-2013-administrative-tools"></a>Pour installer les outils d’administration Lync Server 2013

1.  Sur le serveur frontal Lync Server 2013, exécutez le programme d’installation de Lync Server 2013.

2.  À partir de la page Assistant Déploiement de Lync Server 2013, sélectionnez **installer les outils d’administration**
