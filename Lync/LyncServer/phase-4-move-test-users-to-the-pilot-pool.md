</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-09-26_

Vous pouvez déplacer un ou plusieurs utilisateurs vers votre nouveau déploiement de Microsoft Lync Server 2013 à l’aide des deux méthodes suivantes : Lync Server Control Panel et Lync Server Management Shell. Les rubriques de cette section décrivent les tâches que vous devez effectuer lors du déploiement pilote, ainsi que avant de faire passer votre déploiement de Lync Server 2013 d’un déploiement pilote à un déploiement au niveau de la production.
