# <a name="planning-for-group-call-pickup-in-lync-server-2013"></a>Planification de la prise d’appel de groupe dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2013-02-01_

Mise à jour cumulative pour Lync Server 2013 : février 2013 introduit la prise d’appel de groupe en tant que nouvelle fonctionnalité voix entreprise. La prise d’appel de groupe permet aux utilisateurs de sélectionner des appels qui sonnent pour un autre utilisateur en composant un numéro de groupe de prise d’appel. Cette section comprend des informations de planification spécifiques à la prise d’appel de groupe.
