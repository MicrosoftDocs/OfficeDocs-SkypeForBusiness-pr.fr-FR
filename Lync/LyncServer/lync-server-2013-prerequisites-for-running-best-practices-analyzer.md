# Conditions préalables pour l’exécution de Best Practices Analyzer

 

_**Dernière rubrique modifiée :** 2012-06-25_

Vous pouvez utiliser Best Practices Analyzer pour analyser votre environnement Lync Server 2013 afin d’y détecter des problèmes, pour créer des rapports et pour afficher les résultats d’une analyse actuelle ou précédente. Avant d’exécuter Best Practices Analyzer pour analyser votre environnement, vous devez vérifier que le compte que vous envisagez d’utiliser pour exécuter l’analyse dispose des droits et des autorisations requis et que toutes les conditions préalables d’installation sont respectées.
