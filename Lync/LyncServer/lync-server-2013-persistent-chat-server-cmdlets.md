# <a name="persistent-chat-server-cmdlets-in-lync-server-2013"></a>Applets de commande de serveur de conversation permanente dans Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<span> </span>

_**Dernière modification de la rubrique :** 2012-06-27_

Les applets de commande de conversation permanente vous permettent de gérer et de configurer le service de conversation permanente Microsoft Lync Server 2013 (anciennement appelé service de conversation de groupe). La conversation permanente permet aux utilisateurs de participer à des sessions de messagerie instantanée en ligne. Bien que ces sessions puissent avoir lieu en temps réel, le contenu de chaque session est permanent ; en d’autres termes, ces conversations peuvent être reprises par n’importe qui, à tout moment.
