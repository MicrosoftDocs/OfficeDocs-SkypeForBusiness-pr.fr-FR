# Mise en miroir SQL Server dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2012-09-29_

Établissez la session de mise en miroir SQL Server entre la base de données de conversation permanente principale et son miroir. Pour plus d’informations sur le déploiement de mise en miroir SQL Server, reportez-vous à [Déploiement de la mise en miroir SQL pour la haute disponibilité des serveurs principaux dans Lync Server 2013](lync-server-2013-deploying-sql-mirroring-for-back-end-server-high-availability.md).
