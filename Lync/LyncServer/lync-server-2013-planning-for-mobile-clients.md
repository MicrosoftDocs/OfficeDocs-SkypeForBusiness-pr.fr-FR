# <a name="planning-for-mobile-clients-in-lync-server-2013"></a>Planification des clients mobiles dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2013-04-02_

Les clients mobiles Lync 2013 fournissent une présence améliorée, la messagerie instantanée, les réunions Lync, les appels vocaux et vidéo sur Internet ou la connexion cellulaire. Pour obtenir une matrice qui répertorie les fonctionnalités et les fonctionnalités des clients mobiles et les compare avec le client de bureau, reportez-vous à la rubrique [mobile client Comparison Tables for Lync Server 2013](lync-server-2013-mobile-client-comparison-tables.md).

<div>


> [!NOTE]  
> Lync Server 2013 prend également en charge les clients mobiles Lync 2010. Pour plus d’informations, voir « Planification des clients mobiles » dans la bibliothèque TechNet Lync Server 2010 <A href="http://go.microsoft.com/fwlink/p/?linkid=235955">http://go.microsoft.com/fwlink/p/?LinkID=235955</A>à l’adresse.
