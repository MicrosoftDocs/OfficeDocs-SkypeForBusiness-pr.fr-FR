# <a name="view-current-users-in-lync-server-2010-pool"></a>Afficher les utilisateurs actuels dans le pool Lync Server 2010

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-09-26_

Avant d’apprendre les différentes façons de déplacer des utilisateurs entre des pools, nous devons d’abord déterminer les utilisateurs présents dans le pool Lync Server 2010 hérité. Dans l’image ci-dessous, la colonne pool de serveurs d’inscriptions identifie six utilisateurs configurés pour le pool Lync Server 2010 hérité. Voici les utilisateurs de test que nous allons déplacer vers le pool Lync Server 2013.

**Pour afficher la liste des utilisateurs dans le pool Lync Server 2010**

1.  Ouvrez une session sur le serveur frontal Lync Server 2010 avec un compte membre du groupe RTCUniversalServerAdmins ou du rôle d’administration CsAdministrator ou CsUserAdministrator.

2.  Ouvrez le **Panneau de configuration Lync Server**.

3.  Cliquez sur **Utilisateurs**, sur Rechercher, puis sur **Rechercher**.
