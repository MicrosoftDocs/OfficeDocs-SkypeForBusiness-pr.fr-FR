# <a name="lync-server-2013-best-practices-analyzer"></a>Lync Server 2013 Best Practices Analyzer

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-06-13_

Lync Server 2013, Best Practices Analyzer est un outil de diagnostic qui rassemble les informations de configuration des environnements Lync Server 2013 et détermine si la configuration est définie conformément aux meilleures pratiques de Microsoft.

<div>


> [!NOTE]  
> Lync Server 2013, Best Practices Analyzer analyse et signale les problèmes uniquement avec les composants de Lync Server 2013. Si votre déploiement inclut les composants Lync Server 2010 ou Office Communications Server 2007 R2, utilisez la version précédente de Best Practices Analyzer pour analyser ces composants. Pour plus d’informations, reportez-vous à la rubrique <A href="lync-server-2013-requirements-for-running-best-practices-analyzer.md">Configuration requise pour l’exécution de Best Practices Analyzer dans Lync Server 2013</A>.
