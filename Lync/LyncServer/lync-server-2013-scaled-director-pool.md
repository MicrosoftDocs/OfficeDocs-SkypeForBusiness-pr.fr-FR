# Pool directeur mis à l’échelle dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2012-09-08_

Le directeur peut être déployé dans une configuration basée sur un seul directeur ou un pool de directeurs. Cette section définit une topologie et une configuration pour deux topologies de directeur sous forme de pool. Si vous planifiez le déploiement d’un seul directeur, reportez-vous à la rubrique [Directeur unique dans Lync Server 2013](lync-server-2013-single-director.md) pour plus d’informations sur les éléments à prendre en compte lors de la planification de cette topologie.
