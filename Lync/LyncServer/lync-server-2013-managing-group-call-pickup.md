_**Dernière modification de la rubrique :** 2013-02-22_

Mise à jour cumulative pour Lync Server 2013 : février 2013 introduit la prise d’appel de groupe en tant que nouvelle fonctionnalité voix entreprise. La prise d’appel de groupe permet aux utilisateurs de voix entreprise de choisir les appels qui sonnent pour un autre utilisateur en composant un numéro de groupe de prise d’appel.

Les rubriques de cette section comportent des procédures pas à pas pour les tâches que vous effectuez pour configurer la prise d’appel de groupe dans votre déploiement.
