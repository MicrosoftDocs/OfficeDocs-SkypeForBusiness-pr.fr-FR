# Nouvelles fonctionnalités Voix Entreprise dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2013-05-01_

Lync Server 2013 inclut plusieurs nouvelles fonctionnalité de routage et de gestion des appels qui améliorent Voix Entreprise.

Lync Server 2013 prend en charge plusieurs jonctions entre les serveurs de médiation et les passerelles. Une *jonction* est une association logique entre un numéro de port et le serveur de médiation avec un numéro de port et une passerelle. Cela signifie qu’un serveur de médiation peut avoir plusieurs jonctions vers différentes passerelles, et une passerelle peut avoir plusieurs jonctions vers différents serveurs de médiation. Le routage entre les jonctions permet à Lync Server 2013 d’interconnecter un système IP-PBX à une passerelle RTC ou plusieurs systèmes IP-PBX. Lync Server 2013 sert de liaison (interconnexion) entre les différents systèmes de téléphonie.

Microsoft Lync Server 2013 apporte des améliorations dans les domaines du transfert d’appels, de la sonnerie simultanée de téléphones, de la gestion de la messagerie vocale et de la présentation de l’ID d’appelant. Ces fonctionnalités enrichissent l’expérience d’appel de Voix Entreprise.
