# Affichage des informations relatives aux URL simples

 

_**Dernière rubrique modifiée :** 2012-10-11_

Vous pouvez utiliser le Panneau de configuration Lync Server 2013 pour afficher les détails d’URL simple pour votre environnement Lync Server 2013. Les URL simples permettent aux utilisateurs de prendre part plus facilement à des réunions, et aux administrateurs d’accéder plus aisément aux outils d’administration. Pour plus d’informations, voir [Planification des URL simples dans Lync Server 2013](lync-server-2013-planning-for-simple-urls.md).

## Pour afficher les détails d’URL simple

1.  À partir d’un compte d’utilisateur auquel est affecté un rôle d’administrateur CsServerAdministrator, CsAdministrator, CsHelpDesk ou CsViewOnlyAdministrator pour Lync Server 2013, ouvrez une session sur un ordinateur de votre déploiement interne. Pour plus de détails sur les rôles d’administrateur prédéfinis disponibles dans Lync Server 2013, voir [Planification du contrôle d’accès basé sur un rôle dans Lync Server 2013](lync-server-2013-planning-for-role-based-access-control.md).

2.  Ouvrez une fenêtre de navigateur, puis entrez l’URL d’administration pour ouvrir le Panneau de configuration Lync Server. Pour plus d’informations sur les différentes méthodes de démarrage du Panneau de configuration Lync Server, voir [Ouvrir les outils d’administration Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

3.  Dans la barre de navigation de gauche, cliquez sur **Topologie**, puis sur **URL simple**.

4.  Dans la page **URL simple**, cliquez sur l’en-tête d’une colonne pour trier la liste si nécessaire.

5.  Sélectionnez le nom pour lequel vous souhaitez afficher les détails d’URL simple, puis cliquez sur **Propriétés**.

6.  Lorsque vous avez terminé de consulter les détails, cliquez sur **Fermer**.
