# Création ou modification d’une catégorie dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2012-10-06_

Pour créer une catégorie, reportez-vous à [Configuration des catégories dans Lync Server 2013](lync-server-2013-configure-categories.md) dans la documentation de déploiement. Si vous êtes un administrateur de conversation permanente, vous pouvez créer des catégories à l’aide du Panneau de configuration Lync Server ou des applets de commande Windows PowerShell.
