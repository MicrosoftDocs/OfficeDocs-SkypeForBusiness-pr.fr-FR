# <a name="troubleshooting-persistent-chat-server-configuration-using-windows-powershell-cmdlets-in-lync-server-2013"></a>Résolution des problèmes de configuration du serveur de conversation permanente à l’aide des applets de commande Windows PowerShell dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-10-06_

Utilisez les solutions de contournement suivantes pour résoudre les problèmes de configuration du serveur de conversation permanente avec les applets de commande de l’interface de ligne de commande Windows PowerShell.

  - Si vous obtenez un message d’erreur du type « principal non mis en service », le principal que vous avez essayé d’ajouter n’est pas disponible.
    
    <div>
    

    > [!TIP]  
    > Solution de contournement : aucune. Ce message d’erreur signifie que le principal n’est pas disponible.
