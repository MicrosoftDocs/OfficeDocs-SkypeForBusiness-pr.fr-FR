# Vue NetworkConfigurationSettings

 

_**Dernière rubrique modifiée :** 2015-03-09_

La vue NetworkConfigurationSettings stocke des informations relatives à la configuration réseau. Cette vue est une nouveauté de Microsoft Lync Server 2013.
