_**Dernière modification de la rubrique :** 2012-10-19_

Avant de procéder à la migration vers Lync Server 2013, les mises à jour doivent être appliquées à votre environnement Lync Server 2010. Pour obtenir les informations les plus à jour sur Lync Server 2010, consultez la rubrique **mises à jour du centre de ressources pour Lync** à l’adresse <https://go.microsoft.com/fwlink/p/?linkid=232630> .

Pour installer des mises à jour pour Lync Server 2010, nous vous recommandons de suivre la procédure décrite dans l’article **de la base** de connaissances Microsoft, décrite dans l’article de la base de connaissances Microsoft « mises à jour pour Lync Server 2010 », à l’adresse [https://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=2493736](https://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=2493736) .
