# <a name="checking-for-updates-to-best-practices-analyzer-in-lync-server-2013"></a>Vérification des mises à jour apportées par l’analyseur de meilleures pratiques dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-10-02_

Lorsque vous démarrez le vérificateur de meilleures pratiques, l’outil vous propose une option pour rechercher les dernières mises à jour apportées à l’outil. Si une mise à jour est disponible, vous pouvez télécharger la mise à jour. Si vous choisissez de ne pas télécharger les mises à jour, ou si les meilleurs analyseurs ne sont pas en mesure d’accéder à Internet, vous pouvez continuer à utiliser la version déjà installée sur l’ordinateur.

<div>


> [!NOTE]  
> Si vous avez besoin d’une authentification par proxy pour accéder à Internet, l’analyseur de meilleures pratiques ne peut pas accéder aux nouvelles mises à jour à télécharger. Toutefois, vous pouvez télécharger manuellement la dernière version d’RtcBPA. msi à partir du centre de téléchargement <A href="http://go.microsoft.com/fwlink/p/?linkid=266539">http://go.microsoft.com/fwlink/p/?linkId=266539</A>Microsoft à l’adresse. Après avoir téléchargé le fichier, vous pouvez le copier sur l’ordinateur sur lequel vous voulez mettre à jour les meilleurs analyseurs et utiliser le fichier. msi pour installer la nouvelle version de l’outil sur cet ordinateur.



</div>

Pour mettre à jour les règles de l’analyseur de meilleures pratiques, vous devez exécuter l’outil en tant qu’administrateur sur l’ordinateur local. Si vous n’avez pas ouvert de session à l’aide d’un compte membre du groupe administrateurs et des mises à jour sont détectées, fermez l’analyseur de meilleures pratiques, puis utilisez la procédure suivante pour démarrer le programme.

<div>

## <a name="to-open-best-practices-analyzer-as-administrator-to-check-for-updates"></a>Pour ouvrir l’outil de recherche de meilleures pratiques en tant qu’administrateur pour rechercher les mises à jour

1.  Sur un ordinateur sur lequel le système d’analyse des recommandations est installé, cliquez sur **Démarrer**, pointez sur **Microsoft Lync Server 2013**, cliquez avec le bouton droit sur **Analyseur de meilleures pratiques**, puis cliquez sur **exécuter en tant qu’administrateur**.

2.  Spécifiez les informations d’identification d’un compte membre du groupe administrateurs.
