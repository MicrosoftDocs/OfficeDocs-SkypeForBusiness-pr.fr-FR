## <a name="making-the-best-use-of-the-conference-summary-report"></a><span data-ttu-id="49131-137">Exploiter au mieux le rapport de synthèse de conférence</span><span class="sxs-lookup"><span data-stu-id="49131-137">Making the Best Use of the Conference Summary Report</span></span>

<span data-ttu-id="49131-p104">Les valeurs totales pour la plupart des mesures utilisées sur le rapport de synthèse de conférence se trouvent dans la partie inférieure du rapport.
