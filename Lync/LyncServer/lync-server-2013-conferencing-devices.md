# <a name="conferencing-devices-in-lync-server-2013"></a>Périphériques de conférence dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2014-02-05_

Les périphériques de conférence utilisent les fonctionnalités de Lync Phone Edition dans des salles de conférence, ce qui permet aux personnes de la salle d’écouter et, en fonction de l’appareil, de voir les personnes à d’autres emplacements.

Utilisez les rubriques de cette section pour découvrir comment configurer et gérer vos périphériques de conférence.

<div>


> [!NOTE]  
> Pour plus d’informations sur les périphériques de conférence pris en charge pour une utilisation avec Lync Server 2013, voir la page Web <A href="http://technet.microsoft.com/lync/gg278164.aspx">téléphones et appareils qualifiés pour Microsoft Lync</A>.
