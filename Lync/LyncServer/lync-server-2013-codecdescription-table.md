# <a name="codecdescription-table-in-lync-server-2013"></a>Table CodecDescription dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-10-17_

Le tableau CodecDescription mappe les identificateurs de codecs uniques à leur codec correspondant. Les codecs sont utilisés pour coder les signaux numériques pour la transmission et la diffusion, puis pour décoder ces signaux en vue de leur lecture. Cette table a été introduite dans Microsoft Lync Server 2013
