# Planification et déploiement du magasin de contacts unifié dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2012-06-14_

Le magasin de contacts unifié est une fonctionnalité offrant une expérience de contact cohérente entre les produits Microsoft Office. Cette fonctionnalité permet aux utilisateurs de stocker toutes les informations de contact dans Exchange 2013 pour que ces informations soient disponibles globalement dans Lync, Exchange, Outlook, et Outlook Web Access.
