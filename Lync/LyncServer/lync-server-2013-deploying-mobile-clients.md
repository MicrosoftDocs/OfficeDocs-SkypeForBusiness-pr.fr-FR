# <a name="deploying-mobile-clients-in-lync-server-2013"></a>Déploiement de clients mobiles dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2013-02-19_

Les applications Lync 2013 pour les clients mobiles fournissent une messagerie instantanée, une présence améliorée et la téléphonie pour les utilisateurs de votre organisation qui se connectent à partir d’un smartphone ou d’un téléphone exécutant une édition professionnelle de Windows Mobile. Vous pouvez demander à vos utilisateurs d’installer Lync 2013 en les dirigeant vers le Marketplace de l’application pour leur téléphone mobile.
