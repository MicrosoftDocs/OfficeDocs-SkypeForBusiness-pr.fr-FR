# <a name="whats-new-for-devices-in-lync-server-2013"></a>Nouveautés en termes de périphériques dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique:** 2012-06-22_

Lync Server 2013 inclut Lync Phone Edition, un logiciel qui s’exécute sur les appareils qualifiés et fournit des fonctionnalités de téléphonie traditionnelles et avancées, une sécurité intégrée, une gestion, etc. Le fonctionnement de Lync Phone Edition fonctionne de la même façon que pour Lync Server 2013 comme pour Lync Server 2010. Pour plus d’informations sur les dernières fonctionnalités relatives aux appareils, voir [nouveautés pour les appareils](http://go.microsoft.com/fwlink/p/?linkid=256490) dans la bibliothèque TechNet de Lync Server 2010.
