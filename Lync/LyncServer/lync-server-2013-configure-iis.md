_**Dernière modification de la rubrique :** 2011-12-16_

La configuration des services Internet (IIS) pour Lync Server 2013 implique l’installation des composants appropriés afin de prendre en charge les services Web requis par Lync Server 2013. Pour plus d’informations sur l’installation des services Internet (IIS), voir [configuration IIS dans Lync Server 2013](lync-server-2013-iis-configuration.md). Si vous avez une stratégie pour exécuter l’Assistant Configuration de la sécurité sur les serveurs avant de les mettre en service ou dans le cadre d’une maintenance, consultez la rubrique [réactivation du serveur après que l’Assistant Configuration de la sécurité ferme les ports dans IIS](lync-server-2013-re-activate-server-after-security-configuration-wizard-closes-ports-in-iis.md) pour obtenir des informations sur un effet secondaire de l’exécution de l’Assistant qui fermera les ports sur une configuration IIS Lync Server 2013.
