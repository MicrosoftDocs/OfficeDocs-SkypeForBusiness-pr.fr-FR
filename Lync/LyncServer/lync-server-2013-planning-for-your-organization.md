Les rubriques de cette section vous aident à prendre en main la planification de votre déploiement Lync Server.

<div>

## <a name="in-this-section"></a>Dans cette section

  - [Le choix du mode de déploiement de Lync server 2013](lync-server-2013-deciding-how-to-deploy-microsoft-lync.md) fournit des instructions pour choisir entre ces scénarios de déploiement de base pour Lync Server 2013.

  - [Le processus de planification de Lync Server 2013](lync-server-2013-beginning-the-planning-process.md) vous aide à comprendre comment commencer à planifier un déploiement sur site et comment la documentation de planification fonctionne avec le générateur de topologie.

  - Les concepts de base de la topologie que [vous devez savoir avant la planification de Lync server 2013](lync-server-2013-topology-basics-you-must-know-before-planning.md) décrivent les concepts de base des topologies Lync Server, notamment les sites, les pools de serveurs et les topologies qui prennent en charge la haute disponibilité et la récupération d’urgence.

  - Les [décisions de planification initiales de Lync server 2013](lync-server-2013-initial-planning-decisions.md) vous guident tout au long des questions auxquelles vous devez répondre pour déterminer les charges de travail et les fonctionnalités de Lync Server à déployer.

  - [Clients pour Lync Server 2013](lync-server-2013-clients.md) décrit les différents types de logiciels clients que vous pouvez déployer sur les utilisateurs de votre organisation, notamment les logiciels clients installés sur l’ordinateur, les clients Web et les appareils mobiles.

  - Les [topologies de référence dans Lync Server 2013](lync-server-2013-reference-topologies.md) illustrent trois exemples de topologies illustrant une conception de topologie appropriée dans trois types d’organisations typiques, et expliquent le raisonnement derrière la plupart des décisions de conception de ces topologies.
