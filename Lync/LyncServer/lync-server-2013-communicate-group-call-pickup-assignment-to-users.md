# <a name="communicate-group-call-pickup-assignments-to-users-in-lync-server-2013"></a>Communiquer des attributions de prise d’appel de groupe aux utilisateurs dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2013-01-30_

Une fois que vous avez activé la prise d’appel de groupe pour les utilisateurs, utilisez la messagerie ou un autre mécanisme pour avertir les utilisateurs de leur numéro de groupe de prise d’appel. Avertissez les utilisateurs du numéro de groupe de prise d’appel pour tout groupe qu’ils peuvent souhaiter surveiller. Étant donné que les utilisateurs peuvent récupérer des appels pour d’autres utilisateurs, même s’ils ne se trouvent pas dans le même groupe, les utilisateurs peuvent avoir besoin du numéro de groupe de prise d’appel pour plusieurs groupes.
