# <a name="new-group-call-pickup-feature-in-lync-server-2013"></a>Nouvelle fonctionnalité de prise d’appel de groupe dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2013-02-12_

Mises à jour cumulatives pour Lync Server 2013 : février 2013 introduit la prise d’appel de groupe en tant que nouvelle fonctionnalité voix entreprise. Avec la prise d’appel de groupe, vous pouvez affecter des utilisateurs à des groupes afin que d’autres utilisateurs puissent répondre à des appels entrants à des utilisateurs appartenant au groupe à partir de leurs propres téléphones.

La prise d’appel de groupe est basée sur l’application de parcage d’appel. Pour la prise d’appel de groupe, comme avec le parcage d’appel, vous configurez une plage de numéros virtuels à utiliser en tant que numéros de groupe. Un utilisateur compose le numéro de groupe pour effectuer un appel de sonnerie pour un autre utilisateur qui se trouve dans le groupe.
