# Configuration des clients pour la migration

 

_**Dernière rubrique modifiée :** 2016-12-08_

Cette rubrique recense les étapes de déploiement des clients qu’il est recommandé d’effectuer avant de procéder à la migration vers Lync Server 2013. Ces modifications de configuration doivent être apportées sur Lync Server 2010.

## Pour configurer les clients avant la migration

1.  Déployez les dernières mises à jour (correctifs) des serveurs, clients et appareils pour Lync Server 2010. Pour obtenir les dernières informations sur Lync Server 2010, reportez-vous au Centre de ressources de mises à jour pour Lync à l’adresse <http://go.microsoft.com/fwlink/p/?linkid=232630>.

2.  Sur Lync Server 2010, utilisez le filtrage de la version du client pour autoriser uniquement les clients disposant des mises à jour les plus récentes.
