# Planification des annonces dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2012-09-13_

L’application d’annonceLync Server vous permet de configurer le traitement des appels téléphoniques entrants lorsque le numéro composé est valide pour votre entreprise, mais n’est pas attribué à un utilisateur ou à un téléphone. Vous pouvez transférer ces appels vers une destination prédéterminée (numéro de téléphone, URI SIP ou messagerie vocale), lire une annonce audio, ou les deux. L’application d’annonce permet d’éviter les situations dans lesquelles un appelant compose un mauvais numéro et entend une tonalité de ligne occupée, ou lorsque le client SIP reçoit un message d’erreur. Cette section comporte des informations de planification spécifiques à l’application d’annonce.
