# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Certificats requis pour le serveur de conversation permanente dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-10-03_

Pour installer le serveur Chat permanent, vous devez disposer d’un certificat émis par la même autorité de certification que celle utilisée par les serveurs internes Lync Server 2013 pour chaque serveur exécutant les services Web chat permanent pour le téléchargement/téléchargement de fichiers. Vérifiez que vous disposez des certificats requis avant de démarrer l’installation de chat permanent, en particulier si vous utilisez une autorité de certification externe.
