# Suppression de Lync pour iPhone et iPad

 

_**Dernière rubrique modifiée :** 2013-02-19_

Pour supprimer Lync 2013 pour iPhone ou Microsoft Lync 2010 pour iPad de l’appareil, procédez comme suit :

1.  Sur l’écran d’accueil de l’appareil mobile, tapez sur l’icône Lync et maintenez appuyé.

2.  Lorsque les icônes commencent à vibrer et que le symbole **X** apparaît, tapez sur **X** pour supprimer l’application.
