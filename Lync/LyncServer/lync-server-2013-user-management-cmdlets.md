Les cmdlets de gestion des utilisateurs incluses dans Microsoft Lync Server 2013 vous permettent d’activer, de désactiver et de modifier des comptes d’utilisateurs Lync Server.

<div>

## <a name="user-management-cmdlets"></a>Applets de commande de gestion des utilisateurs

La plupart des tâches de gestion qui s’appliquent aux utilisateurs et aux comptes d’utilisateur peuvent être effectuées à partir du panneau de configuration Lync Server. Les principales exceptions concernent les applets de commande utilisables avec les fournisseurs de services d’audioconférence. Les tâches de gestion des utilisateurs peuvent être réalisées à l’aide d’applets de commande de Lync Server Management Shell ou à partir d’un script. En utilisant un script, vous pouvez automatiser certaines tâches. Vous trouverez ci-dessous une liste des applets de commande qui sont directement liées à la gestion des utilisateurs et des comptes d’utilisateur :
