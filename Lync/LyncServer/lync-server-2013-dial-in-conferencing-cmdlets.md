# <a name="dial-in-conferencing-cmdlets-in-lync-server-2013"></a>Cmdlets de conférence rendez-vous dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique:** 2012-03-21_

Les conférences rendez-vous permettent aux utilisateurs d’utiliser un téléphone «ordinaire» (c’est-à-dire, un appareil sur le réseau téléphonique commuté public) pour accéder à la partie audio d’une conférence.

<div>

## <a name="dial-in-conferencing-cmdlets"></a>Cmdlets de conférence rendez-vous

Si vous ne voulez pas autoriser les conférences rendez-vous, vous pouvez désactiver cette fonctionnalité à l’aide de l’applet de connexion Set-CsConferencingPolicy et définir la propriété EnableDialInConferencing sur false. Par défaut, tous les utilisateurs sont autorisés à organiser des réunions à l’aide de conférences rendez-vous.
