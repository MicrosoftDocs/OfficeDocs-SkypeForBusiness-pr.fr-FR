# <a name="phones-and-devices-cmdlets-in-lync-server-2013"></a>Applets de commande de téléphones et de périphériques dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-06-28_

Microsoft Lync Server 2013 fournit un certain nombre d’applets de commande qui vous permettent de gérer les téléphones et les autres périphériques matériels. Cela inclut notamment les téléphones voix sur IP (VoIP); téléphones de partie commune (par exemple, un téléphone dans une salle d’attente de bâtiment, une cafétéria ou un autre lieu public); et même les téléphones analogiques, les téléphones qui ne sont pas en mesure d’exécuter Lync Phone Edition.

<div>

## <a name="phones-and-devices-cmdlets"></a>Applets de commande pour les téléphones et les appareils

Les applets de commande **CsDeviceUpdate** sont utilisées pour gérer le service Web de mise à jour des périphériques, un composant Lync Server qui permet aux administrateurs de distribuer des mises à jour du microprogramme vers des téléphones et d’autres appareils exécutant Lync Phone Edition.
