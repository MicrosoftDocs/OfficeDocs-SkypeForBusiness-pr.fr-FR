# Gestion des groupes Response Group dans Lync Server 2013 lors d’un sinistre

 

_**Dernière rubrique modifiée :** 2012-11-01_

Lync Server 2013 prend en charge l’exécution de groupes de réponse dans le pool de sauvegarde durant la récupération d’urgence. Cette section décrit comment planifier les groupes de réponse durant une panne, le fonctionnement des groupes de réponse durant la panne et les étapes nécessaires au basculement et à la restauration automatique des groupes de réponse.
