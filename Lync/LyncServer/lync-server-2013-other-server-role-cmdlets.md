# <a name="other-server-role-cmdlets-in-lync-server-2013"></a>Autres cmdlets de rôle de serveur dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-03-21_

Ces applets de action permettent de gérer les rôles serveur « ponctuels » (par exemple, le serveur de conférence) qui ne tiennent pas dans les catégories de rôles de serveur, telles que les serveurs de périphérie ou les serveurs du carnet d’adresses.

**Autres rôles de serveur**

<!-- end list -->

  - <span></span>
