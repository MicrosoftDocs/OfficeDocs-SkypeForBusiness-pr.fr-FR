# <a name="lync-server-2013-hybrid"></a>Lync Server 2013 hybride

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique:** 2016-05-25_

Grâce aux déploiements hybrides Lync, vous pouvez disposer de certains de vos utilisateurs Lync en local et d’autres utilisateurs de Lync Online, qui partagent le même domaine. Cela peut simplifier la fourniture de services Lync aux utilisateurs de votre organisation à partir de différents emplacements géographiques ou utilisateurs qui se connectent à distance. Vous pouvez également tirer parti de configurations hybrides Lync en tant que chemin de migration vers Office 365.

Cette section décrit les configurations hybrides pour Lync Server 2013.
