# Affichage et utilisation des rapports créés par Best Practices Analyzer

 

_**Dernière rubrique modifiée :** 2012-06-14_

Quand vous utilisez l’outil Best Practices Analyzer pour analyser votre environnement, Best Practices Analyzer crée des rapports qui identifient les problèmes et autres informations spécifiques à votre déploiement. Vous pouvez utiliser l’outil Best Practices Analyzer pour afficher les rapports qui contiennent les résultats de l’analyse et comprendre les problèmes identifiés dans ces rapports.
