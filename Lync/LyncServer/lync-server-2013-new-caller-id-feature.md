# Nouvelle fonctionnalité ID de l’appelant dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2012-10-05_

Lync Server 2013 permet à l’administrateur de modifier le format du numéro de téléphone de l’appelant. Grâce à cette fonctionnalité de présentation de l’ID de l’appelant, l’administrateur peut convertir le numéro de téléphone de l’appelant en un format de numérotation compris par l’homologue de jonction, le cas échéant. Par exemple, vous pouvez rédiger une règle pour supprimer la valeur « +44 » au début de la chaîne de numérotation et la remplacer par « 0144 ».

Désormais, avec Lync Server 2013, le numéro de téléphone de l’appelant et le numéro de téléphone de l’appelé peuvent être convertis en différents formats, selon les besoins. Cette flexibilité permet à Lync Server 2013 de servir de convertisseur de jonction entre différents systèmes téléphoniques.
