# <a name="database-and-management-server-cmdlets-in-lync-server-2013"></a>Cmdlets de serveur de base de données et de gestion dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-10-09_

Les cmdlets de serveur de base de données et de gestion sont utilisées pour gérer les bases de données principales de Microsoft Lync Server 2013 et vos services de gestion frontaux. Vous pouvez utiliser ces applets de commande pour installer ou désinstaller des bases de données utilisées par Lync Server 2013, en plus de la configuration du point de contrôle du service Active Directory pour le magasin central de gestion.
