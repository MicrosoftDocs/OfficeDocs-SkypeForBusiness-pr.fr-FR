# Préparation des services de domaine Active Directory pour Lync Server 2013

 

_**Dernière rubrique modifiée :** 2013-02-21_

Avant de déployer et de faire fonctionner Lync Server 2013, vous devez préparer les services de domaine Active Directory en développant le schéma, puis en créant et en configurant des objets. Les extensions de schéma ajoutent les classes et les attributs Active Directory requis par Lync Server.

Les rubriques de cette section expliquent comment préparer les services de domaine Active Directory pour le déploiement de Lync Server et comment attribuer les autorisations d’installation et d’unité d’organisation. Pour plus d’informations sur les modifications du schéma requises pour Lync Server, reportez-vous à [Attributs, classes et extensions des schémas Active Directory utilisés par Lync Server 2013](lync-server-2013-active-directory-schema-extensions-classes-and-attributes-used-by-lync-server.md).
