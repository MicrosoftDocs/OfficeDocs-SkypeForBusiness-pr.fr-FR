# <a name="media-configuration-cmdlets-in-lync-server-2013"></a>Applets de configuration de média dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-03-21_

Les applets de passe relatifs à la configuration de média vous permettent de spécifier des éléments tels que le niveau de cryptage pris en charge et la résolution vidéo maximale autorisée.

<div>

## <a name="media-configuration-cmdlets"></a>Applets de configuration de média

Utilisez les applets de commande suivantes pour la configuration de média.

**Configuration de média**

  - <span></span>  
    [Get-CsMediaConfiguration](https://technet.microsoft.com/en-us/library/Gg398128(v=OCS.15))

  - <span></span>  
    [Nouveau-CsMediaConfiguration](https://technet.microsoft.com/en-us/library/Gg425881(v=OCS.15))

  - <span></span>  
    [Remove-CsMediaConfiguration](https://technet.microsoft.com/en-us/library/Gg398705(v=OCS.15))

  - <span></span>  
    [Set-CsMediaConfiguration](https://technet.microsoft.com/en-us/library/Gg398580(v=OCS.15))

</div>
