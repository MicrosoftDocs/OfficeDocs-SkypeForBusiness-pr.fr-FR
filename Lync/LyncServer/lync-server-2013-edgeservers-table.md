# Table EdgeServers dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2015-03-09_

La table EdgeServers est une table de prise en charge. Chaque enregistrement contient des informations sur un serveur Edge qui est impliqué dans des appels possédant des enregistrements dans la base de données.
