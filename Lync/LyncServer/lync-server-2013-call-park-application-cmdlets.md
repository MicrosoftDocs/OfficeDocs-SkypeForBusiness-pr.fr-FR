# <a name="call-park-application-cmdlets-in-lync-server-2013"></a>Cmdlets d’application de parcage d’appel dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-03-21_

L’application de parcage d’appel permet à un utilisateur de mettre un appel en attente, puis de récupérer cet appel à partir d’un autre téléphone. Utilisez ces applets de commande pour configurer les paramètres des orbites de parcage d’appel et de l’application de parcage d’appel.

<div>

## <a name="call-park-application-cmdlets"></a>Applets de commande de l’application de parcage d’appel

Les applets de commande suivantes peuvent être utilisées pour gérer l’application de parcage d’appel.

**Application de parcage d’appel**

  - <span></span>  
    [Get-CsCallParkOrbit](https://technet.microsoft.com/library/Gg398554(v=OCS.15))

  - <span></span>  
    [New-CsCallParkOrbit](https://technet.microsoft.com/library/Gg398936(v=OCS.15))

  - <span></span>  
    [Remove-CsCallParkOrbit](https://technet.microsoft.com/library/Gg412901(v=OCS.15))

  - <span></span>  
    [Set-CsCallParkOrbit](https://technet.microsoft.com/library/Gg398796(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [Set-CsCallParkServiceMusicOnHoldFile](https://technet.microsoft.com/library/Gg412836(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [Get-CsCpsConfiguration](https://technet.microsoft.com/library/Gg398948(v=OCS.15))

  - <span></span>  
    [New-CsCpsConfiguration](https://technet.microsoft.com/library/Gg412919(v=OCS.15))

  - <span></span>  
    [Remove-CsCpsConfiguration](https://technet.microsoft.com/library/Gg398358(v=OCS.15))

  - <span></span>
