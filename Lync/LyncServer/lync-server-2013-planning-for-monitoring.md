# <a name="planning-for-monitoring-in-lync-server-2013"></a>Planification de la surveillance dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-09-05_

Le service de surveillance de Microsoft Lync Server 2013 permet aux administrateurs de collecter les données d’utilisation, de tendance et de qualité de service pour les sessions de communication qui ont lieu au sein de leur organisation. La surveillance dans Lync Server 2013 ne requiert plus de rôle serveur distinct ; au lieu de cela, le service de surveillance est intégré à chaque serveur frontal. Toutefois, la surveillance par défaut n’est pas activée dans Lync Server 2013. Ce document vous aidera à déterminer si vous devez ou non activer la surveillance dans votre organisation.
