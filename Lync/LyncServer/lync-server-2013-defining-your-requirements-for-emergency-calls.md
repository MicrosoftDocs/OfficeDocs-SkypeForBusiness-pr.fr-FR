# Définition de la configuration requise pour les appels d’urgence dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2015-03-09_

Avant de commencer un déploiement E9-1-1 Microsoft Lync Server 2013, vous devez d’abord répondre aux questions décrites en détail dans les sections suivantes. La planification dont vous avez besoin dépend du type de solution E9-1-1 que vous choisissez de déployer : fournisseur de services E9-1-1 de jonction SIP ou numéro ELIN (Emergency Location Identification Number). Le tableau ci-dessous identifie les sections du document de planification que vous devez consulter pour chacune de ces solutions.
