# <a name="managing-the-lync-server-2013-network-infrastructure"></a>Gestion de l’infrastructure réseau Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2014-02-11_

Microsoft Lync Server 2013 inclut la prise en charge du contrôle d’admission des appels (CAC) et de la déviation du trafic multimédia. Pour implémenter ces fonctionnalités, vous devez configurer un réseau de régions, de sites, de sous-réseaux, et ainsi de suite, qui vous permettra de gérer la bande passante dans les situations dans lesquelles les transmissions audio et vidéo doivent être restreintes. Vous pouvez également utiliser la technologie de mise en réseau QoS (Quality of service) pour obtenir une expérience utilisateur final optimale pour les communications audio et vidéo.

Vous pouvez utiliser le panneau de configuration Lync Server pour configurer et gérer le contrôle d’admission des médias, la déviation du trafic multimédia et la qualité de service. Les rubriques suivantes expliquent comment procéder.
