# <a name="new-conferencing-features-in-lync-server-2013"></a>Nouvelles fonctionnalités de conférence dans Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2012-11-08_

Lync Server 2013 introduit plusieurs nouvelles fonctionnalités qui améliorent la Conférence, comme décrit dans la liste suivante.

  - **Lanceur de participation**
    
    Lync Server 2013 met à jour le lanceur de participation afin de valider chaque réunion avant de lancer un client et de prendre en charge l’ouverture d’une réunion dans les clients suivants :
    
      - Windows Phone 7
    
      - Périphériques Android
    
      - Périphériques iOS Apple
    
      - Windows 8
    
      - Internet Explorer 10

  - **Partage de PowerPoint mis à jour**
    
    Lync Server 2013 utilise désormais Office Web Apps et Office Web Apps Server (anciennement appelé WAC Server) pour gérer les présentations PowerPoint. L’utilisation d’Office Web Apps Server permet des affichages à haute résolution et une meilleure prise en charge des fonctionnalités PowerPoint, ainsi que l’accès à d’autres types d’appareils mobiles (Lync Server 2013 utilise le langage DHTML et JavaScript standard pour diffuser des présentations PowerPoint), et le possibilité pour les utilisateurs disposant des privilèges appropriés de faire défiler une présentation PowerPoint indépendamment de la présentation proprement dite.

  - **Mode d’affichage Galerie et conférence en vidéo haute définition**
    
    Dans les conférences vidéo, les utilisateurs peuvent voir jusqu’à 5 participants simultanément sur une vidéo.
    
    <div>
    

    > [!NOTE]  
    > Le mode d’affichage Galerie est conçu pour les conférences regroupant jusqu’à 75 participants. Lorsque le nombre de participant est plus élevé, l’affichage revient à une vue unique.

    
    </div>

  - **Vidéo HD**
    
    Les utilisateurs peuvent définir une qualité d’affichage allant jusqu’à 1080 P HD dans un appel entre deux interlocuteurs et des conférences de groupes.

  - **Mode vidéo Présentateur uniquement**
    
    Les présentateurs peuvent configurer la conférence pour que seule la vidéo du présentateur soit affichée. Ce mode empêche les participants des grandes conférences d’être distraits lorsque plusieurs flux vidéo sont disponibles et verrouillés sur différentes sources. Ce mode s’applique également aux vidéos capturées et fournies par les périphériques de conférence.

  - **Vidéo à la une**
    
    Les présentateurs peuvent configurer la conférence pour que seule la vidéo d’un participant sélectionné qui est une source de vidéo soit vue par chaque participant de la conférence. Ce mode s’applique également à la vidéo capturée et fournie par les périphériques de conférence pour la vidéo panoramique.

  - **mise en conférence par téléphone pour les personnes n’utilisant pas Voix Entreprise**
    
    Lync Server 2013 permet désormais aux participants pour lesquels la fonctionnalité voix entreprise n’est pas activée de lancer des appels entrants à partir d’une conférence rendez-vous. Cette fonctionnalité peut être configurée par l’administrateur.

  - **Archivage**
    
    Tout document partagé au cours d’une conférence est archivé dans le stockage de données Exchange 2013 si l’intégration d’Exchange Server est activée avec l’archivage. Cela inclut les présentations PowerPoint, les pièces-jointes, les tableaux blancs et les sondages.

  - **Personnalisation de l’invitation à la réunion**
    
    Les administrateurs peuvent personnaliser les invitations par courrier électronique pour les réunions en ligne à l’aide du panneau de configuration Lync Server ou de Lync Server Management Shell. Les personnalisations portent sur les URL pour les logos, les textes d’aide, juridique et de pieds de page. Les personnalisations seront inclues sur toutes les invitations suivantes.
