# Configuration des autorisations pour l’archivage

 

_**Dernière rubrique modifiée :** 2012-10-01_

Dans Lync Server 2013, certaines tâches exigent encore que les utilisateurs qui exécutent ces tâches soient membres d’un ou plusieurs groupes spécifiques. Toutefois, vous pouvez également utiliser le contrôle d’accès en fonction du rôle (RBAC) pour accorder des privilèges en affectant les utilisateurs à des rôles d’administration Lync Server prédéfinis. Avant de commencer à déployer l’archivage, assurez-vous que les autorisations et les droits d’utilisateur appropriés sont définis et que tout utilisateur que vous souhaitez affecter à un rôle RBAC spécifique a été affecté à ce rôle. Pour plus d’informations sur les droits d’utilisateur, les autorisations et les rôles en vue du déploiement de la prise en charge de l’archivage, voir [Liste de vérification du déploiement pour l’archivage dans Lync Server 2013](lync-server-2013-deployment-checklist-for-archiving.md) dans la documentation de planification et de déploiement. Pour plus d’informations sur RBAC, voir [Planification du contrôle d’accès basé sur un rôle dans Lync Server 2013](lync-server-2013-planning-for-role-based-access-control.md) dans la documentation de planification.
