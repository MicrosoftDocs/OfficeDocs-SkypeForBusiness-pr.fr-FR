# Ajout d’une authentification Kerberos à d’autres sites dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2010-11-03_

Si vous devez ajouter des sites à un déploiement que vous avez configuré pour l’authentification Kerberos, vous pouvez utiliser un compte d’authentification Kerberos pour le nouveau site ou créer un compte. Pour plus d’informations sur la création d’un compte pour un site, reportez-vous à [Création d’un compte d’authentification Kerberos dans Lync Server 2013](lync-server-2013-create-a-kerberos-authentication-account.md). Pour plus d’informations sur l’utilisation du même compte pour un nouveau site en attribuant un compte au site, reportez-vous à [Attribution d’un compte d’authentification Kerberos sur un site dans Lync Server 2013](lync-server-2013-assign-a-kerberos-authentication-account-to-a-site.md).
