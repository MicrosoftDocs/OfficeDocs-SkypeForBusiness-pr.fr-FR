# <a name="lync-server-2013-stress-and-performance-tool"></a>Outil de contrainte et de performances de Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Dernière modification de la rubrique :** 2013-01-25_

L’outil stress and performance de Lync Server 2013 inclut des outils qui simplifient la planification de la capacité pour Lync Server 2013. L’outil stress and performance de Lync Server 2013 vous permet d’effectuer les opérations suivantes :

  - Simplifiez votre planification matérielle pour Lync Server 2013.

  - Vous fournir des connaissances et meilleures pratiques accrues pour le réglage des performances.

  - Mesurez les performances de vos déploiements Lync Server 2013 prévus.
