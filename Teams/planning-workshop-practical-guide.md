<a name="planning-workshop-kit-and-practical-guide-for-microsoft-teams"></a>Kit d'atelier de planification et guide pratique pour Microsoft Teams
=============================================================

Téléchargez les outils de planification et le guide pratique suivants pour vous orienter dans le lancement de Teams dans votre organisation.

- [Questionnaire préalable à l'atelier de travail sur Microsoft Teams](https://www.microsoft.com/en-us/download/55975)
    
    Le document Microsoft Teams Pre-workshop Questionnaire.docx est destiné à être utilisé pour collecter des informations de base sur l'environnement Office 365 actuel d'une organisation, et permettre au responsable de l'atelier de personnaliser le contenu avant de tenir un atelier de planification Microsoft Teams.

- [Atelier de planification pour Microsoft Teams](https://www.microsoft.com/en-us/download/55982) 
    
    Le document Planning Workshop for Microsoft Teams.pptx est destiné à être utilisé par une ressource informée pour mener une conversation interactive et aider à prendre en main Microsoft Teams. L'atelier présentera Microsoft Teams aux participants et les orientera vers les points de décision et les étapes suivantes.

- [Résumé des points de décision relatifs à l'atelier de travail sur Microsoft Teams ](https://www.microsoft.com/en-us/download/55981)
    
    Le document Microsoft Teams Workshop Decision Points Summary.docx est un actif unique permettant de suivre tous les points de décision et les étapes suivantes de l'atelier de planification pour Microsoft Teams et le contenu d'aide associé.
