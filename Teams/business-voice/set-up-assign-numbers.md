# <a name="step-5-assign-business-voice-phone-numbers-to-your-users"></a>Étape 5 : affecter des numéros de téléphone Voix Entreprise à vos utilisateurs

Avant que les utilisateurs ne Teams utiliser des numéros de téléphone pour effectuer ou recevoir des appels vers ou à partir de lignes téléphoniques normales, vous devez leur affecter des numéros de téléphone. Dans Microsoft Teams, le numéro de téléphone que vous affectez est répertorié lorsqu'un utilisateur clique sur **Appels.** Pour chaque utilisateur qui a besoin d'un numéro de téléphone, vous pouvez suivre les étapes suivantes.

![Numéro de téléphone de l'utilisateur affiché dans Teams.](../media/teams-phone-number.png)

> [!NOTE]
> Si vous ne voyez aucun numéro de téléphone, veuillez patienter. Plusieurs heures peuvent être avant que vos nouveaux numéros de téléphone deviennent disponibles dans Teams.

1. Dans le navigateur de gauche, cliquez **sur Numéros**  >  **Téléphone.**
2. Dans la page **Téléphone numéros** de téléphone, sélectionnez un numéro non publié dans la liste, puis cliquez sur **Modifier.**  
3. Dans le **volet Édition,** sous **Affecté à,** recherchez l'utilisateur par nom d'affichage ou nom d'utilisateur, puis cliquez sur **Attribuer.**
4. Sous Emplacement d'urgence, vous pouvez sélectionner l'emplacement d'urgence que vous avez ajouté à l'étape Configurer les emplacements d'urgence ou si vous devez créer un nouvel emplacement pour un autre bureau ou un bureau à domicile, cliquez sur Ajouter un **emplacement.**  [](set-up-emergency-locations.md)
5. **Assurez-vous que l'utilisateur de** courrier avec des informations de numéro de téléphone est sélectionné pour recevoir un e-mail de bienvenue avec son nouveau numéro de téléphone.
6. Cliquez sur **Enregistrer**.
7. Répétez les étapes ci-dessus pour chaque utilisateur auquel vous souhaitez attribuer un numéro de téléphone.

> [!NOTE]
> En raison de la latence entre Microsoft 365 ou Office 365 et Teams, l'activé peut prendre jusqu'à 24 heures pour les utilisateurs. Si le numéro de téléphone n'est pas attribué correctement après 24 heures, contactez le support technique pour les produits pour les entreprises [- Aide de l'administrateur.](/microsoft-365/admin/contact-support-for-business-products) Nous sommes là pour vous aider !
