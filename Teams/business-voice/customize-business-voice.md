# <a name="customize-teams-phone-system-with-calling-plan"></a>Personnaliser des Teams Système téléphonique’aide d’un forfait d’appels

Le [Teams Système téléphonique](set-up-overview.md) guide de configuration du plan d’appels vous donne le Système téléphonique avec l’offre groupée Forfait d’appels démarrée avec des options de base pour vous aider à l’utiliser immédiatement. Il est possible que ces options soient suffisantes si vous êtes une petite entreprise ou si vous dirigez un programme pilote au sein d’une organisation de plus grande taille. Vous obtenez les concepts de base avec de nouveaux numéros de téléphone et un menu d’appel.

Il est toutefois probable que vous voudrez personnaliser Système téléphonique’aide de l’forfait d’appels pour mieux répondre aux besoins de votre entreprise. Par exemple, vous souhaiterez probablement déplacer vos numéros de téléphone existants (appelés portage de numéros) Système téléphonique’aide d’un plan d’appels.

| Tâche        | Description       |
|-------------|-------------------|
| [Déplacer des numéros de téléphone vers Système téléphonique’aide de l’forfait d’appels](port-phone-numbers.md) | Apportez vos numéros de téléphone existants Système téléphonique’aide du forfait d’appels. |
| [Configurer vos stratégies d’appel](set-up-policies.md) | Déterminez si les utilisateurs peuvent faire certaines actions telles que passer des appels privés, utiliser le transfert d’appel et déléguer des appels. |
| [Créer des utilisateurs supplémentaires](create-users.md) | Ajoutez des utilisateurs et attribuez-leur des numéros de téléphone nouveaux ou existants. |
| [Configurer des files d’attente d’appels](./create-a-phone-system-call-queue-smb.md) | Créez une file d’attente dans laquelle les appelants peuvent attendre l’agent disponible suivant. |
