# <a name="customize-microsoft-365-teams-phone-with-calling-plan"></a>Personnaliser votre Microsoft 365 Teams Téléphone’aide d’une forfait d’appels

Le [Teams Téléphone guide de](set-up-overview.md) configuration du plan d’appels permet de configurer une Teams Téléphone’aide d’un plan d’appels avec des options de base pour vous aider à l’utiliser immédiatement. Il est possible que ces options soient suffisantes si vous êtes une petite entreprise ou si vous dirigez un programme pilote au sein d’une organisation de plus grande taille. Vous obtenez les concepts de base avec de nouveaux numéros de téléphone et un menu d’appel.

Il est toutefois probable que vous voudrez personnaliser Teams Téléphone’aide de l’forfait d’appels pour mieux répondre aux besoins de votre entreprise. Par exemple, vous voudrez probablement déplacer vos numéros de téléphone existants (appelés portage de numéros) vers un Teams Téléphone’aide d’un plan d’appels.

| Tâche                                                          | Description                                                                                          |
|---------------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [Déplacer des numéros de téléphone vers Teams Téléphone’aide de l’forfait d’appels](port-phone-numbers.md) | Apportez vos numéros de téléphone existants Teams Téléphone’aide du forfait d’appels. |
| [Configurer vos standards automatiques](./create-a-phone-system-auto-attendant-smb.md) | Ajoutez d’autres menus d’appel pour proposer davantage d’options aux appelants qui vous contactent sur votre numéro de téléphone principal.  |
| [Configurer vos stratégies d’appel](set-up-policies.md)                 | Déterminez si les utilisateurs peuvent faire certaines actions telles que passer des appels privés, utiliser le transfert d’appel et déléguer des appels.  |
| [Créer des utilisateurs supplémentaires](create-users.md)                    | Ajoutez des utilisateurs et attribuez-leur des numéros de téléphone nouveaux ou existants.  |
| [Configurer des files d’attente d’appels](./create-a-phone-system-call-queue-smb.md)    | Créez une file d’attente dans laquelle les appelants peuvent attendre l’agent disponible suivant.  |
