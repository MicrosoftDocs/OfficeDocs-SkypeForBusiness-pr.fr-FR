# <a name="whats-the-status-of-your-port-orders"></a>Quel est le statut de vos demandes de transfert ?

Pour voir l’état de votre demande de port, dans le navigation gauche du Centre d’administration Microsoft Teams, allez à > **Voice**  >  **Port orders,** puis cliquez sur **Historique des commandes.** Le statut de chaque commande de port est répertorié dans la **colonne** Statut. Découvrez [combien de temps faut-il pour les numéros de port](../phone-number-calling-plans/port-order-overview.md#how-long-does-it-take-to-port-numbers) pour en savoir plus sur le processus de demande. 

Le tableau suivant répertorie le statut des commandes de port, ainsi que les actions que vous pouvez prendre si nécessaire.

|**État**|**Pouvez-vous afficher la commande ?**|**Pouvez-vous modifier la commande ?**|**Pouvez-vous annuler la commande ?**|**Pouvez-vous supprimer la commande ?**|**Description**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Traitement** <br/> |Oui  <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |L’administrateur a créé la commande et elle a été reçue par Microsoft.  <br/> |
|**Contact de l’opérateur** <br/> |Oui  <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |La demande a été reçue et approuvée par Microsoft, et nous travaillons avec l’opérateur d’approbation pour obtenir son approbation.  <br/> |
|**Transfert approuvé** <br/> |Oui  <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |La commande a été acceptée par l’opérateur d’téléphonie d’entreprise et la date d’engagement de commande ferme (FOC) a été définie.  <br/> |
|**Transfert en attente** <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |Non  <br/> |Le transfert est dans moins de 24 heures, de sorte que la commande ne peut plus être modifiée ni annulée.  <br/> |
|**Error** <br/> |Non  <br/> |Oui  <br/> |Oui  <br/> |Oui (pour le moment, vous ne pouvez pas supprimer la demande de port en cas d’erreur. La demande de port doit être re-créée ou vous devez contacter l’aide [du service PSTN.](../manage-phone-numbers-for-your-organization/contact-pstn-service-desk.md)  <br/> |L’opérateur d’téléphonie d’avant-première a rejeté la demande.  <br/> |
|**Terminé** <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |Non  <br/> |Les numéros ont été transférés.  <br/> |
|**Annulée** <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |L’administrateur a annulé la commande.  <br/> |

Pour obtenir des instructions détaillées complètes, voir Transférer des numéros [de téléphone vers Teams.](transfer-phone-numbers-to-teams.md)

Si vous avez besoin d’aide ou si vous avez besoin d’obtenir plus de numéros de téléphone, contactez le [service PSTN.](../manage-phone-numbers-for-your-organization/contact-pstn-service-desk.md)
