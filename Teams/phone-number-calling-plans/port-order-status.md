# <a name="whats-the-status-of-your-port-orders"></a>Quel est le statut de vos demandes de transfert ?

Pour afficher l’état de votre commande de port, dans le volet de navigation gauche du Centre d’administration Microsoft Teams, accédez à >**commandes de port** **vocal** > , puis cliquez sur **Historique des commandes**. Chaque état de l’ordre de port est répertorié dans la colonne **État** . Découvrez [le temps qu’il faut aux numéros de port](../phone-number-calling-plans/port-order-overview.md#how-long-does-it-take-to-port-numbers) pour en savoir plus sur le processus de commande. 

Le tableau suivant répertorie les états d’ordre de port, ainsi que les actions que vous pouvez effectuer si nécessaire.

|**Statut**|**Pouvez-vous afficher la commande ?**|**Pouvez-vous modifier la commande ?**|**Pouvez-vous annuler la commande ?**|**Pouvez-vous supprimer la commande ?**|**Description**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Traitement** <br/> |Oui  <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |L’administrateur a créé la commande et elle a été reçue par Microsoft.  <br/> |
|**Contacter le transporteur** <br/> |Oui  <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |La commande a été reçue et approuvée par Microsoft, et nous travaillons avec le transporteur perdant pour l’obtenir approuvée.  <br/> |
|**Transfert approuvé** <br/> |Oui  <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |La commande a été acceptée par le transporteur perdant, et la date d’engagement de commande ferme (FOC) a été définie.  <br/> |
|**Transfert en attente** <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |Non  <br/> |Le transfert étant à moins de 24 heures, la commande ne peut plus être modifiée ou annulée.  <br/> |
|**Error** <br/> |Non  <br/> |Oui  <br/> |Oui  <br/> |Oui (pour l’instant, vous ne pouvez pas supprimer l’ordre de port en cas d’erreur. La commande de port doit être recréé, ou vous devez contacter [TNS Service Desk](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).  <br/> |Le transporteur perdant a rejeté la commande.  <br/> |
|**Terminé** <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |Non  <br/> |Les nombres ont été transférés avec succès.  <br/> |
|**Annulé** <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |L’administrateur a annulé la commande.  <br/> |

Pour obtenir des instructions détaillées, consultez [Transférer des numéros de téléphone vers Teams](transfer-phone-numbers-to-teams.md).

Si vous avez besoin d’aide ou si vous devez obtenir d’autres numéros de téléphone, contactez le [service TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).
