# <a name="more-information-about-porting"></a>Plus d’informations sur le portage

Vous trouverez ici plus d’informations sur le portage de vos numéros de téléphone vers Microsoft Teams.

Pour obtenir des instructions détaillées, consultez [Transférer des numéros de téléphone vers Teams](transfer-phone-numbers-to-teams.md).

Si vous avez besoin d’aide ou si vous avez besoin d’obtenir d’autres numéros de téléphone, contactez [l’aide de TNS Service Desk](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).

## <a name="port-order-account-information"></a>Informations sur le compte de demande de transfert

Lorsque vous êtes sur la page **Ajouter des informations de compte** de l’Assistant Portage pour envoyer une commande de port, vous entrez presque toutes les mêmes informations que celles que vous fourniriez dans la LOA, notamment :
  
- Numéro de compte du fournisseur de services ou de l’opérateur
    
- Numéro de téléphone de facturation (BTN)
    
- Code confidentiel - si nécessaire par votre fournisseur de services ou opérateur actuel
    
- Nom de l’organisation
    
    > [!NOTE]
    > Cela n’accepte que 25 caractères, espaces compris. Si le nom de l’organisation dépasse 25 caractères, les 25 premiers caractères du nom sont envoyés et l’ordre de port est toujours traité.
  
- Nom de la personne autorisée à apporter des modifications au compte
    
    > [!NOTE]
    > Cela n’accepte que 15 caractères, espaces compris. Si le nom de la personne autorisant contient plus de 15 caractères, les 15 premiers caractères sont envoyés et la demande de transfert est traitée malgré tout. 
  
- Adresse du service
  
Pour faciliter l’envoi de la commande de port et éviter les erreurs, veillez à effectuer les opérations suivantes :
  
- Supprimez toutes les fonctionnalités (telles que les groupes de chasse) associées à vos numéros. Assurez-vous qu’aucune fonctionnalité de contrôle d’appel avancée, telle que la chasse aux appels ou l’anneau distinctif, n’est activée sur ces numéros de téléphone.
    
- Veillez à ne pas avoir passé de nouvelles commandes de service ou à vous déconnecter de votre fournisseur de services actuel.
    
- Assurez-vous que tous les numéros sont issus du même opérateur et du même compte.
    
- Assurez-vous que les informations de compte que vous indiquez correspondent exactement à celles enregistrées par votre opérateur téléphonique. Les informations incompatibles sont la cause d’erreurs la plus courante et peuvent retarder votre ordre de port.
    
> [!CAUTION]
> Ne déconnectez pas vos services avec votre fournisseur de services ou votre opérateur. Vous devez maintenir votre service précédent actif afin de porter vos numéros de téléphone vers Teams. Ne gèlez pas votre compte auprès de votre fournisseur de services ou de votre opérateur. Le fait de figer le compte empêche la modification des opérateurs téléphoniques sur le compte. L'utilisateur autorisé doit soumettre une demande à l'opérateur téléphonique actuel pour annuler le gel. Ce processus peut prendre une à trois semaines, selon le transporteur.

## <a name="authorized-person-on-the-account"></a>Personne autorisée sur le compte

Dans l’Assistant Portage, vous devez entrer le nom de la personne autorisée à apporter des modifications au compte auprès du fournisseur de services ou de l’opérateur. Le nom n’est pas utilisé pour traiter l’ordre de port, mais il est utilisé en cas de litige, ou si quelque chose est incorrect lorsque des numéros sont portés. Cette personne est responsable de la lettre d’autorisation (LOA) pour une commande de port.
  
> [!NOTE]
> La zone est limitée à 15 caractères (espaces compris). Le fait de ne pas avoir le nom complet dans la zone ne retardera pas ou n’annulera pas la commande de port.
  
## <a name="whats-my-billing-telephone-number"></a>Quel est mon numéro de téléphone de facturation ?

Le numéro de téléphone de facturation (BTN) est le numéro de téléphone principal inclus sur votre facture et facturé par votre fournisseur de services ou opérateur. Si vous transférez un numéro de téléphone à partir d’un compte qui n’a qu’un seul numéro de téléphone, vous devez entrer ce numéro de téléphone. Si vous transférez des numéros de téléphone à partir d’un compte qui en contient plusieurs, vous pouvez consulter votre facture ou contacter votre fournisseur de services ou opérateur pour déterminer ce qu’est le BTN pour votre compte.

## <a name="what-should-i-put-in-for-the-account-number"></a>Que dois-je inclure comme numéro de compte ?

En règle générale, vous pouvez trouver le numéro de compte sur n’importe quelle facture ou facture de votre fournisseur de services ou opérateur, ou vous pouvez vous connecter au site web de votre opérateur. Si vous ne connaissez toujours pas le numéro de compte, vous pouvez contacter votre fournisseur de services ou votre opérateur pour l’obtenir.
  
> [!CAUTION]
>  Il est important de vous assurer que vous n’utilisez pas d’espaces, de tirets ou de traits d’union lors de l’entrée de votre numéro de compte de fournisseur de services ou d’opérateur.

## <a name="what-should-i-put-in-for-the-organization-name"></a>Que dois-je mettre pour le nom de l’organisation ?

Il s’agit du nom de votre organisation. Le nom de l’organisation est limité à 25 caractères, ce qui inclut des espaces. Le nom de l’entreprise n’est pas utilisé pour traiter la demande de commande de port. Il est utilisé en cas de litige ou si quelque chose est incorrect lorsque les numéros de téléphone sont transférés. Si vous ne pouvez pas contenir l’intégralité du nom de l’entreprise dans la zone, cela ne retardera pas ou n’annulera pas la commande de port.
  
## <a name="what-should-i-put-in-for-the-service-address"></a>Que dois-je inclure comme adresse de service ?

L’adresse de service est différente de l’adresse de facturation ou d’urgence que vous avez inscrite auprès de votre fournisseur de services téléphoniques ou opérateur. Si vous ne le savez pas, contactez votre fournisseur de services ou votre opérateur pour connaître l’adresse de service répertoriée sur votre compte.

## <a name="how-should-i-enter-the-phone-numbers"></a>Comment dois-je entrer les numéros de téléphone ?
<a name="bkadding"> </a>

Lorsque vous envoyez une commande de port, vous devez utiliser un fichier CSV correctement mis en forme pour envoyer vos numéros de téléphone. Voici les conditions requises pour le fichier CSV :

 - Vous pouvez donner au fichier le nom de votre choix.
 - Le fichier ne doit avoir qu’une seule colonne avec un en-tête nommé PhoneNumber.
 - Chaque numéro de téléphone doit se trouver sur une ligne distincte.
 - Les numéros de téléphone peuvent être des chiffres uniquement ou au format E.164.
 - Le format du numéro de téléphone doit correspondre au pays ou à la région que vous avez sélectionné. Par exemple, si vous choisissez le Royaume-Uni dans l’Assistant Portage, utilisez 44, qui est le code du pays, suivi du numéro de téléphone avec le nombre correct de chiffres. Par exemple, 4420812341234.
