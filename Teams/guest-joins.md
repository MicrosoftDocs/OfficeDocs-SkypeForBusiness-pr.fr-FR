<a name="how-a-guest-joins-a-team"></a><span data-ttu-id="78aa2-103">Comment un invité rejoint une équipe</span><span class="sxs-lookup"><span data-stu-id="78aa2-103">How a guest joins a team</span></span>
========================

<span data-ttu-id="78aa2-104">Dans Microsoft Teams, le propriétaire d'une équipe peut ajouter et gérer des invités dans ses équipes via le client Web ou de bureau.</span><span class="sxs-lookup"><span data-stu-id="78aa2-104">A team owner in Microsoft Teams can add and manage guests in their teams via the web or desktop.</span></span> <span data-ttu-id="78aa2-105">Seuls les utilisateurs disposant d'une adresse de messagerie correspondant à un compte scolaire ou professionnel Azure Active Directory ou Office 365 peuvent être ajoutés en tant qu'utilisateurs invités.</span><span class="sxs-lookup"><span data-stu-id="78aa2-105">Only users who have an email address corresponding to an Azure Active Directory or Office 365 work or school account can be added as a guest user.</span></span>
  
    
    

> [!NOTE]
> <span data-ttu-id="78aa2-106">Pour permettre aux invités de rejoindre une équipe, l'administrateur doit d'abord activer l'accès invité dans Teams.</span><span class="sxs-lookup"><span data-stu-id="78aa2-106">Before guests can join a team, an admin must enable guest access in Teams.</span></span> <span data-ttu-id="78aa2-107">Pour cela, [connectez-vous à ](https://portal.office.com/adminportal/home) à l'aide de votre compte d'administrateur global Office 365.</span><span class="sxs-lookup"><span data-stu-id="78aa2-107">To do that,  [sign in](https://portal.office.com/adminportal/home) with your Office 365 global admin account.</span></span> <span data-ttu-id="78aa2-108">Puis, sélectionnez **Paramètres** > **Services &amp; compléments** > **Microsoft Teams**.</span><span class="sxs-lookup"><span data-stu-id="78aa2-108">Then, choose **Settings** > **Services &amp; add-ins** > **Microsoft Teams**.</span></span> <span data-ttu-id="78aa2-109">Sélectionnez **Invité** sous la section **Sélectionnez le type d'utilisateur/licence que vous souhaitez configurer**, puis sélectionnez **Activé** sous la section **Activer/désactiver Microsoft Teams pour tous les utilisateurs de ce type**.</span><span class="sxs-lookup"><span data-stu-id="78aa2-109">Select **Guest** in **Select the user/license type you want to configure**, and select **On** in **Turn Microsoft Teams on or off for all users of this type**.</span></span> <span data-ttu-id="78aa2-110">L'application des paramètres peut prendre jusqu'à une heure.</span><span class="sxs-lookup"><span data-stu-id="78aa2-110">It can take up to an hour for the settings to take effect.</span></span> <span data-ttu-id="78aa2-111">Pour plus d'informations, reportez-vous à la section « Activer ou désactiver l'accès invité de Microsoft Teams ».</span><span class="sxs-lookup"><span data-stu-id="78aa2-111">For more details, see "Turn on or off guest access for Microsoft Teams" on this article's Manage tab.</span></span> 
  
    
    

<span data-ttu-id="78aa2-112">Voici comment un invité devient membre d'une équipe :</span><span class="sxs-lookup"><span data-stu-id="78aa2-112">Here's how a guest becomes a member of a team:</span></span>

- <span data-ttu-id="78aa2-113">**Étape 1** Un propriétaire d'équipe ou un administrateur d'Office 365 [ajoute un invité à une équipe](https://support.office.com/en-us/article/adds-a-guest-to-a-team-df38ae23-8f85-46d3-b071-cb11b9de5499#bkmk_addingguests).</span><span class="sxs-lookup"><span data-stu-id="78aa2-113">**Step 1** A team owner or an Office 365 admin [adds a guest to a team](https://support.office.com/en-us/article/adds-a-guest-to-a-team-df38ae23-8f85-46d3-b071-cb11b9de5499#bkmk_addingguests).</span></span>
    
  
- <span data-ttu-id="78aa2-114">**Étape 2** L'administrateur d'Office 365 ou le propriétaire de l'équipe peut gérer les fonctionnalités requises pour l'invité.</span><span class="sxs-lookup"><span data-stu-id="78aa2-114">**Step 2** The Office 365 admin or the team owner can manage a guest's capabilities as necessary.</span></span> <span data-ttu-id="78aa2-115">Par exemple, il peut l'autoriser à ajouter ou supprimer des canaux, ou désactiver son accès aux fichiers.</span><span class="sxs-lookup"><span data-stu-id="78aa2-115">For example, allowing a guest to add or delete channels or disabling access to files.</span></span>

<span data-ttu-id="78aa2-120">Les invités peuvent quitter l'équipe à tout moment via les clients Web ou de bureau Teams.</span><span class="sxs-lookup"><span data-stu-id="78aa2-120">Guests can leave the team at any time via Teams web and desktop clients.</span></span> <span data-ttu-id="78aa2-121">Pour plus d'informations, reportez-vous à l'article [Comment quitter une équipe ?](https://support.office.com/en-us/article/How-do-I-leave-a-team-df38ae23-8f85-46d3-b071-cb11b9de5499#bkmk_howdoileaveateam)</span><span class="sxs-lookup"><span data-stu-id="78aa2-121">For details, see  [How do I leave a team?](https://support.office.com/en-us/article/How-do-I-leave-a-team-df38ae23-8f85-46d3-b071-cb11b9de5499#bkmk_howdoileaveateam)</span></span>
