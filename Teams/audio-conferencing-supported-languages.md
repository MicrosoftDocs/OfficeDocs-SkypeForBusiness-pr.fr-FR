# <a name="audio-conferencing-supported-languages"></a>Langues prises en charge pour l’audioconférence

Lorsque vous configurez et configurez l’audioconférence dans Microsoft 365 ou Office 365, vous pouvez sélectionner les langues principales et secondaires (jusqu’à 4) utilisées lorsque les appelants composent une réunion Microsoft Teams. Voici la liste des langues prises en charge pour l’audioconférence :
  
> [!IMPORTANT]
> Les langues primaires et secondaires ne peuvent être modifiées que sur des numéros de téléphone d’audioconférence dédiés.

## <a name="want-to-know-how-to-manage-with-windows-powershell"></a>Comment utiliser Windows PowerShell pour gérer cette fonction ?

- Pour gagner du temps ou automatiser cela, vous pouvez utiliser les applets de commande [Get-CsOnlineDialInConferencingLanguagesSupported](/powershell/module/skype/Get-CsOnlineDialInConferencingLanguagesSupported) ou [Set-CsOnlineDialInConferencingServiceNumber](/powershell/module/skype/Set-CsOnlineDialInConferencingServiceNumber) .

- Windows PowerShell est axé sur la gestion des utilisateurs et sur les actions qu'ils sont autorisés ou non à effectuer. Avec Windows PowerShell, vous pouvez gérer Microsoft 365 ou Office 365 à l’aide d’un seul point d’administration qui peut simplifier votre travail quotidien lorsque vous avez plusieurs tâches à effectuer.
