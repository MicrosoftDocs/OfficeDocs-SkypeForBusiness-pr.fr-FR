# <a name="overview-of-third-party-apps-in-microsoft-teams"></a>Vue d’ensemble des applications tierces dans Microsoft Teams

Teams Store fournit de nombreuses applications utiles créées par des éditeurs de logiciels tiers et des développeurs, en plus des applications fournies par Microsoft. Pour utiliser ces applications, en tant qu’administrateur, vous définissez un processus de gouvernance des applications qui équilibre les exigences étendues des utilisateurs finaux avec les stratégies informatiques, les normes et les profils de risque de votre organisation. Bien que ces applications soient créées par des fournisseurs tiers, Microsoft effectue une validation rigoureuse des fonctionnalités, de la facilité d’utilisation et de la sécurité de ces applications. Ce n’est qu’après la validation que Microsoft rend les applications disponibles dans Teams Store. Pour que les utilisateurs finaux bénéficient de l’écosystème d’applications enrichi, la gouvernance des applications par les administrateurs est une étape clé.

Les applications tierces dans Microsoft Teams ne sont pas contrôlées ou détenues par Microsoft et ne sont pas régies par le Contrat d’abonnement Microsoft Online. L’utilisation par votre organisation de chaque application tierce est soumise aux conditions générales et à la déclaration de confidentialité de cette application. Le Centre d’administration Microsoft Teams vous permet de vérifier toutes les autorisations d’une application, les déclarations de confidentialité et les conditions d’utilisation avant d’autoriser son utilisation. Comme toujours, vous pouvez autoriser l’utilisation d’une application pour quelques utilisateurs spécifiques ou tous les utilisateurs. Pour les applications tierces, comme pour les applications personnalisées, Teams permet aux administrateurs d’autoriser, de bloquer, de régir et d’analyser l’utilisation des applications.

Microsoft s’assure que les administrateurs contrôlent les applications tierces et que seules les applications de haute qualité sont disponibles dans Teams Store. Toutes les applications passent par un cycle de validation obligatoire qui vérifie les fonctionnalités de l’application, l’identité du développeur, les autorisations demandées par l’application et les vérifications liées à la conformité, à la confidentialité, à la gestion des données et à la sécurité des données. Nous encourageons les applications à faire partie d’un programme facultatif afin de mettre l’accent sur la conformité aux normes du secteur, la gestion des données et la sécurité. En savoir plus sur ces programmes dans les articles suivants :

   - [Attestation Publisher](overview-of-app-certification.md#publisher-attestation)
   - [Certification Microsoft 365](overview-of-app-certification.md#microsoft-365-certification)
