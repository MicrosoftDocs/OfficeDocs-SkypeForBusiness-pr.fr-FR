# <a name="overview-of-third-party-apps-in-microsoft-teams"></a>Vue d’ensemble des applications tierces dans Microsoft Teams

Beaucoup fournissent des applications dans Teams magasin. Ces applications sont créées par les développeurs d’applications et Microsoft valide les fonctionnalités de base, la facilité d’utilisation et la sécurité de base des applications. Ce n’est qu’après la validation que Microsoft rend les applications disponibles dans Teams store. Pour que les utilisateurs finaux bénéficient de l’écosystème d’applications enrichi, la gouvernance des applications par les administrateurs est une étape clé. En tant qu’administrateur, définissez un processus de gouvernance des applications qui équilibre les exigences d’application étendues des utilisateurs finaux et les limitations des stratégies informatiques internes, des normes et des profils de risque.

Les applications tierces dans Microsoft Teams ne sont pas contrôlées ou détenues par Microsoft et ne sont pas régies par le Contrat d’abonnement Microsoft Online. L’utilisation par votre organisation de chaque application tierce est soumise aux conditions générales de cette application et à sa déclaration de confidentialité. Teams centre d’administration vous permet de vérifier toutes les autorisations d’une application, les déclarations de confidentialité et les conditions d’utilisation avant d’autoriser son utilisation. Comme toujours, vous pouvez autoriser l’utilisation d’une application pour quelques utilisateurs spécifiques ou tous les utilisateurs. Pour les applications tierces, comme pour les applications personnalisées, Microsoft Teams permet aux administrateurs d’autoriser, d’interdire, de régir et d’analyser l’utilisation des applications.

En plus de s’assurer que les administrateurs contrôlent les applications tierces, Microsoft garantit également que les applications de haute qualité sont disponibles dans le Store. Nous exigeons que toutes les applications passent par un cycle de validation obligatoire par Microsoft. Nous encourageons les applications à faire partie d’un programme de conformité facultatif afin de mettre l’accent sur la sécurité et la conformité aux normes du secteur. Les différents programmes sont les suivants :

1. [Validation d’application](overview-of-app-validation.md#app-validation-and-testing)
1. [vérification Publisher](overview-of-app-validation.md#publisher-verification)
1. Programme de conformité des applications Microsoft :

   * [attestation Publisher](overview-of-app-validation.md#publisher-attestation)
   * [certification Microsoft 365](overview-of-app-validation.md#microsoft-365-certification)
