# <a name="microsoft-365-apps-for-enterprise"></a>Applications 365 Microsoft pour les entreprises

Si vous avez déjà acheté les [applications 365 Microsoft pour les entreprises](https://products.office.com/en/business/office-365-proplus-business-software) , Voici les licences de compléments que vous devez acheter pour obtenir plus de fonctionnalités d’équipe.

> [!IMPORTANT]
> Même si l’audioconférence, le système téléphonique et les offres d’appels peuvent être achetés pour activer les fonctions d’appel audio, nous vous recommandons d’évaluer une offre entreprise E5 pour voir s’il est plus rentable.


|Pour ajouter cette fonctionnalité |Voici ce que vous devez faire |
|:---------------------------|:--------------------------------|
|Audioconférence <br/> |Achetez le composant additionnel audioconférence. <br/>Acheter 1 licence d’audioconférence pour chaque personne chargée de planifier ou d’organiser une conférence rendez-vous. Les participants n'ont pas besoin de licences. <br/>Une fois la licence achetée, Microsoft sera votre fournisseur d’audioconférence. |
|Numéros gratuits pour un accès rendez-vous à vos conférences et possibilité d’accès par appels sortants à partir d’une conférence pour ajouter une personne en appelant n’importe quel numéro de téléphone dans le monde <br/> |Acheter des [crédits de communication](../set-up-communications-credits-for-your-organization.md).|
|Système téléphonique <br/> |<ol><li>[Basculer vers une autre offre Office 365 pour les entreprises](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a). Si vous disposez d’applications Microsoft 365 pour les entreprises, nous vous recommandons de mettre à niveau votre plan vers une offre entreprise E5 pour obtenir les fonctionnalités vocales de Cloud. Si vous mettez à niveau vers un forfait Entreprise E1 ou E3, vous verrez l’option d’achat du complément Système téléphonique. <br/></li><li>Acheter le module complémentaire Système téléphonique. <br/></li><li>Achetez un [plan d’appel](../calling-plans-for-office-365.md) pour Office 365. <br/>Ou bien, utilisez le [service de téléphone de votre fournisseur existant](microsoft-teams-add-on-licensing.md#bkmk_existing).</li></ol>  <br/> |
|Acheter un forfait d’appels <br/> |<ol><li>[Basculer vers une autre offre Office 365 pour les entreprises](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a). <br/></li><li>Acheter le module complémentaire Système téléphonique.</li><li>Achetez un [forfait d’appels](../calling-plans-for-office-365.md).<br/> Vous pouvez l’acheter uniquement après l’acquisition du complément téléphonique du système. <br/> |
