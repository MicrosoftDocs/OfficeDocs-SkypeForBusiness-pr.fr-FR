# <a name="office-365-enterprise-e5-without-audio-conferencing"></a>Office 365 entreprise E5 (sans conférence audio)

Si vous avez déjà acheté Office 365 entreprise E5 (sans conférence audio), Voici les licences de compléments Microsoft teams que vous devez acheter pour obtenir plus de fonctionnalités.

|Pour ajouter cette fonctionnalité|Voici ce que vous devez faire |
|:------------------|:--------------------------|
|Audioconférence, <br/> | Pour obtenir ce complément, procédez comme suit : <ol><li>Si vous avez déjà acheté une offre E5, mais que la fonctionnalité de conférence rendez-vous n’est pas disponible dans votre pays ou région, vous devez utiliser un fournisseur de services d’audioconférence tiers. Recherchez un fournisseur tiers de services d’audioconférence dans [Microsoft Pinpoint](https://go.microsoft.com/fwlink/?LinkId=797530).  <br/>Consultez cet article pour voir si l’audioconférence est disponible dans votre pays ou région : [Disponibilité des offres d’appels et d’audioconférence selon les régions et les pays](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md) </li><li/>S’il est disponible, utilisez le [bouton changer de plan](http://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a) pour passer à l’offre Office 365 entreprise E5 incluant l’audioconférence. Vous obtiendrez également davantage de fonctionnalités ! </li></ol> |
|Le système téléphonique est inclus ! <br/> | Achetez un [plan d’appel](../calling-plans-for-office-365.md) pour Office 365. <br/>  Ou bien, utilisez le [service de téléphone de votre fournisseur existant](microsoft-teams-add-on-licensing.md#bkmk_existing).  <br/> |
|La messagerie vocale Cloud est incluse. <br/> |Vous n’avez rien à faire.  <br/> |
