## <a name="unclassified-streams"></a><span data-ttu-id="3a6ba-238">Flux non classifiés</span><span class="sxs-lookup"><span data-stu-id="3a6ba-238">Unclassified Streams</span></span>

<span data-ttu-id="3a6ba-239">Dans bord, un flux est marqué comme n’étant pas _classé_ lorsque la connectivité de l’établissement de la connexion interactives échoue ou lorsque toutes les mesures requises pour calculer la classification du flux ne sont pas communiquées.</span><span class="sxs-lookup"><span data-stu-id="3a6ba-239">In CQD, a stream is marked _Unclassified_ when Interactive Connectivity Establishment (ICE) connectivity fails or when all the metrics required to compute the stream classification are not reported.</span></span>

<span data-ttu-id="3a6ba-240">Pour vérifier les échecs de la connectivité ICE, examinez les dimensions « First Connectivity Ice » et « Second Connectivity Ice » pour une valeur « FAILED ».</span><span class="sxs-lookup"><span data-stu-id="3a6ba-240">To check for ICE connectivity failures, examine the dimensions "First Connectivity Ice" and "Second Connectivity Ice" for a "FAILED" value.</span></span> <span data-ttu-id="3a6ba-241">Si l’une des valeurs indique un échec, le flux est marqué comme non _classé_.</span><span class="sxs-lookup"><span data-stu-id="3a6ba-241">If either value indicates a failure, the stream is marked as _Unclassified_.</span></span>

<span data-ttu-id="3a6ba-242">Si la connectivité ICE a réussi pour un flux non _classé_ , le flux est probablement considéré comme non _classé_ car les métriques de flux de clés n’ont pas été communiquées.</span><span class="sxs-lookup"><span data-stu-id="3a6ba-242">If ICE connectivity succeeded for an _Unclassified_ stream, the stream is likely considered _Unclassified_ because key stream metrics were not reported.</span></span> <span data-ttu-id="3a6ba-243">Il existe plusieurs raisons pour lesquelles ces métriques ne peuvent pas être rapportées :</span><span class="sxs-lookup"><span data-stu-id="3a6ba-243">There are a few reasons these metrics may not be reported:</span></span>
