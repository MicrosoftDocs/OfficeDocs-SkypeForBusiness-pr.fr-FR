<a name="configure-an-exchange-hybrid-organization-for-use-with-microsoft-teams"></a>Configurer une organisation Exchange hybride à utiliser avec Microsoft Teams
======================================================================

En règle générale, vous devez pas configurer toutes les fonctionnalités Exchange Online pour une utilisation avec Microsoft Teams. Toutefois, pour les scénarios hybrides d’Exchange, il étapes sont nécessaires pour assurer les appartenances aux groupes sont synchronisés entre Exchange Server (localement) et Exchange Online. Cela implique la prise en charge des fonctionnalités d’écriture différée de groupe dans Azure AD se connecter avec différents scripts d’initialisation : [configurer Office 365 groupes avec locale Exchange hybride](https://go.microsoft.com/fwlink/?linkid=854389).
