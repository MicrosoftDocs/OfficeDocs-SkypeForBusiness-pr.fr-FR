<a name="view-guest-users-in-a-team"></a>Afficher les utilisateurs invités d'une équipe
==========================
Dans une équipe tout le monde peut voir tous les membres d’une équipe, y compris les invités, par le biais **l’équipe de gestion** de Microsoft Teams.  [Voir qui est d’une équipe ou un canal](https://support.office.com/article/see-who-s-on-a-team-or-in-a-channel-5c6be9be-9c45-4a0f-a1a0-f332b23cb6b7) a plus de détails.

En tant qu’administrateur, vous pouvez utiliser le centre d’administration d’Office 365 pour afficher les utilisateurs invités sur le client. Toutefois, cette vue affiche *tous les* utilisateurs invités sur le client, pas seulement les utilisateurs invités dans Microsoft Teams.
1. Connectez-vous à l'aide de votre compte d'administrateur général Office 365 sur [https://portal.office.com/adminportal/home](https://portal.office.com/adminportal/home).
2. Accédez à **Utilisateurs** > **Utilisateurs invités**.
