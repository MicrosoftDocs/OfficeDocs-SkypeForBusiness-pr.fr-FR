<a name="install-microsoft-teams-using-msi-with-sccm"></a>Installer Microsoft Teams à l’aide de MSI (avec SCCM)
===========================================

Microsoft a fourni un fichier MSI pour les administrateurs en bloc déployer équipes pour sélectionner des utilisateurs ou des ordinateurs. Administrateurs pouvant utiliser ce fichier pour déployer à distance des équipes afin que les utilisateurs n’ont pas télécharger manuellement l’application d’équipes. System Center Configuration Manager (SCCM) est le principal outil utilisé avec les fichiers MSI pour les déploiements gérés.
