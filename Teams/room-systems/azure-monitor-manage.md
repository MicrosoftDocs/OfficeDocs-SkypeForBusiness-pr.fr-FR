# <a name="manage-microsoft-teams-rooms-devices-with-azure-monitor"></a>Gérer les périphériques de salles d’équipes Microsoft Azure moniteur

Cet article explique comment gérer les périphériques Microsoft équipes salles d’une manière intégrée, de bout en bout pour l’utilisation du moniteur Azure.

Vous pouvez configurer le moniteur Azure pour fournir la télémétrie base qui vous aideront à gérer Skype réunion équipements de salle (voir [gestion planifier Microsoft équipes salles avec Azure moniteur](azure-monitor-plan.md) et [déployer Microsoft équipes salles avec Azure moniteur]((azure-monitor-deploy.md) for details). Comme votre solution de gestion ailleurs, vous pouvez utiliser des données supplémentaires et des fonctionnalités de gestion pour créer une vue plus détaillée des performances de l’appareil.

## <a name="understand-the-log-entries"></a>Comprendre les entrées de journal

Les descriptions d’événement suivants sont insérées dans le champ de description du journal des événements toutes les cinq minutes, lorsque l’application Microsoft équipes salle enregistre les informations correspondantes dans le journal des événements Windows. L’Agent de surveillance Microsoft transmet ces enregistrements au journal Analytique dans le moniteur Azure, qui analyse les dans le tableau de bord que vous avez créé dans la [gestion de déployer Microsoft équipes salles avec Azure moniteur](azure-monitor-deploy.md). Le tableau de bord vous permet de consulter les entrées de journal individuelles afin de déterminer les mesures à prendre, le cas échéant. 

Les ID d'événement 2000 et 3000 indiquent que l'appareil concerné fonctionne correctement. Les ID d'événement 2001 et 3001 indiquent qu'un problème a été détecté. L'ID d'événement 4000 peut nécessiter une action s'il apparaît plus fréquemment que prévu, par rapport à la ligne de base que vous avez définie ou aux autres appareils déployés dans votre organisation.

Comprendre ces descriptions d'événements vous permet d'être rapidement informé(e) des problèmes et de disposer d'un point de départ pour le dépannage.
