# <a name="getting-started-with-microsoft-teams-rooms-premium-managed-service"></a>Prise en main du service géré par Salles Microsoft Teams Premium

Les services managés Salles Microsoft Teams Premium réduisent la complexité pour les administrateurs salles Teams en offrant des opérations de salle améliorées qui améliorent la productivité des réunions en salle.

## <a name="obtain-and-assign-a-license"></a>Obtenir et attribuer une licence

Salles Microsoft Teams comptes de ressources nécessitent une licence Salles Microsoft Teams Premium pour fonctionner avec le service géré par Salles Microsoft Teams Premium.

Pour attribuer une licence, effectuez les étapes suivantes :

1. Connectez-vous à [Centre d'administration Microsoft 365](https://admin.microsoft.com) en tant qu’administrateur.

    > [!NOTE]
    > Ce type d’administrateur est parfois appelé **administrateur général**. Consultez la vue d’ensemble de [l’Centre d'administration Microsoft 365](/microsoft-365/business-video/admin-center-overview).

2. Choisissez **Facturation > Licences** dans le volet de navigation gauche.
3. Sélectionnez la licence **premium salles Teams** appropriée pour votre utilisation prévue (des licences payantes et d’essai sont disponibles). Pour une comparaison entre les références SKU Salles Microsoft Teams Standard et Premium, consultez [Les solutions de licence pour les appareils de communications partagées](rooms-licensing.md).
4. Dans la page **Comptes de facturation** , sélectionnez **Attribuer des licences**. La page **Attribuer des licences aux utilisateurs** s’affiche.
5. Entrez les comptes **Salles Microsoft Teams** existants pour salles Teams attribution de licence **Premium**. Pour plus d’informations sur la configuration de nouveaux comptes Salles Microsoft Teams ou la validation de comptes existants, consultez [Créer des comptes de ressources pour les salles et les appareils Teams partagés](with-office-365.md).
6. Une fois l’envoi du compte terminé, **sélectionnez Affecter** , puis **Fermez**. Les **licences Salles Microsoft Teams Premium** ont été attribuées à **Salles Microsoft Teams** comptes.

Vous êtes maintenant prêt à inscrire les comptes de ressources et les appareils certifiés Salles Microsoft Teams associés dans le service géré par Salles Microsoft Teams Premium. Pour plus d’informations, consultez [Inscrire un appareil dans le service managé Salles Microsoft Teams Premium](enrolling-mtrp-managed-service.md).
