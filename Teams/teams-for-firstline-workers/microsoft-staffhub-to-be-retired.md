# <a name="microsoft-staffhub-to-be-retired"></a>Fin de parcours pour Microsoft StaffHub

Effet 2019, octobre 1, Microsoft StaffHub sera être retirée. Nous créons StaffHub fonctionnalités, notamment la gestion de tâches et calendrier dans Microsoft Teams. L’application mobile équipes inclut désormais gestion MAJ et une expérience de l’écran d’accueil. Des fonctionnalités supplémentaires pour les travailleurs firstline mettra en place pour les équipes au fil du temps. 

Ces modifications font partie des efforts pour permettre aux employés avec Microsoft 365 que nous. Avec des capacités pour les travailleurs firstline dans des équipes de chaque employé de votre organisation seront en mesure d’utiliser des équipes pour simplifier leur journée de travail, de collaborer avec vos collègues et accéder aux informations et des connaissances pour leur permettre d’effectuer leur travail meilleures.

## <a name="frequently-asked-questions"></a>Forum aux questions

- **Lorsque Microsoft StaffHub être retirée ?**<br> Début avril 1, 2019, Microsoft StaffHub ne sera plus disponible pour les nouveaux clients et tous les autres points d’accès. Les utilisateurs en cours sera toujours en mesure d’utiliser le service jusqu'à octobre 1, 2019, fonctionnalités toutefois ne supplémentaires seront disponibles.

- **Que se passe-t-il lorsque StaffHub est obsolète ?**<br>Microsoft StaffHub cesse de fonctionner pour tous les utilisateurs sur le 1 octobre 2019. Toute personne qui tente d’ouvrir l’application s’affichera un message pronom pour télécharger Microsoft Teams. Condition de l’utilisateur possède une licence active qui inclut des équipes, leurs données et les fonctionnalités principales de StaffHub seront disponibles pour les lors de leur transition vers les équipes.
Entre avril 2019 et octobre 2019, StaffHub d’utilisateurs recevront des notifications dans l’application invitant à utiliser Microsoft Teams.

- **Que dois-je faire pour préparer pour que cette modification ?**<br>Les administrateurs informatiques de clients avec des utilisateurs Microsoft StaffHub actives seront communiquées dans le centre de messages Office 365 sur les actions que les utilisateurs ont besoin pour la transition vers l’application d’équipes dans les équipes. Pour plus d’informations sur la migration à partir de StaffHub pour les déplacements dans les équipes, y compris les outils permettant de faciliter la migration, sera bientôt disponibles.

    Les utilisateurs seront informés de retraite de l’application via les notifications dans l’application, mais vous pouvez communiquer le changement leur, et mettre à jour de la documentation interne qui fait référence à Microsoft StaffHub.

    Visitez cette page pour plus d’informations sur la façon d’une transition réussie de StaffHub pour les déplacements dans les équipes, y compris la migration des données, la gestion des nouvelles fonctionnalités StaffHub que nous créons en équipes (dans le & Microsoft Teams Skype pour administrateur d’entreprise Centre) et d’autres informations sur les actions requises par les administrateurs et les utilisateurs finaux.

- **Microsoft Teams offre toutes les fonctionnalités actuellement proposées dans Microsoft StaffHub ?**<br>Oui, à compter de novembre 2018, Teams Microsoft propose une fonctionnalité d’équipes, qui permet aux responsables de créer, mettre à jour et gérer les planifications et permet aux employés de publier leur disponibilité, échange équipes et le temps de demande. En outre, l’affichage d’accueil de l’application mobile équipes affichera calendrier et annonces de l’entreprise importants de l’employé. 

    Pour plus d’informations, administrateurs peuvent-ils [gérer l’application des équipes de votre organisation dans les équipes](manage-the-shifts-app-for-your-organization-in-teams.md). Pour obtenir de l’utilisateur final sur l’utilisation des déplacements dans les équipes, consultez la rubrique [Équipes aide pour les responsables et employés firstline](https://support.office.com/article/apps-and-services-cc1fba57-9900-4634-8306-2360a40c665b). 

    Voir la [feuille de route](https://www.microsoft.com/microsoft-365/roadmap?filters=) pour en savoir plus sur les fonctionnalités supplémentaires doit être publié dans les équipes.

- **Pourquoi ne puis-je pas accéder mon équipe StaffHub à partir de l’application d’équipes dans les équipes ?**<br>Les équipes qui ont été créées avec Microsoft StaffHub peuvent uniquement être visualisés et gérés dans StaffHub. Lorsque nous version prise en charge pour migrer votre équipe StaffHub à l’application d’équipes dans les équipes, vous serez en mesure d’afficher les planifications StaffHub en équipe.
