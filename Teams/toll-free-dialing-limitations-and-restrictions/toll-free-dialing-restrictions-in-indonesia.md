# <a name="toll-free-dialing-restrictions-in-indonesia"></a>Restrictions d’appel gratuit en Indonésie

[Trouver un autre pays ou région](../toll-free-dialing-limitations-and-restrictions.md), ou en savoir plus sur les restrictions et les limitations de composer le numéro gratuit.


|**Détails**|**Description**|
|:-----|:-----|
|Est gratuit appelant disponibles ?  <br/> |Oui  <br/> |
|Formats de numérotation  <br/> | 001 803 CCC XXXX <br/>  007 803 CCC XXXX <br/> |
|Code de pays/région ISO  <br/> |ID  <br/> |
|Île/territoire couvert  <br/> |Non applicable  <br/> |
|Île/territoire non couvert  <br/> |Non applicable  <br/> |
|Disponibilité de la ligne filaire/fixe  <br/> |Oui  <br/> |
|Disponibilité de réseau mobile  <br/> |Partiel. Voir les commentaires.  <br/> |
|Commentaires, les restrictions et les problèmes d’accès  <br/> |Deux formats de numéro sont fournis pour une meilleure couverture. Le format XXXX CCC 007 803 format numérique est accessible à partir du réseau Telkom PT, y compris Telkomsel et XL mobile. Le format de numéro 001 803 CCC XXXX est accessible via le réseau Indosat. Pour mobile : matrice cellulaire (préfixe 0855, 0858, 0815 et 0816) uniquement.  <br/> |
