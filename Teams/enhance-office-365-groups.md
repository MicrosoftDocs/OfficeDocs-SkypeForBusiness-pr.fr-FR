<a name="enhance-existing-microsoft-365-groups-with-microsoft-teams"></a>Améliorer les groupes Microsoft 365 existants avec Microsoft Teams
=======================================================

Les utilisateurs de Microsoft Teams peuvent améliorer un groupe Microsoft 365 existant avec la fonctionnalité Teams. Les utilisateurs peuvent améliorer un groupe Microsoft 365 public si le nombre de membres est inférieur ou égal à 5 000.

Pour ce faire, les utilisateurs doivent passer par le flux de création d’une équipe à partir du client Microsoft Teams. Sélectionnez **Créer**  >  **dans le groupe Microsoft 365,** puis choisissez le groupe existant que vous voulez améliorer avec Teams. Les membres existants du groupe sont automatiquement ajoutés en tant que membres à l’équipe.

> [!IMPORTANT]
> Seuls les propriétaires de groupe Microsoft 365 sont autorisés à améliorer un groupe existant avec Teams, et ce groupe doit contenir moins de 5 000 utilisateurs. Les équipes qui ont déjà été améliorées ne figureront pas dans la liste.
>
>L’option de création d’une équipe à partir d’un groupe Microsoft 365 ne sera pas disponible sauf si ces conditions ont été remplies.

Les utilisateurs peuvent également inviter une liste de distribution à une équipe, et les membres de cette liste sont ajoutés à l’équipe. Il s’agit d’un ajout une seule fois et les modifications apportées ultérieurement à l’appartenance aux groupes dans la liste de distribution ne seront pas répliquées ni synchronisées avec Teams.

![Capture d’écran de l’option de création d’une équipe à partir d’un groupe Microsoft 365.](media/Enhance_Existing_Office_365_groups_with_Microsoft_Teams_image2.png)

Vous pouvez également ajouter des groupes de sécurité à extension messagerie en tant que membres d'une équipe. En revanche, si par la suite vous ajoutez des membres au groupe de sécurité, ils ne seront pas automatiquement ajoutés à l'équipe. Vous devez ajouter les nouveaux membres séparément ou ajouter à nouveau le groupe de sécurité à l'équipe. Si vous ajoutez à nouveau le groupe de sécurité, la fonction de déduplication vérifie que les membres n'ont été ajoutés qu'une seule fois.

Il existe deux types de paramètres de confidentialité avec les groupes Microsoft 365 : **publics et privés.** Si les deux types de groupes peuvent être activés pour Teams, il y a une légère différence en ce qui concerne le libre-service.

-   Les utilisateurs peuvent rechercher des groupes publics et participer seuls sans approbation du propriétaire de l’équipe.

-   Les groupes privés ne peuvent pas faire l'objet d'une recherche et les utilisateurs peuvent uniquement les rejoindre si le propriétaire d'équipe les ajoutent en tant que membres.

Lors de la création d’une équipe dans Teams, le propriétaire d’un groupe privé Microsoft 365 existant a la possibilité d’utiliser l’appartenance au groupe Microsoft 365 pour créer l’équipe. Les utilisateurs peuvent ajouter leurs fichiers SharePoint et OneNote existants en ajoutant un onglet pour SharePoint et en fusionnant les fichiers OneNote.
