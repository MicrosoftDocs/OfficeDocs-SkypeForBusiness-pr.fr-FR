# <a name="sharing-files-in-microsoft-teams"></a>Partage de fichiers dans Microsoft Teams

> [!INCLUDE [new feature coming soon](includes/new-feature-coming-soon-article.md)]

Les fonctionnalités dans les équipes de partage de fichiers permettent aux utilisateurs de partager du contenu avec d’autres utilisateurs d’équipes dans leur organisation. Dans les équipes de partage est basé sur les paramètres configurés dans SharePoint et OneDrive, afin que ce que vous configurer pour SharePoint et OneDrive contrôle ainsi que le partage dans les équipes.

Les équipes de partage permet aux utilisateurs d’effectuer les opérations suivantes :

- Partager des fichiers à partir de OneDrive.

- Définir des autorisations pour les fichiers qu’ils souhaitent partager avec d’autres personnes.

- Partager des fichiers entre les équipes.

- Partager des fichiers à partir de la liste des derniers fichiers utilisés (en règle générale, ce sont les utilisateurs sont plus intéressés par le partage de fichiers).

- Rester au sein des équipes lorsqu’ils cliquent sur un nom de fichier pour ouvrir un fichier.

Les équipes réduit long URLS SharePoint et les URL qui pointent vers un fichier de navigateur. Les équipes utilise le nom du fichier à lier à un fichier. En outre, l’option **d’obtenir le lien** a été modifiée pour **Copier le lien** afin d’éliminer toute confusion que les utilisateurs peuvent avoir sur d’autres personnes donnant accès à un fichier.

## <a name="configure-sharing-in-onedrive-and-sharepoint"></a>Configurer le partage de OneDrive et SharePoint

Pour plus d’informations sur le partage de fichiers OneDrive et SharePoint, y compris comment configurer le partage et comment activer et désactiver le partage, voir :

- [Vue d’ensemble du partage externe](https://docs.microsoft.com/sharepoint/external-sharing-overview) - décrit ce qui se produit lorsque des utilisateurs partagent, selon qu’ils partagent et avec lesquelles.

- [Activer externe ou désactiver le partage](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) - décrit comment globaux et les administrateurs SharePoint peuvent modifier leurs paramètres de partage de niveau entreprise pour SharePoint et OneDrive.

- [Modification externe pour un site de partage](https://docs.microsoft.com/sharepoint/change-external-sharing-site) – décrit comment globaux et les administrateurs SharePoint peuvent activer externes ou désactiver le partage pour un site.

- [Modifier le type de liaison par défaut lorsque les utilisateurs obtiennent des liens pour partager](https://docs.microsoft.com/sharepoint/change-default-sharing-link) - décrit comment définir le type de lien par défaut afin qu’il soit plus restrictif.
