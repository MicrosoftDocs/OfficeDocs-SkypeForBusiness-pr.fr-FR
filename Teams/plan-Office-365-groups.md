<a name="plan-for-microsoft-365-groups-when-creating-teams-in-microsoft-teams"></a><span data-ttu-id="e883f-103">Planifier les groupes Microsoft 365 lors de la création d’équipes dans Microsoft teams</span><span class="sxs-lookup"><span data-stu-id="e883f-103">Plan for Microsoft 365 Groups when creating teams in Microsoft Teams</span></span>
==========================================================

<span data-ttu-id="e883f-p101">Lorsque vous envisagez d’utiliser les groupes Microsoft 365 ou que vous créez des équipes, considérez l’utilisation de l’équipe, qui doit avoir accès et quel est l’objectif de l’équipe. Soyez vigilant en particulier sur le nombre de canaux que vous créez, car les personnes peuvent facilement devenir saturées en fonction du contenu de la page.</span><span class="sxs-lookup"><span data-stu-id="e883f-p101">When considering the use of Microsoft 365 Groups or when creating teams, consider what the team will be used for, who should have access, and what outcome the team will expect to achieve. Pay special attention to the number of channels you create as people can quickly become overrun by content spread too thin (across too many channels).</span></span>

<span data-ttu-id="e883f-106">Il existe deux scénarios qui nécessitent une présentation de la planification de groupes Microsoft 365 et leurs répercussions sur (ou par) de Microsoft teams :</span><span class="sxs-lookup"><span data-stu-id="e883f-106">There are two scenarios that warrant some discussion around planning of Microsoft 365 Groups and their impact on (or by) Microsoft Teams:</span></span>

-   <span data-ttu-id="e883f-p102">Tout d’abord, dans la mesure où les clients peuvent avoir des investissements en groupes, vous prenez actuellement en charge les groupes publics et privés de moins de 5000 membres. Comme mentionné précédemment, vous souhaitez gérer l’appartenance des personnes à une équipe à l’aide du client teams plutôt que du centre d’administration 365 Microsoft. Étant donné que ce scénario est utilisé pour les conversations thématiques dans les groupes Microsoft 365, il est intéressant de noter qu’une conversation de groupe est essentiellement un message électronique et non une conversation dans un canal d’équipe. Informez vos interlocuteurs de cette différence et Suggérez-leur d’adopter le format de messages de chat plus flexible dans Teams, ou d’envoyer le groupe à l’aide d’Outlook ou OWA.</span><span class="sxs-lookup"><span data-stu-id="e883f-p102">First, since customers could have existing investments in Groups, we currently support both Public and Private groups of less than 5000 members. As mentioned previously, you want to manage the membership of people to a team using the Teams client rather than the Microsoft 365 admin center. Given this scenario, if people are used to threaded conversations in Microsoft 365 Groups, it is worthwhile noting that a Groups conversation is essentially email and not the same as a chat message in a Teams channel. Educate your people about this difference and suggest they adopt the more flexible chat message format in Teams versus emailing the Group using Outlook or OWA.</span></span>

-   <span data-ttu-id="e883f-p103">Deuxièmement, pour les clients qui n’ont pas de groupes existants définis dans Microsoft 365, vous pouvez les créer à l’aide du centre d’administration Microsoft 365, du Web teams ou des clients de bureau. Comme nous l’avons mentionné précédemment, gérez tout le futur de votre appartenance au groupe Microsoft 365 à l’aide du client Teams. Dans la mesure où l’appartenance à une équipe définit également l’appartenance aux groupes Microsoft 365, vous devez préparer les utilisateurs à ce changement.</span><span class="sxs-lookup"><span data-stu-id="e883f-p103">Second, for customers who don't have existing Groups defined in Microsoft 365, you can either create them using the Microsoft 365 admin center, the Teams web, or desktop clients. As mentioned previously, manage all future membership to the Microsoft 365 group using the Teams client. Since membership to a team is also defining membership to Microsoft 365 Groups, you should prepare people for this change.</span></span>
