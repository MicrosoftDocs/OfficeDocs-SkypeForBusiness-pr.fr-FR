<a name="support-resources-for-microsoft-teams"></a>Ressources de support pour Microsoft Teams
=====================================

Voici les ressources disponibles pour vous aider à utiliser Microsoft Teams :

-   [*Conseiller de déploiement pour Microsoft Teams*](https://go.microsoft.com/fwlink/?linkid=843465)

-   [*successwithteams.com*](https://go.microsoft.com/fwlink/?linkid=854396)

-   [*Version d’évaluation de Microsoft Teams pour développeurs*](https://go.microsoft.com/fwlink/?linkid=854397)

-   [*Aide de Microsoft Teams*](https://support.office.com/en-us/Teams)

-   [*Kit de réussite des clients de Microsoft Teams*](https://go.microsoft.com/fwlink/?linkid=846006)

-   [*Communauté technique Microsoft Teams*](https://go.microsoft.com/fwlink/p/?linkid=832751)

-   Canal vidéo YouTube : [*Microsoft Teams*](https://go.microsoft.com/fwlink/?linkid=854398)

-   [Aide de l'utilisateur final](https://support.office.com/en-us/teams)

-   [Problèmes connus](https://support.office.com/en-US/article/Known-issues-for-Microsoft-Teams-04b35d1b-bdca-420a-991b-878da5157650)

Les options de commentaires et de support incluent les éléments suivants :

-   Microsoft Teams : [*Envoyez vos questions, bogues, demandes de fonctionnalité et contributions | MSDN*](https://go.microsoft.com/fwlink/?linkid=854399)

-   [*Portail de commentaires sur Microsoft Teams sur UserVoice*](https://go.microsoft.com/fwlink/?linkid=854400)

-   [*Communauté technique Microsoft Teams*](https://go.microsoft.com/fwlink/p/?linkid=832751)

-   Dans le client Microsoft Teams (application de bureau ou Web), dans la partie inférieure gauche, vous pouvez cliquer ou appuyer sur le bouton **Commentaires**, puis sur **Partager une idée**.
