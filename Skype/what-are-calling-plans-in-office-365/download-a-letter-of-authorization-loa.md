# <a name="download-a-letter-of-authorization-loa"></a>Télécharger un courrier dʼautorisation 

La **Lettre d’autorisation (LOA)** nous donne l’autorisation de demander, de votre part, transférer l’ou les numéros existant à partir d’un autre fournisseur de services en cas de litige. Utilisez une lettre d’accord, si vous avez des numéros de service pour les ponts de conférence audio, des surveillances automatiques ou des autres numéros de service, le numéro d’appel gratuit, numéros de téléphone, ou ont plus de 999 numéros de téléphone de l’utilisateur (abonné) que vous devez transférer à Skype pour les entreprises (ou si vous ne souhaitez pas utiliser l' **Assistant de portage de numéro Local**). Vous devez [soumettre manuellement une demande de service personnalisé](manually-submit-a-custom-service-request.md) avec cette LOA pour obtenir les numéros de téléphone à partir de votre fournisseur de service en cours de transfert.
  
Si vous avez des numéros de service pour les ponts de conférence audio, des surveillances automatiques ou des autres numéros de service, le numéro d’appel gratuit, numéros de téléphone, ou plus de 999 numéros de téléphone utilisateur dont vous avez besoin pour transférer vers Skype pour entreprise, vous devez envoyer manuellement un ordre de port.
  
Si vous n’êtes pas sûr des types de numéros de téléphone que vous demandera à transférer, voir [différents types de numéros de téléphone utilisés pour les Plans d’appel](different-kinds-of-phone-numbers-used-for-calling-plans.md).
  
> [!IMPORTANT]
> Lorsque vous transférez manuellement les numéros de téléphone à l'aide d'un courrier d'autorisation (LOA), vérifiez que vous **sélectionnez le type de numéro de téléphone approprié**. Vous devez **soumettre des demandes de transfert distinctes** pour chaque type de numéro de téléphone que vous voulez transférer.
