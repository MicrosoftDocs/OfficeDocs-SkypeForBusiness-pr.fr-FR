# Quel est le statut de mes demandes de transfert ?

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'avertissement.  
  
Vous pouvez consulter l'état de votre commande port en accédant à l' **Skype centre d'administration entreprise** > **vocale** > **commandes Port**. Chaque statut de la commande port apparaîtront dans la colonne **statut**. Si vous aider, [Contacter le support Office 365 pour les entreprises - Aide de l'administrateur](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
  
Le tableau suivant répertorie les statuts de commande port, ainsi que les actions à qu'entreprendre si nécessaire.
  
|**État**|**Pouvez-vous afficher la demande ?**|**Pouvez-vous modifier la demande ?**|**Pouvez-vous annuler la demande ?**|**Pouvez-vous supprimer la demande ?**|**Description**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Traitement** <br/> |Oui  <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |L'administrateur a créé l'ordre, et il est été reçu par Microsoft.  <br/> |
|**Contact de l'opérateur** <br/> |Oui  <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |La commande a été reçue et approuvée par Microsoft et nous travaillons avec l'opérateur perdre obtenir approuvé.  <br/> |
|**Transfert approuvé** <br/> |Oui  <br/> |Date effective de transfert (FOC)  <br/> |Oui  <br/> |Non  <br/> |La commande a été acceptée par le support de perdre, et la date de produits gratuits a été définie.  <br/> |
|**Transfert en attente** <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |Non  <br/> |Le transfert est inférieure à 24 heures absent (e), pour que l'ordre n'est plus peut être modifié ou annulé.  <br/> |
|**Erreur** <br/> |Non  <br/> |Oui  <br/> |Oui  <br/> |Oui (pour l'instant, vous ne pouvez pas supprimer l'ordre de port si une erreur est générée. L'ordre de port doit être recréée, ou vous devez contacter le [Contacter le support Office 365 pour les entreprises - Aide de l'administrateur](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).  <br/> |L'opérateur d'origine a rejeté la demande.  <br/> |
|**Terminée** <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |Non  <br/> |Les numéros ont été transférés.  <br/> |
|**Annulée** <br/> |Non  <br/> |Oui  <br/> |Non  <br/> |Non  <br/> |L'administrateur a annulé la demande.  <br/> |
   
 **Pour obtenir des instructions étape par étape terminées, voir [Transférer des numéros de téléphone vers Office 365](transfer-phone-numbers-to-office-365.md).**
