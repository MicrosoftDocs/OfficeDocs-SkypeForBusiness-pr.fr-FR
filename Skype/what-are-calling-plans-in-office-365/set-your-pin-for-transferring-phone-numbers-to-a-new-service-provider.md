# Définir votre code confidentiel pour le transfert des numéros de téléphone à un fournisseur de services

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'avertissement.  
  
Pour transférer ou  *port des*  numéros de téléphone à partir de Skype entreprise Online vers un autre fournisseur de services de téléphone ou opérateur, vous devez définir manuellement un code confidentiel. Après avoir défini le code confidentiel, vous devez inclure lorsque vous demandez à un numéro de téléphone déconnecter le port. Ce n'est pas différente de celle du Office 365 **Prend en charge le code confidentiel** qui est utilisé pour vous authentifier lorsque vous appelez dans prise en charge. La prise en charge le code confidentiel est un code généré dynamiquement que vous entrez dans notre système téléphonique automatisé mais n'est pas utilisé dans Skype entreprise Online. Si vous recherchez pour le support de votre code confidentiel, accédez[Obtenir votre code PIN du support](https://support.office.com/article/cb0d3ad7-da00-4b71-96a2-5a1902f144aa).
  
> [!IMPORTANT]
> Un port arrière code confidentiel est utilisé uniquement pour les organisations aux États-Unis. 
  
SEEE [Transférer des numéros de téléphone vers Office 365](transfer-phone-numbers-to-office-365.md) pour plus d'informations sur le transfert et porter en/des numéros de téléphone.
  
Voici quelques informations spécifiques sur ce code confidentiel que vous devez savoir :
  
- Si un code confidentiel n'est pas défini, vous ne pourrez pour transférer ou port des numéros de téléphone à partir de Skype entreprise Online.
    
- Il peut contenir 6 à 10 chiffres (nombres).
    
- Il ne peut pas contenir des lettres ou des caractères spéciaux.
    
- Le code confidentiel par défaut est vide, mais si vous insérez une dans, vous ne peut pas supprimer ou définissez-la sur vide.
    
- Vous pouvez mettre à jour ou modifier le code confidentiel après avoir une mis en.
    
## Configurer votre code confidentiel

1. Connectez-vous à Office 365 à l'aide de votre compte professionnel ou scolaire.
    
2. Accédez au **Centre d'administration Office 365** > **administrateur centres** > **Skype Entreprise**.
    
3. Dans le volet de navigation gauche, sélectionnez **vocale** > **commandes Port**.
    
4. Cliquez sur **définir vers le haut et de gérer le code confidentiel** qui est utilisé pour transférer ou migrer des numéros à un autre transporteur de service.
    
5. Dans la **définir ou modifier le port de votre absence code confidentiel** panel, entrez votre code confidentiel, cliquez sur **Enregistrer**.
