# Combien de numéros de téléphone vous pouvez obtenir ?

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'avertissement.  
  
Lorsque vous recherchez et obtenir des numéros de téléphone pour votre organisation, vous pouvez obtenir des numéros de téléphone supplémentaires que vous avez attribué les licences. Mais cela dépend de types de licences que vous avez achetées et affectées et les types de numéros de téléphone. Vous pouvez cliquer [Différents types de numéros de téléphone utilisés pour l'appel d'offre](different-kinds-of-phone-numbers-used-for-calling-plans.md) pour savoir comment les numéros de téléphone différentes qui sont utilisées dans Skype entreprise Online.
  
Vous pouvez voir le nombre de numéros de téléphone que vous pouvez accéder dans la page de **numéros de téléphone** dans la Skype centre d'administration entreprise, ou vous pouvez exécuter l'applet de commande[Get-CsOnlineTelephoneNumberAvailableCount](https://technet.microsoft.com/en-us/library/mt634605.aspx) .
  
> [!IMPORTANT]
> Les limites ci-dessous n'incluent pas les numéros transférés vers Microsoft.
