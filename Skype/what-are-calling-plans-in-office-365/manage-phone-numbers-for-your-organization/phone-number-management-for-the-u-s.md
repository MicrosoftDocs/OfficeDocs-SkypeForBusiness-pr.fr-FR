# <a name="phone-number-management-for-the-us"></a>Gestion de numéros de téléphone pour les États-Unis

Cette table permet de rechercher des informations sur l’obtention et la gestion des numéros de téléphone des États-Unis (Porto Rico) pour Skype pour l’activité en ligne. 
  
Pour plus d’informations, voir [Gérer les numéros de téléphone pour votre organisation](manage-phone-numbers-for-your-organization.md).
  
|**Ce que vous voulez faire ?**|**Effectuez ces étapes et télécharger les formulaires corrects**|
|:-----|:-----|
|Vous devez obtenir les numéros des utilisateurs qui ne figurent pas dans le Skype pour centre d’administration de Business.  <br/> |Téléchargez le [nouvelle demande de numéro de téléphone pour les États-Unis (numéros d’utilisateur) (v.3.0(en.us)](../../downloads/new-phone-number-request-for-the-u.s.-(user-numbers)-(v.3.0)-(en.us).pdf) former, remplir les extraire et les [Envoyer pour nous](mailto:gcstnmsd@microsoft.com).  <br/> |
|Vous devez obtenir les numéros de téléphone transférées vers notre serveur.  <br/> | Téléchargez un de ces formulaires : <br/> [Lettre d’autorisation (LOA) pour les États-Unis (numéros d’utilisateur et de service) (v.3.1)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(user-and-service-numbers)-(v.3.1)-(en.us).pdf) <br/> [Lettre d’autorisation (LOA) pour les États-Unis (numéros verts) (v.3.1)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(toll-free-numbers)-(v.3.1)-(en.us).pdf) <br/><br/>**Remarque :** Si vous tentez d’obtenir plus de 999 numéros de téléphone de l’abonné ou l’utilisateur, vous devrez remplir à une lettre d’autorisation (LOA) et nous l’envoyez. Pour plus d’informations, consultez [transfert des numéros de téléphone à Office 365](../../what-are-calling-plans-in-office-365/transfer-phone-numbers-to-office-365.md) .           Remplir en et [nous l’envoyer](mailto:ptn@microsoft.com).  <br/> |
|Vous devez modifier le numéro de l’utilisateur à un numéro de service.  <br/> |[Envoyez-nous](mailto:ptn@microsoft.com) votre code confidentiel et la liste des numéros d’utilisateur que vous souhaitez convertir en nombres de service. Pour configurer un code confidentiel, allez [ici](../../what-are-calling-plans-in-office-365/set-your-pin-for-transferring-numbers-to-a-new-service-provider.md).  <br/> |
|Vous devez transférer des nombres d’une organisation d’Office 365 à l’autre.  <br/> |[Envoyer](mailto:ptn@microsoft.com) la liste des numéros de téléphone de l’utilisateur (abonné) à transférer, ainsi que le code confidentiel de l’organisation source, dans lequel vous souhaitez transférer les numéros de. Pour configurer un code confidentiel, allez [ici](../../what-are-calling-plans-in-office-365/set-your-pin-for-transferring-numbers-to-a-new-service-provider.md).  <br/> |
