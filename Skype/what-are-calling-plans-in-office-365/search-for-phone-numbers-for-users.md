# Rechercher des numéros de téléphone pour les utilisateurs

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'avertissement.  
  
Lorsque vous configurez les utilisateurs de votre organisation d'émission et réception d'appels téléphoniques, vous devez utiliser le **Centre d'administration Skype Entreprise**et obtenez d'abord les numéros de téléphone peuvent être affectées aux utilisateurs. Le numéro de téléphone que vous attribuez à un utilisateur sera un numéro de téléphone que vous avez choisi pour votre organisation et apparaîtront dans la liste déroulante lorsque vous modifiez les propriétés de l'utilisateur, cliquez sur **attribuer**.
  
Avant d'affecter des numéros de téléphone à vos utilisateurs, vous devez utiliser la page **nouveaux numéros** pour rechercher des nombres qui sont disponibles pour vous dans une zone. Vous pouvez effectuer des recherches par **pays/région**, ** état** et **Ville** et puis entrez le nombre de numéros de téléphone que vous aurez besoin pour vos utilisateurs.
  
Si vous avez besoin d'aide pour obtenir des numéros de téléphone, vous pouvez voir [Gérer les numéros de téléphone pour votre organisation](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) ou[Contacter le support Office 365 pour les entreprises - Aide de l'administrateur](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
  
## Recherche des numéros de téléphone

 **Pour rechercher des numéros de téléphone pour vos utilisateurs**
  
1. Connectez-vous à Office 365 à l'aide de votre compte professionnel ou scolaire.
    
2. Sélectionnez **Centre d'administration Office 365** > **Skype Entreprise**.
    
3. Dans le volet de navigation gauche, accédez à **voix** > **numéros de téléphone**, puis cliquez sur **Ajouter nouveau numéro** > **nouveaux numéros utilisateur**.
    
    **IMPORTANT**: pour que l'option **Voix** apparaisse dans le volet de navigation gauche du Centre d'administration Skype Entreprise, vous devez d'abord acheter au moins une **licence Entreprise E5**, une licence de composant additionnel **Système téléphonique**, ou une licence de composant additionnel **Audioconférence**.
    
4. Dans la page **Ajouter des numéros de nouvel utilisateur**, cliquez sur le **pays** et les listes **d'état** pour sélectionner l'état que vous souhaitez sélectionner des numéros et puis sélectionnez l'indicatif régional de la ville dans la liste des **villes**.
    
5. Sous **quantité**, entrez le nombre de numéros de téléphone que vous voulez utiliser pour votre organisation et cliquez sur **Ajouter** pour créer une réservation. Vous avez 10 minutes pour sélectionner vos numéros de téléphone. Si vous prenez plus de 10 minutes, les nombres apparaîtront au pool de numéros de téléphone.
    
    > [!NOTE]
    > Vous pouvez voir le nombre de numéros de téléphone, qui sont basés sur le nombre de licences, répertorié en regard de **nombres Total utilisateur que vous pouvez acquérir**. 
  
6. Cliquez sur **Afficher les numéros** pour consulter la liste complète des numéros de téléphone. Cela est utile si vous ne voulez pas sélectionner un numéro de téléphone spécifique dans la liste.
    
7. Sélectionnez les numéros de téléphone souhaités, puis cliquez sur **numéros acquérir**.
    
    > [!IMPORTANT]
    > Vous pouvez acquérir plusieurs numéros de téléphone que vous avez des licences. Pour déterminer le nombre de numéros de téléphone vous pouvez acquérir, prendre le nombre de licences, ajouter 10 pour cent du nombre de licences, puis ajouter 10. Par exemple, si vous avez 100 licences **Nationaux appelant planifier** et/ou **International appelant planifier**, vous pouvez réserver des numéros de téléphone 120, en supposant que vous n'avez pas déjà acquis des numéros de téléphone pour ces 100 utilisateurs. > Pour plus d'informations, voir[Combien de numéros de téléphone vous pouvez obtenir ?](how-many-phone-numbers-can-you-get.md).
