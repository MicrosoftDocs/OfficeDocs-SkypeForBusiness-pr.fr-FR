# <a name="search-for-phone-numbers-for-users"></a>Rechercher des numéros de téléphone pour les utilisateurs

Lorsque vous configurez les utilisateurs de votre organisation d’effectuer et de recevoir des appels téléphoniques, vous devez utiliser le **Skype pour le centre d’administration Business**et tout d’abord obtenir les numéros de téléphone qui peuvent être affectés aux utilisateurs. Le numéro de téléphone que vous attribuez à un utilisateur sera un numéro de téléphone que vous avez choisi pour votre organisation et apparaît dans la liste déroulante lorsque vous modifiez les propriétés de l’utilisateur et que vous cliquez sur **attribuer**.
  
Avant d’affecter des numéros de téléphone à vos utilisateurs, vous devez utiliser la **nouveaux numéros de** page pour rechercher des nombres qui sont disponibles pour vous dans une zone. Vous pouvez effectuer une recherche par **pays/région**, **état**et **Ville**et puis entrez le nombre de numéros de téléphone que vous avez besoin pour vos utilisateurs. 
  
Si vous avez besoin d’aide pour obtenir des numéros de téléphone, vous pouvez voir [Gérer les numéros de téléphone pour votre organisation](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) ou un [Contact de prise en charge des produits professionnels - aide de l’administrateur](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
  
## <a name="search-for-phone-numbers"></a>Recherche des numéros de téléphone

 **Pour rechercher des numéros de téléphone pour vos utilisateurs**
  
1. Connectez-vous à Office 365 à l'aide de votre compte professionnel ou scolaire.
    
2. Sélectionnez **Centre d'administration Office 365** > **Skype Entreprise**.
    
3. Dans la navigation de gauche, accédez à **voix** > de**numéros de téléphone**, puis cliquez sur **Ajouter un nouveau numéro** > **nouveaux numéros d’utilisateur**.
    
    **IMPORTANT**: pour que l'option **Voix** apparaisse dans le volet de navigation gauche du Centre d'administration Skype Entreprise, vous devez d'abord acheter au moins une **licence Entreprise E5**, une licence de composant additionnel **Système téléphonique**, ou une licence de composant additionnel **Audioconférence**.
    
4. Dans la page **Ajouter de nouveaux numéros d’utilisateur** , cliquez sur le **pays** et les listes **d’état** sélectionnez l’état que vous souhaitez sélectionner des numéros de, puis sélectionnez le code zone de la ville à partir de la liste des **villes** .
    
5.  Sous **quantité**, entrez le nombre de numéros de téléphone que vous souhaitez pour votre organisation et cliquez sur **Ajouter** pour créer une réservation. Vous avez 10 minutes pour sélectionner vos numéros de téléphone. Si vous prenez plus de 10 minutes, les numéros apparaîtront dans le pool de numéros de téléphone.
    
    > [!NOTE]
    > Vous pouvez voir le nombre de numéros de téléphone, qui sont basés sur le nombre de licences, répertoriées en regard des **numéros d’utilisateur Total que vous pouvez acquérir**. 
  
6. Cliquez sur **Afficher les numéros** pour consulter la liste complète des numéros de téléphone. Cela est utile si vous ne voulez pas sélectionner un numéro de téléphone spécifique dans la liste.
    
7. Sélectionnez les numéros de téléphone que vous souhaitez, puis cliquez sur **les numéros d’acquisition**.
    
    > [!IMPORTANT]
    > Vous pouvez acquérir plus de numéros de téléphone que vous avez de licences. Pour déterminer le nombre de numéros de téléphone vous pouvez acquérir, prendre votre nombre de licences, ajouter 10 pour cent du nombre de licences et puis ajouter 10. Par exemple, si vous avez 100 licences **Domestiques Plan d’appel** ou **Envisagez d’appel International** , vous pouvez réserver des numéros de téléphone 120, en supposant que vous n’avez pas déjà acquis certains numéros de téléphone pour ces 100 utilisateurs. Pour plus d’informations, consultez [le nombre de numéros de téléphone vous obtiendrez ?](how-many-phone-numbers-can-you-get.md)
