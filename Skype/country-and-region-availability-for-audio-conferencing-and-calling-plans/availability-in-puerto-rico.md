# <a name="availability-in-puerto-rico"></a>Availability in Puerto Rico

[Find another country or region](country-and-region-availability-for-audio-conferencing-and-calling-plans.md), or learn more about country and region availability for Audio Conferencing and calling plans.

Here are the Office 365 services for Skype for Business and Microsoft Teams available for Puerto Rico.
  
## <a name="audio-conferencing"></a>Audioconférence

|**Fonctionnalité**|**Détails**|
|:-----|:-----|
|Is Audio Conferencing available for purchase?  <br/> |Oui  <br/> |
|Are phone numbers automatically assigned for Audio Conferencing?  <br/> |Non  <br/> |
|Dial-out capability using the Call Me feature  <br/> |Disponibles  <br/> |
   
## <a name="phone-system"></a>Phone System

|**Fonctionnalité**|**Détails**|
|:-----|:-----|

| Auto attendants <br/> |Disponibles  <br/> |
|Call queues  <br/> |Disponibles  <br/> |
|Messagerie vocale  <br/> |Disponibles  <br/> |
   
## <a name="calling-plans"></a>Calling Plans

|**Fonctionnalité**|**Détails**|
|:-----|:-----|
|Calling Plans are available to purchase?  <br/> |Oui  <br/> |
|Minutes available for Calling Plans  <br/> |3000 minutes nationales ou 600 minutes internationales (en fonction de ce qui arrive en premier par mois civil). Les appels nationaux aux États-Unis incluent ceux à destination de l'ensemble des 50 États des États-Unis et de Porto Rico. Les utilisateurs ayant reçu l'emplacement d'utilisation sous licence des États-Unis et/ou de Porto Rico partagent le même nombre de minutes.  <br/> |
|Can make outbound calls to?  <br/> | To 196 countries, including [these](users-can-make-outbound-calls-to-these-countries-and-regions.md).  <br/> |
