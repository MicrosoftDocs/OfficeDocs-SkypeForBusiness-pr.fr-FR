# <a name="set-up-your-computer-for-windows-powershell"></a>Configurer votre ordinateur pour Windows PowerShell

La gestion à distance de Skype entreprise Online à l’aide de Microsoft PowerShell est uniquement prise en charge sur les ordinateurs 64 bits exécutant l’un des systèmes d’exploitation suivants :

- Windows Server 2016

- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
Outre le système d’exploitation pris en charge, l’ordinateur doit également exécuter les éléments suivants :
  
- PowerShell
    
- Module Skype entreprise Online Connector
    
Vous trouverez des informations sur le téléchargement et l’installation de ces éléments dans les rubriques suivantes :
  
- [Télécharger et installer le module Skype entreprise Online Connector](download-and-install-the-skype-for-business-online-connector.md)
    
- [Télécharger et installer Windows PowerShell 5.1](download-and-install-windows-powershell-5-1.md)

- [Gestion de Skype entreprise Online avec Office 365 PowerShell](https://github.com/MicrosoftDocs/OfficeDocs-Enterprise/blob/live/Enterprise/powershell/manage-skype-for-business-online-with-office-365-powershell.md)
