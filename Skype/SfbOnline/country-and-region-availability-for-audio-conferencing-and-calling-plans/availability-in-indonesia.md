# <a name="availability-in-indonesia"></a>Disponibilité en Indonésie

Voici les services Office 365 pour Skype pour les entreprises et les Teams Microsoft disponible pour l’Indonésie.

|**Fonctionnalité**|**Détails**|

|L'audioconférence est-elle disponible à l'achat ?  <br/> |Oui  <br/> |

| Standards automatiques <br/> |Disponibles  <br/> |

|Messagerie vocale  <br/> |Disponibles  <br/> |
