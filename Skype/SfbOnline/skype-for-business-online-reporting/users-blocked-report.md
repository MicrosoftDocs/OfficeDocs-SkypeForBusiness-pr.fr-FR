# <a name="users-blocked-report"></a>Rapport sur les utilisateurs bloqués

Le nouveau Skype pour des **rapports** de tableau de bord affiche la vue d’ensemble de l’activité sur le Skype pour les produits d’entreprise de votre organisation. Il vous permet de réaliser un zoom avant les rapports au niveau du produit individuel à vous fournir des informations plus précises sur les activités au sein de chaque produit. Par exemple, vous pouvez utiliser le rapport **Skype pour les utilisateurs professionnels bloqué** pour voir les utilisateurs de votre organisation qui ont été bloqués d’effectuer les appels RTPC. Ce rapport, ainsi que d’autre Skype pour les rapports d’entreprise, vous donne des détails sur l’activité, y compris l’utilisation de TLS au sein de votre organisation.
  
 Consultez la [vue d’ensemble des rapports](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) pour les rapports supplémentaires qui sont disponibles.
  
> [!NOTE]
> Vous pouvez accéder à tous les rapports Skype Entreprise lorsque vous vous connectez au centre d'administration d'Office 365 en tant qu'administrateur. 
  
## <a name="how-to-get-to-the-skype-for-business-users-blocked-report"></a>Comment obtenir le Skype pour les utilisateurs professionnels bloqué rapport

- Accédez au **Centre d’administration Office 365** > **Admin centres** > **Skype pour le centre d’administration Business** > **rapports** > **les utilisateurs bloqués**.
    
## <a name="interpret-the-skype-for-business-users-blocked-report"></a>Interpréter le Skype pour le rapport des utilisateurs bloqués professionnels

Vous pouvez obtenir une vue en utilisateurs bloqués en examinant chacun des colonnes affichées.
  
Voici une illustration du rapport. 
  
![Rapport des utilisateurs bloqués](../images/df50a413-7a51-4340-a59b-3f83de941762.png)

La table affiche le détail de tous les utilisateurs qui ne peuvent pas effectuer des appels. Affiche tous les utilisateurs ayant système téléphonique ou une conférence Audio qui leur sont affectées. Vous pouvez ajouter ou supprimer des colonnes dans ce tableau.
***
![Numéro 1](../images/sfbcallout1.png)
*   **L’ID utilisateur** est l’utilisateur signe.
*   **Numéro de téléphone** est le numéro qui est affecté à un utilisateur. 
*   **Heure de l’action Bloquer** est l’heure (UTC) que l’utilisateur n’a pas pu effectuer des appels.
*   **Action de blocage** est le type d’action qui a été prise pour interdire à l’utilisateur.
*   **Motif d’action de bloc** est la raison pour laquelle que l’utilisateur a été bloqué à partir de l’appel.
***
![Numéro 2](../images/sfbcallout2.png)<br/>
Cliquez pour déplacer une colonne vers **Pour regrouper les informations autour d'une colonne en particulier, faites glisser et déplacez l'en-tête de la colonne ici** si vous souhaitez visualiser toutes les données regroupées dans une ou plusieurs colonnes.
***
![Numéro 3](../images/sfbcallout3.png)<br/>
Vous pouvez également exporter les données du rapport vers un fichier Excel .csv, en cliquant ou en appuyant sur le bouton **Exporter vers Excel**.

Cela exportera les données de tous les utilisateurs et vous permettra de trier et de filtrer plus simplement pour une analyse ultérieure. Si vous avez moins de 2000 utilisateurs, vous pouvez trier et filtrer dans la table dans le rapport lui-même. Si vous avez plus de 2 000 utilisateurs, vous devrez exporter les données pour pouvoir les trier et les filtrer.
***
