# <a name="pstn-minute-pools-report"></a>Rapport de pools minute RTPC

>[!NOTE]
>Ce rapport n’est disponible pour afficher un aperçu des clients.

Le nouveau Skype pour la zone Centre d’administration Business **rapports** affiche vous appelant et audio conférence activité dans votre organisation. Il vous permet de vous y plonger le rapports pour vous fournir des informations plus précises sur les activités de chaque utilisateur. Par exemple, vous pouvez utiliser le rapport **Skype pour les pools d’entreprise RTPC minutes** pour voir le nombre de minutes consommées au cours du mois en cours au sein de votre organisation.
  
Consultez la [vue d’ensemble des rapports](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) pour les rapports supplémentaires qui sont disponibles.
  
Ce rapport, ainsi que l’autre Skype pour les rapports d’entreprise, vous donne plus d’informations sur l’activité au sein de votre organisation. Ces informations sont très utiles lors de l’analyse, la planification et les décisions autres pour votre organisation et de la configuration des [Crédits de Communications](../skype-for-business-and-microsoft-teams-add-on-licensing/what-are-communications-credits.md)
  
> [!NOTE]
> Vous pouvez accéder à tous les rapports Skype Entreprise lorsque vous vous connectez au centre d'administration d'Office 365 en tant qu'administrateur. 
  
## <a name="how-to-get-to-the-skype-for-business-pstn-minute-pools-report"></a>Comment obtenir le Skype pour rapport de pools minute Business PSTN

- Accédez au **Centre d’administration Office 365** > **Admin centres** > **Skype pour le centre d’administration Business** > **rapports** > **pools de minute RTPC**.
    
> [!NOTE]
> En fonction de l’abonnement à Office 365 que vous avez, vous ne voyez pas tous les mêmes coordonnées indiquées ici. 
  
## <a name="interpret-the-skype-for-business-pstn-minute-pools-report"></a>Interpréter le Skype pour rapport de pools minute Business PSTN

Vous pouvez obtenir un affichage dans Skype de l’utilisateur pour les pools d’entreprise minutes en examinant les colonnes qui s’affichent.
  
Voici une illustration du rapport.
  
## 

![Rapport de pools de Skype pour minute d’entreprise RTPC](../images/f5da5ca9-3466-4234-8f33-ab50ac5eb781.png)
  
***
![Numéro 1](../images/sfbcallout1.png)<br/>Le tableau montre la répartition des pools minutes par licence (capacité) et l’emplacement d’utilisation. 
*    **Fonctionnalité** est le plan de licence/service utilisé pour l’appel. Les plans de service/licence que peuvent apparaître dans ce rapport sont les suivantes :
     * MCOPSTNPP - crédits de Communications
     * MCOPSTN1 - intérieur appelant Plan (3000-minute US/1200-minute UE les plans
     * MCOPSTN2 - Plan d’appel International
     * MCOPSTN5 - intérieur appelant Plan (plan d’appel 120 minutes)
     * MCOMEETADD - conférence Audio
     * MCOMEETACPEA - paie par Minute audioconférence
*    **Description de la fonction** est une description du type de licence utilisé pour l’appel.
*    **Pool de Minute de pays** est l’emplacement de l’utilisation de licence l’ou les utilisateurs qui partagent le pool minute. 
*    **Minutes d’utilisé** est le nombre de minutes utilisée chaque mois.
*    **Nombre total de Minutes** est le nombre total de minutes pour le mois. 
*    **Pourcentage utilisé** est le pourcentage de temps en minutes pour le mois. 
***
![Numéro 2](../images/sfbcallout2.png)<br/>Cliquez pour déplacer une colonne vers **Pour regrouper les informations autour d'une colonne en particulier, faites glisser et déplacez l'en-tête de la colonne ici** si vous souhaitez visualiser toutes les données regroupées dans une ou plusieurs colonnes. 
***
![Numéro 3](../images/sfbcallout3.png)<br/>Vous pouvez également exporter les données du rapport vers un fichier Excel .csv, en cliquant ou en appuyant sur le bouton **Exporter vers Excel**. <br/><br/> Cela exportera les données de tous les utilisateurs et vous permettra de trier et de filtrer plus simplement pour une analyse ultérieure. Si vous avez moins de 2000 utilisateurs, vous pouvez trier et filtrer dans la table dans le rapport lui-même. Si vous avez plus de 2 000 utilisateurs, vous devrez exporter les données pour pouvoir les trier et les filtrer.
