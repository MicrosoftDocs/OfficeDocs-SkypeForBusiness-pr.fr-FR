# <a name="user-statistics-and-organization-information"></a>Statistiques utilisateur et informations sur l’organisation

Le tableau de bord Skype Entreprise vous donne un aperçu de la configuration des services pour votre entreprise. Si vous n’avez pas encore configuré Skype Entreprise, reportez-vous à la rubrique [Configurer Skype Entreprise Online](set-up-skype-for-business-online.md) pour des instructions étape par étape.
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>Comment modifier les statistiques utilisateur de Skype Entreprise Online

|||
|:-----|:-----|
|**Propriété** <br/> |**Comment la modifier** <br/> |
|Utilisateurs de Skype Entreprise Online  <br/> |Pour ajouter ou bloquer des personnes à l’aide de l’application Skype entreprise, reportez-vous à la section [attribution ou suppression de licences pour microsoft 365 pour les entreprises](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) ou [Suppression de licences de utilisateurs dans Microsoft 365 pour les entreprises](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Pour obtenir la liste complète des étapes à suivre lorsqu’un utilisateur quitte votre entreprise, reportez-vous à la rubrique [supprimer un ancien employé de Microsoft 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Utilisateurs pour lesquels l’audioconférence est activée  <br/> |Pour activer les services d’audioconférence, reportez-vous à [essayer ou acheter une audioconférence dans Microsoft 365 ou Office 365](../audio-conferencing-in-office-365/try-or-purchase-audio-conferencing-in-office-365.md).  <br/> |
|Utilisateurs dans le cloud  <br/> |Nombre d’utilisateurs de Skype entreprise Online disposant de comptes Microsoft 365 ou Office 365.  <br/> Pour ajouter des personnes à votre abonnement Microsoft 365 ou Office 365 (c’est-à-dire, créer des comptes Microsoft 365 ou Office 365 pour eux), consultez la rubrique [Ajouter des utilisateurs individuellement ou en bloc à Office 365-aide de l’administrateur](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Utilisateurs synchronisés et hébergés en ligne  <br/> |Nombre d’utilisateurs hybrides créés dans un déploiement Skype entreprise Server local, mais qui sont synchronisés avec Microsoft 365 ou Office 365.  <br/> Pour configurer une connectivité hybride Skype Entreprise, reportez-vous à la rubrique [Déployer une connectivité hybride entre Skype Entreprise Server et Skype Entreprise Online](https://technet.microsoft.com/library/jj204669.aspx).  <br/> |
   
> [!NOTE]
> Si Non disponible est indiqué, cela signifie qu’il n’y a pas d’informations ou qu’il y avait des erreurs. 
  
## <a name="how-to-change-information-about-your-organization"></a>Comment modifier les informations sur votre organisation ?

|||
|:-----|:-----|
|**Propriété** <br/> |**Comment la modifier** <br/> |
|Nom de l’organisation  <br/> |Il s’agit du nom que vous avez entré lors de l’inscription à Microsoft 365 ou Office 365. Pour le modifier, reportez-vous à la rubrique [Modifier l’adresse, le contact technique et les autres informations de votre organisation](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|ID de l’organisation  <br/> |Vous ne pouvez pas modifier l’ID de votre organisation.  <br/> |
|Organisation créée  <br/> |Vous ne pouvez pas modifier cette valeur.  <br/> |
|Synchronisation d’Active Directory activée  <br/> |Indique si votre organisation est synchronisée avec une organisation Skype Entreprise Online sur site.  <br/> Pour configurer votre environnement Active Directory local, reportez-vous à la rubrique [configurer la synchronisation d’annuaires pour Microsoft 365 ou Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Domaines de l'organisation  <br/> |Lorsque vous configurez initialement Microsoft 365 ou Office 365, il s’agit des domaines enregistrés.  <br/> Pour ajouter un domaine, voir [ajouter plusieurs domaines à Microsoft 365 ou Office 365](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207).  <br/> Pour supprimer un domaine, voir [supprimer un domaine de Microsoft 365 ou Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c).  <br/> |
