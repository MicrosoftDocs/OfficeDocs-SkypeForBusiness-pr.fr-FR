# <a name="skype-for-business-online-admin-center-users-page-limitations"></a>Skype Entreprise Limitations de la page utilisateurs du Centre d’administration en ligne

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

Pour des raisons de **Skype Entreprise centre d’administration** affiche les 1 000 premiers utilisateurs dans votre liste d’utilisateurs. Si vous avez plus de 1 000 utilisateurs, utilisez  ![ filtrer.](../images/d360528a-a05a-4f0d-a7fd-25a6c696206a.png) et **recherchez** ![ pour trouver les personnes ](../images/5cc29d50-4eec-48a4-bdad-bbeae230f9ae.png) spécifiques que vous recherchez.
  
## <a name="bulk-editing"></a>Modification en bloc
<a name="__top"> </a>

1. Sélectionnez les 1 000 premiers utilisateurs, ou toutes les personnes  renvoyées par votre filtre ou recherche, en cochant la case en haut de la liste.
    
2. Cliquez **sur Modifier.** ![ ](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png)
    
Vous pouvez modifier le nombre d’utilisateurs affichés sur chaque page avec le contrôle en bas de la liste :
  
![Navigation dans la page utilisateurs pour modifier le nombre d’utilisateurs sur la page.](../images/7d8ba24e-e3e8-4ff8-92f7-98ac143aab6d.png)
