# <a name="download-a-letter-of-authorization-loa"></a>Télécharger un courrier d'autorisation (LOA) 

Le **courrier d’autorisation (LOA)** nous donne l’autorisation de demander, pour votre compte, le transfert du ou des numéros existants d’un autre fournisseur de services en cas de différend. Utilisez un LOA si vous avez des numéros de service pour des ponts d’audioconférence, des standards automatiques ou d’autres types de numéro de service, des numéros de téléphone gratuits ou plus de 999 numéros de téléphone d’utilisateur (abonné) que vous devez transférer vers Skype Entreprise (ou si vous ne souhaitez pas utiliser l’**assistant de transfert de numéros locaux**). Vous devez [envoyer manuellement une demande de service personnalisé](manually-submit-a-custom-service-request.md) avec ce LOA pour obtenir les numéros de téléphone transférés depuis votre fournisseur de services actuel.
  
Si vous avez des numéros de service pour des ponts d’audioconférence, des standards automatiques ou d’autres types de numéro de service, des numéros de téléphone gratuits ou plus de 999 numéros de téléphone d’utilisateur que vous devez transférer vers Skype Entreprise, vous devez envoyer manuellement une demande de transfert.
  
Si vous ne savez pas quels types de numéros de téléphone vous nous demanderez de transférer, consultez la rubrique [Différents types de numéros de téléphone utilisés pour les forfaits d'appels](different-kinds-of-phone-numbers-used-for-calling-plans.md).
  
> [!IMPORTANT]
> Lorsque vous transférez manuellement les numéros de téléphone à l'aide d'un courrier d'autorisation (LOA), vérifiez que vous **sélectionnez le type de numéro de téléphone approprié**. Vous devez **soumettre des demandes de transfert distinctes** pour chaque type de numéro de téléphone que vous voulez transférer.

## <a name="downloads-for-phone-number-management-forms"></a>Téléchargements de formulaires de gestion des numéros de téléphone

- Vous pouvez afficher la de tous les courriers d’autorisation et les télécharger en accédant à la rubrique [Courriers d’autorisation Skype Entreprise Online](https://go.microsoft.com/fwlink/?LinkID=623745).
    
- Vous pouvez afficher la liste de tous les formulaires pour obtenir de nouveaux numéros de téléphone et les télécharger en accédant à la rubrique [Demandes de nouveaux numéros de téléphone Skype Entreprise](https://go.microsoft.com/fwlink/?linkid=851581).

> [!NOTE]
> Si vous devez obtenir davantage de numéros de téléphone, veuillez [contacter le support pour les entreprises - Aide de l'administrateur](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
