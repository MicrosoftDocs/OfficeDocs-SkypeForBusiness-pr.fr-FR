## <a name="change-the-emergency-address-for-a-user"></a><span data-ttu-id="e9aa8-110">Modification de l'adresse d'urgence d'un utilisateur</span><span class="sxs-lookup"><span data-stu-id="e9aa8-110">Change the emergency address for a user</span></span>

1. <span data-ttu-id="e9aa8-111">Connectez-vous à Office 365 à l'aide de votre compte professionnel ou scolaire.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-111">Sign in to Office 365 with your work or school account.</span></span>

3. <span data-ttu-id="e9aa8-113">Dans la navigation de gauche, accédez à la **voix** > **utilisateurs Enterprise Voice**.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-113">In the left navigation, go to **Voice** > **Voice users**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="e9aa8-114">Vous permet de voir l’option de **voix** dans le volet de navigation gauche dans le Skype entreprise centre d’administration, vous devez d’abord acheter une licence de module complémentaire de **Conférence Audio** , une licence de module complémentaire **Système téléphonique** ou au moins une **licence Enterprise E5**.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-114">For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>
    
4. <span data-ttu-id="e9aa8-115">Dans la page **Utilisateurs Voix**, cherchez et sélectionnez le ou les utilisateurs pour lesquels vous souhaitez modifier l'adresse de secours.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-115">On the **Voice users** page, locate and select the user you want to change the emergency address for.</span></span>
    
5. <span data-ttu-id="e9aa8-116">Dans le volet Action, sous **Emplacement d'urgence**, cliquez sur **Modifier**.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-116">In the Action pane, under **Emergency location**, click **Change**.</span></span>
    
6. <span data-ttu-id="e9aa8-117">Dans la page **Attribuer un numéro**, cliquez sur **Changer l'emplacement**.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-117">On the **Assign number** page, click **Change location**.</span></span> 
    
7. <span data-ttu-id="e9aa8-118">Sous **Modifier l’adresse d’urgence à**, entrez le nom de la ville dans la zone, cliquez sur **Rechercher**.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-118">Under **Change emergency address to**, enter the name of the city in the box and click **Search**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="e9aa8-119">Vous pouvez modifier une adresse de secours à condition qu'elle ait été validée.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-119">You can only change an emergency address that has already been validated.</span></span> <span data-ttu-id="e9aa8-120">Pour modifier une adresse d’urgence qui n’a pas été validée, supprimer et en créer une autre adresse en cas d’urgence.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-120">To change an emergency address that hasn't been validated, delete it and create another emergency address.</span></span> 
  
8. <span data-ttu-id="e9aa8-121">Sélectionnez une nouvelle adresse d’urgence dans la liste, puis cliquez sur **Enregistrer**.</span><span class="sxs-lookup"><span data-stu-id="e9aa8-121">Select a new emergency address from the list, and then click **Save**.</span></span>
