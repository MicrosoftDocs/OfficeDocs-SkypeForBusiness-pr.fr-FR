    > [!IMPORTANT]
    > <span data-ttu-id="f8f97-114">Pour pouvoir voir l’option **voix** dans la navigation de gauche dans le Skype pour le centre d’administration métier, vous devez d’abord acheter au moins une **licence d’entreprise E5**, une licence de module complémentaire de **Système téléphonique** ou une licence de module complémentaire **d’Audioconférence** .</span><span class="sxs-lookup"><span data-stu-id="f8f97-114">For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>
    
4. <span data-ttu-id="f8f97-115">Sur la page **utilisateurs de voix** , recherchez et sélectionnez l’utilisateur que vous souhaitez modifier l’emplacement d’urgence.</span><span class="sxs-lookup"><span data-stu-id="f8f97-115">On the **Voice users** page, locate and select the user you want to change the emergency location for.</span></span>

7. <span data-ttu-id="f8f97-118">Sous **Modifier l’adresse d’urgence à**, entrez le nom de la ville dans la zone et cliquez sur **Rechercher**.</span><span class="sxs-lookup"><span data-stu-id="f8f97-118">Under **Change emergency address to**, enter the name of the city in the box and click **Search**.</span></span>
