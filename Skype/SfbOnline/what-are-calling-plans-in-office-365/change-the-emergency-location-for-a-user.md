    > [!IMPORTANT]
    > <span data-ttu-id="b4285-114">Vous permet de voir l’option de **voix** dans le volet de navigation gauche dans le Skype entreprise centre d’administration, vous devez d’abord acheter une licence de module complémentaire de **Conférence Audio** , une licence de module complémentaire **Système téléphonique** ou au moins une **licence Enterprise E5**.</span><span class="sxs-lookup"><span data-stu-id="b4285-114">For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>

6. <span data-ttu-id="b4285-117">Dans la page **Attribuer un numéro**, cliquez sur **Changer l'emplacement**.</span><span class="sxs-lookup"><span data-stu-id="b4285-117">On the **Assign number** page, click **Change location**.</span></span> 
    
7. <span data-ttu-id="b4285-118">Sous **Modifier l’adresse d’urgence à**, entrez le nom de la ville dans la zone, cliquez sur **Rechercher**.</span><span class="sxs-lookup"><span data-stu-id="b4285-118">Under **Change emergency address to**, enter the name of the city in the box and click **Search**.</span></span>

8. <span data-ttu-id="b4285-119">Sélectionnez **Rechercher par emplacement** dans la liste déroulante, entrez un nom partiel de l’emplacement (par exemple, entrez **floor**), puis cliquez sur **Rechercher**.</span><span class="sxs-lookup"><span data-stu-id="b4285-119">Select **Search by location** from the drop-down list, enter a partial name for the location (for example, enter **floor**), and then click **Search**.</span></span>

    <span data-ttu-id="b4285-121">Si vous souhaitez ajouter un nouvel emplacement en cas d’urgence qui apparaîtra dans la liste, voir [Ajouter, modifier ou supprimer un emplacement en cas d’urgence pour votre organisation](add-change-or-remove-an-emergency-location-for-your-organization.md).</span><span class="sxs-lookup"><span data-stu-id="b4285-121">If you want to add a new emergency location that will appear on the list, see [Add, change, or remove an emergency location for your organization](add-change-or-remove-an-emergency-location-for-your-organization.md).</span></span>
