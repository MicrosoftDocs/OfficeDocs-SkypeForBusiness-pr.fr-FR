2. <span data-ttu-id="d17c4-112">|||UNTRANSLATED_CONTENT_START|||Go to the **Office 365 admin center** > **Skype for Business**.|||UNTRANSLATED_CONTENT_END|||</span><span class="sxs-lookup"><span data-stu-id="d17c4-112">Go to the **Office 365 admin center** > **Skype for Business**.</span></span>
    
3. <span data-ttu-id="d17c4-113">Dans le volet de navigation de gauche, sélectionnez **Voix** > **Utilisateurs Voix**.</span><span class="sxs-lookup"><span data-stu-id="d17c4-113">In the Skype for Business admin center, in the left navigation, go to **Voice** > **Voice users**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="d17c4-114">Pour que l’option **Voix** apparaisse dans le volet de navigation gauche du Centre d’administration Skype Entreprise, vous devez d’abord acheter au moins une **licence Entreprise E5**, une licence de complément **Système téléphonique**, ou une licence de complément **Audioconférence**.</span><span class="sxs-lookup"><span data-stu-id="d17c4-114">IMPORTANT: For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>

5. <span data-ttu-id="d17c4-116">Dans le volet Action, sous **Emplacement d'urgence**, cliquez sur **Modifier**.</span><span class="sxs-lookup"><span data-stu-id="d17c4-116">In the Action pane, under **Emergency location**, click **Change**.</span></span>
    
6. <span data-ttu-id="d17c4-117">Dans la page **Attribuer un numéro**, cliquez sur **Changer l'emplacement**.</span><span class="sxs-lookup"><span data-stu-id="d17c4-117">On the **Assign number** page, click **Change location**.</span></span> 
    
7. <span data-ttu-id="d17c4-118">Sous **Remplacer l’adresse de secours**, saisissez le nom de la ville dans le champ et cliquez sur **Rechercher**.</span><span class="sxs-lookup"><span data-stu-id="d17c4-118">Under **Change emergency address to** put the name of the city in the Find City box and click **Search**.</span></span>
    
8. <span data-ttu-id="d17c4-119">Sélectionnez l’emplacement d’urgence dans la liste, puis cliquez sur **Enregistrer**.</span><span class="sxs-lookup"><span data-stu-id="d17c4-119">Select the right emergency location from the Select associated location (optional) list, then click **Save**.</span></span>
    
    <span data-ttu-id="d17c4-120">Si vous souhaitez ajouter un nouvel emplacement d’urgence, voir [Ajouter, modifier ou supprimer un emplacement d’urgence pour votre organisation](add-change-or-remove-an-emergency-location-for-your-organization.md).</span><span class="sxs-lookup"><span data-stu-id="d17c4-120">If you want to add a new emergency location, see [Add, change, or remove an emergency location for your organization](add-change-or-remove-an-emergency-location-for-your-organization.md).</span></span>
