# <a name="getting-service-phone-numbers"></a>Numéros de téléphone des services de mise en route

Outre les numéros de téléphone pour vos utilisateurs à partir d’Office 365, vous pouvez rechercher et acquérir le numéro ou les numéros de téléphone pour les services de conférence audio (pour les ponts de conférence), les standards automatiques et les files d’attente de l’appel (également appelés numéros de service). Les numéros de téléphone de service disposent d'une capacité d'appels simultanés plus élevée que les numéros de téléphone d'utilisateur ou d'abonné. Par exemple, un numéro de service peut gérer 100 s d’appels simultanément, alors que le numéro de téléphone d’un utilisateur ne peut gérer que quelques appels simultanément.
  
> [!NOTE]
> Crédits de Communications Office 365 doit être définies en premier pour acquérir des numéros verts. Reportez-vous à la section [configurer les crédits de Communications pour votre organisation](../skype-for-business-and-microsoft-teams-add-on-licensing/set-up-communications-credits-for-your-organization.md). 
  
Vous avez deux façons d’obtenir numéros de service, vous pouvez les utiliser avec Skype pour les entreprises et les Teams de Microsoft :
  
- Obtenir les nouveaux numéros à partir d’Office 365.
    
- Port ou transférer vos numéros existants à partir de votre fournisseur de services ou votre opérateur de téléphonie à Office 365.
    
    > [!NOTE]
    > Lorsque vous transférez vos numéros de service, il est fortement recommandé de contacter [le support Microsoft](https://support.office.com/en-us/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b) pour vous assurer que la capacité d’appel simultanée supérieure est pris en compte et correctement configurée.
  
## <a name="get-new-service-numbers"></a>Obtenir de nouveaux numéros de service

1. Connectez-vous à Office 365 à l'aide de votre compte professionnel ou scolaire.
    
2. Sélectionnez **Centre d'administration Office 365** > **Skype Entreprise**.
    
3. Dans la navigation de gauche, accédez à **voix** > **les numéros de téléphone** > **d’Ajouter un nouveau numéro**, puis cliquez sur **nouveaux numéros de service**.
    
    > [!IMPORTANT] 
    > Pour pouvoir voir l’option **voix** dans la navigation de gauche dans le Skype pour le centre d’administration métier, vous devez d’abord acheter au moins une **licence d’entreprise E5**, une licence de module complémentaire de **Système téléphonique** ou une licence de module complémentaire **d’Audioconférence** .
    
4. Dans la page **Ajouter de nouveaux numéros de service** , cliquez sur les éléments suivants :
    
  - **Pays/Région**
    
  - **État/Région**
    
  - **Ville**
    
5. Sous **quantité**, entrez le nombre de numéros de téléphone que vous souhaitez pour votre organisation et cliquez sur **Ajouter** pour créer une réservation. Vous disposez de 10 minutes pour sélectionner vos numéros de téléphone ; Si vous prenez plus de 10 minutes, les numéros de téléphone apparaîtront dans le pool de numéros de téléphone.
    
    > [!NOTE]
    > Vous pouvez voir le nombre de numéros de téléphone, qui sont basés sur le nombre de licences, listée en regard de **Service Total numéros vous pouvez acquérir**. Pour plus d’informations, consultez [le nombre de numéros de téléphone vous obtiendrez ?](../what-are-calling-plans-in-office-365/how-many-phone-numbers-can-you-get.md)
  
6. Cliquez sur **Afficher les numéros** pour consulter la liste complète des numéros de téléphone. Ceci est utile si vous ne souhaitez pas sélectionner un numéro de téléphone spécifique dans la liste.
    
7. Sélectionnez les numéros de téléphone appropriés, puis cliquez sur **Acquérir les numéros**.
    
### <a name="assign-service-numbers"></a>Affecter des numéros de service

Une fois que vous avez vos numéros de service, il peuvent ensuite être assignées à un pont de téléconférence audio. Pour ce faire, reportez-vous à la section [Modifier le numéro ou les numéros gratuits sur le pont de conférence de données Audio](../audio-conferencing-in-office-365/change-the-phone-numbers-on-your-audio-conferencing-bridge.md).
  
### <a name="port-or-transfer-existing-service-numbers"></a>Transférer des numéros de service existants

Si vous voulez transférer des numéros de service depuis votre fournisseur ou opérateur de services actuel, vous devez envoyer une demande de transfert manuellement à Microsoft. Vous devez soumettre des commandes port distinct pour chaque type de numéro de service (appel payant et gratuit) que vous transférez à l’aide d’une lettre d’autorisation (LOA). Dans la lettre d’autorisation (LOA), vous devez sélectionner le type de numéro de service approprié. Lorsque vous contactez le support technique de Microsoft, assurez-vous que vous spécifiez que vous transférez un numéro de service (*et non un nombre d’utilisateur ou de l’abonné*), ou la capacité d’appel simultanée peuvent ne pas suffire à gérer des volumes d’appels. Si vous souhaitez transférer les numéros de téléphone ou effectuer d’autres opérations avec vos numéros de téléphone, voir [Gérer les numéros de téléphone pour votre organisation](../what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md).

> [!NOTE]
> Si vous devez obtenir plus de numéros de téléphone, vous pouvez contacter le [support pour les entreprises - Aide de l'administrateur](https://support.office.com/en-us/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
