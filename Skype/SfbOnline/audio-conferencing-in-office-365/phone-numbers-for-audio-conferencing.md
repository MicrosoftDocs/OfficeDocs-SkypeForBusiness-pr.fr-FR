# <a name="phone-numbers-for-audio-conferencing-in-skype-for-business-online"></a>Numéros de téléphone pour l'Audioconférence dans Skype Entreprise Online

> [!NOTE]
> Pour plus d’informations sur les numéros de téléphone dans Microsoft Teams, voir [Numéros de téléphone pour l'Audioconférence dans Microsoft Teams](/MicrosoftTeams/phone-numbers-for-audio-conferencing-in-teams).

Lorsque vous configurez l'**Audioconférence** dans Skype Entreprise, des numéros de téléphone d'appel sont automatiquement attribués à votre organisation. Vous pouvez voir les numéros de téléphone qui sont affectés à votre pont d'audioconférence en allant dans le **Centre d'administration Skype Entreprise** > **Audioconférence** > **Pont Microsoft**. Voir [Afficher la liste des numéros d'Audioconférence](see-a-list-of-audio-conferencing-numbers.md).
  
> [!NOTE]
> Il n'existe pas de ressource qui contienne un listing de tous les numéros d'appel de l'Audioconférence. Si vous voulez voir si des numéros de téléphone d'appel sont disponibles dans votre zone ou votre pays/région, utilisez le **Centre d'administration Skype Entreprise** > **Voix** > **Numéros de téléphone**, cliquez sur **Ajouter** puis sur **Nouveaux numéros de service**. Utilisez les listes de **Pays/Région**, **État/Région** et **Ville** pour filtrer votre recherche. Également, si vous recherchez des numéros de service gratuits, sélectionnez **Gratuits** dans la liste **État/Région**.
  
## <a name="audio-conferencing-coverage-and-pricing"></a>Couverture et tarification de l'audioconférence

Pour une liste complète de tous les pays/régions et villes où l'Audioconférence est disponible, voir [Disponibilité par pays et par région de l'Audioconférence et des Plans d’appels](/microsoftteams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans). Pour les informations de tarification, consultez [Tarification de l’Audioconférence](https://products.office.com/en-us/skype-for-business/audio-conferencing#Requirements).
  
## <a name="dial-in-phone-numbers-in-a-meeting-invite"></a>Numéros de téléphone d'appel dans une invitation à une réunion

Lorsqu'un utilisateur Skype Entreprise Online planifie une réunion dans Outlook ou dans l'application web Outlook, le numéro d'audioconférence par défaut qui est défini pour l'utilisateur est inclus dans l'invitation à la réunion. Si vous souhaitez sélectionner un autre numéro par défaut pour un ou plusieurs utilisateurs, vous pouvez le modifier en allant dans le **Centre d'administration Skype Entreprise** > **Audioconférence** > **Utilisateurs**. Voir [Définir les numéros de téléphone inclus dans les invitations](set-the-phone-numbers-included-on-invites.md).
  
D'autres numéros d'appel peuvent être affichés en cliquant sur le lien **Trouver un numéro local** dans l'invitation à une réunion.
  
## <a name="dial-in-phone-numbers-set-on-an-audio-conferencing-bridge"></a>Numéros de téléphone d'appel définis pour un pont d'audioconférence

Il existe deux types de numéros de téléphone d'audioconférence pouvant être affectés à votre pont de conférence : **Partagé** et **Dédié**. Ces deux types de numéros peuvent être utilisés par n'importe quel appelant pour se joindre aux réunions audio qui se tiennent dans votre organisation.
  
 Les **numéros de téléphone dédiés** sont les numéros disponibles uniquement pour les utilisateurs au sein de votre organisation. Vous pouvez modifier les langues utilisées lorsque quelqu'un appelle l'un de ces numéros.
  
 Les **numéros de téléphone partagés** sont des numéros qui peuvent être partagés avec d'autres organisations Office 365. Vous ne pouvez pas modifier les langues utilisées lorsque quelqu'un appelle l'un de ces numéros.
  
Bien que le numéro d'audioconférence par défaut qui est affecté à un organisateur soit seulement inclus dans l'invitation à une réunion, un appelant peut utiliser n'importe lequel des numéros de téléphone affectés à votre pont de conférence pour se joindre à une réunion. La liste des numéros de téléphone qui peuvent être utilisés pour participer à une réunion est disponible en cliquant sur le lien **Trouver un numéro national** figurant sur chaque invitation à la réunion.
  
## <a name="automatically-assigned-audio-conferencing-phone-numbers"></a>Numéros de téléphone d'audioconférence automatiquement affectés

Des numéros de téléphone d'audioconférence partagés sont automatiquement affectés à des organisations lorsqu’elles sont autorisées pour l’audioconférence. Lorsque les numéros de téléphone sont affectés, un numéro de téléphone est affecté comme numéro de téléphone par défaut du pont de conférence. Le numéro de téléphone affecté comme numéro par défaut du pont sera l'un de ceux du pays/de la région de l'organisation.
  
> [!NOTE]
> L'emplacement du pays ou de la région de votre organisation peut être trouvé en se connectant au **Centre d'administration Office 365** et en regardant sous **Profil de l'organisation**. 
  
> [!CAUTION]
> En raison de la disponibilité limitée de numéros payants au Venezuela, en Indonésie et aux Émirats Arabes Unis (EAU), les organisations situées dans ces pays/régions ne se verront pas affecter automatiquement un numéro payant d'audioconférence. Des numéros gratuits dans ces endroits sont disponibles en fonction des stocks. 
  
Les numéros de téléphone d'audioconférence dédiés sont des numéros de service que vous pouvez obtenir, puis affecter à votre organisation. Les numéros de service peuvent être trouvés en utilisant le **Centre d'administration Skype Entreprise**. Pour les détails, voir [Obtention de numéros de téléphone de service](/SkypeForBusiness/what-is-phone-system-in-office-365/getting-service-phone-numbers).
  
Pour voir une liste des pays/régions qui ont des numéros de téléphone automatiquement affectés aux organisations, voir [Disponibilité par pays et par région de l'Audioconférence et des Plans d’appels](/microsoftteams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans).
  
## <a name="what-else-should-you-know"></a>Que devriez-vous savoir d'autre ?

- Pour voir la liste des langues prises en charge par l’audioconférence, voir [Langues prises en charge par l’audioconférence](/MicrosoftTeams/audio-conferencing-supported-languages).
    
- Vous pouvez utiliser le cmdlet [Get-CsOnlineDialInConferencingServiceNumber](https://go.microsoft.com/fwlink/?LinkId=617691) pour voir les numéros de téléphone dédiés pour l'audioconférence dans votre organisation.
    
- Vous pouvez utiliser le cmdlet [Get-CsOnlineDialInConferencingLanguagesSupported](https://go.microsoft.com/fwlink/?LinkId=617684) pour voir les langues qui peuvent être définies sur un numéro de téléphone d'appel dédié.
    
- Vous pouvez définir jusqu'à quatre langues pour chaque numéro de téléphone d'audioconférence : une langue principale et trois langues secondaires. Et vous pouvez également définir les langues d'un numéro de téléphone d'audioconférence dédié.
    
- Pour définir le numéro de téléphone d'appel pour un utilisateur, voir [Définir les numéros de téléphone inclus dans les invitations](set-the-phone-numbers-included-on-invites.md).
