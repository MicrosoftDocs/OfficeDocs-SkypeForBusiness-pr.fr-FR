# Voici ce que vous obtenez avec système téléphonique dans Office 365

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'[avertissement](bc9756d1-8a2f-42c4-98f6-afb17c29231c.md#MT_Footer). Vous pouvez consulter la version en anglais de cet article [ici](https://support.office.com/en-us/article/bc9756d1-8a2f-42c4-98f6-afb17c29231c). 
  
Un PBX est un système téléphonique au sein de l'entreprise. Le système téléphonique dans Office 365 module complémentaire vous propose les fonctionnalités PBX, mais sans l'équipement complexe et coûteuse.
  
Avec le module complémentaire système téléphonique, vous pouvez utiliser Skype Entreprise pour effectuer les tâches de base telles qu'émission et réception d'appels, transfert d'appels et les appels silence ou unmuting, à partir de pratiquement n'importe où ayant accès à Internet. Il est disponible dans le monde !
  
Pour plus d'informations, voir [tarification pour système téléphonique](https://products.office.com/en-us/skype-for-business/cloud-pbx#requirements).

- Vous pouvez utiliser le module complémentaire système téléphonique avec des lignes téléphoniques existantes de votre société ou le service [Quelle est l'appel d'offre dans Office 365 ?](../what-are-calling-plans-in-office-365/what-are-calling-plans-in-office-365.md) pour les appels entrants et sortants.
    
- Vous pouvez obtenir de nouveaux numéros de téléphone d'Office 365 ou transférer des numéros de téléphone existants vers Skype Entreprise.
