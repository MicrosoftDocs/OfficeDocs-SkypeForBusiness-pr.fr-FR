# <a name="heres-what-you-get-with-phone-system-in-office-365"></a>Voici ce que vous obtenez avec système de téléphone dans Office 365

Un PBX est un système téléphonique au sein de l’entreprise. Le système téléphonique dans Office 365 complémentaire vous offre des capacités de PBX, mais sans l’équipement complexe et coûteuse. 
  
Avec le module complémentaire de système téléphonique, vous pouvez utiliser Skype pour les entreprises pour effectuer des tâches de base telles que la fabrication et de recevoir des appels, transfert d’appels et les appels en sourdine ou unmuting, à partir de presque n’importe où avec accès à Internet. Il est disponible dans le monde entier ! 
  
Pour plus d’informations, reportez-vous à la section [tarification pour système téléphonique](https://products.office.com/en-us/skype-for-business/cloud-pbx#requirements).

## <a name="plan-and-set-up-the-phone-system-in-office-365"></a>Planifier et configurer le système téléphonique dans Office 365
<a name="bkmk_setup"> </a>

Configurer système téléphonique le mieux adapté à votre entreprise :
  
- Vous pouvez utiliser le module complémentaire du système téléphonique avec des lignes téléphoniques existantes de votre société, ou le [Quels sont les Plans d’appel dans Office 365 ?](../what-are-calling-plans-in-office-365/what-are-calling-plans-in-office-365.md) service pour les appels entrants et sortants.
    
- Vous pouvez obtenir des nouveaux numéros de téléphone à partir d’Office 365, ou transférer les numéros de téléphone existant à Skype pour les entreprises. 
    
Consultez ces articles pour planifier et configurer le système téléphonique de votre entreprise :

Il comporte également des informations sur la planification d’une solution entièrement nuage seule. 
    
- [Skype pour les professionnels et les équipes Microsoft complémentaire licences](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md): Découvrez les licences que vous devez acheter pour obtenir le système téléphonique dans Office 365.
