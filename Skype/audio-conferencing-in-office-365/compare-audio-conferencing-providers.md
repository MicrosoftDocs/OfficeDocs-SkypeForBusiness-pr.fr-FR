# <a name="compare-audio-conferencing-providers"></a>Comparaison de fournisseurs de conférence audio

Lorsque vous configurez une conférence Audio pour Skype pour les entreprises et les Teams de Microsoft, vous devez choisir un fournisseur de conférence audio. Vous pouvez choisir soit Microsoft ou un tiers en tant que votre fournisseur de conférence audio. 
  
> [!IMPORTANT]
> Les utilisateurs de Microsoft Teams ne pouvez pas utiliser un fournisseur de conférence audio de tiers. 
  
Fournisseurs d’audioconférences effectuer les opérations suivantes : 
  
- Fournir un *pont de conférence audio*. Le pont de conférence définira les numéros de téléphone, les broches et les ID de conférence pour les réunions.
    
- Donner le numéro payant, gratuit et les numéros de téléphone internationaux dont les utilisateurs seront connecteront à. Ce sont les numéros de téléphone inclus dans l’invitation à une réunion.
    
- Définir la langue que vous entendez les utilisateurs lorsqu’ils se connectent à une réunion.
    
- Créer des ID de conférence qui sont utilisés lorsque les utilisateurs sont rejoindre une réunion de la réunion.
    
- Vous donner de l’organisateur de la broche pour lancer des réunions par téléphone.
