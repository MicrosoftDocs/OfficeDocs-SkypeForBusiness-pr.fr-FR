# <a name="turn-on-or-off-mobile-phone-notifications"></a>Activer ou désactiver les notifications de téléphone mobile

**[Attribuer des rôles d’administrateur dans Office 365 pour les professionnels](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** de votre organisation, vous pouvez choisir si votre Skype pour les utilisateurs professionnels recevoir les alertes concernant les messages instantanés entrants et manqués lorsqu’ils sont sur leurs téléphones ou leurs tablettes.
  
Android et téléphones Windows, Skype pour Business Notification affiche en temps réel. Pour Windows Phone, iPhone et iPad périphériques, toutefois, notification de transmission est utilisée pour afficher les alertes à chaque fois que vous utilisez pas activement Skype pour entreprise sur votre téléphone ou votre Tablet PC.
  
## <a name="turn-push-notifications-off-for-all-the-windows-phone-or-apple-devices-in-your-organization"></a>Désactiver les notifications de type Pousser pour tous les périphériques Windows Phone ou Apple dans votre organisation
<a name="__top"> </a>

1. Se connecter à Office 365.
    
2. Sélectionnez **Centre d'administration Office 365** > **Skype Entreprise**.
    
3. Accédez à **l’organisation** > **Général**. 
    
4. Sous **notifications de téléphone Mobile**, désactivez la case en regard du service de notification que vous souhaitez désactiver, puis cliquez sur **Enregistrer**.
    
À retenir : 
  
- Si vous désactivez les notifications de transmission, les utilisateurs reçoivent toujours toutes les alertes lorsqu’ils démarrent Skype pour les entreprises à nouveau sur son appareil mobile.
    
- Les notifications de transmission sont activées par défaut. Les utilisateurs individuels peuvent les désactiver en choisissant le Skype appropriée pour l’option entreprise sur son appareil mobile.
    
- Lorsque vous désactivez les notifications de transmission, les utilisateurs ne peuvent pas les réactiver.
    
> [!IMPORTANT]
>  Microsoft utilise des autres sociétés pour fournir des Skype en temps réel pour les notifications de professionnels mobiles pour Windows Phone, iPhone et iPad utilisateurs. Consultez la [déclaration de confidentialité de Skype de Microsoft pour les produits de l’entreprise](https://go.microsoft.com/fwlink/p/?linkid=247732).
