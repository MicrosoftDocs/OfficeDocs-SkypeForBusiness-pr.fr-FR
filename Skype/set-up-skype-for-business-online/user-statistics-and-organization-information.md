# Statistiques utilisateur et informations sur l'organisation

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'[avertissement](3e55c5a2-baad-4573-9ad9-37d0c42dcb53.md#MT_Footer). Vous pouvez consulter la version en anglais de cet article [ici](https://support.office.com/en-us/article/3e55c5a2-baad-4573-9ad9-37d0c42dcb53). 
  
Le tableau de bord Skype Entreprise vous donne un aperçu de la façon dont le service est configuré pour votre entreprise. Si vous n'avez pas encore configuré Skype Entreprise, consultez la rubrique [Configurer Skype Entreprise Online](set-up-skype-for-business-online.md) pour obtenir des instructions détaillées.
  
## Comment modifier les statistiques utilisateur de Skype Entreprise Online

|||
|:-----|:-----|
|**Propriété** <br/> |**Comment modifier** <br/> |
|Utilisateurs Skype Entreprise Online  <br/> |Pour ajouter ou bloquer des utilisateurs à l'aide de l'application Skype Entreprise, consultez [Attribuer ou supprimer des licences pour Office 365 pour les entreprises](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) ou[Remove licenses from users in Office 365 for business](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Pour obtenir une liste complète de la procédure à suivre lorsqu'un utilisateur quitte votre entreprise, voir [Supprimer un ancien employé d'Office 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Utilisateurs invités à une conférence rendez-vous  <br/> |Pour permettre aux utilisateurs de conférences rendez-vous, voir [Configurer la conférence Audio pour Skype entreprise et Teams Microsoft](../audio-conferencing-in-office-365/set-up-audio-conferencing-for-skype-for-business-and-microsoft-teams.md).  <br/> |
|Utilisateurs du cloud  <br/> |Nombre d'utilisateurs de Skype Entreprise Online ayant des comptes Office 365.  <br/> Pour ajouter des personnes à votre abonnement Office 365 (autrement dit, créer des comptes Office 365 pour eux), consultez la rubrique [Ajouter des utilisateurs individuellement ou en bloc à Office 365 - Aide de l'administrateur](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Utilisateurs synchronisés et hébergés en ligne  <br/> |Nombre d'utilisateurs de Skype Entreprise Online synchronisés à partir d'une instance locale d'Active Directory.  <br/> Pour configurer votre instance locale Active Directory, consultez la rubrique [Configurer la synchronisation d'annuaires pour Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
   
> [!NOTE]
> Si Indisponible s'affiche dans la liste, il n'y a pas d'information ou des erreurs se sont produites. 
  
## Comment modifier les informations sur votre organisation

|||
|:-----|:-----|
|**Propriété** <br/> |**Comment modifier** <br/> |
|Nom de l'organisation  <br/> |Voici le nom que vous avez saisi au moment de votre inscription à Office 365. Pour le modifier, consultez la rubrique [Modifier les adresses de votre organisation, contact technique, etc.](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|OrganizationID  <br/> |Vous ne pouvez pas modifier l'ID de votre organisation.  <br/> |
|Organisation créée  <br/> |Vous ne pouvez pas modifier cette valeur.  <br/> |
|Synchronisation Active Directory sur  <br/> |Il s'agit si votre organisation est synchronisée avec une organisation Skype Entreprise Online qui se trouve en local.  <br/> Pour configurer votre instance locale Active Directory, consultez la rubrique [Configurer la synchronisation d'annuaires pour Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Domaines de l'organisation  <br/> |Lorsque vous avez configuré initialement Office 365, ces domaines ont été enregistrés.  <br/> Pour ajouter un domaine, voir [Ajout de domaines supplémentaires à Office 365](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207).  <br/> Pour supprimer un domaine, consultez la rubrique [Supprimer un domaine d'Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c).  <br/> |
