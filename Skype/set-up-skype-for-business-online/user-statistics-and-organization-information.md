# <a name="user-statistics-and-organization-information"></a>Informations statistiques et de l’organisation utilisateur

Le Skype à Business dashboard vous donne un aperçu rapide de la façon dont le service est configuré pour votre entreprise. Si vous n’avez pas encore configuré Skype pour entreprise, consultez [définir Skype pour l’activité en ligne](set-up-skype-for-business-online.md) pour obtenir des instructions pas à pas.
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>La modification de Skype pour les statistiques d’utilisateurs professionnels en ligne

|||
|:-----|:-----|
|**Propriété** <br/> |**Comment le modifier** <br/> |
|Skype pour les utilisateurs professionnels en ligne  <br/> |Pour ajouter ou bloquer des personnes d’utiliser le Skype pour l’application d’entreprise, consultez [attribuer ou supprimer des licences pour Office 365 pour professionnels](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) ou [Supprimer des licences aux utilisateurs dans Office 365 pour entreprises](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Pour obtenir une liste complète des actions à effectuer lorsqu’un utilisateur quitte votre entreprise, voir [Supprimer un ancien employé à partir d’Office 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Utilisateurs activés pour la conférence à distance  <br/> |Pour permettre aux utilisateurs pour les conférences à distance, consultez [définir audioconférence pour Skype pour les entreprises et les équipes de Microsoft](../audio-conferencing-in-office-365/set-up-audio-conferencing.md).  <br/> |
|Utilisateurs dans le nuage  <br/> |Ceci est le nombre de Skype pour les utilisateurs professionnels en ligne qui disposent de comptes d’Office 365.  <br/> Pour ajouter des personnes à votre abonnement à Office 365 (en d’autres termes, créer des comptes d’Office 365 pour eux), reportez-vous à la section [Ajouter des utilisateurs individuellement ou en bloc vers Office 365 - aide de l’administrateur](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Utilisateurs synchronisés et d’hébergement en ligne  <br/> |Ceci est le nombre de Skype pour des utilisateurs professionnels en ligne qui sont synchronisés à partir d’Active Directory sur site.  <br/> Pour configurer Active Directory sur site, reportez-vous à la section [configurer la synchronisation d’annuaire pour Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
   
> [!NOTE]
> Si apparaît non disponible, cela signifie qu’il n’y a aucune information ou erreurs se sont produites. 
  
## <a name="how-to-change-information-about-your-organization"></a>Comment faire pour modifier les informations sur votre organisation

|||
|:-----|:-----|
|**Propriété** <br/> |**Comment le modifier** <br/> |
|Nom de l’organisation  <br/> |Il s’agit du nom que vous avez entré lorsque vous vous abonnez à Office 365. Pour la modifier, consultez [modifier de votre organisation adresse contact technique et bien plus encore](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|ID de l’organisation  <br/> |Vous ne pouvez pas modifier votre nom d’organisation.  <br/> |
|Organisation créée  <br/> |Vous ne pouvez pas modifier cette valeur.  <br/> |
|Synchronisation de Active Directory  <br/> |C’est si votre organisation est en cours de synchronisation à un Skype pour une organisation commerciale en ligne qui est sur site.  <br/> Pour configurer Active Directory sur site, reportez-vous à la section [configurer la synchronisation d’annuaire pour Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Domaines de l’organisation  <br/> |Lorsque vous définissez initialement Office 365, voici les domaines qui ont été enregistrés.  <br/> Pour ajouter un domaine, consultez [Ajouter plusieurs domaines à Office 365](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207).  <br/> Pour supprimer un domaine, consultez [suppression d’un domaine à partir d’Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c).  <br/> |
