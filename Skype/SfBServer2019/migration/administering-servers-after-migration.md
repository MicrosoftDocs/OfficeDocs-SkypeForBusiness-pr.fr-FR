# <a name="administering-servers-after-migration"></a>Administration des serveurs après la migration

En règle générale, vous devez utiliser l’outil d’administration qui correspond à la version du serveur que vous souhaitez gérer. Vous ne pouvez pas installer les outils d’administration à partir de versions différentes sur le même ordinateur.
  
> [!IMPORTANT]
> Après avoir déployé un Skype pour le pool pilote Business Server, vous ne pouvez pas utiliser le Générateur de topologie ou le panneau de configuration pour gérer les ressources 2019. Vous devez utiliser les outils 2019 pour gérer toutes les ressources de la version actuelle et précédentes.
