# <a name="administering-servers-after-migration"></a>Administration des serveurs après la migration

En règle générale, vous devez utiliser l’outil d’administration qui correspond à la version du serveur que vous voulez gérer. Vous ne pouvez pas installer d’outils d’administration de différentes versions sur le même ordinateur.
  
> [!IMPORTANT]
> Après le déploiement d’un pool de pilotes Skype entreprise Server, vous ne pouvez pas utiliser le générateur de topologie ou le panneau de configuration pour gérer des ressources 2019. Vous devez utiliser les outils 2019 pour gérer toutes les ressources de version actuelles et antérieures.
