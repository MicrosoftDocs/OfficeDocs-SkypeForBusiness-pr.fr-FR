# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Étape 3: déploiement du pool de pilotes Skype entreprise Server 2019

Cette section présente les étapes nécessaires au déploiement d’un pool de pilotes de Skype entreprise Server 2019. Le déploiement de Skype entreprise Server 2019 nécessite l’utilisation du générateur de topologie pour définir votre topologie et les composants que vous voulez déployer, en préparant votre environnement pour le déploiement des composants 2019 de Skype entreprise Server et la publication de votre topologie Concevez sur le premier serveur frontal, puis installez et configurez le logiciel 2019 de Skype entreprise Server pour les composants de votre déploiement. Lorsque vous avez terminé, votre déploiement de la liste de pilotes de déploiement 2019 de Skype entreprise Server peut cohabiter avec un pool hérité existant.
