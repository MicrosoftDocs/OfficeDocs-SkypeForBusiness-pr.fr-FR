# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Phase 3 : déploiement du pool pilote Skype entreprise Server 2019

Cette section décrit les étapes nécessaires au déploiement d’un pool pilote de Skype entreprise Server 2019. Le déploiement de Skype entreprise Server 2019 nécessite l’utilisation du générateur de topologie pour définir votre topologie et les composants que vous souhaitez déployer, la préparation de votre environnement au déploiement des composants de Skype entreprise Server 2019, la publication de votre conception de topologie sur le premier serveur frontal, puis l’installation et la configuration du logiciel Skype entreprise Server 2019 pour les composants de votre déploiement. Lorsque vous avez terminé, votre déploiement de pool pilote Skype entreprise Server 2019 coexiste avec un pool hérité existant.
