# <a name="view-current-users-in-legacy-pool"></a>Afficher les utilisateurs actuels dans le pool hérité

Avant d’apprendre les différentes façons de déplacer des utilisateurs entre des pools, nous devons d’abord déterminer les utilisateurs qui existent dans le pool hérité. La colonne pool de serveurs d' **inscriptions** identifie les utilisateurs qui sont configurés pour le pool hérité. Voici les utilisateurs de test que nous allons déplacer vers le pool Skype entreprise Server 2019.
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>Pour afficher la liste des utilisateurs dans le pool hérité

1. Ouvrez une session sur le serveur frontal hérité à l’aide d’un compte membre du groupe RTCUniversalServerAdmins ou du rôle d’administration CsAdministrator ou CsUserAdministrator.
    
2. Ouvrez **le panneau de configuration de Skype entreprise Server**.
    
3. Cliquez sur **utilisateurs**, sur **recherche**, puis sur **Rechercher**.
