# <a name="run-best-practices-analyzer"></a>Exécutez l’outil Best Practices Analyzer

L’outil Best Practices Analyzer rassemble les informations de configuration d’un déploiement et détermine si la configuration est définie en fonction des meilleures pratiques de Microsoft. Vous pouvez installer l’outil sur un ordinateur client qui exécute Microsoft .NET Framework 3.5 Service Pack 1 ou directement sur le serveur qui exécute l’installation héritée. Nous vous recommandons d’installer et exécuter cet outil sur un ordinateur client. Les outils d’administration doit également être installés localement sur l’ordinateur client afin que Best Practices Analyzer peut collecter un ensemble complet de données. 
  
Vous pouvez télécharger Best Practices Analyzer à partir de la Center Download Microsoft à [https://go.microsoft.com/fwlink/p/?linkid=246173](https://go.microsoft.com/fwlink/p/?linkid=246173).
