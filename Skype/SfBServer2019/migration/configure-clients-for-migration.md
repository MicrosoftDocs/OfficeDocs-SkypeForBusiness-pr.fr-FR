# <a name="configure-clients-for-migration"></a>Configurer des clients pour la migration

Cette rubrique contient les étapes du déploiement client recommandé que vous devez prendre avant de migrer vers Skype pour Business Server 2019. Ces modifications de configuration doivent porter sur Lync Server 2013 ou Skype pour Business Server 2015 (la version **à partir**de laquelle vous effectuez la migration).
  
### <a name="to-configure-clients-before-migration"></a>Pour configurer les clients avant la migration

1. Déployer les plus récentes serveur, clients et périphériques mises à jour (correctifs logiciels) pour votre installation existante.
    
2. Dans la version précédente de Skype pour Business Server, utilisez le filtrage de Version Client pour autoriser uniquement les clients avec les dernières mises à jour installées.
