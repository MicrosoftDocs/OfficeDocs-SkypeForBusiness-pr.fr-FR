# <a name="configure-clients-for-migration"></a>Configuration des clients pour la migration

Cette rubrique contient les étapes recommandées en matière de déploiement de clients que vous devez prendre avant de procéder à la migration vers Skype entreprise Server 2019. Ces modifications de configuration doivent être apportées sur Lync Server 2013 ou Skype entreprise Server 2015 (version **à partir de**laquelle vous effectuez la migration).
  
### <a name="to-configure-clients-before-migration"></a>Pour configurer des clients avant la migration

1. Déploiement de la version la plus récente des mises à jour de serveur, de client et d’appareil (HotFix) pour votre installation existante.
    
2. Dans la version précédente de Skype entreprise Server, utilisez les filtres de version de client pour autoriser uniquement les clients sur lesquels les mises à jour les plus récentes sont installées.
