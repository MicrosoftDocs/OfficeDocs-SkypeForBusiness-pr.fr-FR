# <a name="prepare-active-directory-for-skype-for-business-server"></a>Préparer Active Directory pour Skype Entreprise Server

Avant de déployer Skype Entreprise Server 2019 dans un état de coexistence, vous devez effectuer des tâches Active Directory supplémentaires pour configurer le schéma, la forêt et le domaine pour Skype Entreprise Server 2019. Les extensions de schéma ajoutent les classes et attributs Active Directory requis par Skype Entreprise Server 2019. <!-- For additional information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>Pour préparer Active Directory pour Skype Entreprise Server 2019

1. Sur le Skype Entreprise Server frontal 2019, exécutez Skype Entreprise Server configuration 2019.
    
2. Sélectionnez **Préparer Active Directory.**
  
3. Complétez les étapes 1 à 5 de l’Assistant.
