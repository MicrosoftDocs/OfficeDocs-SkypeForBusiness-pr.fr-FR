# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Phase 4 : déplacer les utilisateurs test vers le pool pilote

Vous pouvez déplacer un seul utilisateur ou groupe d’utilisateurs vers votre nouveau déploiement de Microsoft Skype entreprise Server 2019 à l’aide des deux méthodes suivantes : le panneau de configuration Skype entreprise Server et Skype entreprise Server Management Shell. Les rubriques de cette section décrivent les tâches que vous devez effectuer lors du déploiement pilote, ainsi que avant de faire passer votre déploiement de Skype entreprise Server 2019 d’un déploiement pilote à un déploiement au niveau de la production.
