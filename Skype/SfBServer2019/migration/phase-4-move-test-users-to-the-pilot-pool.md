# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Phase 4 : Déplacer les utilisateurs de test vers le pool pilote

Vous pouvez déplacer un ou plusieurs groupes d’utilisateurs vers votre nouveau déploiement Microsoft Skype Entreprise Server 2019 à l’aide des deux méthodes suivantes : Skype Entreprise Server Control Panel et Skype Entreprise Server Management Shell. Les rubriques de cette section décrivent les tâches que vous devez effectuer lors du déploiement pilote, ainsi que avant de déplacer votre déploiement de Skype Entreprise Server 2019 d’un déploiement pilote vers un déploiement de niveau production.
