# <a name="microsoft-telephony-solutions"></a>Solutions de téléphonie Microsoft

Microsoft prend en charge plusieurs options lorsque vous commencez votre parcours aux équipes dans le nuage de Microsoft. Cet article vous aide à déterminer lequel Microsoft solution de téléphonie (système téléphonique dans le nuage ou Enterprise Voice sur site) est le bon choix pour les utilisateurs dans votre organisation, et comment votre organisation peut se connecter à la Public téléphone réseau commuté (RTC). 

Vous devez utiliser cet article ainsi que le diagramme technique associé, qui fournit une aide visuelle pour rendre le bon choix pour votre organisation :

- [Solutions de téléphonie Microsoft - PDF](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/telephony-solutions/microsoft-telephony-solutions-12-18.pdf)

- [Solutions de téléphonie Microsoft - Visio](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/telephony-solutions/microsoft-telephony-solutions-12-18.vsdx)




## <a name="private-branch-exchange-pbx-options"></a>Options d’autocommutateur (PBX) privée

### <a name="phone-system-office-365"></a>Système téléphonique (Office 365)
  
Système téléphonique est la technologie de Microsoft pour activer le contrôle d’appel et les fonctionnalités d’autocommutateur privé (PBX) dans le nuage Office 365 avec Microsoft Teams et/ou Skype pour Business en ligne. 

Système téléphonique fonctionne avec des équipes ou Skype pour les clients professionnels en ligne et les périphériques certifiés. Système téléphonique vous permet de remplacer votre système PBX existant par un ensemble de fonctionnalités remis directement à partir d’Office 365 et étroitement intégré à l’expérience de productivité de l’entreprise dans le nuage. Pour connecter le système téléphonique pour le Public téléphone réseau commuté (RTC), vous pouvez choisir votre propre opérateur de téléphonie ou de Plan de l’appel de Microsoft.

Pour plus d’informations, voir [What ' s système téléphonique dans Office 365](https://docs.microsoft.com/en-us/MicrosoftTeams/what-is-phone-system-in-office-365).

### <a name="enterprise-voice-skype-for-business-server"></a>Enterprise Voice (Skype pour Business Server)

Enterprise Voice est la technologie de Microsoft pour l’activation du contrôle d’appel et les fonctionnalités d’autocommutateur privé (PBX) dans le locaux Skype pour Business Server. Cette option peut uniquement être connectée sur le réseau téléphonique commuté à l’aide de votre opérateur de téléphonie. 

Pour plus d’informations, voir [Plan for Enterprise Voice dans Skype pour Business Server](https://docs.microsoft.com/en-us/SkypeForBusiness/plan-your-deployment/enterprise-voice-solution/enterprise-voice?toc=/SkypeForBusiness/toc.json&bc=/SkypeForBusiness/breadcrumb/toc.json).

## <a name="connection-to-the-public-switched-telephone-network-pstn-options"></a>Connexion aux options Public réseau de téléphonique commuté (RTC)

Vous pouvez choisir de se connecter à la Public téléphone réseau commuté (RTC) par :

- À l’aide du Plan d’appel de Microsoft dans Office 365 
- Connexion de votre opérateur de téléphonie

### <a name="calling-plan-office-365"></a>Plan d’appel (Office 365)

Cette option connecte système téléphonique de Microsoft Office 365 pour les publics téléphone réseau commuté (RTC) pour activer les appels vers des réseaux terrestres et téléphones mobiles dans le monde entier. Planification de l’appelant, Microsoft est votre opérateur RTC.

Pour plus d’informations, voir [Appel Plans pour Office 365](https://docs.microsoft.com/en-us/MicrosoftTeams/calling-plans-for-office-365).

### <a name="connect-your-own-telephony-carrier-office-365-and-skype-for-business-on-premises"></a>Se connecter à votre propre opérateur de téléphonie (Office 365 et Skype pour les professionnels sur site)

Cette option connecte système téléphonique dans Office 365 ou système Enterprise Voice dans Skype pour Business local à votre réseau de téléphonie. Cette option requiert un contrôleur de frontière de Session pris en charge (SBC). Dans certains cas, cette option peut nécessiter des logiciels Microsoft supplémentaires déploiement en local.

## <a name="which-solution-is-right-for-your-organization"></a>Solution qui consiste le mieux pour votre organisation ?

Vous pouvez choisir une solution tout-en nuage, une solution de se connecter à votre-propriétaire-opérateur ou un mélange entre opérateurs tout-en nuage et de tiers :

- Système téléphonique avec l’appel planifier (sur le nuage)

- Système téléphonique avec propre opérateur via le routage Direct

- Système téléphonique avec propre opérateur via Skype pour le serveur ou sur le nuage connecteur Professionnel

- Enterprise Voice dans Skype pour Business Server avec propre opérateur

La solution que vous choisissez dépend de vos besoins actuels et futurs, tels que :

- Si vous souhaitez que :- ou sont nécessaires :-pour conserver les fonctionnalités fournies par votre déploiement sur site.
- Le client que vous souhaitez déployer pour vos utilisateurs.
- Quel est votre plan pour déplacer des utilisateurs vers le nuage.
- Si vous devez interagir avec 3ème partie PBX et d’autres équipements de téléphonie.

Posez-vous les questions suivantes pour déterminer la meilleure solution pour votre organisation :

- Existe-t-il un Skype existant pour le déploiement de serveur d’entreprise ?
- Vos utilisateurs sont hébergés sur Skype pour les entreprises sur site, dans le nuage sur Skype pour Business Online ou les deux ? 
- Vous souhaitez déplacer les utilisateurs locaux vers le nuage ?
- Prévoit RTC appelant planifier de Microsoft dans votre région ?
- Souhaitez-vous ou souhaitez conserver votre opérateur de téléphonie actuel ?  Par exemple, vous devez conserver votre opérateur actuel en raison d’un contrat existant ?
- Avez-vous un système PBX hérité local existant que vous souhaitez ou devez garder ?
- Votre système PBX hérité actuel offre-t-il des fonctionnalités uniques essentielles à votre entreprise ?
- Tout ou partie de vos utilisateurs ne nécessitent pas actuellement proposées dans le système téléphonique de fonctionnalités ?

Notez les points suivants :

- Les quatre options peuvent coexister avec eux au cas où vous avez besoin concevoir une solution pour un environnement complexe ou gestion de migration en plusieurs étapes.
- Système téléphonique avec propre opérateur via Skype pour Business Server ou sur le nuage connecteur Edition peut uniquement être déployé avec soit Skype pour le connecteur Cloud Business Server ou. Coexistence de Skype Business Server et le nuage n’est pas pris en charge dans une seule société.
