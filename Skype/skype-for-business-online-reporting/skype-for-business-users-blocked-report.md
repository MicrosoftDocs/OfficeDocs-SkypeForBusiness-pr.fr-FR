# Skype pour les utilisateurs professionnels bloqués rapport

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'[avertissement](0ac844b2-1b08-4e5a-addf-03cde7af7a40.md#MT_Footer). Vous pouvez consulter la version en anglais de cet article [ici](https://support.office.com/en-us/article/0ac844b2-1b08-4e5a-addf-03cde7af7a40). 
  
La nouvelle Skype pour des **rapports** de tableau de bord affiche la vue d'ensemble de l'activité au sein de la Skype pour les produits d'entreprise de votre organisation. Il vous permet d'exploration les rapports au niveau du produit individuels à vous fournir des informations plus précises sur les activités au sein de chaque produit. Par exemple, vous pouvez utiliser le rapport **Skype pour les utilisateurs professionnels bloqués** pour afficher les utilisateurs de votre organisation qui ont été bloqués d'effectuer des appels PSTN. Ce rapport, ainsi que d'autres Skype pour les rapports d'entreprise, vous donne plus d'informations sur l'activité, y compris l'utilisation PSTN au sein de votre organisation.
  
Consultez la [Rapports d'activité dans le Centre d'administration Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) pour les autres rapports disponibles.
  
> [!NOTE]
> Vous pouvez accéder à tous les rapports Skype Entreprise lorsque vous vous connectez au centre d'administration d'Office 365 en tant qu'administrateur. 
  
## Comment accéder à la Skype pour les utilisateurs professionnels bloqués rapport

- Accédez au **Centre d'administration Office 365** > **Centre d'administration** > ** Skype centre d'administration entreprise** > **rapports** > **utilisateurs bloqués**.
    
## Interpréter la Skype pour le rapport sur les utilisateurs bloqués entreprise

Vous pouvez obtenir un affichage dans utilisateurs bloqués en recherchant dans chacune des colonnes affichées.
  
Voici une illustration du rapport.
  
![Blocked users report](../images/df50a413-7a51-4340-a59b-3f83de941762.png)

Vous pouvez ajouter/supprimer des colonnes à la table. <br/> **ID d'utilisateur** est connexion de l'utilisateur. <br/> **Numéro de téléphone** est le numéro de téléphone attribué à un utilisateur. <br/> **Heure du blocage** est l'heure (UTC) à laquelle l'utilisateur a été bloqué. <br/> **Action de blocage** est le type d'action qui a été prise pour bloquer l'utilisateur. <br/> **Raison du blocage** est la raison pour laquelle l'utilisateur a été bloqué. <br/> |
