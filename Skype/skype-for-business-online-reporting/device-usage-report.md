# Skype pour le rapport d'utilisation entreprise appareil

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'[avertissement](b9019c36-034f-40c7-acb0-c2a0400b03c3.md#MT_Footer). Vous pouvez consulter la version en anglais de cet article [ici](https://support.office.com/en-us/article/b9019c36-034f-40c7-acb0-c2a0400b03c3). 
  
 **Rapports**, le nouveau tableau de bord d'Office 365, vous donne une vue d'ensemble de l'activité de tous les produits Office 365 dans votre organisation. Il vous permet d'explorer les rapports au niveau de chaque produit pour obtenir une vue d'ensemble plus précise des activités qu'ils contiennent. Par exemple, vous pouvez utiliser le rapport sur les **clients de conférence utilisés de Skype Entreprise** pour visualiser les appareils, mobiles ou non, y compris ceux qui utilisent des systèmes d'exploitation Windows, sur lesquels l'application Skype Entreprise est installée et utilisée pour la messagerie instantanée et les réunions. Consultez la[Rapports d'activité dans le Centre d'administration Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) pour en savoir plus.
  
Tout comme les autres rapports Skype Entreprise, ce rapport vous fournit des détails sur les types de clients/appareils utilisés dans votre organisation. Ces informations détaillées sont très utiles pour analyser, planifier et prendre des décisions pour votre entreprise.
  
> [!NOTE]
> Vous pouvez accéder à tous les rapports Skype Entreprise lorsque vous vous connectez au centre d'administration d'Office 365 en tant qu'administrateur. 
  
## Comment obtenir le rapport sur les clients utilisés de Skype Entreprise

1. Accédez au **Centre d'administration Office 365** > **rapports**
    
2. Dans le menu situé à gauche, sélectionnez **Rapports**, ou cliquez sur le widget **Rapports**.
    
3. Sur le tableau de bord, cliquez sur le widget **Activité Skype Entreprise**, puis sélectionnez **Clients utilisés de Skype Entreprise** ou sélectionnez-le depuis la liste intitulée **Sélectionner un rapport** dans la page **Utilisation** des rapports.
    
     ![Skype for Business Clients Used Report widget.](../images/37fde85f-b829-41ab-b3fa-1ac417eb683d.png)
  
    > [!IMPORTANT]
    > En fonction de l'abonnement Office 365 que vous avez souscrit, tous les rapports sur les produits et les activités présentés dans cet article ne seront peut-être pas disponibles. 
  
## Interpréter le rapport sur les clients utilisés de Skype Entreprise

Vous pouvez obtenir un aperçu des clients qui utilisent l'application Skype Entreprise en consultant les graphiques **Utilisateur** et **Distribution**.
  
![Skype for Business Clients used report.](../images/cb8b777a-67d4-43ad-a301-3cbeedbb45b5.png)
  
|||
|:-----|:-----|
|**1** <br/> |Le **rapport sur les clients utilisés de Skype Entreprise** peut être consulté pour les tendances au cours des 7, 30, 90 ou 180 derniers jours. <br/> |
|**2** <br/> |Chaque rapport comporte la date à laquelle il a été généré. Les rapports reflètent généralement une latence de 24 à 48 heures par rapport à l'heure de l'activité.  <br/> |
|**3** <br/> |Utiliser les données du graphique interactif sur le graphique **utilisateurs** pour comprendre les tendances d'utilisation et pour afficher le nombre d'utilisateurs qui sont connectés à l'aide de la Skype entreprise App qui sont utilisées dans votre organisation. Il indique le nombre total d'utilisateurs et les types d'appareils **Windows**, **Windows Phone**, **téléphone Android**, **iPhone** et **iPad** qui ont la Skype entreprise App installé et sont utilisés au sein de votre organisation. <br/> |
|**4** <br/> | Utilisez le graphique interactif du graphique à barres **Distribution** afin de comprendre l'utilisation et de voir le nombre d'utilisateurs qui utilisent des appareils uniques dans votre organisation. Il affiche le nombre total d'utilisateurs utilisant un appareil **Windows**, **Windows Phone**, **Android**, **iPhone** ou **iPad**.  <br/> |
|**5** <br/> |Vous pouvez filtrer la série présentée dans le graphique en cliquant sur un élément dans la légende. Par exemple, dans le graphique **Utilisateurs**, cliquez ou appuyez sur **Windows**, **Windows Phone**, **téléphone Android**, **iPhone** ou **iPad** pour voir uniquement les informations liées à chaque catégorie. Modifier cette sélection ne change pas les informations dans le tableau. <br/> |
|**6** <br/> | Chaque graphique a un axe « X » (horizontal) et « Y » (vertical). <br/>  Sur le graphique **Utilisateurs**, l'axe vertical représente le nombre d'utilisateurs connectés disposant d'un appareil sur lequel est installée l'application Skype Entreprise.  <br/>  Sur le graphique d'activités **Distribution**, l'axe vertical représente le nombre d'utilisateurs connectés disposant d'un certain type d'appareil sur lequel est installée l'application Skype Entreprise.  <br/>  L'axe X sur les deux graphiques représente la plage de dates sélectionnée pour ce rapport particulier. <br/> |
|**7** <br/> | Le tableau présente une répartition par utilisateur de toutes les activités de conférence. Il vous indique tous les utilisateurs auxquels Skype Entreprise a été affecté ainsi que leur utilisation des conférences. Vous pouvez ajouter des colonnes supplémentaires dans ce tableau. <br/> **Nom d'utilisateur** est le nom de l'utilisateur. <br/> **Dernière date d'activité (UTC)** représente la dernière fois que l'utilisateur a participé à une session d'égal à égal, a organisé une conférence ou a participé à une conférence. <br/> **Windows** est sélectionné si l'utilisateur dispose de l'application Skype Entreprise installée sur un système d'exploitation Windows et l'utilise actuellement. <br/> **Windows Phone** est sélectionné si l'utilisateur dispose de l'application Skype Entreprise installée sur un appareil Windows Phone et l'utilise actuellement. <br/> **Téléphone Android** est sélectionné si l'utilisateur dispose de l'application Skype Entreprise installée sur un téléphone Android et l'utilise actuellement. <br/> **iPhone** est sélectionné si l'utilisateur dispose de l'application Skype Entreprise installée sur un iPhone et l'utilise actuellement. <br/> **iPad** est sélectionné si l'utilisateur dispose de l'application Skype Entreprise installée sur un iPad et l'utilise actuellement. <br/>  Si la politique de votre organisation vous empêche de consulter les rapports sur lesquels figurent des informations propres aux utilisateurs, vous pouvez modifier les paramètres de confidentialité de tous ces rapports. |**8** <br/> |Cliquez ou appuyez sur **Colonnes** pour ajouter ou supprimer des colonnes du rapport.           <br/> ![Skype for Business Online Reporting Manage Button.](../images/4c8f5387-cebb-4d6c-b7d3-05c954a2c234.png)|
|**9** <br/> |Vous pouvez également exporter les données du rapport vers un fichier Excel .csv, en cliquant ou en appuyant sur **Exporter**.           <br/> ![Skype for Business Reporting Export Button.](../images/de7e2ab7-d70c-422f-a0ec-178b10f7dd51.png)Cela permet d'exporter les données de tous les utilisateurs et vous permet de trier et de filtrer plus simplement pour une analyse plus approfondie. Si vous avez moins de 2 000 utilisateurs, vous pouvez trier et filtrer directement dans le tableau du rapport.
