# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
La fonctionnalité de mobilité dans Skype pour Business Server 2015 prend en charge Skype pour les fonctionnalités d’entreprise sur des appareils mobiles.
  
Fonctionnalités de mobilité sont identique à celle Lync Server 2013. Pour plus d’informations sur la préparation de votre environnement pour la mobilité, voir [Plan pour la mobilité pour Skype pour Business Server](../../plan-your-deployment/mobility.md).
