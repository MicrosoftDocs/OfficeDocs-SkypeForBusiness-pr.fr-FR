# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
La fonction de mobilité de Skype entreprise Server 2015 prend en charge la fonctionnalité de Skype entreprise sur les appareils mobiles.
  
La fonctionnalité de mobilité est identique à celle de Lync Server 2013. Pour plus d’informations sur la préparation de votre environnement pour la mobilité, voir [planifier la mobilité pour Skype entreprise Server](../../plan-your-deployment/mobility.md).
