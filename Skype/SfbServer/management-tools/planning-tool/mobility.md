# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
La fonctionnalité de mobilité de Skype Entreprise Server 2015 prend en charge la fonctionnalité Skype Entreprise sur les appareils mobiles.
  
La fonctionnalité de mobilité est la même que dans Lync Server 2013. Pour plus d’informations sur la préparation de votre environnement pour la mobilité, voir [Plan for Mobility for Skype for Business Server](../../plan-your-deployment/mobility.md).
