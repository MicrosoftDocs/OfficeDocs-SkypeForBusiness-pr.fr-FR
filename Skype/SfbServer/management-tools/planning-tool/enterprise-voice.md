# <a name="enterprise-voice-planning-tool"></a>Voix Entreprise (outil de planification)
 
Avec la Voix Entreprise, Skype pour Business Server 2015 offre un autonome protocole voix sur IP (VoIP) offre à améliorer ou à remplacer les systèmes exchange (PBX) traditionnel branche privé. Les utilisateurs de Voix Entreprise peuvent appeler des collègues sur le réseau de votre organisation VoIP ou PBX, et elles peuvent appeler des numéros de téléphone traditionnel à l’extérieur de votre organisation. La solution Voix Entreprise inclut des fonctionnalités d’appel courantes telles que les réponses, transmettre, transférer, maintenez, détourner, mise à jour et s’et d’appel Enhanced 9-1-1 (E9-1-1) (E9-1-1 est disponible uniquement aux États-Unis). Voix Entreprise prend également en charge une large gamme de périphériques actuels et anciens de IP et USB.
  
Pour plus d’informations sur la planification de Voix Entreprise, voir [planification de Voix Entreprise dans Skype pour Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/enterprise-voice.md). Pour plus d’informations sur le déploiement d’un serveur de médiation autonome, voir [déploiement d’un serveur de médiation dans le Générateur de topologie dans Skype pour Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
