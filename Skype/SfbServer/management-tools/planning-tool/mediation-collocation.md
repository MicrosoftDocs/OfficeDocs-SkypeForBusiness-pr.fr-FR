# <a name="mediation-collocation-planning-tool"></a>Colocalisation de médiation (outil de planification)
 
Par défaut, le serveur de médiation est colocalisé avec le serveur frontal. Le serveur de médiation peuvent également être déployé dans un pool autonome pour des raisons de performances, ou si vous déployez SIP trunking, auquel cas le pool autonome est fortement recommandé. Colocalisation dans Skype pour les fonctions de serveur de l’entreprise telle qu’elle a fait de Lync Server 2013. Pour obtenir des détails, consultez les rubriques suivantes :
  
- [Colocalisation de serveurs pris en charge dans Lync Server 2013](https://technet.microsoft.com/en-us/library/gg425885%28v=ocs.15%29.aspx)
    
- [Composant de serveur de médiation dans Skype pour Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
