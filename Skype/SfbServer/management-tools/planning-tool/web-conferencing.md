# <a name="web-conferencing-planning-tool"></a>Conférence sur le Web (outil de planification)
 
Skype pour les conférences web Business Server 2015 inclut la collaboration sur les documents, partage d’application et le partage de bureau.
  
> [!NOTE]
> Skype pour Business Server 2015 utilise Office Web Apps et l’Office Web Apps Server pour gérer le partage et le rendu des présentations PowerPoint. Fonctionnalités de conférence Web fonctionnent exactement comme dans Microsoft Lync 2013. Pour plus d’informations sur l’installation et la configuration du serveur d’applications Web Office, consultez [intégration de configuration avec Office Web Apps Server et Lync Server 2013](https://technet.microsoft.com/en-us/library/jj204792%28v=ocs.15%29.aspx).

- [Plan pour les conférences à distance dans Skype pour Business Server 2015](../../plan-your-deployment/conferencing/dial-in-conferencing.md)
