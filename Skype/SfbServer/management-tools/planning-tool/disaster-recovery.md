# <a name="disaster-recovery-planning-tool"></a>Récupération d’urgence (outil de planification)
 
Skype pour Business Server 2015 comprend l’urgence des mesures de récupération en vous permettant de paire de pools de Front-End situés dans deux centres de données.
  
Pour plus d’informations sur la préparation de votre environnement de reprise après sinistre, consultez [planification de la haute disponibilité et reprise après sinistre dans Skype pour Business Server 2015](../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md).
