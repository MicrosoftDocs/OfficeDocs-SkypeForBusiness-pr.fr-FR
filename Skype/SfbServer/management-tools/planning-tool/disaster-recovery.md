# <a name="disaster-recovery-planning-tool"></a>Disaster Recovery (Planning Tool)
 
Skype Entreprise Server 2015 inclut des mesures de récupération d’urgence en vous permettant de jumeler des pools frontaux situés dans deux centres de données.
  
Pour plus d’informations sur la préparation de votre environnement pour la récupération d’urgence, voir [Plan for high availability and disaster recovery in Skype for Business Server 2015](../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md).
