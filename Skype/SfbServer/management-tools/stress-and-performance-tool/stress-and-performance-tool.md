# <a name="skype-for-business-server-2015-stress-and-performance-tool"></a>Skype pour Business Server 2015 Stress et l’outil performances
 
Le Skype pour Business Server 2015 Stress et l’outil performances est utilisé lors de la planification de la capacité et le réglage des performances dans les environnements non-production ou de test.
  
Le Skype pour Business Server 2015 Stress et l’outil performances inclut des outils qui simplifient votre planification des capacités pour Skype pour Business Server 2015. Le Skype pour Business Server 2015 Stress et l’outil performances vous aidera à :
  
- Simplifier votre planification pour Skype pour Business Server de matériel
    
- Améliorent connaissances et meilleures pratiques pour l’optimisation des performances
    
- Mesurer les performances de votre Skype pour les déploiements de serveur d’entreprise
    
Vous utiliserez en général cet outil après que le [Skype pour outil de planification de 2015 Business Server](../../management-tools/planning-tool/planning-tool.md) vous permet de concevoir la topologie et l’affinage de la topologie avec le [Skype pour le calculateur de planification de capacité Business Server 2015](../../management-tools/capacity-planning-calculator.md). 
  
## <a name="tests"></a>Tests

Le Stress et l’outil performances peuvent de simuler ces types de charge de l’utilisateur :
  
|||
|:-----|:-----|
|Instant Messaging (IM) et présence  <br/> |Conférence audio  <br/> |
|Partage d'application  <br/> |Voix sur IP (VoIP), y compris le réseau téléphonique public commuté, simulation de (PTSN)  <br/> |
|Conférence sur le Client d’accès Web  <br/> |Surveillance automatique de conférence  <br/> |
|Groupes de réponse  <br/> |Extension des listes de distribution  <br/> |
|Téléchargement du carnet d’adresse et de requête de carnet d’adresses  <br/> |Enhanced 911 (E911 Enhanced) appels et profil d’emplacement (plan de numérotation)  <br/> |
|MultiView  <br/> |Collaboration de données  <br/> |
|Mobilité  <br/> ||
   
## <a name="applications-and-files-included-with-the-skype-for-business-server-2015-stress-and-performance-tool"></a>Les applications et fichiers inclus avec le Skype pour Business Server 2015 Stress et l’outil performances

Ces applications sont une partie de la Skype pour Business Server Stress et l’outil de Performance :
  
|**Outil**|**Description**|
|:-----|:-----|
|UserProvisioningTool.exe  <br/> |Cet outil est utilisé pour créer des utilisateurs et des contacts.  <br/> |
|UserProfileGenerator.exe  <br/> |Permet de configurer les caractéristiques de la charge utilisateur simulée.  <br/> |
|LyncPerfTool.exe  <br/> |L’outil qui simule la charge utilisateur.  <br/> |
|Default.TMX  <br/> |Requise pour utiliser le Skype pour outil de journalisation Business Server 2015.  <br/> |
|Mise en service des exemples de scripts  <br/> |Permet de configurer la topologie pour l’exécution des tests de charge basés sur des scénarios spécifiques. Vous devrez probablement les modifier pour qu’elles soient adaptées à votre environnement particulier.  <br/> |
