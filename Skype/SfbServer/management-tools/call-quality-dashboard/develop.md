# <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Développer l’appel du tableau de bord qualité pour Skype pour Business Server 2015
 
**Résumé :** Apprenez à développer des solutions pour appeler le tableau de bord qualité. Tableau de bord qualité appel est un outil de Skype pour Business Server 2015.
  
Cette rubrique fournit la documentation de développement pour appeler qualité du tableau de bord (CQD) de Skype pour Business Server 2015.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Développer l’appel du tableau de bord qualité pour Skype pour Business Server 2015

CQD peut être étendu et intégré à votre environnement de création de rapports à l’aide de développement personnalisé. 
  
Deux ensembles de web API sont disponibles, une pour l’accès aux données dans le cube (via les API de données) et une pour accéder aux métadonnées de rapport (par le biais de l’API de référentiel). 
  
- [Données API pour le tableau de bord (CQD) appel qualité dans Skype pour Business Server 2015](data-api.md)
    
- [API de référentiel pour le tableau de bord (CQD) appel qualité dans Skype pour Business Server 2015](repository-api.md)
    
- [Exemples de développement CQD](cqd-development-samples.md)
