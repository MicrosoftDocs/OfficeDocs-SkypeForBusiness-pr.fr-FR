# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Développement du tableau de bord de qualité des appels pour Skype entreprise Server
 
**Résumé:** Apprenez à développer des solutions pour le tableau de bord de qualité des appels. Le tableau de bord de qualité des appels est un outil pour Skype entreprise Server.
  
Cette rubrique fournit une documentation de développement pour le tableau de bord de qualité des appels pour Skype entreprise Server (bord).
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Développement du tableau de bord de qualité des appels pour Skype entreprise Server

BORD peut être étendu et intégré à votre environnement de création de rapports à l’aide du développement personnalisé. 
  
Deux jeux d’API Web sont disponibles, un pour l’accès aux données du cube (via les API de données) et l’autre pour l’accès aux métadonnées du rapport (via l’API du référentiel). 
  
- [API de données pour le tableau de bord de qualité des appels (bord) dans Skype entreprise Server](data-api.md)
    
- [API du référentiel pour le tableau de bord de qualité des appels (bord) dans Skype entreprise Server](repository-api.md)
    
- [Exemples de développement du tableau de bord de qualité des appels](cqd-development-samples.md)
