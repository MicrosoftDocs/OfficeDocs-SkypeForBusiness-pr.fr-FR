# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Développer le tableau de bord appel qualité pour Skype pour Business Server
 
**Résumé :** Apprenez à développer des solutions pour appeler le tableau de bord qualité. Tableau de bord de qualité des appels est un outil de Skype pour Business Server.
  
Cette rubrique fournit une documentation de développement pour appeler la qualité du tableau de bord (CQD) pour Skype pour Business Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Développer le tableau de bord appel qualité pour Skype pour Business Server

CQD peut être étendu et intégré à votre environnement de création de rapports à l’aide de développement personnalisé. 
  
Deux jeux d’API de web sont disponibles, un pour l’accès aux données du cube (via les API de données) et un pour l’accès aux métadonnées des rapports (par le biais de l’API de référentiel). 
  
- [API de données pour l’appel de tableau de bord qualité (CQD) dans Skype pour Business Server](data-api.md)
    
- [API de référentiel pour l’appel de tableau de bord qualité (CQD) dans Skype pour Business Server](repository-api.md)
    
- [Exemples de développement du tableau de bord de qualité des appels](cqd-development-samples.md)
