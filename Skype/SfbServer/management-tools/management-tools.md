# <a name="skype-for-business-server-2015-management-tools"></a>Outils de gestion Skype Entreprise Server 2015
 
**Résumé :** Découvrez les outils de gestion de service dans Skype pour Business Server 2015.
  
Skype pour (anciennement Lync Server), le logiciel de communication Business Server 2015 offre la messagerie instantanée (MI), de présence, de conférence et de solutions de téléphonie qui prennent en charge les besoins de collaboration au niveau de l’entreprise. Les outils servant à gérer ces services sont à la fois flexibles et puissants.
