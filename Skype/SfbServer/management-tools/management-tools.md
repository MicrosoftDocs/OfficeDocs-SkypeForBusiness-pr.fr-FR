# <a name="skype-for-business-server-2015-management-tools"></a>Outils de gestion de Skype entreprise Server 2015
 
**Résumé :** En savoir plus sur les outils de gestion des services dans Skype entreprise Server 2015.
  
Le logiciel de communications Skype entreprise Server 2015 (anciennement Lync Server) fournit des solutions de messagerie instantanée, de présence, de conférence et de téléphonie qui prennent en charge les besoins de collaboration au niveau de l’entreprise. Les outils permettant de gérer ces services sont flexibles et puissants.
