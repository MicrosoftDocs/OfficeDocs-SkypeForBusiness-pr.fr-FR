# <a name="skype-for-business-server-2015-management-tools"></a>Outils de gestion Skype Entreprise Server 2015
 
**Résumé :** En savoir plus sur les outils de gestion des services dans Skype entreprise Server 2015.
  
Le logiciel de communications Skype entreprise Server 2015 (anciennement Lync Server) offre des solutions de messagerie instantanée, de présence, de conférence et de téléphonie qui prennent en charge les exigences de collaboration au niveau de l’entreprise. Les outils servant à gérer ces services sont à la fois flexibles et puissants.
