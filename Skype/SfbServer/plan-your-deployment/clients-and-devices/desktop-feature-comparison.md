# <a name="desktop-client-feature-comparison-for-skype-for-business"></a>Comparaison des fonctionnalités des clients de bureau pour Skype Entreprise
 
**Résumé :** Skype pour Business Server 2015 ou Skype pour les administrateurs de l’entreprise en ligne peut utiliser ces tables pour comprendre quelles fonctionnalités sont prises en charge sur les clients.
  
 Avant de déployer ou de mettre à niveau vers Skype pour entreprise, vérifiez que les clients qui sont déjà en cours d’utilisation dans votre organisation. Les tableaux ci-dessous permet de comprendre l’impact de la prise en charge de fonctionnalité sur ces clients. Cela peut vous aider à communiquer les modifications aux utilisateurs, opérer le processus de déploiement et bien comprendre les avantages de la mise à niveau vers le dernier client.
  
Certaines fonctionnalités disponibles avec Skype pour Business Server 2015 ne sont pas disponibles dans Skype pour professionnels en ligne, voir les [restrictions de compte d’utilisateur en ligne ou hybride](desktop-feature-comparison.md#Online-Hybrid) pour les caractéristiques de l’objet. Skype pour les administrateurs de l’entreprise en ligne peut souhaiter [Skype pour la Description de Service Business en ligne](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) pour plus d’informations sur les différents plans disponibles pour les.
  
Les tableaux suivants répertorient les fonctionnalités disponibles avec chaque client qui fonctionne avec Skype pour Business Server 2015 ou Skype pour l’activité en ligne. Vous pouvez également faire référence à la [comparaison des fonctionnalités de client Mobile pour Skype pour entreprise](mobile-feature-comparison.md) de téléphone intelligent et de comparaison des fonctionnalités de client de Tablet PC. Les licences d’accès client ou USL (User Subscription License) qu’achète votre organisation auront également un impact sur les fonctionnalités auxquelles les utilisateurs ont accès. Le déploiement du client complet ou de base pour les utilisateurs dépend de la licence ou de l’offre que votre organisation choisit d’acheter. Consultez le [Guide de gestion des licences](https://products.office.com/en-us/skype-for-business/it-pros) pour plus de détails.
  
> [!IMPORTANT]
> Skype pour Business Server 2015 et Skype pour Business Online prend en charge les clients précédemment publiées suivantes : Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Phone Edition et le service de surveillance du Lync 2010. Pour plus d’informations sur ces clients lorsqu’elle est utilisée avec d’autres serveurs, consultez les [tableaux de comparaison Client de Lync Server 2013](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.15%29.aspx) et les [tables de comparaison de Client pour Lync Server 2010](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.14%29.aspx). 
  
> [!NOTE]
> Le client **Lync 2010 surveillance** n’est pas pris en charge dans Skype pour l’activité en ligne.
  
> [!NOTE]
> Le Skype pour le client du navigateur Business Web App et Skype réunions application Windows 10 app assurent uniquement la [prise en charge des réunions](desktop-feature-comparison.md#BKMK_Conferencing). Pour plus d’informations sur ces clients, voir la rubrique [Plan pour les clients de réunions (application Web et application de réunions)](meetings-clients.md) .

## <a name="client-limitations"></a>Restrictions relatives aux clients 
<a name="Types"> </a>

### <a name="basic-client-limitations"></a>Restrictions relatives aux clients de base
<a name="Full-Basic"> </a>

Les fonctionnalités ci-après sont disponibles avec le client complet. Elles ne sont pas intégrées au client de base :  
  
- Gérer les paramètres d’appel d’équipe
    
- Gérer les délégués
    
- Passer un appel pour le compte d’un autre contact (scénario responsable/délégué)
    
- Gérer les appels d’une autre personne, si celle-ci est configurée en tant que déléguée
    
- Gérer un volume d’appels important
    
- Démarrer un appel à un groupe Response Group
    
- Parcage d’appel
    
- Changer le message d’accueil
    
- Prise d’appel de groupe
    
### <a name="online-or-hybrid-user-account-limitations"></a>Restrictions relatives aux comptes d’utilisateur en ligne ou hybrides
<a name="Online-Hybrid"> </a>

Les comptes d’utilisateur peuvent exister en ligne ou dans les locaux, ce qui affectera les fonctionnalités auxquelles l’utilisateur correspondant peut accéder. Les utilisateurs possédant des comptes sur Skype pour Business Online n’aura pas accès aux fonctions suivantes, même si le client complet : 
  
- Présence améliorée : Utiliser une photo à partir d’un site public pour Mon image
    
- Contacts : Rechercher des groupes Response Group
    
- Prise en charge de la messagerie instantanée : Intégration de la conversation permanente (Conversation de groupe)
    
- Prise en charge de la messagerie instantanée : Faire remonter une salle de conversation permanent à un Skype pour une réunion d’affaires en un seul clic
    
- Utilisateurs externes : Mener des appels à deux ou à plusieurs avec des utilisateurs externes
