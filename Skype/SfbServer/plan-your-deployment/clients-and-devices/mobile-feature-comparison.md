# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>Comparaison des fonctionnalités de client mobile pour Skype pour entreprise
 
**Résumé :** Passez en revue la prise en charge de la fonctionnalité pour le client mobile lors de la planification pour Skype pour Business Server 2015.
  
Cet article compare les fonctionnalités entre Skype pour les clients professionnels mobiles et le Skype pour client ordinateur de bureau d’entreprise dans les catégories suivantes :
  
- Connexion, notifications Push et fonctionnalités générales
    
- Présence avancée
    
- Contacts et groupes de contacts
    
- Messagerie instantanée
    
- Skype pour entreprise de Skype Business audio et vidéo
    
- Téléconférence
    
- Téléphonie
    
- Utilisateurs externes
    
- Archivage et conformité
    
-  Authentification moderne
    
Les tableaux suivants répertorient les fonctionnalités disponibles à Skype pour les utilisateurs professionnels dans un déploiement sur site de Skype pour Business Server 2015. Les mêmes fonctions sont également disponibles pour Skype pour les utilisateurs Business Online et de Microsoft Office 365, sauf indication contraire dans les notes de fin de la table.
  
> [!NOTE]
> Pour l’aide en ligne et des ressources pour les utilisateurs finaux, consultez [Découverte de Skype pour les entreprises](https://go.microsoft.com/fwlink/p/?LinkId=528686). 
  
> [!NOTE]
> Pour comparer les fonctionnalités disponibles dans les autre Skype pour les clients de l’entreprise, consultez [comparaison des fonctionnalités client de bureau pour Skype pour les entreprises](desktop-feature-comparison.md).
