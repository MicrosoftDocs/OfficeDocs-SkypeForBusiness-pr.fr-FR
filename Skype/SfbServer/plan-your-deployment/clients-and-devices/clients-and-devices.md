# <a name="plan-for-clients-and-devices"></a>Planifier des clients et des appareils
 
**Résumé :** Examen des clients pris en charge et des applications pour Skype pour les entreprises.
  
À notre époque, la population active est toujours en déplacement. Les employés doivent pouvoir communiquer et collaborer, qu’ils travaillent dans les bureaux du siège, sur un site régional, à leur domicile ou sur la route. Skype pour Business Server 2015 prend en charge ces besoins dans une collection d’interfaces de client que vous pouvez déployer pour les utilisateurs de votre organisation. Une bonne planification ainsi que les employés ont satisfait leurs besoins que Skype pour entreprises est disponible pour les partout où ils se trouvent. 
  
## <a name="available-clients"></a>Clients disponibles

Skype pour Business Server 2015 prend en charge plusieurs types de clients, y compris le logiciel client d’ordinateur installé, les clients basés sur le web et les clients pour les périphériques mobiles. Chacun de ces clients est abordé dans cette section. Si vous avez déjà utilisé une combinaison de clients Lync, notez qu’il n’y a pas de prise en charge [des clients hérités](clients-and-devices.md#Legacy) qui ne sont pas compatibles avec Skype pour les entreprises. Les mises à jour ayant lieu régulièrement, consultez cette rubrique périodiquement pour recevoir des informations à jour sur les clients.
  
### <a name="skype-for-business-2016"></a>Skype pour entreprise 2016

Skype pour entreprise 2016 est le client complet recommandé pour Skype pour Business Server 2015. Pour obtenir une description des nouvelles fonctionnalités, consultez [Nouveautés de Skype pour 2016 de l’entreprise](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) . Prise en charge de la fonctionnalité client est détaillée dans la [comparaison des fonctionnalités client de bureau pour Skype pour les entreprises](desktop-feature-comparison.md)et la documentation utilisateur est à [l’aide de Skype pour les entreprises](https://support.office.com/en-US/Skype-for-business). Ce client est inclus lorsqu’un utilisateur installe Office 365.
  
Un client de base gratuit, qui prend en charge moins de fonctionnalités, est également disponible. Les deux versions sont disponibles pour téléchargement à [Télécharger de Skype pour entreprise sur tous les périphériques](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Les différences entre les clients complet et Basic sont décrites dans la section [limitations du client de base](desktop-feature-comparison.md#Full-Basic) .

### <a name="online-meeting-add-in-for-skype-for-business"></a>Réunion en ligne complément pour Skype pour entreprise

Le complément de réunion en ligne pour Skype pour entreprise prend en charge la gestion de réunion à partir de client de messagerie et de collaboration Microsoft Outlook. Le complément de réunion en ligne pour Skype pour le logiciel Professionnel installe automatiquement avec Skype pour les entreprises.
  
### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype pour Business Web App et application de réunions Skype

Skype pour Business Web App est le client de plug-in de navigateur conférence web pour Skype pour les réunions d’entreprise serveur 2015. Les participants à une réunion ont accès à toutes les fonctionnalités de collaboration et de partage, tandis que le présentateur de la réunion dispose du contrôle. Ce client permet également invités à partir de l’extérieur de votre organisation qui ne possèdent pas un client régulier à assister aux réunions créées par votre Skype pour les utilisateurs professionnels. 
  
Si Skype pour entreprise n’est pas installé sur l’ordinateur d’un utilisateur et que l’utilisateur clique sur un lien d’une réunion dans une demande de réunion, Skype pour Business Web App sera installé et l’ouvrir. Vous pouvez aussi configurer la page Participer à la réunion pour permettre aux utilisateurs de rejoindre la réunion en utilisant des versions antérieures des clients. Skype pour Business Web App est le client idéal pour les participants à l’extérieur de votre organisation. Avec Skype pour Business Web App, aucune installation de client locale n’est requise, bien que les fonctionnalités audio, vidéo et partage nécessitent l’installation d’un plug-in au cours de la première utilisation. Afficher un [Plan pour les clients de réunions (application Web et application de réunions)](meetings-clients.md) pour la configuration requise pour utiliser ce client.
  
L’application de réunions Skype est une application Windows 10 directe qui utilise un navigateur lors de l’installation et s’exécute comme une application native Windows 10 uniquement. Il requiert la configuration spécifique du serveur pour activer la distribution à partir du serveur frontal. Comme Skype pour Business Web App, ce client permet également invités à partir de l’extérieur de votre organisation qui ne possèdent pas un client régulier à assister aux réunions créées par votre Skype pour les utilisateurs professionnels. 
  
### <a name="skype-for-business-web-scheduler"></a>Skype pour le Planificateur Web de professionnels

Skype pour le Planificateur Web Business est basé sur un web réunion d’outil de planification et de gestion pour les utilisateurs qui n’ont pas accès à Microsoft Outlook, ou qui sont sur un système d’exploitation non basé sur Windows. Avec Skype pour le Planificateur Web de professionnels, les utilisateurs peuvent créer de nouvelles réunions, modifier les réunions existantes et envoyer des invitations à l’aide de son programme de messagerie par défaut.
  
[Skype pour le Planificateur Web Business](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) documentation fournit des informations complémentaires.
  
### <a name="vdi-plugins"></a>Plug-ins VDI

Un environnement VDI (Virtual Desktop Infrastructure) est utilisé dans certaines entreprises pour lesquelles les aspects liés à la sécurité et à la conformité sont particulièrement cruciaux. À l’aide de Skype pour entreprise avec audio et vidéo sur une connexion complète que nécessitent des charges lourdes de données audio et vidéo de traitement sur le client hébergés sur un bureau virtuel. Un plug-in logiciel VDI supplémentaire n’est disponible qui décharge le traitement à l’ordinateur local de l’utilisateur final et réduit la charge sur le bureau virtuel. Pour plus d’informations sur l’utilisation de ces plug-ins, consultez [planification de Skype pour l’entreprise dans les environnements VDI](vdi-environments.md) .
  
### <a name="skype-room-systems-v2"></a>Skype Room System v2

Systèmes de salle Skype v2 est plus récente conférence solution Microsoft qui utilise un familier Skype pour l’interface de l’entreprise et qui est facile à déployer et géré, exploitant des équipements existants tels que des panneaux LCD pour faciliter l’installation. Systèmes de salle Skype v2 utilise une application UWP spécialisée en cours d’exécution sur un 4 de Surface Pro ou Surface Pro en mode console (une fois déployé l’application UWP est la seule application qui s’exécute sur le périphérique) et qu’il requiert son propre compte de dispositif sur votre Skype pour la mise en oeuvre de l’entreprise. Est mis à jour via Windows Update et le Windows store. Pour plus d’informations sur l’utilisation de ces consoles de pièce dans votre déploiement, consultez [planification de systèmes de salle Skype v2](skype-room-systems-v2-0.md) .
  
### <a name="skype-for-business-on-surface-hub"></a>Skype pour le commerce sur le concentrateur de Surface

Concentrateur de Microsoft Surface est un périphérique de la productivité de tout-en-un qui vise pour le brainstorming, la collaboration et des présentations. Il possède son propre itération de la Skype pour client d’entreprise, documentée dans le [guide de l’administrateur Microsoft Surface concentrateur](https://docs.microsoft.com/en-us/surface-hub/).

## <a name="ways-to-deploy-the-client-to-your-users"></a>Méthodes de déploiement du client pour vos utilisateurs
<a name="BK_User_Deploy"> </a>

Programmes d’installation du client sont disponibles pour les MSI et les programmes d’installation du type de clic. Le Skype pour la stratégie d’entreprise-client sustainment peut-être affecter vos choix, afin que vous devez comprendre les points suivants : 
  
- En règle générale Skype pour entreprise n’ajoute pas de nouvelles fonctionnalités aux clients précédemment publiées
    
- En général, Skype pour entreprise ne prévoit pas sur l’expédition des nouvelles fonctionnalités dans le Skype pour MSI de l’entreprise après sa version initiale. Les améliorations MSI entre chaque sortie concerneront principalement la qualité/sécurité. 
    
- Le toutes dernières Skype pour une expérience client se trouve dans le Skype pour le programme d’installation de Business 2016 click-to-run. 
    
Vous pouvez effectuer un déploiement personnalisé du client comme indiqué dans [l’installation du client Windows de personnaliser dans Skype pour Business Server 2015](../../deploy/deploy-clients/customize-windows-client-installation.md). Méthodes d’installation sont décrits en détail dans [les clients de déployer pour Skype pour Business Server 2015](../../deploy/deploy-clients/deploy-clients.md)
  
### <a name="click-to-run"></a>Démarrer en un clic

Clic est une diffusion en continu de Microsoft et la technologie de virtualisation que vous pouvez utiliser pour installer et mettre à jour des produits Office, y compris de Skype pour les entreprises. Ces fonctionnalités de diffusion en continu et de virtualisation reposent sur les technologies de Microsoft Application Virtualization (App-V). Démarrer en un clic possède les avantages suivants : 
  
- Installation en flux continu de la suite Office induisant une durée d’installation courte
    
- Mises à jour et correctifs « slipstreamed »
    
- Occupe moins d’espace sur le disque
    
- Licences utilisateur : 5 installations par utilisateur
    
- Personnalisable via l’éditeur XML pour l’installation de programmes indépendants
    
Vous souhaiterez peut-être utiliser l' [Outil de déploiement de Office 2016](https://www.microsoft.com/en-us/download/details.aspx?id=49117) pour ce type de programme d’installation.
  
À la fois les versions de client de base et complète (avec le choix entre les versions 32 bits et 64 bits) sont disponibles à l’aide d’un programme d’installation de démarrer en un clic, vos utilisateurs peuvent télécharger à [Télécharger de Skype pour entreprise sur tous les périphériques de votre](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). 
  
### <a name="msi"></a>MSI

MSI est une méthode d’installation plus traditionnelle. Il vous permet d’installer manuellement les mises à jour et correctifs, utilisation licence en volume et l’activation et il est personnalisable à l’aide de l' [Outil de personnalisation Office](http://www.microsoft.com/en-us/download/details.aspx?id=49030). Vous pouvez répartir les clients en appliquant des stratégies de groupe, à l’aide de System Center Configuration Manager, ou à l’aide d’un outil tiers.
