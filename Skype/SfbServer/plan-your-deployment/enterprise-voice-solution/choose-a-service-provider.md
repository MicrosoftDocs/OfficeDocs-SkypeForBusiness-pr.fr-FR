# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server-2015"></a>Sélection d’un fournisseur de services E9-1-1 pour Skype Entreprise Server 2015
 
Choix de votre fournisseur de services pour un déploiement E9-1-1 dans Skype pour Business Server Voix Entreprise.
  
Les E9-1-1 Services fournisseur itinéraires appels d’urgence provenant de Skype pour Business Server à la correcte Public sécurité répondant Point (PSAP) basé sur les informations d’emplacement contenues dans l’appel. 
  
Prise en charge E9-1-1 dans le cadre d’un Skype pour le déploiement du serveur de l’entreprise, vous devez vous procurer E9-1-1 service de routage à partir d’un programme d’interopérabilité d’ouvrir Lync qualifié de fournisseur de service E9-1-1. Choisissez le fournisseur qui répond le mieux aux besoins de votre organisation.
