# <a name="define-your-requirements-for-emergency-calls-in-skype-for-business-server"></a>Définir vos exigences pour les appels d’urgence dans Skype entreprise Server
 
Résume les étapes nécessaires à l’activation de E9-1-1 dans Skype entreprise Server Voice, selon que vous disposez d’un fournisseur de services SIP Trunk E9-1-1 ou d’une passerelle ELIN.
  
Avant de commencer un déploiement de Skype entreprise Server E9-1-1, vous devez d’abord répondre aux questions détaillées des sections suivantes. La planification dont vous avez besoin dépend du type de solution E9-1-1 que vous choisissez de déployer : fournisseur de services E9-1-1 de jonction SIP ou numéro ELIN (Emergency Location Identification Number).
