# <a name="capacity-planning-for-skype-for-business-server"></a>Planification de la capacité pour Skype entreprise Server

> Toutes les recommandations de cette section supposent que la mise à jour cumulative de Skype Entreprise de novembre 2015, ou version ultérieure, soit installée sur vos serveurs.
