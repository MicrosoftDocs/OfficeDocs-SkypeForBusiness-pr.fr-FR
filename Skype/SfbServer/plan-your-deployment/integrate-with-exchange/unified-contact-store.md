# <a name="plan-for-unified-contact-store-in-skype-for-business-server-2015"></a>Planification d’un magasin de contacts unifié dans Skype Entreprise Server 2015
 
**Résumé :** Lors de la planification intégrer Skype de 2015 de serveur d’entreprise avec Exchange 2013, consultez cette rubrique.
  
Magasin de contacts unifiée offre une expérience cohérente de contact sur les produits Microsoft Office et permet aux utilisateurs de stocker toutes les informations de contact dans Exchange 2013, mais autorise les informations doivent être globalement disponibles sur Skype pour entreprise, Exchange, Outlook et Outlook Web Access.
  
## <a name="requirements-for-unified-contact-store"></a>Configuration requise pour le magasin de contacts unifié

Pour implémenter le magasin de contacts unifiée dans Skype pour Business Server 2015 :
  
- Vous devez utiliser Skype pour Business Server 2015 et Exchange 2013.
    
- Les utilisateurs doivent utiliser Skype pour les entreprises de migrer leurs contacts Skype pour Business Server 2015 vers Exchange 2013.
    
- Boîtes aux lettres de l’utilisateur doivent être migrés vers Exchange 2013.
    
- Vous devez disposer de l’authentification de serveur à serveur configurée entre Skype pour Business Server 2015 et Exchange 2013.
    
    > [!NOTE]
    > Pour plus de détails sur la configuration de l’authentification entre Skype pour Business Server 2015 et Exchange 2013, consultez [authentification de serveur à gérer (OAuth) et les applications partenaires dans Skype pour Business Server 2015](../../manage/authentication/server-to-server-and-partner-applications.md) dans les opérations documentation.
