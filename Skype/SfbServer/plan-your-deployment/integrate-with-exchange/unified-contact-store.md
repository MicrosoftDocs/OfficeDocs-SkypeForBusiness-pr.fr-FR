# <a name="plan-for-unified-contact-store-in-skype-for-business-server-2015"></a>Planification d’un magasin de contacts unifié dans Skype Entreprise Server 2015
 
**Résumé :** Consulter cette rubrique lors de la planification intégrer Skype pour Business Server avec Exchange 2013 ou 2016.
  
Magasin de contacts unifié fournit une expérience cohérente contact entre les produits Microsoft Office et permet aux utilisateurs de stocker toutes les informations de contact dans Exchange 2013, mais autorise les informations doivent être disponibles globalement sur Skype pour Business, Exchange, Outlook et Outlook Web Access.
  
## <a name="requirements-for-unified-contact-store"></a>Configuration requise pour le magasin de contacts unifié

Pour implémenter le magasin de contacts unifié de Skype pour Business Server :
  
- Vous devez exécuter Skype pour Business Server et Exchange 2013 ou 2016.
    
- Utilisateurs doivent utiliser Skype pour les entreprises à migrer leurs contacts à partir de Skype pour Business Server à Exchange 2013 ou 2016.
    
- Boîtes aux lettres utilisateur doivent être migrées vers Exchange 2013.
    
- Vous devez disposer de l’authentification de serveur à serveur configurée entre Skype pour Business Server et Exchange 2013 ou 2016.
    
    > [!NOTE]
    > Pour plus de détails sur la configuration de l’authentification entre Skype pour Business Server et Exchange 2013 ou 2016, voir [gérer l’authentification de serveur à serveur (OAuth) et des applications partenaires dans Skype pour Business Server](../../manage/authentication/server-to-server-and-partner-applications.md) dans les opérations documentation.
