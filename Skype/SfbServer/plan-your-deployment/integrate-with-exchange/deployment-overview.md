# <a name="deployment-process-overview-for-integrating-on-premises-unified-messaging-and-skype-for-business"></a>Vue d’ensemble de la procédure de déploiement pour l’intégration de la messagerie unifiée locale et de Skype Entreprise
 
**Résumé :** Lors de la planification intégrer Skype de 2015 de serveur d’entreprise avec Exchange 2013, consultez cette rubrique.
  
 Si vous souhaitez intégrer Exchange la messagerie unifiée (MU) avec Skype pour Business Server 2015, vous devez effectuer les tâches décrites dans cette rubrique. Assurez-vous également que vous passez en revue les meilleures pratiques de planification et de déploiement, décrites dans le [Plan de l’intégration de la messagerie unifiée Exchange dans Skype pour les entreprises](unified-messaging.md). Cette rubrique suppose que vous avez déployé Skype pour 2015 de serveur d’entreprise avec un serveur de médiation colocalisée et que vous avez activé des utilisateurs de Skype pour Business Server 2015, mais vous n’avez ne peut-être pas effectué toutes les étapes de configuration et de déploiement pour permettre d’entreprise Vocale, comme indiqué dans [Déploiement de Voix Entreprise dans Skype pour Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) dans la documentation de déploiement.
