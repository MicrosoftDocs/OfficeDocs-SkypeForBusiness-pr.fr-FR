# <a name="synchronize-users-to-the-cloud-and-assign-licenses"></a>Synchronisation des utilisateurs avec le cloud et affectation des licences
 
Avant de réellement passer l’utilisateur à Office 365, vous devez d’abord confirmer qu’ils sont synchronisés avec le nuage et leur attribuer une licence. À cet effet, vous devez utiliser le Centre d’administration Office 365.
  
### <a name="to-confirm-that-a-user-is-synchronized-with-office-365"></a>Pour confirmer qu’un utilisateur est synchronisé avec Office 365

1. Ouvrir le centre d’administration Office 365 (https://portal.office.com).
    
2. Dans le volet de navigation de gauche, cliquez sur **Utilisateurs**, puis sur **Utilisateurs actifs**.
    
3. Cliquez sur **Rechercher un utilisateur** et tapez le nom de l’utilisateur.
    
4. Vérifiez que l’utilisateur s’affiche et que son statut est **Synchronisé avec Active Directory**.
    
    Si l’utilisateur n’est pas encore synchronisé, la synchronisation automatique doit avoir lieu dans les trois heures. Ou bien, vous pouvez forcer une synchronisation plus tôt. Pour plus d’informations, consultez [Force la synchronisation d’annuaire](https://msdn.microsoft.com/en-us/library/azure/JJ151771.aspx).
    
## <a name="to-assign-the-license"></a>Pour affecter la licence

Pour attribuer la licence dans Office 365, voir [attribuer ou supprimer l’attribution d’une licence de nuage PBX pour un utilisateur](https://support.office.com/article/Assign-or-unassign-a-Cloud-PBX-license-for-a-user-36c6d5a6-5ea8-4c44-9f18-fea33d5a847e).
