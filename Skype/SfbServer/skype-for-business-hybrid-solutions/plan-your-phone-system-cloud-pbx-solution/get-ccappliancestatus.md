# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
L’applet de connexion Get-CcApplianceStatus fournit des informations de diagnostic pour l’appliance Skype entreprise Cloud Connector Edition.
  
Cette cmdlet s’applique à Skype entreprise version Cloud Connector 2,1.
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>Description détaillée

L’applet de connexion Get-CcApplianceStatus, lors de l’exécution sur l’ordinateur hôte du Cloud Connector, fournit des informations de diagnostic pour le matériel Skype entreprise Cloud Connector.
  
## <a name="parameters"></a>Paramètres

Aucun
  
## <a name="input-types"></a>Types d’entrées

Aucun. L’applet de commande Get-CcApplianceStatus n’accepte pas les entrées pipelines.
  
## <a name="return-types"></a>Types de retours

La commande renvoie l’état de l’appliance.
