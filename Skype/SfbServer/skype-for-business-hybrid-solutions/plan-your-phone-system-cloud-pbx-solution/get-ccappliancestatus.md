# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
L’applet de commande Get-CcApplianceStatus fournit des informations de diagnostic pour le Skype pour appliance nuage connecteur édition.
  
Cette applet de commande s’applique à Skype Business nuage connecteur Edition 2.1.
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>Description détaillée

L’applet de commande Get-CcApplianceStatus, sur l’ordinateur hôte sur le nuage connecteur, fournit des informations de diagnostic pour le Skype pour appliance Business Cloud Connector.
  
## <a name="parameters"></a>Paramètres

Aucun
  
## <a name="input-types"></a>Types d’entrées

Aucun. L’applet de commande Get-CcApplianceStatus n’accepte pas la saisie de données redirigées.
  
## <a name="return-types"></a>Types de retours

La commande renvoie l’état de l’application.
