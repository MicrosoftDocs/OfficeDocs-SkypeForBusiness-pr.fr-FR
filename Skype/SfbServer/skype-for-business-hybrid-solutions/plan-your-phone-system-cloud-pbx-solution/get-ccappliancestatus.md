# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
L’applet de commande Get-CcApplianceStatus fournit des informations de diagnostic pour le Skype pour appareil de connecteur de Cloud Business Edition.
  
Cette applet de commande s’applique à Skype Business Cloud connecteur Édition 2.1.
  
```

    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>Description détaillée

L’applet de commande Get-CcApplianceStatus, lorsqu’il est exécuté sur l’ordinateur hôte du connecteur de nuage, fournit des informations de diagnostic pour le Skype pour la solution matérielle-logicielle de nuage de Business Connector.
  
## <a name="parameters"></a>Paramètres

Aucun
  
## <a name="input-types"></a>Types d’entrées

Aucun. L’applet de commande Get-CcApplianceStatus n’accepte pas les entrées de pipeline.
  
## <a name="return-types"></a>Types de retours

La commande renvoie l’état de la solution matérielle-logicielle.
