# <a name="configure-and-manage-skype-for-business-cloud-connector-edition"></a>Configuration et gestion de Skype entreprise, version Cloud Connector
 
Découvrez comment configurer Skype entreprise version Cloud Connector, une topologie sur site minimale permettant l’intégration de votre infrastructure vocale locale au système téléphonique dans les services vocaux Office 365 (Cloud PBX) dans Skype entreprise online. 
  
Avant de commencer, passez en revue les conditions préalables dans [plan for Skype for Business Cloud Connector Edition](plan-skype-for-business-cloud-connector-edition.md).
  
> [!IMPORTANT]
> Les étapes de cette rubrique s’appliquent uniquement à la version de Cloud Connector 1.4.1 et versions ultérieures. Si vous n’avez pas encore effectué la mise à niveau vers Cloud Connector Edition 2,1, consultez [la rubrique Upgrade to a New version of Cloud Connector](upgrade-to-a-new-version-of-cloud-connector.md). Vous pouvez télécharger le fichier d’installation [https://aka.ms/CloudConnectorInstaller](https://aka.ms/CloudConnectorInstaller)à partir de.
