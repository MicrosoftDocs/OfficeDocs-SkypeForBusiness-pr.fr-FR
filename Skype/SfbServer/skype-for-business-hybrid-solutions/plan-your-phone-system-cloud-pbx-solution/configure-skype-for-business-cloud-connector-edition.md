# <a name="configure-and-manage-skype-for-business-cloud-connector-edition"></a>Configuration et gestion de Skype Entreprise, version Cloud Connector
 
Découvrez comment configurer Skype pour l’édition de connecteur Business Cloud, une topologie minimale locale pour activer l’intégration de votre infrastructure de voix sur site avec un système téléphonique dans Office 365 (en nuage PBX) des services de téléphonie dans Skype pour Business Online. 
  
Avant de commencer, vous devez examiner les conditions préalables décrites dans [Plan for Skype pour l’édition de connecteur Business Cloud](plan-skype-for-business-cloud-connector-edition.md).
  
> [!IMPORTANT]
> Les étapes de cette rubrique s'appliquent uniquement à la version 1.4.1 et suivantes de Cloud Connector. Si vous n'avez pas encore effectué la mise à niveau vers Cloud Connector Edition 2.1, consultez [Upgrade to a new version of Cloud Connector](upgrade-to-a-new-version-of-cloud-connector.md). Vous pouvez télécharger le fichier d’installation à partir de [https://aka.ms/CloudConnectorInstaller](https://aka.ms/CloudConnectorInstaller).

|[Deploy a single site in Cloud Connector](deploy-a-single-site-in-cloud-connector.md) <br/> |Créer un site dans votre déploiement de Cloud Connector.  <br/> |
|[Déployer plusieurs sites dans Cloud Connector](deploy-multiple-sites-in-cloud-connector.md) <br/> |Ajouter des sites à votre déploiement de la version CC et découvrir les différences entre les déploiement sur site unique et ceux sur plusieurs sites.  <br/> |
|[Configure Cloud Connector integration with your Office 365 tenant](configure-cloud-connector-integration-with-your-office-365-tenant.md) <br/> |Ajoutez des enregistrements DNS, configurez l'hybridité, organisez les passerelles PSTN et activez la messagerie vocale du système téléphonique d'Office 365 pour les utilisateurs.  <br/> |
