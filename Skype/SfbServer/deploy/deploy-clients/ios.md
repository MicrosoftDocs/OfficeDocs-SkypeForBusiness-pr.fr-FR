# <a name="install-and-test-skype-for-business-for-ios"></a>Installation et test de Skype Entreprise pour iOS
 
**Résumé :** Comment faire pour installer et tester le Skype pour Business application iOS.
  
Le Skype pour l’entreprise pour une application iOS apporte Skype pour entreprise présence, messagerie instantanée (MI) et appel vocal et vidéo pour les périphériques d’e/s. Les utilisateurs de Lync 2013 obtiendront l’application mise à jour automatiquement ou seront invités à faire la mise à jour manuellement, selon leurs paramètres utilisateur. Les nouveaux utilisateurs peuvent la télécharger sur l’App Store d’Apple. Le Skype pour l’entreprise pour une application iOS est uniquement disponible sur iOS, versions 8.0 et ultérieures.
  
## <a name="installing-skype-for-business-for-ios"></a>L’installation de Skype pour entreprise pour iOS

1. À partir de votre périphérique iOS, cliquez sur **App Store**et recherchez **Skype pour entreprise pour iOS**.
    
2. Cliquez sur **Télécharger** > **ouvert**. 
    
3. Connectez-vous à l’application.
    
## <a name="verifying-mobile-client-installation"></a>Vérification de l’installation du client mobile

Après que vous être connecté avec succès, effectuez les tests suivants pour vérifier que votre Skype pour l’installation de l’entreprise fonctionne correctement sur votre périphérique iOS. 
  
### <a name="search-for-a-contact-in-the-corporate-directory"></a>Recherche d’un contact dans l’annuaire d’entreprise

1. Dans la liste Contacts, tapez dans la barre de recherche affichée en haut et commencez à taper le nom d’un contact qui existe uniquement dans la liste d’adresses global. 
    
2. Vérifiez que le nom du contact figure dans les résultats de la recherche. 
    
### <a name="test-instant-messaging-and-presence"></a>Test de la messagerie instantanée et de la présence

1. Dans la liste Contacts, tapez sur un contact. 
    
2. Vérifiez qu’une fenêtre de messagerie instantanée apparaît et que vous pouvez taper et envoyer un message instantané. 
    
### <a name="test-dial-out-conferencing"></a>Test de la conférence rendez-vous

1. Dans Outlook, planifiez un Skype pour une réunion d’affaires. 
    
2. Sur l’appareil mobile, ouvrez l’invitation à la réunion. 
    
3. Cliquez sur le bouton **Rejoindre la réunion**.
    
4. Répondez à l’appel du service de conférence et vérifiez que vous êtes connecté au système audio de la réunion. 
    
### <a name="test-push-notifications"></a>Test des notifications Push

1. Sur le périphérique mobile de l’utilisateur A, connectez-vous à Skype pour entreprises avec le compte de l’utilisateur A. 
    
2. Ouvrez une autre application sur l’appareil mobile. 
    
3. Sur un autre client, connectez-vous à Skype pour entreprises avec le compte de l’utilisateur B. 
    
4. Envoyez un message instantané de l’utilisateur B à l’utilisateur A. 
    
5. Vérifiez que la notification de la messagerie instantanée s’affiche sur le périphérique mobile de l’utilisateur A.
