## <a name="configuring-skype-for-business-server-to-be-a-partner-application-for-exchange-server"></a><span data-ttu-id="04e36-113">Configuration de Skype Entreprise Server comme application partenaire pour Exchange Server</span><span class="sxs-lookup"><span data-stu-id="04e36-113">Configuring Skype for Business Server to be a Partner Application for Exchange Server</span></span>

<span data-ttu-id="04e36-118">Après avoir configuré l’application partenaire, il est recommandé d’arrêter et de redémarrer Internet Information Services (IIS) sur vos serveurs de boîtes aux lettres et d’accès au client Exchange.</span><span class="sxs-lookup"><span data-stu-id="04e36-118">After configuring the partner application it is recommended that you stop and restart Internet Information Services (IIS) on your Exchange mailbox and client access servers.</span></span> <span data-ttu-id="04e36-119">Vous pouvez redémarrer les services Internet (IIS) à l’aide d’une commande ressemblant à celle-ci, qui permet de redémarrer le service sur l’ordinateur atl-exchange-001 :</span><span class="sxs-lookup"><span data-stu-id="04e36-119">You can restart IIS by using a command similar to this, which restarts the service on the computer atl-exchange-001:</span></span>
