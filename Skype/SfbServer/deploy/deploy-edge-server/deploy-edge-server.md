# <a name="deploy-edge-server-in-skype-for-business-server"></a>Déployer le serveur Edge dans Skype Entreprise Server
 
**Résumé :** Découvrez comment déployer un serveur Edge ou un pool edge dans votre environnement Skype Entreprise Server.
  
Pourquoi déployer un serveur Edge ou un pool de serveurs Edge dans votre environnement Skype Entreprise Server ? Il est nécessaire si vous avez besoin d’utilisateurs externes qui ne sont pas connectés au réseau interne de votre organisation pour pouvoir interagir avec des utilisateurs internes. Ces utilisateurs externes peuvent être des utilisateurs distants authentifiés et anonymes, des partenaires fédérés ou d’autres clients mobiles.
  
## <a name="deployment-checklist-for-the-edge-for-skype-for-business-server"></a>Liste de vérification du déploiement pour Edge, pour Skype Entreprise Server

Comme indiqué ci-dessus, un déploiement de serveur Edge pour Skype Entreprise Server est important. Cette liste de vérification vous donne une vue d’ensemble des tâches que vous devez effectuer et fournit des liens vers des étapes plus détaillées.
  
Nous espérons que vous avez commencé dans la section Planifier les déploiements de serveurs [Edge dans Skype Entreprise Server.](../../plan-your-deployment/edge-server-deployments/edge-server-deployments.md) Si ce n’est pas le cas, la plupart des éléments que nous faisons référence y sont détaillés. La section déploiement contient uniquement des procédures. Par donc, si vous souhaitez connaître le raisonnement sous-fondé de ces étapes, la planification est l’endroit où commencer.
  
Cette documentation suppose également que vous avez déjà effectué le déploiement de base de Skype Entreprise Server.
