# <a name="deploy-edge-server-in-skype-for-business-server"></a>Déployer le serveur de périphérie dans Skype pour Business Server
 
**Résumé :** Découvrez comment déployer un serveur de périphérie ou un pool de serveurs Edge dans votre Skype pour un environnement de serveur d’entreprise.
  
Raisons pour lesquelles déployer un serveur de périphérie ou un pool de serveurs Edge dans votre Skype pour un environnement Business Server ? C’est nécessaire si vous avez besoin que des utilisateurs externes non connectés à votre réseau interne puissent interagir avec des utilisateurs internes. Ces utilisateurs externes peuvent être authentifiés comme des utilisateurs distants anonymes, des partenaires fédérés ou d’autres clients mobiles.
