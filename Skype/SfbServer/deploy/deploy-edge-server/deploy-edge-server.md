# <a name="deploy-edge-server-in-skype-for-business-server-2015"></a>Déploiement d’un serveur Edge dans Skype Entreprise Server 2015
 
**Résumé :** Découvrez comment déployer un serveur de périphérie ou un pool de serveurs Edge dans votre Skype pour environnement Business Server 2015.
  
Raisons pour lesquelles déployer un serveur de périphérie ou un pool de serveurs Edge dans votre Skype pour Business Server 2015 environnement ? C’est nécessaire si vous avez besoin que des utilisateurs externes non connectés à votre réseau interne puissent interagir avec des utilisateurs internes. Ces utilisateurs externes peuvent être authentifiés comme des utilisateurs distants anonymes, des partenaires fédérés ou d’autres clients mobiles.
