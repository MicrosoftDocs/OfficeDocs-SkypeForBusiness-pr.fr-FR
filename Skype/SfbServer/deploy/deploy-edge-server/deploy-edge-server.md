# <a name="deploy-edge-server-in-skype-for-business-server-2015"></a>Déploiement d’un serveur Edge dans Skype Entreprise Server 2015
 
**Résumé :** Découvrez comment déployer un serveur de transport Edge ou un pool d’arête dans votre Skype pour environnement d’entreprise serveur 2015.
  
Pourquoi déployer un serveur de transport Edge ou un pool d’arête dans votre Skype pour Business Server 2015 environnement ? C’est nécessaire si vous avez besoin que des utilisateurs externes non connectés à votre réseau interne puissent interagir avec des utilisateurs internes. Ces utilisateurs externes peuvent être authentifiés comme des utilisateurs distants anonymes, des partenaires fédérés ou d’autres clients mobiles.

Nous espérons que vous avez commencé dans la section [planifier les déploiements de serveur de transport Edge dans Skype pour Business Server 2015](../../plan-your-deployment/edge-server-deployments/edge-server-deployments.md) .
