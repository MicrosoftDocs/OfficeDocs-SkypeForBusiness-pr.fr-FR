# <a name="optional-verify-response-group-deployment-in-skype-for-business"></a>Module Vérifier le déploiement de Response Group dans Skype entreprise
 
Vérifiez que votre déploiement Response Group réussit, dans Skype entreprise Server Voice voix.
  
Après avoir configuré Response Group, vous devez vérifier la configuration pour vous assurer que les groupes Response Group fonctionnent comme prévu. Au minimum, vérifiez les scénarios suivants en vous basant sur les types d’utilisateur indiqués ci-dessous :
  
 **Utilisateurs**
  
- Un utilisateur hébergé sur Skype entreprise
    
- Utilisateur externe qui utilise le réseau téléphonique commuté (PSTN) public
    
- Un agent hébergé sur Skype entreprise
    
  **Scenarios**
  
- L’utilisateur de Skype entreprise appelle le groupe Response Group.
    
- L’utilisateur externe appelle le service Response Group.
    
- Un utilisateur appelle le service Response Group alors que l’agent traite un autre appel. L’appel de l’utilisateur est alors transmis à la file d’attente.

    > [!NOTE]
    > Si le groupe Response Group ne fonctionne pas comme prévu, vérifiez le lien suivant :https://support.office.com/article/troubleshooting-for-response-groups-ca72d8f8-4054-4974-b832-4f173611bd89
