# <a name="optional-verify-response-group-deployment-in-skype-for-business"></a>Facultatif Vérifier le déploiement de Response Group dans Skype entreprise
 
Confirmez la réussite du déploiement de votre groupe de réponses dans Skype entreprise Server Voice.
  
Après avoir configuré Response Group, vous devez vérifier la configuration pour vous assurer que les groupes de réponse fonctionnent comme prévu. Au minimum, vérifiez les scénarios ci-dessous en vous basant sur les types d’utilisateur indiqués ci-dessous :
  
 **Utilisateurs**
  
- Utilisateurs hébergés sur Skype entreprise
    
- Utilisateur externe utilisant le réseau téléphonique commuté (RTC)
    
- Un agent hébergé sur Skype entreprise
    
  **Scénarios**
  
- L’utilisateur de Skype entreprise appelle le groupe réponse.
    
- L’utilisateur externe appelle le service Response Group.
    
- Un utilisateur appelle le service Response Group pendant que l’agent traite un autre appel. L’appel de l’utilisateur est alors transmis à la file d’attente.

    > [!NOTE]
    > Si le groupe réponse ne fonctionne pas comme prévu, consultez le lien suivant:https://support.office.com/en-us/article/troubleshooting-for-response-groups-ca72d8f8-4054-4974-b832-4f173611bd89
