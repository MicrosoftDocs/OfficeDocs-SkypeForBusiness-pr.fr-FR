# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a>(Facultatif) Vérifier le déploiement des annonces dans Skype Entreprise
 
Vérification de votre déploiement de l’application Annonce dans Skype Entreprise Server Voix Entreprise. Cela affecte la façon dont les appels vers des numéros non affectés sont gérés.
  
Une fois que vous avez installé et configuré le service Annonce, vérifiez sa configuration pour vous assurer que les appels passés aux numéros non attribués fonctionneront comme prévu. Vérifiez au minimum les éléments suivants :
  
- Appelez un numéro valide pour votre organisation, mais qui n’est pas un numéro non signé.
    
- Appelez le numéro non attribué et vérifiez que l’annonce appropriée est lue.
