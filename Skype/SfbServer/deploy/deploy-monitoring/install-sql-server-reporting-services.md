# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>Installer SQL Server Reporting Services dans Skype Entreprise Server 
 
**Résumé :** Découvrez où trouver des informations sur les SQL Server Reporting Services utilisées par Skype Entreprise Server.
  
Skype Entreprise Server pouvez utiliser SQL Server Reporting Services (SSRS) pour l’affichage et la surveillance des rapports. Pour utiliser cette fonctionnalité, reporting Services doit être installé et configuré.
  
## <a name="install-sql-server-reporting-services"></a>Installer SQL Server Reporting Services

Si vous avez l’intention d’utiliser Skype Entreprise Server monitoring reports (voir l’article suivant de cette documentation pour plus d’informations), vous devez d’abord installer SQL Server Reporting Services ; Reporting Services peut être installé en même temps que vous installez Microsoft SQL Server ou à tout moment après SQL Server’installation. Vous pouvez le faire lors de l’installation de Microsoft SQL Server ou à tout moment après l’installation de SQL Server. Si vous n’avez pas installé SQL Server, suivez les instructions fournies plus haut dans cette documentation. Lorsque vous installez SQL Server, dans la page Sélection des composants, veillez à sélectionner Reporting Services pour installer SQL Server Reporting Services.
  
Pour savoir comment installer SQL Server Reporting Services, voir [SQL Server Reporting Services (SSRS).](/sql/reporting-services/create-deploy-and-manage-mobile-and-paginated-reports)
  
Si vous avez déjà installé SQL Server mais pas SQL Server Reporting Services vous pouvez ajouter cette fonctionnalité en suivant l’ensemble approprié d’instructions pour SQL Server Reporting Services.
