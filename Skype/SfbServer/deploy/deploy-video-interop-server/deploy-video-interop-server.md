# <a name="deploy-video-interop-server-in-skype-for-business-server-2015"></a>Déployer le serveur d’interopérabilité vidéo dans Skype Entreprise Server 2015
 
**Résumé :** Déployer le rôle de serveur de VIS dans Skype pour Business Server 2015.
  
Skype pour Business Server peut désormais intégrer directement aux systèmes de téléconférence (VTCs) Cisco comme Cisco C60 ou Cisco MX300. Cela requiert l’introduction d’un nouveau rôle de serveur appelée serveur d’interopérabilité vidéo (VIS) et de la configuration correcte de la VIS et l’équipement qu’il fonctionnera avec. Un VTC s’inscrit auprès de l’infrastructure Cisco existante, telle que le Cisco Unified Communication Manager (CUCM), et une jonction SIP vidéo est utilisée entre le CUCM et le pool VIS.
  
## <a name="in-this-section"></a>Contenu de cette section

La configuration de l’interopérabilité entre un serveur ou un pool VIS et des systèmes VTC requiert la réalisation des cinq procédures suivantes : 
  
- [Créer un pool de VIS dans Skype Business Server 2015](create-a-vis-pool.md)
    
- [Déployer le rôle de serveur de VIS dans Skype pour Business Server 2015](deploy-the-vis-server-role.md)
    
- [Configurer le serveur d’interopérabilité vidéo dans Skype pour Business Server 2015](configure-the-vis.md)
    
- [Configurer CUCM pour l’interopérabilité avec Skype pour Business Server 2015](configure-cucm-for-interoperation.md)
    
- [Configurer un VTC pour l’interopérabilité avec Skype pour Business Server 2015](configure-a-vtc-for-interoperation.md)
