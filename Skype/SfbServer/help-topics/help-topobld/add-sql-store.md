# <a name="add-sql-store"></a>Ajouter banque de SQL
 
Pour définir un nouveau magasin SQL, ce qui signifie que vous spécifiez une base de données SQL Server et une instance de SQL Server, soit une instance par défaut ou une instance nommée, vous spécifiez les options suivantes.
  
Spécifiez le nom de domaine pleinement qualifié (FQDN) de le SQL Server qui hébergera l’instance de base de données que vous définissez.
  
Spécifiez l’instance de SQL Server qui héberge les données. Vous pouvez spécifier l’instance par défaut, ou vous pouvez spécifier une instance nommée.
  
Colocalisation des bases de données dans des instances spécifiques doit être très clairement comprise. Pour plus d’informations sur la colocalisation de serveurs et de colocalisation d’instance de base de données, consultez [Colocalisation de serveurs dans un déploiement de Pool fin avant](http://technet.microsoft.com/library/0516b18d-14c0-4237-9279-0f92e341b1bd.aspx) et [Colocalisation de serveurs dans un déploiement Standard Edition Server](http://technet.microsoft.com/library/0763ffab-4fd6-463a-8e62-d97876b376d3.aspx).
