# <a name="edge-next-hop-settings-expander"></a>Expanseur des paramètres du tronçon suivant du serveur Edge

Pour définir ou modifier le **pool de tronçon suivant** que les serveurs Edge utiliseront pour communiquer avec l’infrastructure interne, sélectionnez un directeur, un pool Directeur, un serveur frontal ou un pool de serveurs frontaux à partir de la liste. Seuls les directeurs ou les serveurs frontaux qui ont été configurés dans le générateur de topologie apparaissent dans la liste.

Pour plus d’informations sur la définition et la configuration des paramètres des services Edge, voir [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)
