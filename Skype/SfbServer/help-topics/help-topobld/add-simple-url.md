# <a name="add-simple-url"></a>Ajouter une URL Simple
 
Pour ajouter une URL simple supplémentaires à la définition du déploiement, spécifiez l’URL de la nouvelle URL simple. Le format de l’URL est _http://\<type d’URL simple\>.\< nom de domaine\> _ ou _http://\<nom de domaine\>/\<type d’URL simple\>_, où le type d’URL simple est une des opérations suivantes :
  
- Meet
    
- Effectuer un appel entrant
    
> [!IMPORTANT]
> Vous pouvez disposer d’un accès dans une URL simple et une URL simple respect active à tout moment. Ajout d’une nouvelle URL simple et rend l’URL active définit toutes les demandes futures à utiliser la nouvelle URL simple. Vous laissez l’URL simple inactive pour gérer des conférences et autres demandes qui ont été apportées lorsque l’ancienne URL simple était actif. 
  
Pour plus d’informations sur l’ajout d’une URL simple, reportez-vous à la section [configuration DNS requise pour simples URL dans Skype pour Business Server 2015](../../plan-your-deployment/network-requirements/simple-urls.md).
