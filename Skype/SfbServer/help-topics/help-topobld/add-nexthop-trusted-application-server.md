# <a name="add-nexthop-trusted-application-server"></a>Ajouter le serveur d’Application de confiance NextHop
 
Pour définir le saut suivant pour le pool ou le serveur d’Applications de confiance, effectuez les opérations suivantes :
  
Dans la liste déroulante **pool du tronçon suivant** , sélectionnez le pool Enterprise Edition ou Standard Edition serveur frontal que vous voulez que le pool d’applications de confiance à utiliser comme un tronçon suivant.
  
Cliquez sur **Terminer** pour achever la création de pool d’Application sécurisée.
