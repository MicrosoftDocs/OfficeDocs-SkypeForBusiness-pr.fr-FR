# <a name="define-the-ip-address-of-the-ip-or-pstn-gateway"></a>Définir l’adresse IP de l’adresse IP ou une passerelle PSTN
 
Vous définissez les adresses IPv4 ou IPv6 pour la passerelle IP ou téléphone téléphonique commuté network (PSTN) en sélectionnant **Activer un IPv4** ou **IPv6 d’activer**. Vous pouvez choisir **d’utiliser toutes les adresses IP configurées** ou **limiter l’utilisation de service aux adresses IP sélectionnées**.
  
Si vous sélectionnez **utiliser toutes les adresses IP configurées**, toutes les adresses configurées sur la passerelle va être utilisés. Sélectionnez éventuellement **limiter l’utilisation de service pour les adresses IP sélectionnées**, ce qui permet l’entrée de **l’adresse IP du réseau RTPC** . Entrez l’adresse IPv4 ou IPv6 qui a été attribué pour cette passerelle.
