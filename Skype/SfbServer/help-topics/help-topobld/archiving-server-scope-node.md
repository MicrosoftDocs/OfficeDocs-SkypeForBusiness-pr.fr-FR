# <a name="archiving-server-scope-node"></a>Nœud d’étendue du serveur d’archivage
 
Dans le générateur de topologies, vous pouvez effectuer des tâches liées au serveur exécutant l’archivage en cliquant avec le bouton droit sur le pool dans l’arborescence de la console, puis en sélectionnant **modifier les propriétés**. À partir de la page **modifier les propriétés** , vous pouvez modifier le nom de domaine complet (FQDN) du serveur exécutant l’archivage, la base de données SQL Server d’archivage et le magasin de fichiers de l’archivage.
