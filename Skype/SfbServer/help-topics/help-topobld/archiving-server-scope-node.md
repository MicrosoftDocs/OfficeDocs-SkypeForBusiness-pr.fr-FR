# <a name="archiving-server-scope-node"></a>Nœud d’étendue du serveur d’archivage
 
Dans le Générateur de topologie, vous pouvez effectuer des tâches relatives au serveur exécutant l’archivage en cliquant avec le bouton droit sur le pool dans l’arborescence de la console, puis en sélectionnant **Modifier les propriétés.** Dans la **page** Modifier les propriétés, vous pouvez modifier le nom de domaine complet (FQDN) du serveur exécutant l’archivage, la base de données SQL Server d’archivage et le magasin de fichiers d’archivage.
