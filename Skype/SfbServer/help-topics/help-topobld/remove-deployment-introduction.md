# <a name="remove-deployment-introduction"></a>Assistant Suppression du déploiement
 
Exécuter l' **Assistant Suppression du déploiement** uniquement si vous souhaitez retirer de tout votre déploiement. Cet Assistant crée une topologie vide qui se compose d’un seul serveur frontal. Une fois que vous exécutez cet Assistant et publiez la topologie vide, vous ne pouvez pas revenir à votre topologie d’origine.
  
> [!IMPORTANT]
> Avant de cliquer sur **suivant**, veillez à supprimer tous les ordinateurs inactifs, de désactiver tous les utilisateurs et les systèmes d’extrémité de communications unifiées et supprimer tous les annuaires des conférences comme spécifié dans la première page de l’Assistant.
