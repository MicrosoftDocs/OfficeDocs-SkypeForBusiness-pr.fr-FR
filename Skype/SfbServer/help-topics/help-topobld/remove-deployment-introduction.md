# <a name="remove-deployment-introduction"></a>Supprimer l’Introduction du déploiement
 
Exécutez l' **Assistant de déploiement de suppression** uniquement si vous souhaitez retirer tout votre déploiement. Cet Assistant crée une topologie vide qui se compose d’un seul serveur frontal. Après avoir exécuté cet Assistant et que vous publiez la topologie vide, vous ne pouvez pas revenir à votre topologie d’origine.
  
> [!IMPORTANT]
> Avant de cliquer sur **suivant**, veillez à supprimer tous les ordinateurs inactifs, désactiver tous les utilisateurs et les points de terminaison des communications unifiées et supprimer tous les répertoires de conférence comme spécifié sur la première page de l’Assistant.
