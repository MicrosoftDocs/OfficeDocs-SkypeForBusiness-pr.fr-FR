# <a name="av-mcu-settings-expander"></a>Développement de paramètres AV MCU
 
Pour modifier les propriétés d’un A / pool V Conferencing Server que vous avez déjà définies, effectuez les opérations suivantes :
  
## <a name="general-settings"></a>Paramètres généraux

Mettre à jour le nom de domaine complet (FQDN) du A / V Conferencing Server pool, tapez le nom du pool tel qu’il est défini dans le système DNS (Domain Name System). Vous devez disposer d’un enregistrement d’hôte (A) DNS qui correspond à l’A / V Conferencing Server nom du pool.
  
## <a name="av-conferencing-service"></a>A / V Conferencing Service

Pour associer un A / pool de V Conferencing Server, sélectionnez le pool de Front-End ou des pools qui vous devez associer le A / pool V Conferencing Server.
  
> [!NOTE]
> Si aucun pool de Front-End n’est affichés dans la **Sélectionnez les pools de Front-End à associer à ce A / pool de V Conferencing** volet, vous avez colocalisés soit de l’un / V Conferencing Server avec le pool frontal, ou vous n’avez pas sélectionné ajouter conférence dans les fonctionnalités de ** fonctionnalités et** section des propriétés de pool du Front-End.
  
## 

Lorsque vous avez terminé, cliquez sur **OK** pour terminer la modification du A / V Conferencing Server pool des paramètres. Cliquez sur **Annuler** pour annuler les modifications.
