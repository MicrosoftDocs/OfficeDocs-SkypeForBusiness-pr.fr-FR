# <a name="install-mirror-database-option-page"></a>Page des options Installer une base de données miroir
 
Vous configurez **Paramètres de base de données miroir** en définissant les éléments suivants :
  
- Tapez le **chemin d’accès au partage de fichiers** pour définir l’emplacement des fichiers SQL Server de sauvegarde pour la base de données mise en miroir.
    
    > [!NOTE]
    > L’instance SQL Server principale (instance nommée ou instance par défaut) doit disposer d’autorisations d’écriture pour le partage de fichiers que vous définissez ici. L’instance de SQL Server miroir (instance nommée ou instance par défaut) doit disposer d’autorisations de lecture sur le même partage de fichiers. 
  
  **OK** permet d’accepter et de valider les modifications de la boîte de dialogue.
  
  **Annuler** permet d’annuler les modifications et de fermer la boîte de dialogue.
  
  **Aide** permet d’afficher cet écran d’aide.
