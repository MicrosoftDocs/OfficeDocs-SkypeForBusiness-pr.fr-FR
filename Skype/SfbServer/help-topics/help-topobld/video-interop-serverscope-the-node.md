# <a name="video-interop-server---scope-the-node"></a>Serveur d’interopérabilité vidéo - Étendue du nœud
 
La procédure à suivre pour définir une nouvelle passerelle vidéo dans le générateur de topologie est décrite en ligne dans [la rubrique Create a vis pool in Skype for Business Server 2015](../../deploy/deploy-video-interop-server/create-a-vis-pool.md).
  
Les étapes nécessaires à l’attribution d’un rôle de serveur d’interopérabilité vidéo sur un serveur à l’aide de Windows PowerShell sont décrites en ligne dans [Deploy the vis Server Role in Skype for Business server 2015](../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md)
  
Les étapes de configuration d’un nouveau serveur d’interopérabilité vidéo utilisant Windows PowerShell sont décrites en ligne à [la rubrique Configure the Video Interop Server in Skype for Business server 2015](../../deploy/deploy-video-interop-server/configure-the-vis.md).
  
 Les paramètres d’une passerelle vidéo qui doivent être définis dans l’interface utilisateur graphique (GUI) de Cisco pour l’interopérabilité avec une VIS sont décrits en ligne à l’adresse [configurer CUCM pour l’interopérabilité avec Skype entreprise Server 2015](../../deploy/deploy-video-interop-server/configure-cucm-for-interoperation.md).
  
 Les paramètres d’un VTC qui doivent être définis dans l’interface utilisateur graphique (GUI) de Cisco pour l’interopérabilité avec une passerelle vidéo et un VIS sont décrits en ligne dans la rubrique [Configure a VTC for interopérabilité with Skype for Business Server 2015](../../deploy/deploy-video-interop-server/configure-a-vtc-for-interoperation.md).
