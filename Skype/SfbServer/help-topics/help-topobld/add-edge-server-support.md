# <a name="add-edge-server-support"></a>Ajouter la prise en charge côté serveur
 
Vous pouvez ajouter la prise en charge de l’accès des utilisateurs externes lorsque vous déployez votre topologie initiale ou à une date ultérieure. Pour plus d’informations sur l’ajout de serveurs Edge à une topologie existante, consultez [Définition de votre topologie de bord](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) dans la documentation de déploiement de serveur Edge.
