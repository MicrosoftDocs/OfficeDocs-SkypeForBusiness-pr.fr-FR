# <a name="add-mediation-server-nexthop"></a>Ajouter un tronçon suivant pour le serveur de médiation
 
Pour définir le tronçon suivant d’un serveur de médiation que vous définissez, sélectionnez dans la liste déroulante **pool du tronçon suivant** le serveur Enterprise Edition ou le serveur Standard Edition qui fera office de tronçon suivant pour le serveur de médiation. Cliquez sur **Suivant** pour continuer.
