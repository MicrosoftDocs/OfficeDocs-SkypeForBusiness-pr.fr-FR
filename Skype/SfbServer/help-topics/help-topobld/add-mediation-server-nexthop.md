# <a name="add-mediation-server-nexthop"></a>Ajouter un tronçon suivant pour le serveur de médiation
 
Pour définir le tronçon suivant d’un serveur de médiation que vous définissez, faites votre choix dans la liste déroulante de la liste déroulante de l' **ensemble des tronçons** du serveur Enterprise Edition Server ou Standard Edition Server qui fera office de tronçon suivant pour le serveur de médiation. Cliquez sur **Suivant** pour continuer.
