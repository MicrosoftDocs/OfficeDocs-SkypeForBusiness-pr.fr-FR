# <a name="legacy-merge-landing"></a>Lancement de fusion hérités
 
La page **Spécifier les paramètres de bord** vous permet de configurer vos informations de serveur de transport Edge héritées. Cliquez sur **Ajouter** pour spécifier la version de serveur de transport Edge héritée de votre déploiement d’Office Communications Server. Si vous ne disposez pas de tous les serveurs Edge dans votre environnement existant, cliquez sur **suivant** pour continuer.
