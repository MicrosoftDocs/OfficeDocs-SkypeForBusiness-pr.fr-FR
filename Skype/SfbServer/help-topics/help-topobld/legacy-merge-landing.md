# <a name="legacy-merge-landing"></a>Fusion héritée : accueil
 
La page **spécifier la configuration Edge** vous permet de configurer les informations de votre serveur Edge hérité. Cliquez sur **Ajouter** pour spécifier la version du serveur Edge hérité de votre déploiement Office Communications Server. Si vous n’avez aucun serveur Edge dans votre environnement hérité, cliquez sur **suivant** pour continuer.
