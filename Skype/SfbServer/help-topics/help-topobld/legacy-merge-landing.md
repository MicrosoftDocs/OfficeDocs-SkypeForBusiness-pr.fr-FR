# <a name="legacy-merge-landing"></a>Fusion héritée : accueil
 
La page **Spécifiez la configuration du bord** vous permet de configurer les informations de votre serveur de bord antérieur. Cliquez sur **Ajouter** pour spécifier la version serveur de périphérie hérité de votre déploiement d’Office Communications Server. Si vous n’avez pas de serveur Edge dans votre environnement hérité, cliquez sur **suivant** pour continuer.
