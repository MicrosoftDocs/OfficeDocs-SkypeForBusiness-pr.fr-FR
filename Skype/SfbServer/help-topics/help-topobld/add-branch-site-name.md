# <a name="add-branch-site-name"></a>Ajouter le nom du site de succursale

Pour définir un nouveau site de succursale, spécifiez les éléments suivants :

Un nom pour le nouveau site de succursale. Le nom choisi doit être unique dans votre déploiement. Vous pouvez utiliser la convention d’affectation de noms de votre choix. En règle générale, les recommandations préconisent de mentionner la ville, le nom d’emplacement et le nom de succursale. La valeur Nom doit être renseignée.

Le champ description permet, si vous le souhaitez, d’ajouter d’autres informations ne faisant pas partie du nom de l’infrastructure, mais qui présentent votre site plus en détail.

Pour plus d’informations, reportez-vous à la rubrique [Sites](https://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx).
