# <a name="publish-topology-reset-pool-page"></a>Publier la topologie : page Réinitialiser les pools
 
Cette documentation est préliminaire et sujette à modification. Des rubriques vides sont incluses comme espaces réservés.
  
Pour ajouter de nouveaux ordinateurs à un pool existant, vous devez réinitialiser le pool pour permettre au Windows Fabric de redéfinir les connexions entre tous les membres du pool. Pour ce faire, sélectionnez le pool à qui vous avez ajouté des serveurs, puis cliquez sur **Suivant**.
