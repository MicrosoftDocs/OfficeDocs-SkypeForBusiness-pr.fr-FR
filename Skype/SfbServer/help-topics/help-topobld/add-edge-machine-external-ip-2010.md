# <a name="add-edge-machine-external-ip-2010"></a>Ajouter l’adresse IP 2010 externe de l’ordinateur Edge

Vous pouvez configurer les propriétés sur cette page pour faire pointer le serveur de périphérie vers les emplacements externes du service Edge d’accès, du service Edge de conférence Web et du service Edge A/V.

- Sous **adresses IPv4 externes**, tapez les adresses IPv4 externes du serveur Edge dans **service Edge d’accès**, **service Edge de conférence Web**et **service Edge A/V**.

- Cliquez sur **OK** pour enregistrer et valider les modifications apportées au document topologique.

- Cliquez sur **Annuler** pour ignorer vos modifications et fermer l’Assistant **Ajouter un serveur au pool de périphériques** .

- Cliquez sur **aide** pour lire cette rubrique d’aide.

Pour plus d’informations sur l’ajout de serveurs Edge à une topologie existante, voir [définir votre topologie de bord](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Pour plus d’informations sur la migration des serveurs Edge, voir [Phase 5 : Ajouter Lync server 2013 Preview Edge Server au pool de pilotes](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
