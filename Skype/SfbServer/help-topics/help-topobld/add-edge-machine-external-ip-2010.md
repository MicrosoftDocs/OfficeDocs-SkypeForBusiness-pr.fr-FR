# <a name="add-edge-machine-external-ip-2010"></a>Ajouter IP externe de l’ordinateur de bord 2010
 
Vous pouvez configurer les propriétés de cette page pour pointer le serveur de transport Edge pour les emplacements externes du service Edge d’accès, le service Web Conferencing Edge et A / V Edge service.
  
- Sous **adresses IPv4 externe**, tapez les adresses IPv4 externes du serveur Edge dans le **service Edge d’accès**, **le service Web Conferencing Edge**, et **A / V Edge service**.
    
- Cliquez sur **OK** pour enregistrer et valider vos modifications dans le document de la topologie.
    
- Cliquez sur **Annuler** pour ignorer vos modifications et fermer l’Assistant **Ajouter un serveur au Pool de bord** .
    
- Cliquez sur **aide** pour lire cette rubrique d’aide.
    
Pour plus d’informations sur l’ajout de serveurs Edge à une topologie existante, consultez [Définition de votre topologie de bord](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Pour plus d’informations sur la migration de serveurs Edge, consultez [Phase 5 : ajouter Lync Server 2013 aperçu serveur Edge au Pool de pilote](http://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
