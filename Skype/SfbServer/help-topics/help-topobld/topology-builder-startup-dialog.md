# <a name="topology-builder-startup-dialog"></a>Boîte de dialogue de démarrage du Générateur de topologie
 
Vous démarrez le Générateur de topologies pour configurer les propriétés et la configuration de votre déploiement Microsoft Lync Server 2013. Au démarrage, le Générateur de topologies affiche une boîte de dialogue qui vous demande ce que sera la source du document de la topologie. Vous sélectionnez une des trois options :
  
- **Télécharger la topologie de déploiement existante** Sélectionnez cette option si vous avez exécuté le Générateur de topologies et que vous avez publié votre topologie sur le serveur d’administration centrale. Cette option demande et télécharge une copie locale de la configuration actuelle, ce qui vous permet de modifier et de configurer le déploiement existant.
    
- **Ouvrir la topologie à partir d’un fichier local** Sélectionnez cette option si vous configurez un déploiement mais que vous n’avez pas encore publié la configuration. Cette option vous permet d’enregistrer une copie locale de la tâche de configuration dans le Générateur de topologie et d’y revenir ultérieurement.
    
- **Nouvelle topologie** Sélectionnez cette option pour lancer la configuration d’un nouveau document de topologie. Cette option ouvre un nouveau document de topologie que vous configurez pour la configuration requise pour le déploiement qui vous permettra de configurer.
