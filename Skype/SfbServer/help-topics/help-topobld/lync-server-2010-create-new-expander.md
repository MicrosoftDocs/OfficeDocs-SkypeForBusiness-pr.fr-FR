# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 : créer un expandeur
 
Vous pouvez créer des rôles de serveur dans le nœud Lync Server 2010 du générateur de topologie en cliquant avec le bouton droit sur **Lync server 2010**, sélectionnez **nouveau**, puis sélectionnez le rôle de serveur à créer.
