# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 : créer un expandeur
 
Vous pouvez créer de nouveaux rôles serveur dans le nœud Lync Server 2010 du Générateur de topologie en cliquant avec le bouton droit sur **Lync Server 2010,** sélectionnez **Nouveau,** puis sélectionnez le rôle serveur à créer.
