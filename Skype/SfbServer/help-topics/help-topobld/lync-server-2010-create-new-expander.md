# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 créer nouveau Expander
 
Vous pouvez créer de nouveaux rôles de serveur dans le nœud de Lync Server 2010 de générateur de topologie en cliquant avec le bouton droit sur **Lync Server 2010**, sélectionnez **Nouveau**, puis sélectionnez le rôle de serveur à créer.
