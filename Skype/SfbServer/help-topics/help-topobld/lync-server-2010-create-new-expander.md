# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 : créer un expandeur
 
Vous pouvez créer de nouveaux rôles de serveur dans le nœud du Générateur de topologies Lync Server 2010 en cliquant avec le bouton droit sur **Lync Server 2010**, sélectionnez **Nouveau**, puis sélectionnez le rôle de serveur à créer.
