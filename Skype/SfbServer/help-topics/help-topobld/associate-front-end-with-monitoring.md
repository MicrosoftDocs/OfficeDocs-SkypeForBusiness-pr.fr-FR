# <a name="associate-front-end-with-monitoring"></a>Associer un pool frontal à un serveur de surveillance
 
Chaque pool frontal et Survivable Branch Appliance ne peut être associé qu’à un seul serveur de surveillance. Lorsque la surveillance est activée pour le site, le serveur de surveillance fournit la collecte et le rapport des données de qualité de l’expérience (QoE) et d’enregistrement des détails.
  
Tous les pools d’un site et les pools de sites centraux multiples peuvent utiliser le même serveur de surveillance, si l’utilisation ne dépasse pas la capacité du serveur de surveillance.
