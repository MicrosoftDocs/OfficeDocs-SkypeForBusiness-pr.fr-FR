# <a name="associate-front-end-with-monitoring"></a>Associer l’analyse de Front-End
 
Chaque pool de Front-End et le Survivable Branch Appliance peuvent avoir qu’un seul que serveur de surveillance qui lui sont associé. Lorsque l’analyse est activée pour le site, Monitoring Server fournit les détails de l’appel d’enregistrement (CDR) et de collecte de données de qualité d’expérience (QoE) et de reporting.
  
Tous les pools sur un site et les pools de plusieurs sites central peuvent utiliser le même serveur de surveillance, si l’utilisation ne dépasse pas la capacité du Monitoring Server.
