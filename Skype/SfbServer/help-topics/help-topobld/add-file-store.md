# <a name="add-file-store"></a>Ajouter un magasin de fichiers
 
Pour définir un nouveau magasin de fichiers, spécifiez les éléments suivants :
  
- Le nom de domaine complet (FQDN) du serveur qui hébergera le nouveau magasin de fichiers. Nom de domaine complet du serveur doit exister dans le système DNS (Domain Name System) avant de publier la topologie.
    
- Le nom du partage de fichier sur le serveur hôte. Le partage de fichiers doit exister avant de publier la topologie.
