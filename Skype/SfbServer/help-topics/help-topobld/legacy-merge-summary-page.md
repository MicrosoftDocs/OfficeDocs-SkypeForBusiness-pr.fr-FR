# <a name="legacy-merge-summary-page"></a>Page de résumé de la fusion héritée

La migration de Microsoft Office Communications Server 2007 R2 vers Lync Server 2013 utilise les fonctionnalités de générateur de topologie pour fusionner les topologies existantes. L’écran de synthèse spécifie toutes les actions et sélections que vous avez apportées lors de la configuration de la fusion du serveur Office Communications Server 2007 R2 dans la configuration du générateur de topologie Lync Server 2013.

Passez en revue le résumé et déterminez si vos sélections ont été correctement définies. Procédez à la fusion si vous êtes satisfait des sélections réfléchies dans le résumé.
