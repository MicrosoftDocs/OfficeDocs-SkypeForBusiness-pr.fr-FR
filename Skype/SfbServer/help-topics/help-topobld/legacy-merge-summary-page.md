# <a name="legacy-merge-summary-page"></a>Page de résumé de fusion hérités
 
Migration à partir de Microsoft Office Communications Server 2007 R2 vers Lync Server 2013 utilise les fonctionnalités du Générateur de topologies pour fusionner les topologies existantes. L’écran Résumé indique toutes les actions et les sélections que vous avez apportées au cours de la configuration de la fusion de l’Office Communications Server 2007 R2 dans la configuration de générateur de topologie Lync Server 2013.
  
Passez en revue le résumé et déterminer si vos sélections ont été correctement définies. Procéder à la fusion, si vous avez effectué les sélections apparaissent dans le résumé.
