# <a name="publish-topology-select-cms-page"></a>Publier la topologie : page Sélectionner un CMS

Vous publiez la topologie que vous avez configurée à l’aide du générateur de topologies. Vous êtes invité à sélectionner dans une liste le serveur frontal ou le pool frontal qui assumera le rôle de maintien du magasin central de gestion. Un seul serveur frontal ou pool frontal peut conserver ce rôle à un moment donné.

Pour plus d’informations sur le magasin central de gestion, reportez-vous [à la rubrique Move the Lync server 2010 Central Management Server to Lync Server 2013](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx).
