# <a name="add-front-end-associations-2010"></a>Ajouter des Associations de Front-End 2010
 
Vous pouvez activer la prise en charge des fonctionnalités spécifiques qui nécessitent le déploiement des autres serveurs en associant les rôles de serveur avec le pool frontal pour l’instant. Vous pouvez également associer les rôles serveur et le pool frontal par la suite. Les rôles serveur pouvant être associés à un pool frontal peuvent être les suivants :
  
1. Serveur d’archivage. Pour plus d’informations sur la planification et la mise en œuvre de l’archivage de prise en charge, consultez [définir vos exigences de l’organisation pour l’archivage](http://technet.microsoft.com/library/ce0fc0f6-7704-4b80-bf19-a1fa9818fc7a.aspx) dans la documentation de planification et [définir et configurer un serveur Standard Edition Server ou un Pool de fin avant](http://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) et [déploiement L’archivage](http://technet.microsoft.com/library/a89edd16-12d5-4602-ad2f-194b47d1188e.aspx) dans la documentation de déploiement.
    
2. Serveur d’analyse. Pour plus d’informations sur la mise en œuvre de la prise en charge de surveillance, consultez [planification de surveillance](http://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx) dans la documentation de planification et [définir et configurer un serveur Standard Edition Server ou un Pool de fin avant](http://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) et le [Déploiement de surveillance](http://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) dans le déploiement documentation.
    
3. A / V Edge service. Pour plus d’informations sur l’implémentation d’un A / V Edge service, consultez la [Vue d’ensemble de l’accès des utilisateurs externes](http://technet.microsoft.com/library/97aded6c-5fa3-4225-95a6-9ad094d61654.aspx).
