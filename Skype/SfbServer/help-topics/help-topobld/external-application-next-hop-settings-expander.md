# <a name="external-application-next-hop-settings-expander"></a>Développeur des paramètres du tronçon suivant d’une application externe
 
Vous pouvez spécifier le serveur du tronçon suivant du pool serveur d’applications approuvées en sélectionnant le pool frontal Enterprise Edition ou Standard Edition serveur frontal dans la liste déroulante. Un directeur ou un directeur pool n’est pas valide pour un Application approuvée tronçon suivant du serveur, et il n’apparaît pas dans la liste. Cliquez sur **OK** pour accepter et enregistrer vos modifications. Cliquez sur **Annuler** pour annuler les modifications et quitter la page des propriétés.
