# <a name="external-application-next-hop-settings-expander"></a>Expanseur des paramètres du tronçon suivant d’une application externe
 
Vous pouvez spécifier le serveur du tronçon suivant du pool de serveurs d’applications approuvées en sélectionnant le pool frontal Enterprise Edition défini ou le serveur frontal Standard Edition dans la liste déroulante. Un directeur ou un pool de directeurs n’est pas un choix valide pour un tronçon suivant de serveur d’applications approuvées et n’apparaîtra pas dans la liste. Cliquez sur **OK** pour accepter et enregistrer vos modifications. Cliquez sur **Annuler** pour annuler les modifications et quitter la page des propriétés.
