# <a name="external-application-next-hop-settings-expander"></a>Application externe prochain tronçon paramètres Expander
 
Vous pouvez spécifier le serveur du tronçon suivant du pool de serveur Application sécurisée en sélectionnant le pool frontal de Enterprise Edition ou Standard Edition serveur frontal dans la liste déroulante. Un directeur ou un directeur pool n’est pas une sélection valide pour un saut suivant du serveur Application sécurisée, et il n’apparaît pas dans la liste. Cliquez sur **OK** pour accepter et enregistrer vos modifications. Cliquez sur **Annuler** pour annuler les modifications et quitter la page des propriétés.
