# <a name="add-additional-sip-domains"></a>Ajouter des domaines SIP supplémentaires
 
Spécifiez le nom de domaine de chaque domaine supplémentaire que vous voulez prendre en charge dans votre déploiement. Par exemple, si vous avez spécifié contoso.net en tant que domaine SIP (Session Initiation Protocol) et que vous souhaitez également prendre en charge fabrikam.net pour le trafic SIP dans votre déploiement, spécifiez fabrikam.net en tant que domaine supplémentaire pris en charge.
