# <a name="add-additional-sip-domains"></a>Ajouter des domaines SIP supplémentaires
 
Spécifiez le nom de domaine de chaque domaine supplémentaire que vous souhaitez prendre en charge dans votre déploiement. Par exemple, si vous avez spécifié contoso.net comme domaine SIP (Session Initiation Protocol) principal et que vous souhaitez également prendre en charge les fabrikam.net pour le trafic SIP dans votre déploiement, spécifiez fabrikam.net comme domaine supplémentaire pris en charge.
