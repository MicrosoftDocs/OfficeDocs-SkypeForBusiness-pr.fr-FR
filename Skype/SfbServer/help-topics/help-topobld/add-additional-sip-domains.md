# <a name="add-additional-sip-domains"></a>Ajouter des domaines SIP supplémentaire
 
Spécifiez le nom de domaine de chaque domaine supplémentaire que vous souhaitez prendre en charge dans votre déploiement. Par exemple, si vous avez spécifié contoso.net comme le domaine principal de la Session Initiation Protocol (SIP) et vous souhaitez également prise en charge de fabrikam.net pour le trafic SIP dans votre déploiement, spécifiez fabrikam.net comme un domaine de prise en charge supplémentaire.
