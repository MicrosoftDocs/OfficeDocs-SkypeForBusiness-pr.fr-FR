# <a name="add-survivable-branch-appliance-pool"></a>Ajouter un pool Survivable Branch Appliance
 
Pour **définir le nom de domaine complet du Survivable Branch Appliance**, tapez le nom de domaine complet (FQDN) (tel que défini dans DNS (Domain Name System)) du Survivable Branch Appliance ou du serveur Survivable Branch Server dans la zone de texte nom de domaine **complet** . Cliquez sur **Suivant** pour continuer.
  
> [!NOTE]
> Sauf indication contraire, la définition d’un Survivable Branch Appliance ou d’un serveur Survivable Branch Server est la même procédure.
