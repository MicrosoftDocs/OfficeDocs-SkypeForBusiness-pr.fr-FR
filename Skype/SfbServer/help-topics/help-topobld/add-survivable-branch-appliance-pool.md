# <a name="add-survivable-branch-appliance-pool"></a>Ajouter un Pool Survivable Branch Appliance
 
Pour **définir le nom de domaine complet Survivable Branch Appliance**, tapez le nom de domaine complet (FQDN) (tel que défini dans le système DNS (Domain Name System)) du serveur Survivable Branch Server Survivable Branch Appliance ou dans la zone de texte **nom de domaine complet** . Cliquez sur **Suivant** pour continuer.
  
> [!NOTE]
> Sauf indication contraire, la définition d’un Survivable Branch Appliance ou un serveur de succursale Survivable est la même procédure.
