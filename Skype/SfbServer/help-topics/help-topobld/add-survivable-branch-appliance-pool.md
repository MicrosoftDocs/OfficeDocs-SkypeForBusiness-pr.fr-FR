# <a name="add-survivable-branch-appliance-pool"></a>Ajouter un pool Survivable Branch Appliance
 
Pour définir le nom de domaine complet de **l’application branche Survivable**, tapez le nom de domaine complet (FQDN) (tel qu’il est défini dans DNS) de l’appareil de succursale ou du serveur de succursales survivant dans la zone de texte nom de domaine **complet** . Cliquez sur **Suivant** pour continuer.
  
> [!NOTE]
> Sauf indication contraire, la définition d’une unité de branchement survivant ou d’un serveur de succursales survivant est la même procédure.
