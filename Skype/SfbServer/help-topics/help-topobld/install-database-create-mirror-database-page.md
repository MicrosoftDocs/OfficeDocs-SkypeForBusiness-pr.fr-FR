# <a name="install-database-create-mirror-database-page"></a>Installer base de données de créer la Page de base de données mise en miroir
 
Cette rubrique fait partie de la documentation préliminaire. Elle est susceptible d’être modifiée lors des prochaines mises à jour. Les rubriques vides apparaissent sous la forme d’espaces réservés.
  
Si vous choisissez ne pas à installer les bases de données de mise en miroir de SQL Server pendant le processus de publication, vous les installez à l’aide **d’Installer la base de données** à partir d’un clic droit sur n’importe quel nœud dans le Générateur de topologies. Vous sélectionnez la banque mise en miroir à installer. Cliquez sur **paramètres** pour définir les paramètres avancés.
