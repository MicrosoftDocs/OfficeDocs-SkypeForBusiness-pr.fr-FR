# <a name="install-database-create-mirror-database-page"></a>Page Installer une base de données : créer une base de données miroir
 
Cette documentation est préliminaire et sujette à modification. Des rubriques vides sont incluses comme espaces réservés.
  
Si vous avez choisi de ne pas installer les bases de données miroir SQL Server pendant le processus de publication, installez-les à l’aide de l' **installation de base de données** à partir d’un clic droit sur un nœud dans le générateur de topologie. Vous sélectionnez le magasin miroir à installer. Cliquez sur **Paramètres** pour définir tous les paramètres avancés.
