# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Associer un directeur surveillance Page pour Lync Server 2010
 
Pour configurer un serveur de surveillance et l’associer au directeur ou le directeur pool :
  
Sous **Serveur de surveillance** sélectionnez dans la liste, un serveur de surveillance.
  
Si aucun serveur de surveillance, et que vous souhaitez créer un serveur de surveillance maintenant, cliquez sur **Nouveau**.
  
 **Précédent**: revient à la boîte de dialogue précédente
  
 **Terminer**: valide les modifications et met fin à l’Assistant
  
 **Annuler**: ignore les modifications et annule la configuration
  
 **Aide**: affiche des informations, telles que cette rubrique d’aide
