# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Associer l’analyse de Page pour Lync Server 2010 de directeur
 
Pour configurer un serveur d’analyse et l’associer avec le directeur ou le directeur pool :
  
Sous **Serveur de surveillance** sélectionnez dans la liste, un serveur d’analyse.
  
Si aucun serveur de surveillance et que vous souhaitez créer un serveur d’analyse maintenant, cliquez sur **Nouveau**.
  
 **Précédente**: revient à la boîte de dialogue précédente
  
 **Terminer**: valide les modifications et termine l’Assistant
  
 **Annuler**: annule les modifications et annule la configuration
  
 **Aide**: affiche des informations, par exemple de cette rubrique d’aide
