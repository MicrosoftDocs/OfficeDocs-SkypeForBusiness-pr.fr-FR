# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Page Ajouter un magasin Sql pour le pool frontal pour Lync Server 2010
 
Vous définissez les propriétés des magasins SQL Server pour le nouveau pool frontal que vous configurez.
  
> [!IMPORTANT]
> Si vous configurez un serveur Standard Edition, cette boîte de dialogue s’affiche, mais vous ne pouvez pas configurer les paramètres. Cliquez sur **Suivant** pour continuer.
  
Paramètres que vous pouvez configurer :
  
- **Utilisez un magasin SQL Server précédemment défini**: sélectionnez un magasin SQL Server existant dans la liste pour le magasin de pools frontaux.
    
- Sélectionnez **définir un nouveau magasin SQL Server** pour créer un nouveau magasin SQL Server.
    
    Vous tapez le nom de domaine complet du serveur SQL Server dans le nom de domaine **complet SQL Server**
    
    L’**instance par défaut** est initialement sélectionnée. Si vous préférez définir une instance, sélectionnez **Instance nommée** et tapez le nom de l’instance que vous devez créer.
    
  **Précédent** : renvoie à la boîte de dialogue précédente
  
  **Suivant** : valide les modifications et vous permet d’accéder à la boîte de dialogue suivante
  
  **Annuler** : ignore les modifications et annule la configuration
  
  **Aide** : affiche des informations d’aide, à l’instar de cette rubrique
