# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Page Ajouter un magasin Sql pour le pool frontal pour Lync Server 2010
 
Vous définissez les propriétés des banques SQL Server pour le nouveau pool frontal que vous configurez.
  
> [!IMPORTANT]
> Si vous configurez un serveur Standard Edition Server, cette boîte de dialogue apparaît, mais vous ne pouvez pas configurer les paramètres. Cliquez sur **Suivant** pour continuer.
  
Paramètres que vous pouvez configurer:
  
- **Utiliser un magasin SQL Server déjà défini**: sélectionnez un magasin SQL Server existant dans la liste du magasin frontal.
    
- Sélectionnez **définir une nouvelle banque SQL Server** pour créer une nouvelle banque SQL Server.
    
    Entrez le nom de domaine complet du serveur SQL Server dans le **FQDN SQL Server** .
    
    L' **instance par défaut** est sélectionnée initialement. Si vous préférez définir une instance, sélectionnez **instance nommée** , puis tapez le nom de l’instance que vous souhaitez créer.
    
  **Retour**: revenir à la boîte de dialogue précédente
  
  **Suivant**: valide les modifications et passe à la boîte de dialogue étapes suivantes
  
  **Annuler**: supprime les modifications et annule la configuration.
  
  **Aide**: affiche des informations d’aide, telles que ce sujet.
