# <a name="add-front-end-features"></a>Ajouter des fonctionnalités frontales

Sélectionnez chaque fonctionnalité que vous souhaitez activer pour le pool frontal. Au minimum, vous devez déployer la messagerie instantanée (MI), qui est toujours activé et ne peut pas être désactivée. Également, vous avez la possibilité de déployer d’autres fonctionnalités, telles que des conférences, Enterprise Voice et le contrôle d’admission des appels (CAC). Certaines fonctionnalités dépendent de votre topologie.

- Conférence

- Voix Entreprise

- Le contrôle d’admission des appels (CAC)

- Archivage

- Surveillance

> [!NOTE]
> Si vous souhaitez activer CAC dans votre déploiement, vous devez activer CAC exactement un pool par site central. Nous vous recommandons de CAC si vous déployez des fonctionnalités vocales ou A / V Conferencing.

Pour plus d’informations sur la spécification des fonctionnalités pour un pool Enterprise Edition, voir [Define and Configure a Front End Pool](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) dans la documentation de déploiement.
