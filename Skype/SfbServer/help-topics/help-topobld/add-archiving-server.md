# <a name="add-archiving-server"></a>Ajouter un serveur d’archivage
 
Pour déployer un serveur d’archivage dans votre conception de la topologie, vous devez spécifier le nom de domaine complet du serveur sur lequel vous voulez déployer le serveur d’archivage. Avant de publier une topologie incluant l’archivage et l’installation de Skype entreprise Server, vous devez avoir rempli toutes les conditions préalables pour le déploiement de l’archivage.
