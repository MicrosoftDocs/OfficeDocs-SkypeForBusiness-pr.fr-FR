# <a name="add-archiving-server"></a>Ajouter un serveur d’archivage
 
Pour déployer un serveur d’archivage dans votre conception de la topologie, vous devez spécifier le nom de domaine complet du serveur sur lequel vous voulez déployer le serveur d’archivage. Avant de publier une topologie qui inclut l’archivage et l’installation de Skype entreprise Server, vous devez avoir complété tous les éléments prérequis pour le déploiement de l’archivage.
