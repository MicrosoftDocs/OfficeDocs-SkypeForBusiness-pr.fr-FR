# <a name="remove-deployment"></a>Supprimer le déploiement
 
Cliquez sur **Terminer** pour créer une topologie vide comme première étape de mise hors service de l’ensemble de votre déploiement.
  
> [!CAUTION]
> Cette action ne peut pas être annulée.
