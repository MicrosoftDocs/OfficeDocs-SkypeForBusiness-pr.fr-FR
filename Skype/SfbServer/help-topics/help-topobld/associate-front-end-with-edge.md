# <a name="associate-front-end-with-edge"></a>Associer un pool frontal à un serveur Edge

Chaque pool frontal ne peut être associé qu’à un seul serveur Edge ou pool de serveurs Edge. Lorsque vous activez l’accès des utilisateurs externes à un site, vous pouvez prendre en charge les utilisateurs distants. Vous pouvez également activer la prise en charge pour les utilisateurs fédérés, qui peuvent prendre en charge les utilisateurs de fournisseurs de connectivité de messagerie instantanée publics spécifiques (tels que Windows Live), ainsi que la prise en charge des utilisateurs anonymes.

Tous les pools d’un site et les pools de sites centraux multiples peuvent utiliser le même serveur Edge, si l’utilisation ne dépasse pas la capacité du serveur Edge. Pour plus d’informations sur la surveillance, y compris la mise à l'échelle, voir [Planification de l’accès des utilisateurs externes](https://technet.microsoft.com/library/ea098933-eff5-461e-aba3-e7f128784dc2.aspx) dans la documentation de planification. Pour plus d’informations sur la conception d’une topologie en vue de prendre en charge l’accès des utilisateurs externes, voir [Définir la topologie Edge](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) dans la documentation de déploiement.
