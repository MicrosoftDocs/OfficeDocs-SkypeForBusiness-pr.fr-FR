# <a name="publish-topology"></a>Topologie de publication
 
La page **publier la topologie** s’affiche lorsque vous commencez le processus de publication. Lire et vérifier que les tâches sont terminées avant de continuer. Lorsque vous êtes prêt à continuer, cliquez sur **suivant**. Si vous devez fermer le processus de publication, cliquez sur **Annuler**.
