# <a name="add-edge-server-associations"></a>Ajouter des Associations de serveur Edge
 
Utilisez cette page pour sélectionner les pools de Front-End et les serveurs Standard Edition Server utilisera ce pool de bord pour un / le trafic de V. 
  
Chaque pool frontal peut être associé à un seul pool de bord. Si vous sélectionnez un pool qui est déjà associé à un autre pool de bord, l’association de la grappe de bord précédente est modifiée.
  
Vous pouvez également choisir d’associer ce serveur de transport Edge ou d’un pool de bord avec les serveurs de médiation ou pools de serveur de médiation pour configurer et activer des Voix Entreprise pour les utilisateurs externes.
