# <a name="associate-branch-office-appliance-with-front-end"></a>Associer un Branch Office Appliance à un serveur frontal
 
Pour associer le Survivable Branch Appliance ou le Serveur Survivable Branch Server à un pool frontal ou à un serveur Édition Standard, dans la page Sélectionner le **pool** frontal, sélectionnez le pool frontal ou un serveur Édition Standard dans la liste de listes bas du **pool** frontal. Cliquez sur **Suivant** pour continuer.
