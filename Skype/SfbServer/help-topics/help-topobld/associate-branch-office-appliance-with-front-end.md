# <a name="associate-branch-office-appliance-with-front-end"></a>Associer un Branch Office Appliance à un serveur frontal
 
Pour associer le Survivable Branch Appliance ou serveur Survivable Branch Server avec un pool frontal ou un serveur Standard Edition, dans la page **Sélectionnez le pool frontal** , sélectionnez le pool frontal ou un serveur Standard Edition dans le **pool frontal** liste déroulante. Cliquez sur **Suivant** pour continuer.
