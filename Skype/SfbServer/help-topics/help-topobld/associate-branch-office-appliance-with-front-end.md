# <a name="associate-branch-office-appliance-with-front-end"></a>Associer un Branch Office Appliance à un serveur frontal
 
Pour associer le Survivable Branch Appliance ou le serveur Survivable Branch Server à un pool frontal ou un serveur Standard Edition, dans la page **Sélectionner le pool** frontal, sélectionnez le pool frontal ou un serveur Standard Edition Server dans la liste déroulante **pool frontal** . Cliquez sur **Suivant** pour continuer.
