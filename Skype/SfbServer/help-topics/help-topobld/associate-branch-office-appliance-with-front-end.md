# <a name="associate-branch-office-appliance-with-front-end"></a>Associer un équipement de la succursale avec le Front-End
 
Pour associer le Survivable Branch Appliance ou le serveur Survivable Branch Server avec un pool frontal ou un serveur Standard Edition, sur la page **Sélectionner le pool frontal** , sélectionnez le pool frontal ou un serveur Standard Edition dans le **pool frontal** menu déroulant. Cliquez sur **Suivant** pour continuer.
