# <a name="publish-topology-next-step"></a>Publier l’étape suivante de topologie
 
À la fin de la publication de la topologie, la page de **l’Assistant publication terminé** s’affiche.
  
Que vous avez terminé les étapes au cours du processus de publication de topologie, les journaux sont créés qui contiennent des informations sur les tâches qui ont été effectuées. Pour afficher un fichier journal lié à une étape donnée, sélectionnez l’étape que vous devez passer en revue le fichier journal pour, puis cliquez sur **Afficher les journaux**. 
  
> [!NOTE]
> Pas toutes les étapes au cours du processus de publication de topologie seront sont associés à des fichiers journaux. Si vous sélectionnez une étape et **Afficher les journaux** est disponible, cette étape a créé un fichier journal. Si le bouton **Afficher les journaux** n’est pas disponible (grisée), aucun fichier journal n’a été créé pour cette étape.
  
Pour passer en revue les étapes suggérées pour terminer votre topologie, cliquez sur le lien **Cliquez ici pour ouvrir la liste de tâches**. Fichier texte qui est ouvert en cliquant sur ce lien indique les serveurs que vous devez exécuter le programme d’installation local et les étapes supplémentaires que vous devez prendre votre déploiement. Pour chaque tâche dans la liste, reportez-vous à la section appropriée dans la documentation de déploiement. Pour plus d’informations, consultez [Déploiement de Lync Server 2013 Preview](http://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx)et reportez-vous à la section pour chaque rôle de serveur indiqué dans la liste de tâches.
  
Cliquez sur **Terminer**
  
Pour plus d’informations, reportez-vous à la section [publier la topologie](http://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx).
