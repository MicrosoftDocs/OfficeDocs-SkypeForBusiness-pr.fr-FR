# <a name="publish-topology-next-step"></a>Étape ultérieure à la publication de la topologie

Une fois la topologie publiée, la page **Assistant Publication achevé** s’affiche.

Tandis que vous terminez les étapes de la procédure de publication de la topologie, des journaux sont créés avec des informations sur les tâches qui ont été effectuées. Pour afficher un fichier journal lié à une étape donnée, sélectionnez l’étape pour laquelle vous devez passer en revue le journal, puis cliquez sur **Afficher les journaux**.

> [!NOTE]
> Toutes les étapes de la procédure de publication de la topologie n’ont pas de fichiers journaux associés. Si vous sélectionnez une étape et que **Afficher les journaux** est disponible, cette étape a créé un fichier journal. Si le bouton **Afficher les journaux** n’est pas disponible (grisé), aucun fichier journal n’a été créé pour cette étape.

Pour passer en revue les étapes suivantes suggérées pour achever votre topologie, cliquez sur le lien **Cliquez ici pour ouvrir la liste de tâches**. Le fichier texte qui s’ouvre lorsque vous cliquez sur ce lien vous indiquera quels sont les serveurs à exécuter localement, ainsi que toutes les étapes supplémentaires à suivre pour achever votre déploiement. Pour chacune des tâches répertoriées, consultez la section appropriée dans la documentation de déploiement. Pour plus d’informations, voir [Deploying Lync Server 2013 Preview](https://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx) et consultez la section correspondante pour chaque rôle serveur mentionné dans la liste de tâches.

Cliquez sur **Terminer**.

Pour plus d’informations, voir [Publish the Topology](https://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx).
