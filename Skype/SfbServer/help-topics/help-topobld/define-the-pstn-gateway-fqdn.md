# <a name="define-the-pstn-gateway-fqdn"></a>Définir la nom de domaine complet de passerelle PSTN
 
Vous spécifiez le nom de domaine pleinement qualifié (FQDN) de la passerelle réseau (RTPC) public commuté que vos composants de Voix Entreprise se connectera à un RTC ou à autocommutateur privé (PBX).
  
Pour plus d’informations, reportez-vous à la section [composants de connectivité RTPC dans Skype pour Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)
