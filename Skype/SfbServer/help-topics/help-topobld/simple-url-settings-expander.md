# <a name="simple-url-settings-expander"></a>Expander de paramètres d’URL simple
 
Pour spécifier de nouveaux, supprimer l’existant, ou pour modifier l’actif les URL simples, effectuez le des opérations suivantes :
  
## <a name="simple-urls"></a>URL simples 

Pour modifier ou définir les URL simples, tout d’abord décider des trois URL simples que vous souhaitez modifier. Vous pouvez sélectionner les URL d’accès téléphonique, les URL de réunion ou les URL d’accès administratif.
  
Pour modifier l’URL d’accès téléphonique ou l’URL de réunion, sélectionnez l’URL à modifier. Cliquez sur **Modifier l’URL**. Modifier l’URL, puis cliquez sur **OK** pour enregistrer l’URL. Cliquez sur **Annuler** pour annuler les modifications.
  
Pour ajouter une nouvelle URL, cliquez sur **Ajouter**. Dans la boîte de dialogue **Ajouter des URL simples** , spécifiez l’URL et puis cliquez sur **OK** pour enregistrer l’URL. Sélectionnez **Définir cette URL en tant qu’URL active pour le domaine sélectionné** si vous avez besoin de définir la nouvelle URL comme URL active. Cliquez sur **Annuler** pour annuler les modifications.
  
Pour rendre une URL différente de l’URL active (comme indiqué par la coche verte en regard de l’URL), sélectionnez l’URL, puis cliquez sur **Activer**.
  
> [!NOTE]
> Il ne peut exister qu’une seule URL active pour chaque domaine de protocole SIP (Session Initiation). 
  
Pour supprimer une URL, sélectionnez l’URL, puis cliquez sur **Supprimer**.
  
> [!CAUTION]
> Veillez à lire les informations sur la boîte de dialogue Paramètres URL simple. Suppression d’une URL de la réunion peut rendre les existants de réunions planifiées par des utilisateurs n’est plus accessibles. Envisager l’URL existante après activation de la nouvelle URL de la réunion. Lorsque vous êtes sûr que les utilisateurs n’utilisent plus l’ancienne URL de la réunion, vous pouvez le supprimer en toute sécurité. 
  
Pour modifier l’URL d’accès administratif, modifiez l’entrée.
  
Lorsque vous avez terminé avec toutes les modifications dans la page Modifier les propriétés, cliquez sur **OK** pour enregistrer les modifications. Cliquez sur **Annuler** pour annuler les modifications.
