# <a name="add-director-association-for-lync-server-2010"></a>Ajouter une Association de directeur pour Lync Server 2010
 
Vous définissez si le directeur ou le directeur du pool va avoir un Monitoring Server associé. Activez la case à cocher pour associer un serveur d’analyse avec le directeur ou le directeur pool.
  
> [!TIP]
> Si vous n’avez pas encore créé une base de données de Monitoring Server (et l’instance en option), vous aurez la possibilité de créer ultérieurement si vous activez cette case à cocher pour associer un serveur d’analyse avec le directeur ou le directeur pool.
