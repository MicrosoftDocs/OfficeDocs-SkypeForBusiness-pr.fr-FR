# <a name="add-director-association-for-lync-server-2010"></a>Ajouter une association de directeurs pour Lync Server 2010
 
Vous pouvez déterminer si un serveur de surveillance est associé au directeur ou au pool de directeurs. Activez la case à cocher pour associer un serveur de surveillance au directeur ou au pool de réalisateurs.
  
> [!TIP]
> Si vous n’avez pas encore créé de base de données du serveur de surveillance (et instance facultative), vous aurez la possibilité de le créer ultérieurement si vous activez cette case à cocher pour associer un serveur de surveillance au directeur ou au pool de réalisateurs.
