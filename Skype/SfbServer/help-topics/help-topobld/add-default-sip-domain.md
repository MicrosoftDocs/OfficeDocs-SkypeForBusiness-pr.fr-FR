# <a name="add-default-sip-domain"></a>Ajouter un domaine Sip de par défaut
 
Spécifiez le domaine principal de protocole SIP (Session Initiation) que vous souhaitez que votre déploiement pour prendre en charge. Le domaine SIP principal peut être identique ou différent de votre domaine des Services de domaine Active Directory. Par exemple, votre domaine Active Directory peut être fabrikam.net, et vous pouvez spécifier votre domaine SIP principal pour être contoso.net.
