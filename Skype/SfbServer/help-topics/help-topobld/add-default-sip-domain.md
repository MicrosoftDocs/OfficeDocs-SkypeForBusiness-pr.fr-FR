# <a name="add-default-sip-domain"></a>Ajouter un domaine SIP par défaut
 
Spécifiez le domaine SIP (Session Initiation Protocol) principal qui devra être pris en charge par votre déploiement. Le domaine SIP principal peut être identique ou différent de votre domaine Services de domaine Active Directory. Par exemple, votre domaine Active Directory peut être fabrikam.net, et vous pouvez indiquer contoso.net comme nom de domaine SIP principal.
