# <a name="edit-office-web-apps-server-settings"></a>Modifier les paramètres Office Web Apps Server

Vous modifiez les propriétés du serveur Office Web Apps Server configuré. Les propriétés suivantes peuvent être modifiées :

 Nom de domaine **complet (FQDN) du serveur Office Web Apps Server**: cette propriété définit le nom de domaine complet du serveur Office Web Apps Server et doit correspondre à un enregistrement d’hôte DNS (Domain Name System) ou AAAA (si IPv6 est utilisé).

 **URL de découverte d’Office Web Apps Server**: l’URL (Uniform Resource Locator) pour l’accès client au serveur Office Web Apps Server, il se peut que vous deviez modifier cette adresse à partir de sa valeur par défaut si le serveur est placé dans une autre zone réseau que le réseau interne de votre déploiement.

Activez la case à cocher **Le serveur Office Web Apps Server est déployé sur un réseau externe (périmètre/Internet)** si le serveur est déployé dans le réseau de votre périmètre ou dans une autre zone réseau qui se situe à l’extérieur de votre pare-feu interne séparant le réseau de périmètre, les réseaux moyennement fiables et Internet de votre déploiement interne.
