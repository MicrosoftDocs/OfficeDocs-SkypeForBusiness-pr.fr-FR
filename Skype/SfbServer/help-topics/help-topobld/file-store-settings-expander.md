# <a name="file-store-settings-expander"></a>Magasin de fichiers paramètres Expander
 
Pour modifier les propriétés des banques de fichiers définies :
  
Pour le nom de partage de fichiers, vous pouvez définir le nom d’un nouveau partage. Vous devez ensuite créer le partage sur le serveur qui est nommé dans la propriété de nom de domaine qualifié de serveur de fichiers.
  
> [!NOTE]
> Vous ne pouvez pas modifier le nom de domaine complet du serveur qui est spécifié dans la propriété de nom de domaine complet de fichier serveur. Si vous devez placer le partage de fichiers de banque d’informations de fichier sur un autre serveur, vous créez une nouvelle entrée de banque d’informations de fichier.
