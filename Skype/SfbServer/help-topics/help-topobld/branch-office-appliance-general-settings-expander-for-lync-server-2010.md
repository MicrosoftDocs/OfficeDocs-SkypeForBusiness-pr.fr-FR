# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Expanseur des paramètres généraux d’un Branch Office Appliance pour Lync Server 2010
 
Pour modifier les propriétés de Survivable Branch Appliance ou du serveur Survivable Branch Server, sous **général**, configurez les éléments suivants :
  
- Nom de domaine **complet (FQDN)**: tapez le nom de domaine complet du Survivable Branch Appliance ou du serveur Survivable Branch Server.
    
    **Utiliser toutes les adresses IP configurées** utilise les adresses IP configurées sur le Survivable Branch Appliance ou le serveur Survivable Branch Server pour tous les besoins.
    
    **Limiter l’utilisation des services aux adresses IP sélectionnées** Vous configurez l’adresse distincte qui définit le serveur et l’adresse IP à utiliser pour le réseau téléphonique commuté.
    
    **Adresse IP principale** : l’adresse IP est définie et configurée à toutes fins, excepté pour les fonctions associées au réseau téléphonique commuté.
    
    **Adresse IP PSTN** : l’adresse IP associée aux fonctions du réseau téléphonique commuté.
    
- Vous configurez des **associations** pour vous assurer que d’autres rôles serveur sont configurés et associés au Survivable Branch Appliance ou au serveur Survivable Branch Server.
    
    **Associer un serveur d’archivage** Sélectionnez dans la liste le serveur d’archivage que vous souhaitez associer au Survivable Branch Appliance ou au serveur Survivable Branch Server.
    
    Cliquez sur **nouveau** si vous n’avez pas créé le serveur d’archivage que vous souhaitez associer à ce Survivable Branch Appliance ou à un serveur Survivable Branch Server.
    
    **Associer un serveur de surveillance** Dans la liste, sélectionnez le serveur de surveillance que vous souhaitez associer au Survivable Branch Appliance ou au serveur Survivable Branch Server.
    
    Cliquez sur **nouveau** si vous n’avez pas créé le serveur de surveillance que vous souhaitez associer à ce Survivable Branch Appliance ou à un serveur Survivable Branch Server.
    
    **Associer un pool Edge (pour les composants multimédias)** Sélectionnez dans la liste le serveur Edge ou le pool de serveurs Edge que vous souhaitez associer au Survivable Branch Appliance ou au serveur Survivable Branch Server.
    
  **OK** permet d’accepter et de valider les modifications de la boîte de dialogue.
  
  **Annuler** permet d’annuler les modifications et de fermer la boîte de dialogue.
  
  **Aide** permet d’afficher cet écran d’aide.
