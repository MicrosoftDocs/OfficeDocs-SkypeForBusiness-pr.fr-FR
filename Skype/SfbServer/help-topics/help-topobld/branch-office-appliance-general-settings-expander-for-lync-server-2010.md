# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Expanseur des paramètres généraux d’un Branch Office Appliance pour Lync Server 2010
 
Pour modifier les propriétés de Survivable Branch Appliance ou serveur Survivable Branch Server, sous **Général**, vous configurez :
  
- **Nom de domaine complet**: vous tapez le nom de domaine complet du serveur Survivable Branch Server ou Survivable Branch Appliance.
    
    **Utiliser toutes les adresses IP** utilise les adresses IP configurées sur le Survivable Branch Appliance ou le serveur Survivable Branch Server pour tous les usages.
    
    **Limiter l’utilisation des services aux adresses IP sélectionnées** : vous configurez l’adresse distincte qui définit le serveur et l’adresse IP à utiliser pour le réseau téléphonique commuté (RTC).
    
    **Adresse IP principale** : l’adresse IP est définie et configurée à toutes fins, à l’exception des fonctions associées au réseau téléphonique commuté.
    
    **Adresse IP RTC** : adresse IP associée aux fonctions du réseau téléphonique commuté.
    
- Vous configurez les **Associations** pour vous assurer que les autres serveurs de rôles sont configurés et associés au Survivable Branch Appliance ou le serveur Survivable Branch Server.
    
    **Associer le serveur d’archivage** Sélectionnez dans la liste, le serveur d’archivage que vous souhaitez associer au programme Survivable Branch Appliance ou le serveur Survivable Branch Server
    
    Cliquez sur **Nouveau** si vous n’avez pas créé le serveur d’archivage que vous souhaitez associer à ce Survivable Branch Appliance ou le serveur Survivable Branch Server
    
    **Associer le serveur de surveillance** Sélectionnez dans la liste, le serveur d’analyse que vous souhaitez associer au programme Survivable Branch Appliance ou le serveur Survivable Branch Server
    
    Cliquez sur **Nouveau** si vous n’avez pas créé le serveur d’analyse que vous souhaitez associer à ce Survivable Branch Appliance ou le serveur Survivable Branch Server
    
    **Pool d’associer le bord (pour les composants de support)** Sélectionnez dans la liste, le serveur de transport Edge ou le pool de bord que vous souhaitez associer au programme Survivable Branch Appliance ou le serveur Survivable Branch Server
    
 **OK** : permet d’accepter et de valider les modifications de la boîte de dialogue.
  
 **Annuler** : permet d’annuler les modifications et de fermer la boîte de dialogue.
  
 **Aide** : permet d’afficher cet écran d’aide.
