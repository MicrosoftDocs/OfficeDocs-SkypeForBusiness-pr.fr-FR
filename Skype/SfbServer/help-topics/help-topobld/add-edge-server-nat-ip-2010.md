# <a name="add-edge-server-nat-ip-2010"></a>Ajouter IP de NAT Edge Server 2010
 
Si l’adresse IP publique de ce pool de bord est traduit par une traduction d’adresses réseau (NAT), vous devez spécifier l’adresse IP publique à traduire. Ce doit être l’adresse IP externe de l’A / V Edge service.
  
Pour changer si le pool de bord utilise la traduction NAT, cliquez sur le nom du pool de bord et cliquez sur **Modifier les propriétés**.
