# <a name="associate-av-mcu-with-front-end"></a>Associer le service MCU audio/vidéo au serveur frontal
 
Toutes les plages frontales d’Enterprise Edition dont la fonction de conférence est activée doivent disposer d’un service de conférence A/V localisé ou doivent utiliser un pool de conférence A/V autonome. Tout pool frontal Enterprise Edition existant qui dispose actuellement d’un service de conférence A/V, il est possible de sélectionner l’utilisation de ce nouveau pool de conférence A/V autonome.
