# <a name="legacy-merge-next-hop"></a>Tronçon suivant de la fusion héritée
 
Le **nom de domaine complet du tronçon suivant** identifie le serveur du tronçon suivant vers lequel le serveur Edge hérité achemine le trafic interne. Tapez le nom de domaine complet (FQDN) du serveur du tronçon suivant vers lequel ce serveur Edge achemine le trafic interne, puis déterminez s’il s’agit d’un **pool frontal** ou d’un **directeur**.
