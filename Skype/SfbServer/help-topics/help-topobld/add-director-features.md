# <a name="add-director-features"></a>Ajouter des fonctions de directeur
 
Si vous souhaitez que le pool de directeur à inclure Monitoring Server, sélectionnez **Monitoring (CD-r et QoE métriques)**.
  
Pour plus d’informations sur l’ajout d’un pool de directeur, reportez-vous à la section [scénarios pour le directeur](http://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx) et le [Paramètre de configuration le directeur](http://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx).
