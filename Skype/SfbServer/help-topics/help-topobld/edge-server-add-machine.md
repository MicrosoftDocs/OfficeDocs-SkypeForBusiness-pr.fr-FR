Après la publication de la topologie, la modification du nom de domaine complet nécessite la suppression du serveur dans le générateur de topologie et l’ajout d’un nouveau serveur au pool avec le nouveau nom de domaine complet. Pour plus d’informations sur la conception de la topologie d’un pool de bords, voir définir votre topologie de bord dans la documentation de déploiement Edge.
