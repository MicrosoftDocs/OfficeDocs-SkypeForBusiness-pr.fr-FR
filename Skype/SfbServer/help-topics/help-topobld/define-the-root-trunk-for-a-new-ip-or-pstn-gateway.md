# <a name="define-the-root-trunk-for-a-new-ip-or-pstn-gateway"></a>Définir la jonction racine d’une nouvelle passerelle IP ou PSTN

Vous définissez la jonction racine pour la passerelle IP ou PSTN en configurant les options suivantes :

- **Nom de la jonction** : définissez le nom de domaine complet qui est associé à la jonction

- **Port d’écoute pour la passerelle IP/PSTN** : définissez le port sur lequel la jonction procédera à l’écoute

- **Protocole de transport SIP** : sélectionnez soit **TCP**, soit **TLS** dans la liste, selon les exigences de la jonction

- **Serveur de médiation associé**: sélectionnez-le dans la liste des serveurs de médiation disponibles dans votre déploiement

- **Port du serveur de médiation associé**: définir le port d’écoute du serveur de médiation sélectionné
