# <a name="define-the-root-trunk-for-a-new-ip-or-pstn-gateway"></a>Définir le tronc racine pour une nouvelle période d’enquête ou d’une passerelle PSTN
 
Vous définissez le tronc racine d’IP ou le réseau téléphonique public commuté (RTPC) en configurant les éléments suivants :
  
- **Nom de jonction**: définir le nom de domaine pleinement qualifié associé du tronc
    
- **Un port d’écoute pour la passerelle IP/RTPC**: définissez cette jonction sera à l’écoute sur le port
    
- **Protocole de Transport SIP**: sélectionnez dans la liste **TCP** ou **TLS**, selon les exigences du tronc
    
- **Associé à un serveur de médiation**: sélectionnez dans la liste des serveurs de médiation disponibles dans votre déploiement.
    
- **Port de serveur de médiation associé**: définissez le port est à l’écoute sur le serveur de médiation sélectionné
