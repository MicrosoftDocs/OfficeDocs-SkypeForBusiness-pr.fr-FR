# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 : créer un expandeur
 
Vous pouvez créer de nouveaux rôles de serveur dans le nœud du Générateur de topologie Lync Server 2013 en cliquant avec le bouton droit sur **Lync Server 2013**, sélectionnez **Nouveau**, puis sélectionnez le rôle de serveur à créer.
