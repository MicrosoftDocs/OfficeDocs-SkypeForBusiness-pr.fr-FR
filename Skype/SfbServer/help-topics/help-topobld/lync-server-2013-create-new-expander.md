# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 : créer un expandeur
 
Vous pouvez créer des rôles serveur dans le nœud Lync Server 2013 du Générateur de topologie en cliquant avec le bouton droit sur **Lync Server 2013,** sélectionnez **Nouveau,** puis sélectionnez le rôle serveur à créer.
