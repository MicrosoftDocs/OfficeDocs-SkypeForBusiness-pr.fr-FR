# <a name="director-general-settings-expander-for-lync-server-2010"></a>Développement de paramètres de directeur général pour Lync Server 2010
 
Pour configurer les propriétés **générales** d’un directeur ou d’un pool de directeur :
  
 **Nom de domaine complet**: tapez le nom de domaine complet du directeur ou d’un pool de directeur
  
 **Associations**
  
 **Stockage de fichier**: sélectionnez dans la liste, le magasin de fichiers pour le directeur ou d’un pool de directeur
  
Cliquez sur **Nouveau** pour définir une nouvelle banque de fichiers
  
Activez la case à cocher **Associer Monitoring Server** et sélectionnez un serveur d’analyse dans la liste
  
Cliquez sur **Nouveau** pour définir un nouveau serveur de surveillance
  
 **OK** : permet d’accepter et de valider les modifications de la boîte de dialogue.
  
 **Annuler** : permet d’annuler les modifications et de fermer la boîte de dialogue.
  
 **Aide** : permet d’afficher cet écran d’aide.
