# <a name="install-database-options-page"></a>Installez la Page des Options de base de données
 
Vous configurez des options avancées pour l’emplacement des fichiers journaux et de base de données sur votre SQL Server. Les options disponibles sont les suivantes :
  
> [!IMPORTANT]
> Sélectionnez l’option qui correspond le mieux à vos exigences et politiques de placement des fichiers journaux et de données sur les ordinateurs SQL Server. 
  
 **Déterminer automatiquement le dossier de base de données**: l’option par défaut utilise un algorithme qui détermine l’espace disponible sur le SQL Server et distribue les fichiers journaux et de base de données pour des performances optimales.
  
 **Utilisez SQL Server, instance par défaut**: sélectionnez cette option pour placer le fichier de base de données et les journaux basés sur les paramètres de l’instance de SQL Server. Les options sont en général gérées et configurées par votre administrateur de base de données.
  
 **Nous ces chemin sur la cible de SQL Server**: sélectionnez cette option pour définir vos propres chemins pour les fichiers journaux et de base de données SQL Server en tapant le chemin d’accès complet au lecteur et au dossier où les fichiers journaux et de base de données seront placés.
  
> [!IMPORTANT]
> Les chemins d’accès que vous entrez peuvent être modifiées, fondés sur des algorithmes d’optimisation des performances dans l’installation. Pour plus d’informations, consultez la [Base de données Installation à l’aide de Communications Server Management Shell](http://technet.microsoft.com/library/c90a6449-4dd5-4b18-b21c-ea2c2a64dc3c.aspx). 
  
 **OK**: cliquez sur le bouton OK pour valider vos modifications.
  
 **Annuler**: cliquez sur Annuler pour ignorer les modifications et revenir à l’écran d’installation de la base de données.
  
 **Aide**: cliquez sur le bouton aide pour accéder à cette page d’aide.
