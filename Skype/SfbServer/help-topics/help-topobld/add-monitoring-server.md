# <a name="add-monitoring-server"></a>Ajouter un serveur de surveillance

Pour incorporer un serveur de surveillance dans votre conception topologique, vous devez spécifier le nom de domaine complet (FQDN) du serveur sur lequel vous voulez déployer le serveur de surveillance. Avant de publier une topologie incluant le serveur de surveillance et l’installation de Skype entreprise Server sur le serveur, vous devez effectuer toutes les conditions préalables pour le déploiement de la surveillance. Pour plus d’informations sur ces conditions préalables, reportez-vous à la rubrique [Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) de la documentation de déploiement.
