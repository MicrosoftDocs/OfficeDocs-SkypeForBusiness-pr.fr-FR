# <a name="add-monitoring-server"></a>Ajouter un serveur de surveillance
 
Pour incorporer un serveur de surveillance dans la conception de la topologie, vous devez spécifier le nom de domaine complet (FQDN) du serveur sur lequel vous souhaitez déployer Monitoring Server. Avant de publier une topologie qui inclut le serveur d’analyse et d’installation Skype pour Business Server sur le serveur, vous devez effectuer tous les composants requis pour le déploiement de surveillance. Pour plus d’informations sur ces conditions préalables, voir [Déploiement de surveillance](http://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) dans la documentation de déploiement.
