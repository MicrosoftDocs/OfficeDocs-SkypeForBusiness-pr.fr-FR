# <a name="select-pstn-usage-record"></a>Sélectionnez l’enregistrement de l’utilisation de TLS
 
Permet de rechercher un enregistrement utilisation réseau téléphonique public commuté existant que vous pouvez utiliser la boîte de dialogue **Sélectionner un enregistrement de l’utilisation de TLS** .
  
> [!NOTE]
> Pour récupérer tous les enregistrements de l’utilisation de TLS disponibles, pas rien taper dans la zone de recherche et puis cliquez sur l’icône de recherche. 
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide de la Skype pour Business Server du Panneau de configuration, voir [Gérer les Skype pour Business Server 2015](../../manage/manage.md).
