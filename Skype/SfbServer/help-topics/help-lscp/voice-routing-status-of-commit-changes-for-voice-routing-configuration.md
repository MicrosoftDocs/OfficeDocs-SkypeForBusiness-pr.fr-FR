# <a name="voice-routing-status-of-commit-changes-for-voice-routing-configuration"></a>Routage des communications vocales : état de la validation des modifications pour la configuration du routage des communications vocales
 
Si les modifications n’ont pas été effectuées, passez en revue les messages de cette boîte de dialogue, modifiez votre configuration de routage de la voix, puis réessayez.
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide du panneau de configuration Skype entreprise Server, reportez-vous à la rubrique [gestion de Skype entreprise server 2015](../../manage/manage.md).
