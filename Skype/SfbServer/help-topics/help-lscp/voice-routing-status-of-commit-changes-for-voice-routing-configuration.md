# <a name="voice-routing-status-of-commit-changes-for-voice-routing-configuration"></a>Routage des communications vocales : état de la validation des modifications pour la configuration du routage des communications vocales
 
Si les modifications n’ont pas été terminées, passez en revue les messages de cette boîte de dialogue, modifiez votre configuration du routage des communications vocales et réessayez.
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide du Panneau de Skype Entreprise Server, voir [Manage Skype Entreprise Server 2015](../../manage/manage.md).
