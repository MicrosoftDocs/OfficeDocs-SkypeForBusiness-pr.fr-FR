# <a name="select-trunk"></a>Sélectionnez Trunk
 
La boîte de dialogue **Sélectionner un tronc** permet de rechercher un trunk existant que vous pouvez utiliser.
  
> [!NOTE]
> Pour récupérer tous les trunks disponibles, pas rien taper dans la zone de recherche et puis cliquez sur l’icône de recherche. 
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide de la Skype pour Business Server du Panneau de configuration, voir [Gérer les Skype pour Business Server 2015](../../manage/manage.md).
