# <a name="pin-policy"></a>Stratégie de code confidentiel
 
Que voulez-vous faire ? Cliquez sur le lien de la tâche que vous voulez effectuer.
  
 **Travail avec la stratégie de code PIN pour les téléphones IP** Cette stratégie de (code confidentiel PIN) numérique personnel d’identification prend en charge l’authentification pour les utilisateurs qui se connectent à Skype pour Business Server à l’aide d’un téléphone IP.
- [Créer une nouvelle stratégie de code PIN dans Skype pour Business Server 2015](../../manage/authentication/create-a-new-pin-policy.md)
    
- [Modifier une stratégie de code PIN existant dans Skype pour Business Server 2015](../../manage/authentication/modify-an-existing-pin-policy.md)
    
- [Supprimer une stratégie de code PIN dans Skype pour Business Server 2015](../../manage/authentication/delete-a-pin-policy.md)
    
 **Travail avec la stratégie de code PIN pour les conférences à distance** Cette stratégie de code PIN permet aux utilisateurs qui ont des informations d’identification des Services de domaine Active Directory dans votre organisation pour rejoindre une conférence à distance en tant qu’utilisateurs authentifiés en entrant un code confidentiel.
- [Créer ou modifier les paramètres de code PIN à distance conférence d’un Site ou un groupe d’utilisateurs](http://technet.microsoft.com/library/c29bab5c-2b93-48e0-ae0b-29564daaff9a.aspx)
    
- [Modifier les paramètres de code PIN par défaut dans l’accès à la conférence](http://technet.microsoft.com/library/2d110e94-ad29-4755-b17f-d8c2da9b78a4.aspx)
    
- [Supprimer des conférences à distance dans les paramètres de code PIN pour un Site ou un groupe d’utilisateurs](http://technet.microsoft.com/library/15a9faee-d024-4c0e-b2a0-fe7e7dc00589.aspx)
    
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide de la Skype pour Business Server du Panneau de configuration, voir [Gérer les Skype pour Business Server 2015](../../manage/manage.md).
