# <a name="topology-status"></a>Topologie : état

La **page Topologie**: **État** affiche l’état des serveurs dans Skype Entreprise topologie.

## <a name="tasks-you-can-perform"></a>Tâches que vous pouvez effectuer

Vous pouvez effectuer les tâches suivantes dans la page **Topologie** : **Statut** :

- [Afficher la liste des ordinateurs exécutant Lync Server 2010](/previous-versions/office/lync-server-2013/lync-server-2013-view-a-list-of-computers-running-lync-server-2013)

- [Afficher les détails d’un service](/previous-versions/office/lync-server-2013/lync-server-2013-view-details-about-a-service)

- [Démarrer ou arrêter les services Lync Server 2010](/previous-versions/office/lync-server-2013/lync-server-2013-start-or-stop-lync-server-services)

- [Empêcher les nouvelles connexions aux services](/previous-versions/office/lync-server-2013/lync-server-2013-prevent-sessions-for-services)

Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide du Panneau de Skype Entreprise Server, voir [Manage Skype Entreprise Server 2015](../../manage/manage.md).
