1. [Règles de conversion dans Skype Entreprise Server 2015](../../plan-your-deployment/enterprise-voice-solution/translation-rules.md)

2. [Configuration de jonction et règles de traduction](https://technet.microsoft.com/library/0c339511-a185-484e-94f0-dbe918b7e48a.aspx)

3. [Création ou modification d’une règle de conversion pour la présentation de l’ID de la personne appelée dans Skype Entreprise Server 2015](../../deploy/deploy-enterprise-voice/called-id-presentation-rules.md)
