# <a name="select-overflow-queue"></a>Sélectionnez la file d’attente de dépassement de capacité
 
La boîte de dialogue **Sélectionnez une file d’attente de dépassement de capacité** permet de sélectionner une file d’attente où les appels sont transférés lors d’une file d’attente du groupe de réponses est plein.
  
Dans la zone Rechercher, tapez tout ou partie du nom de la file d’attente que vous souhaitez utiliser. Pour renvoyer les files d’attente disponibles, laissez vide la zone de recherche, puis cliquez sur l’icône de recherche.
  
Dans la liste des files d’attente renvoyés par la recherche, sélectionnez la file d’attente que vous souhaitez utiliser pour transférer les appels lorsqu’une file d’attente du groupe de réponses est plein.
