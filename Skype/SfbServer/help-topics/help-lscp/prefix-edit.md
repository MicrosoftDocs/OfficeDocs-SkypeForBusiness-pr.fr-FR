# <a name="prefix-edit"></a>Modification du préfixe
 
Pour définir des préfixes ou à partir de chiffres, ajouter des préfixes de numéro qui doit gérer cet itinéraire, y compris le + si nécessaire. Par exemple, vous pouvez faire correspondre tous les numéros commençant par +123.
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide de la Skype pour Business Server du Panneau de configuration, voir [Gérer les Skype pour Business Server 2015](../../manage/manage.md).
