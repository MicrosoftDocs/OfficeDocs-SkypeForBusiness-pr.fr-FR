# <a name="save-voice-routing-test-case-information"></a>Enregistrer le routage des informations sur les cas de Test de voix
 
Vous pouvez utiliser la boîte de dialogue **Enregistrer les informations de cas de Test routage voix** pour enregistrer le cas de test routage réceptiond vocale. **Enregistrer les informations de cas de Test routage voix**, tapez le nom que vous voulez donner à ce cas de test dans la zone **nom** et puis cliquez sur **OK**. 
  
> [!NOTE]
> Lorsque vous enregistrez un cas de test temporaire, vous le n'avez pas enregistré dans le magasin Central de gestion. Au lieu de cela, il rend le cas de test disponibles à partir de l’onglet **Routage des communications vocales de Test** de la page **Routage des communications vocales** . Le cas de test s’affiche avec un état de **Uncommitted**. Pour enregistrer le cas de test dans le magasin Central de gestion, vous devez valider la configuration de la voix en cliquant sur **L’ensemble de validation** à partir de n’importe quel menu **actions** dans la page **Routage des communications vocales** .
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide de la Skype pour Business Server du Panneau de configuration, voir [Gérer les Skype pour Business Server 2015](../../manage/manage.md).
