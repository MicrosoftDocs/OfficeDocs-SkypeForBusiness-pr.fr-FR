# <a name="save-voice-routing-test-case-information"></a>Enregistrer les informations de cas de test de routage des communications vocales
 
Vous pouvez utiliser la boîte de dialogue Enregistrer les informations de cas de **test** de routage des voix pour enregistrer le cas de test de routage des voix. Dans **Enregistrer les informations de cas de test de routage des communications vocales**, tapez le nom que vous souhaitez donner à ce cas de test dans **Nom** puis cliquez sur **OK**. 
  
> [!NOTE]
> Lorsque vous enregistrez un cas de test temporaire, vous ne l’enregistrez pas dans le magasin central de gestion. Au lieu de cela, le cas de test devient disponible dans l’onglet **Tester le routage des communications vocales** de la page **Routage des communications vocales**. Le cas de test apparaît avec le statut **Non validé**. Pour enregistrer le cas de test dans le magasin central  de gestion, vous devez valider la configuration vocale en cliquant sur Valider tout à partir de n’importe quel menu **Action** de la page **Routage des voix.**
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide du Panneau de Skype Entreprise Server, voir [Manage Skype Entreprise Server 2015](../../manage/manage.md).
