# <a name="normalization-rule-type-a-regular-expression"></a>Règle de normalisation : Tapez une Expression régulière
 
Tapez une expression régulière du.NET Framework pour spécifier le modèle de correspondance pour les numéros de téléphone de destination à laquelle l’itinéraire est appliqué. 
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide de la Skype pour Business Server du Panneau de configuration, voir [Gérer les Skype pour Business Server 2015](../../manage/manage.md).
