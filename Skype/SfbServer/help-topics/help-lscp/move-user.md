# <a name="move-user"></a>Déplacer un utilisateur

Dans la page **Déplacer des** utilisateurs, vous pouvez sélectionner un pool de destination vers qui déplacer l’utilisateur sélectionné.

Pour plus d’informations sur le déplacement d’utilisateurs, voir [Move Users to Another Pool](/previous-versions/office/lync-server-2013/lync-server-2013-move-users-to-another-pool).
