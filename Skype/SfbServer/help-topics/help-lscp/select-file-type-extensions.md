# <a name="select-file-type-extensions"></a>Sélectionner des extensions de type de fichier
 
Utilisez la boîte de dialogue **Sélectionner les extensions de type de fichier** pour sélectionner les extensions de types de fichiers que vous souhaitez bloquer ou autoriser en activant ou en désactivant les cases à cocher en regard des extensions sous extensions de type de **fichier**. Cliquez sur la flèche pour développer une catégorie et afficher toutes les extensions de type de fichier pour cette catégorie. Si vous voulez sélectionner toutes les extensions de type de fichier pour une catégorie, sélectionnez le nom de la catégorie. 
  
Si vous ne voyez pas l’extension correspondant au type de fichier que vous voulez sélectionner répertorié dans **extensions de type de fichier**, tapez l’extension dans la zone de texte sous **ajouter de nouvelles extensions de fichier à la liste**, puis cliquez sur **Ajouter**. 
  
Après avoir ajouté toutes les extensions de type de fichier de votre choix à la liste **extensions de type de fichier sélectionnées** , cliquez sur **OK**.
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide du panneau de configuration Skype entreprise Server, reportez-vous à la rubrique [gestion de Skype entreprise server 2015](../../manage/manage.md).
