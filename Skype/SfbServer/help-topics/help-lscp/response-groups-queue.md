# <a name="response-groups-queue"></a>File d’attente des groupes de réponses
 
Les groupes de réponse sont constitués de flux de travail, les files d’attente et les groupes d’agents. Files d’attente de la réponse groupe contiennent des appels à un groupe de réponse jusqu'à ce qu’un agent répond à l’appel. 
  
Les **Groupes de réponse** - page de**file d’attente** affiche la liste de toutes les files d’attente de réponse groupe sont définies pour votre organisation.
  
## <a name="tasks-you-can-perform"></a>Tâches que vous pouvez effectuer

Vous pouvez effectuer les tâches suivantes à partir des **Groupes de réponse** - page de**file d’attente** :
  
- Créer une nouvelle file d’attente
    
- Modifier une file d’attente existante
    
- Supprimer une file d’attente
    
## <a name="ui-reference"></a>Référence d’interface utilisateur

La liste ci-dessous décrit les commandes de la page.
  
- **Nouveau** Démarre une nouvelle file d’attente.
    
- **Modifier** Ouvre la file d’attente sélectionnée pour la modifier, sélectionne tous les de file d’attente dans la liste ou supprime la file d’attente sélectionnée.
    
- **Actualiser** Actualise la liste des files d’attente.
    
La liste ci-dessous décrit les champs de la page.
  
- **Nom** Le nom unique qui identifie la file d’attente.
    
- **Service** Le service **ApplicationServer** qui héberge la file d’attente.
    
- **Description** Description de la file d’attente.
    
Pour plus d’informations sur les fonctionnalités de groupe de la réponse de, afficher un [Plan pour l’application de groupe de réponse dans Skype pour Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/response-group.md) dans la documentation de planification. Pour plus d’informations sur l’utilisation des files d’attente du groupe de réponse, voir [Gérer les files d’attente de réponse groupe](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) dans la documentation sur les opérations.
