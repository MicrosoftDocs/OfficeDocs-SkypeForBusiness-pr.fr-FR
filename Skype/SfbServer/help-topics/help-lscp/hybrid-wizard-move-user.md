# <a name="hybrid-wizard-move-user"></a>Assistant hybride : déplacement utilisateur

Pour déplacer des comptes d’utilisateur hébergement sur un pool localement pour être hébergés sur Skype pour Business Online, utilisez l’Assistant déplacer des utilisateurs. Pour plus d’informations, voir [utiliser l’Assistant de la gestion hybride](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
