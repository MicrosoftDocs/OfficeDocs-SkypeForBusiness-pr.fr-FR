# <a name="select-groups"></a>Sélectionnez les groupes
 
Files d’attente de la réponse groupe contiennent des appels à des groupes de réponse jusqu'à ce qu’un agent répond à l’appel. Files d’attente ont affecté des groupes d’agents. Lorsque vous créez ou modifiez une file d’attente, la boîte de dialogue **Sélectionner des groupes** vous permet de sélectionner un ou plusieurs groupes d’agents à attribuer à la file d’attente.
  
## <a name="ui-reference"></a>Référence d’interface utilisateur

La liste suivante décrit les contrôles et les champs dans la boîte de dialogue **Sélectionner des groupes** .
  
- Recherche de zone de recherche pour le nom d’un groupe de l’agent. Entrez tout ou partie du nom. Renseignez la zone de recherche pour afficher tous les groupes de l’agent pour votre organisation. 
    
- **Nom** Affiche les noms des groupes agent retourné.
    
- **Stratégie de participation** Affiche la stratégie de participation définie pour le groupe de l’agent : formelle ou informelle.
    
- **Méthode de routage** Affiche le type de routage définis pour le groupe de l’agent.
    
Pour plus d’informations sur l’utilisation des files d’attente, voir [Gérer les files d’attente de réponse groupe](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) dans la documentation sur les opérations.
