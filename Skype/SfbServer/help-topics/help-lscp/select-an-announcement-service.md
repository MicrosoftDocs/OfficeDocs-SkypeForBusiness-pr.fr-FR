# <a name="select-an-announcement-service"></a>Sélectionner un service d’annonce
 
Utilisez la boîte de dialogue **Sélectionner un service d’annonce** pour sélectionner un service d’annonce qui gèrera les appels entrants destinés à une plage de numéros non attribués.
  
Dans la zone de recherche, tapez entièrement ou partiellement le nom du service d’annonce que vous souhaitez utiliser. Pour retourner tous les services d’annonce disponibles, laissez la zone de recherche vide et cliquez sur l’icône de recherche.
  
Dans la liste des services d’annonce retournés par la recherche, sélectionnez le service que vous souhaitez utiliser pour gérer les appels entrants destinés à la plage de numéros non attribués.
