# <a name="select-an-announcement-service"></a>Sélectionnez un Service d’annonce
 
Utilisez la boîte de dialogue **Sélectionner un Service d’annonce** pour sélectionner un service d’annonce qui gère les appels entrants à une plage de numéros non attribués.
  
Dans la zone Rechercher, tapez tout ou partie du nom du service annonce que vous souhaitez utiliser. Pour renvoyer tous les services d’annonce disponibles, laissez vide la zone de recherche, puis cliquez sur l’icône de recherche.
  
Dans la liste des services d’annonce renvoyés par la recherche, sélectionnez le service que vous souhaitez utiliser pour traiter les appels entrants vers la plage de numéros non attribués.
