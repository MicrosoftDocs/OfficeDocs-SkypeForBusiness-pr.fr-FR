# <a name="select-time-out-queue"></a>Sélectionnez la file d’attente du délai d’attente
 
La boîte de dialogue **Sélectionnez une file d’attente** permet de sélectionner une file d’attente où les appels sont transférés lorsqu’ils expirent.
  
Dans la zone Rechercher, tapez tout ou partie du nom de la file d’attente que vous souhaitez utiliser. Pour renvoyer les files d’attente disponibles, laissez vide la zone de recherche, puis cliquez sur l’icône de recherche.
  
Dans la liste des files d’attente renvoyés par la recherche, sélectionnez la file d’attente que vous souhaitez utiliser pour les appels d’arrivées à expiration.
