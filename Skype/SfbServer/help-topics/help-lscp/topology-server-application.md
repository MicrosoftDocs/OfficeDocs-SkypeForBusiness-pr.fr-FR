# <a name="topology-server-application"></a>Topologie : Application de serveur
 
La **topologie**: page **d’Application de serveur** affiche les applications de serveur dans votre déploiement.
  
## <a name="tasks-you-can-perform"></a>Tâches que vous pouvez effectuer

Vous pouvez effectuer les tâches suivantes de la **topologie**: page de **l’Application serveur** :
  
- Afficher les applications de serveur dans votre déploiement.
    
- Activer ou désactiver une application serveur
    
- Sélectionner ou désélectionner une application comme critique
