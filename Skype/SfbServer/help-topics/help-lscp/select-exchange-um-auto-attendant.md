# <a name="select-exchange-um-auto-attendant"></a>Sélectionnez de Standard automatique de messagerie unifiée Exchange
 
Utilisez la boîte de dialogue **Sélectionnez Exchange UM Standard automatique** pour sélectionner un Exchange UM Standard automatique qui gère les appels entrants à une plage de numéros non attribués.
  
Dans la zone Rechercher, tapez tout ou partie du nom de l’Exchange UM Standard automatique que vous souhaitez utiliser. Pour renvoyer tous les disponible Exchange UM surveillances automatiques, laissez vide la zone de recherche, puis cliquez sur l’icône de recherche.
  
Dans la liste d’Exchange UM surveillances renvoyés par la recherche, sélectionnez le Standard automatique que vous souhaitez utiliser pour traiter les appels entrants vers la plage de numéros non attribués.
