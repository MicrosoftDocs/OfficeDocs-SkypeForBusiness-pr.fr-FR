# <a name="prepare-current-forest-universal-group-location"></a>Préparer la forêt actuelle (emplacement du groupe universel)
 
Vous devez indiquer l’emplacement où vous avez besoin de créer les groupes universels dans la forêt. La page **emplacement du groupe universel** utilise le domaine à partir duquel l’ordinateur auquel vous êtes connecté est membre par défaut. Si vous préférez créer des groupes universels dans un domaine autre que le **domaine local**par défaut, sélectionnez **FQDN du domaine** et tapez le nom de domaine complet (FQDN) dans la zone de texte. La fourniture d’un autre domaine est utile lorsque vous exécutez la tâche de préparation de la forêt à partir d’un domaine autre que celui dans lequel vous souhaitez créer des groupes universels. Cliquez sur **Suivant** pour continuer.
