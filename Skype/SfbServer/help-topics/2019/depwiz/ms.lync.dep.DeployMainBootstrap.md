# <a name="setup-or-remove-skype-for-business-server-components"></a>Installer ou supprimer des composants Skype Entreprise Server
 
Pour installer et activer, ou désactiver ou désinstaller les composants Skype entreprise Server, vous devez utiliser l' **étape 2 : installer ou supprimer les composants serveur Skype**. Vous devez être connecté en tant qu’administrateur local sur l’ordinateur sur lequel vous installez ou modifiez et vous devez pouvoir lire les utilisateurs et groupes Active Directory Domain Services dans le domaine actuel. Pour commencer, cliquez sur **Exécuter**. Cette opération provoque la lecture la définition de la topologie basée sur le magasin central de gestion. Les composants logiciels nécessaires sont installés et configurés en fonction du rôle défini dans le magasin central de gestion. Lorsque l’installation est terminée, consultez le Résumé, puis cliquez sur **Terminer**.
  
> [!TIP]
> Si vous avez besoin de passer en revue les fichiers journaux créés par l’Assistant Déploiement, vous pouvez les retrouver sur l’ordinateur sur lequel l’Assistant déploiement a été exécuté, dans l’annuaire utilisateurs de l’utilisateur Active Directory qui a exécuté l’étape. Par exemple, si l’utilisateur connecté en tant qu’administrateur de domaine dans le domaine Contoso.net, les fichiers journaux se trouvent dans : > C:\Users\Administrator.Contoso\AppData\Local\Temp 
  
> [!NOTE]
> Si vous avez déjà installé les composants Skype entreprise Server sur cet ordinateur, l’Assistant déploiement le reconnaît et le bouton à l’étape 2 sera affiché comme exécuté de **nouveau**. Cela vous permet d’exécuter l’étape autant de fois que nécessaire afin de configurer ou modifier correctement le serveur.
