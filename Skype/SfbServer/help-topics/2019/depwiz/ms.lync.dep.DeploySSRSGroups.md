# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services (groupes)

Dans le champ **Spécifier un groupe en lecture seule**, spécifiez le groupe auquel vous souhaitez accorder l’accès en lecture seule aux rapports de surveillance. Utilisez le format &lt;domaine&gt;\\&lt;groupe accès&gt;. Par exemple : contoso\RTCUniversalReadOnlyAdmins.
  
Affectation d’un accès en lecture seule de groupe est facultative. Si vous ne souhaitez pas autoriser l’accès en lecture seule, vous pouvez la configurer ultérieurement directement dans SQL Server Reporting Services.
