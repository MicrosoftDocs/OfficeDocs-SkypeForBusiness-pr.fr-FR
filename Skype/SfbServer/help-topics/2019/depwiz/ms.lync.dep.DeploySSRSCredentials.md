# <a name="sql-server-reporting-services-credentials"></a>SQL Server Reporting Services (informations d’identification)
 
Pour spécifier les informations d’identification afin que SQL Server Reporting Services (SSRS) accède à la base de données de surveillance, procédez comme suit :
  
Dans la page **Spécifier des informations d’identification**, spécifiez les informations d’identification que vous souhaitez utiliser pour accéder à la base de données de surveillance (QoEMetrics et LcsCDR). 
  
> [!NOTE]
> Pour plus d’informations, voir « Spécification des informations d’identification et des informations de connexion pour les sources de données de rapport (SSRS) » sur [https://go.microsoft.com/fwlink/p/?LinkId=263431](/sql/reporting-services/report-data/specify-credential-and-connection-information-for-report-data-sources). 
  
Pour plus d’informations sur le déploiement des rapports Reporting Services pour le serveur exécutant la surveillance, voir Installer les rapports de surveillance [dans Skype Entreprise Server](../../../deploy/deploy-monitoring/install-monitoring-reports.md).
