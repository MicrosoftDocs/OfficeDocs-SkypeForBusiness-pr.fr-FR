# <a name="sql-server-reporting-services-credentials"></a>SQL Server Reporting Services (informations d’identification)
 
Pour spécifier les informations d’identification pour le SQL Server Reporting Services (SSRS) pour accéder à la base de données de surveillance, procédez comme suit :
  
Dans la **Page spécifier des informations d’identification**, spécifiez les informations d’identification utilisateur que vous souhaitez utiliser pour accéder à la base de données de surveillance (QoEMetrics et LcsCDR). 
  
> [!NOTE]
> Pour plus d’informations, voir « Spécification d’informations d’identification et connexion informations pour rapport données Sources (SSRS) » à [https://go.microsoft.com/fwlink/p/?LinkId=263431](https://go.microsoft.com/fwlink/p/?LinkId=263431). 
  
Pour plus d’informations sur le déploiement des rapports Reporting Services pour le serveur de surveillance, voir [Installer des rapports de surveillance dans Skype pour Business Server 2015](../../deploy/deploy-monitoring/install-monitoring-reports.md).
