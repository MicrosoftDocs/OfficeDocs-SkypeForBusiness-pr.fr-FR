# <a name="prepare-current-forrest-intro"></a>Préparer la forêt actuelle (introduction)
 
Pour commencer la tâche **Préparer la forêt**, cliquez sur **Suivant**. Exécution de la tâche **Préparer la forêt** crée des paramètres globaux et crée des groupes universels pour une utilisation par Skype pour Business Server 2015.
