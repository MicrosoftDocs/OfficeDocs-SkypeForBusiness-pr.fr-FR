# <a name="certificate-request-sip-domains"></a>Demande de certificat (domaines SIP)
 
Sur la page **Paramètre du domaine SIP sur les autres noms du sujet**, vous devez activer la case à cocher pour votre domaine SIP principal, ainsi que tout autre domaine SIP si votre infrastructure et votre déploiement sont conformes aux conditions répertoriées.
  
> [!NOTE]
> Vous pouvez choisir de ne sélectionner aucun des domaines SIP répertoriés. Mais nous vous recommandons de sélectionner le domaine SIP principal que vous allez utiliser. Si vous choisissez de ne sélectionner aucun des domaines SIP répertoriés et que votre configuration ou vos conditions requises changent, vous devrez demander un nouveau certificat et sélectionner les domaines SIP dont vous avez besoin pour la nouvelle configuration.
