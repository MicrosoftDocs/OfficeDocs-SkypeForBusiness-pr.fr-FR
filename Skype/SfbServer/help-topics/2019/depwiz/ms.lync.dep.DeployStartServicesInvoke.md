# <a name="start-services-invoke"></a>Démarrer les services (Invoke)
 
Le volet Résumé du volet **commandes en cours d’exécution** affiche le statut des tâches émises pour démarrer les services du serveur de rôles Skype entreprise Server que vous déployez. Le résumé dans le volet des tâches ne représente pas une indication en temps réel du démarrage du service. L’exécution du processus de démarrage initial nécessite un temps étendu pour certains services de Skype entreprise Server. Les tâches exécutent la commande de démarrage, mais n’attendent pas de savoir si le service est lancé avec succès. Si vous devez surveiller le démarrage et l’état du service, utilisez la console MMC (Microsoft Management Console) Services Windows.
  
Sous le volet des tâches, la liste déroulante affiche le fichier journal **Démarrer les services**. Pour afficher le fichier journal, cliquez sur **Afficher le journal**. Cliquez sur **Terminer** pour terminer la tâche.
