# <a name="media-bypass-planning-tool"></a>Le contournement de média (outil de planification)
 
Le contournement de média fait référence à la suppression du serveur de médiation dans le chemin d’accès des médias autant que possible pour les appels dont la signalisation traverse le serveur de médiation. Pour plus d’informations, voir [Plan for media ignorer dans Skype pour les entreprises](../../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
