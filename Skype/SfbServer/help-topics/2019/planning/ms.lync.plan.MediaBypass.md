# <a name="media-bypass-planning-tool"></a>Media Bypass (Planning Tool)
 
Le contournement de média fait référence à la suppression du serveur de médiation du chemin d’accès des médias, dans la mesure du possible, pour les appels dont la signalisation traverse le serveur de médiation. Pour plus d’informations, voir [Plan for media bypass in Skype Entreprise](../../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
