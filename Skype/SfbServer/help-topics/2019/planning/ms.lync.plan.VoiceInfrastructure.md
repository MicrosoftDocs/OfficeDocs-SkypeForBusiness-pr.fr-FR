# <a name="voice-infrastructure-planning-tool"></a>Voice Infrastructure (Planning Tool)
 
Skype entreprise Server fournit une connectivité PSTN fiable et évolutive à l’aide de Trunks SIP, de connexions SIP directes vers une passerelle PSTN ou de connexions SIP directes vers un PBX. Pour plus d’informations, reportez-vous à la rubrique [planification de la connectivité PSTN dans Skype entreprise Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md).
