# <a name="edge-network-planning-tool-in-skype-for-business"></a>Réseau de serveur Edge (outil de planification) dans Skype pour les entreprises 
 
La plupart des composants Edge sont déployés sur un réseau de périmètre. Les composants suivants forment la topologie Edge du réseau de périmètre. Sauf mention contraire, les composants font partie des [scénarios de serveur de transport Edge dans Skype pour Business Server](../../../plan-your-deployment/edge-server-deployments/scenarios.md) et se trouvent dans le réseau de périmètre. Ces composants Edge sont les suivants :
  
- Serveur(s) Edge
    
- Proxy inverse
    
- Pare-feu
    
- Directeurs (facultatif et logiquement situé sur le réseau interne)
    
- Équilibrage de la charge pour les topologies Edge mises à l’échelle (avec charge DNS équilibrée ou un équilibreur de la charge matérielle)
