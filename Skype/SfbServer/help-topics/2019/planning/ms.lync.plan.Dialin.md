# <a name="dial-in-conferencing-planning-tool"></a>Dial-in Conferencing (Planning Tool)
 
Skype pour la conférence rendez-vous Business Server permet aux utilisateurs de participer à la partie audio d’un Skype pour conférence Business Server à l’aide d’un téléphone de réseau téléphonique commuté public sans faire appel à un fournisseur de services d’audioconférence tiers.
  
Pour des informations sur la planification de l’activation de la conférence rendez-vous dans votre organisation, consultez la rubrique suivante et les rubriques auxquelles elle renvoie : 
  
- [Planifier des conférences dans Skype Business Server](../../../plan-your-deployment/conferencing/conferencing.md)
