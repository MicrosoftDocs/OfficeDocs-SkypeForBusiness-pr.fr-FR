# <a name="web-conferencing-planning-tool"></a>Conférence Web (outil de planification)
 
Skype pour les conférences web Business Server 2015 inclut la collaboration sur des documents, partage d’application et le partage du bureau.
  
> [!NOTE]
> Skype pour Business Server 2015 utilise Office Web Apps et Office Web Apps Server pour gérer le partage et le rendu des présentations PowerPoint. Fonctionnalités de conférence Web fonctionnent exactement comme ils le font dans Microsoft Lync 2013. Pour plus d’informations sur l’installation et la configuration du serveur Office Web Apps Server, voir [Configuring l’intégration avec Office Web Apps Server et Lync Server 2013](https://technet.microsoft.com/en-us/library/jj204792%28v=ocs.15%29.aspx).
