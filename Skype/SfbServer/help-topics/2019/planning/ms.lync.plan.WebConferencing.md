# <a name="web-conferencing-planning-tool"></a>Conférence Web (outil de planification)
 
La conférence Web Skype entreprise Server inclut la collaboration sur des documents, le partage d’application et le partage de bureau.
  
> [!NOTE]
> Skype entreprise Server utilise Office Web Apps et Office Web Apps Server pour gérer le partage et le rendu des présentations PowerPoint. Les fonctionnalités de conférence Web fonctionnent exactement de la même manière que dans Microsoft Lync 2013. Pour plus d’informations sur l’installation et la configuration d’Office Web Apps Server, voir Configuration de l' [intégration à Office Web Apps Server et Lync Server 2013](https://technet.microsoft.com/library/jj204792%28v=ocs.15%29.aspx).
