# <a name="skype-for-business-server-mobility-planning-tool"></a>Skype Entreprise Server outil de planification de la mobilité
 
La fonctionnalité de mobilité dans Skype Entreprise Server prend en charge Skype Entreprise fonctionnalités sur les appareils mobiles.
  
La fonctionnalité de mobilité est la même dans Skype Entreprise Server que dans Lync Server 2013. Pour plus d’informations sur la préparation de votre environnement pour la mobilité, voir [Planning for mobility in Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-mobility).
