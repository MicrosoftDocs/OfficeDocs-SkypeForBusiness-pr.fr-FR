# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
La fonctionnalité de mobilité dans Skype pour Business Server prend en charge Skype pour les fonctionnalités d’entreprise sur des appareils mobiles.
  
Fonctionnalités de mobilité sont le même dans Skype pour Business Server comme dans Lync Server 2013. Pour plus d’informations sur la préparation de votre environnement pour la mobilité, voir [Planning for mobility dans Lync Server 2013](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx).
