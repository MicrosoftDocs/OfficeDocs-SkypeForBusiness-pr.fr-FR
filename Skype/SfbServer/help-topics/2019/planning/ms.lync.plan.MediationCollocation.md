Le serveur de médiation est coqueté avec le serveur frontal par défaut. Le serveur de médiation peut également être déployé dans un pool autonome pour des raisons de performances, ou si vous déployez une trunking SIP, auquel cas le pool autonome est vivement recommandé. La cocation Skype Entreprise Server fonctionne comme dans Lync Server 2013. Pour plus de détails, consultez les rubriques suivantes :
