# <a name="add-edge-server-associations"></a>Ajouter des Associations du serveur Edge
 
Utilisez cette page pour sélectionner les pools frontaux et les serveurs Standard Edition Server utiliseront ce pool Edge a / trafic V. 
  
Chaque pool frontal peut être associé qu’un seul pool de serveurs Edge. Si vous sélectionnez un pool est déjà associé à un autre pool de serveurs Edge, il modifie l’association de pool de serveurs Edge précédente.
  
Vous pouvez également choisir d’associer ce serveur Edge ou le pool de serveurs Edge avec les serveurs de médiation ou des pools de serveur de médiation pour configurer et activer Enterprise Voice pour les utilisateurs externes.
