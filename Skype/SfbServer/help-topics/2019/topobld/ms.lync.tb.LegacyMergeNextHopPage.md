# <a name="legacy-merge-next-hop"></a>Fusion héritée : tronçon suivant
 
Le **nom de domaine complet du tronçon suivant** identifie le serveur du tronçon suivant sur lequel le serveur de périphérie hérité route le trafic interne. Entrez le nom de domaine complet (FQDN) du prochain tronçon serveur sur lequel ce serveur Edge route le trafic interne, puis sélectionnez s’il s’agit d’un **pool frontal** ou d’un **réalisateur**.
