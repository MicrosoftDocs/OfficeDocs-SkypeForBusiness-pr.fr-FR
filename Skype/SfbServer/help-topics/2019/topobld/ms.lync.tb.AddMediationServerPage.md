# <a name="add-mediation-server"></a>Ajouter un serveur de médiation
 
Pour définir le serveur de médiation, spécifiez le nom de domaine complet (FQDN) du pool de serveurs de médiation. Si vous créez une liste de serveurs de médiation, vous devez spécifier le nom du pool pour les serveurs de médiation que vous déployez. You will define the FQDNs of the individual servers in another step. Si vous déployez un serveur de médiation unique, le FQDN que vous spécifiez sera le nom de domaine complet du serveur de médiation unique. Pour plus d’informations, reportez-vous à [la rubrique déploiement d’un serveur de médiation dans le générateur de topologie de Skype entreprise Server](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
