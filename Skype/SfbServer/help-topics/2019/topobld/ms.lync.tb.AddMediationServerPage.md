# <a name="add-mediation-server"></a>Ajouter un serveur de médiation
 
Pour définir le serveur de médiation, spécifiez un nom de domaine complet (FQDN) du pool de serveurs de médiation. Si vous créez un pool de serveurs de médiation, vous devez spécifier le nom du pool pour les serveurs de médiation que vous déployez. Vous allez définir les noms de domaine complets des serveurs individuels dans une autre étape. Si vous déployez un serveur de médiation unique, le nom de domaine complet que vous spécifiez sera le nom de domaine complet du serveur de médiation unique. Pour plus d’informations, voir [déploiement d’un serveur de médiation dans le Générateur de topologie dans Skype pour Business Server](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
