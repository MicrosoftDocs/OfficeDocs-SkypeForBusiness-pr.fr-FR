# <a name="director-general-settings-expander-for-lync-server-2010"></a>Expanseur des paramètres du directeur général pour Lync Server 2010
 
Pour configurer les propriétés **générales** du directeur ou un pool directeur :
  
 **Nom de domaine complet**: tapez le nom de domaine complet du directeur ou un pool directeur
  
 **Associations**
  
 **Stocker les fichiers**: sélectionnez dans la liste le magasin de fichiers pour le directeur ou un pool directeur
  
Cliquez sur **Nouveau** pour définir un nouveau magasin de fichiers
  
Activez la case à cocher **Associer un serveur de surveillance** et sélectionnez un serveur de surveillance dans la liste
  
Cliquez sur **Nouveau** pour définir un nouveau serveur de surveillance
  
 **OK** : permet d’accepter et de valider les modifications de la boîte de dialogue.
  
 **Annuler** : permet d’annuler les modifications et de fermer la boîte de dialogue.
  
 **Aide** : permet d’afficher cet écran d’aide.
