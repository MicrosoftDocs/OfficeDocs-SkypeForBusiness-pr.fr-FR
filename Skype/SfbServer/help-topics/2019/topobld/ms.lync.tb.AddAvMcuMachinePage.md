# <a name="add-av-mcu-machine"></a>Ajouter un ordinateur MCU audio/vidéo
 
Spécifiez le nom de domaine complet (FQDN) de chaque ordinateur que vous voulez ajouter comme serveur de conférence A/V dans ce pool de conférence A/V. Après avoir ajouté un ordinateur à la liste, vous pouvez mettre à jour le nom de domaine complet de l’ordinateur ou le supprimer du pool à tout moment avant la publication de la topologie. Après la publication de la topologie, la modification du nom de domaine complet nécessite la suppression du serveur dans le générateur de topologie et l’ajout d’un nouveau serveur au pool avec le nouveau nom de domaine complet.
