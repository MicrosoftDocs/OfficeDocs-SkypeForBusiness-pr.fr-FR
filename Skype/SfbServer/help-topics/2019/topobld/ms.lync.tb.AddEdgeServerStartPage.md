# <a name="add-edge-server-support"></a>Ajouter la prise en charge du serveur Edge
 
Vous pouvez ajouter la prise en charge pour l’accès des utilisateurs externes lorsque vous déployez votre topologie initiale ou à une date ultérieure. Pour plus d’informations sur l’ajout de serveurs de périphérie à une topologie existante, voir [Define Your Edge Topology](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) dans la documentation de déploiement de serveur Edge.
