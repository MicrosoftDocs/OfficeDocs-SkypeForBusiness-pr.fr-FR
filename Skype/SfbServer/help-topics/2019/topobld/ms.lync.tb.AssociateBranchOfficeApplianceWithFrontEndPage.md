# <a name="associate-branch-office-appliance-with-front-end"></a>Associer un Branch Office Appliance à un serveur frontal
 
Pour associer l’unité de branchement ou le serveur de succursale Survivable à un pool frontal ou un serveur Standard Edition Server, dans la page **Sélectionner le pool frontal** , sélectionnez le pool frontal ou un serveur Standard Edition Server dans la liste déroulante du **pool frontal** . Cliquez sur **Suivant** pour continuer.
