# <a name="legacy-merge-sip-port"></a>Port SIP de la fusion héritée
 
Le port SIP TCP par défaut pour la communication avec les clients, les pools et serveurs frontaux hérités est le port 5061. Si la valeur par défaut n’est pas utilisée, sélectionnez **Port SIP pour tous les pools et serveurs** et tapez la nouvelle valeur.
  
Si votre déploiement hérité est configuré pour la connexion automatique du client, sélectionnez **Découverte automatique de la configuration de port SIP pour chaque pool** et cliquez sur **Suivant** pour continuer.
