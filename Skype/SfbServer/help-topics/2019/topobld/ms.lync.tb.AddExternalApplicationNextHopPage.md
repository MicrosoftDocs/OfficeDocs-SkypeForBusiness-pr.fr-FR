# <a name="add-nexthop-trusted-application-server"></a>Ajouter le serveur d’application approuvée NextHop
 
Pour configurer le tronçon suivant pour un serveur ou un pool d’applications approuvées, procédez comme suit :
  
Dans la liste déroulante **Pool du tronçon suivant**, sélectionnez le pool Enterprise Edition ou le serveur frontal Standard Edition que le pool d’applications approuvées utilisera comme tronçon suivant.
  
Cliquez sur **Terminer** pour terminer la création du pool d’applications approuvées.
