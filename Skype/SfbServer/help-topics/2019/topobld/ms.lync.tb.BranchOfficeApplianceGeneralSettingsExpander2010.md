# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Expanseur des paramètres généraux d’un Branch Office Appliance pour Lync Server 2010
 
Pour modifier les propriétés du Survivable Branch Appliance ou Survivable Branch Server, sous **Général**, vous configurez :
  
- **Nom de domaine complet**: vous tapez le nom de domaine complet du Survivable Branch Appliance ou serveur Survivable Branch Server.
    
    **Utiliser toutes les adresses IP configurées** utilise les adresses IP configurées sur le serveur Survivable Branch Appliance ou le serveur Survivable Branch Server à toutes fins.
    
    **Limiter l’utilisation des services aux adresses IP sélectionnées** : vous configurez l’adresse distincte qui définit le serveur et l’adresse IP à utiliser pour le réseau téléphonique commuté (RTC).
    
    **Adresse IP principale** : l’adresse IP est définie et configurée à toutes fins, à l’exception des fonctions associées au réseau téléphonique commuté.
    
    **Adresse IP RTC** : adresse IP associée aux fonctions du réseau téléphonique commuté.
    
- Vous configurez **Associations** pour vous assurer que tout autre serveur rôles sont configurés et associés au Survivable Branch Appliance ou un serveur Survivable Branch Server.
    
    **Associer le serveur d’archivage** Sélectionnez dans la liste, le serveur d’archivage que vous souhaitez associer le Survivable Branch Appliance ou serveur Survivable Branch Server
    
    Cliquez sur **Nouveau** si vous n’avez pas créé le serveur d’archivage que vous souhaitez associer à ce Survivable Branch Appliance ou un serveur Survivable Branch Server
    
    **Associez le serveur de surveillance** Sélectionnez dans la liste, le serveur de surveillance que vous souhaitez associer le Survivable Branch Appliance ou serveur Survivable Branch Server
    
    Cliquez sur **Nouveau** si vous n’avez pas créé le serveur de surveillance que vous souhaitez associer à ce Survivable Branch Appliance ou un serveur Survivable Branch Server
    
    **Pool Edge associé (pour les composants multimédias)** Sélectionnez dans la liste, le serveur Edge ou le pool de serveurs Edge que vous souhaitez associer le Survivable Branch Appliance ou serveur Survivable Branch Server
    
 **OK** : permet d’accepter et de valider les modifications de la boîte de dialogue.
  
 **Annuler** : permet d’annuler les modifications et de fermer la boîte de dialogue.
  
 **Aide** : permet d’afficher cet écran d’aide.
