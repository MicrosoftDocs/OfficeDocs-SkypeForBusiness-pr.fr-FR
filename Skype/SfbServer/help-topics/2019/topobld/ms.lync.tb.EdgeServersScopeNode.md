# <a name="edge-servers-scope-node"></a>Nœud d’étendue des serveurs Edge
 
Le nœud **Pool de serveurs Edge** répertorie les pools de serveurs de périphérie de votre topologie. Vous pouvez effectuer l’action suivante :
  
- **Nouveau pool de serveurs Edge**. Lance l’assistant **Définir un nouveau pool Edge** qui permet d’ajouter un nouveau pool de serveurs de périphérie à la topologie.
    
Outre les options de pool de serveur de périphérie, vous pouvez effectuer les tâches générales suivantes :
  
- **Topologie**. Vous pouvez utiliser cette option pour définir une nouvelle topologie, ouvrir une topologie existante, télécharger la topologie actuelle, enregistrer une copie de la topologie, publier la topologie, installer une base de données, fusionner des composants des versions précédentes ou supprimer le déploiement.
    
- **Afficher**. Vous pouvez utiliser cette option pour personnaliser l’affichage du Générateur de topologies en sélectionnant les éléments à afficher et à masquer dans l’arborescence de la console.
    
- **Aide**. Vous pouvez utiliser cette option pour accéder à l’aide du nœud actif.
