# <a name="publish-topology-next-step"></a>Étape ultérieure à la publication de la topologie

À l’issue de la publication de la topologie, l' **Assistant Publication** est affiché.

Lorsque vous terminez les étapes du processus de la topologie de publication, des journaux sont créés contenant des informations sur les tâches effectuées. Pour afficher un fichier journal lié à une étape donnée, sélectionnez l’étape dont vous avez besoin pour examiner le fichier journal, puis cliquez sur **afficher les journaux**.

> [!NOTE]
> Toutes les étapes du processus de topologie de publication seront associées à des fichiers journaux. Si vous sélectionnez une étape et que les **journaux d’affichage** sont disponibles, cette étape a créé un fichier journal. Si le bouton **afficher les journaux** n’est pas disponible (grisé), aucun fichier journal n’a été créé pour cette étape.

Pour consulter les étapes suivantes proposées pour terminer votre topologie, cliquez sur le lien **cliquez ici pour ouvrir la liste des tâches**. Le fichier texte qui est ouvert en cliquant sur ce lien indique les serveurs nécessaires à l’exécution de la configuration locale ainsi que les étapes supplémentaires que vous devez suivre pour terminer votre déploiement. Pour chacune des tâches répertoriées, reportez-vous à la section appropriée dans la documentation de déploiement.

Cliquez sur **Terminer** .

Pour plus d’informations, voir [publier la topologie](https://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx).
