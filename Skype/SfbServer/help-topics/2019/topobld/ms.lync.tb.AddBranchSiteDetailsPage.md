# <a name="add-branch-site-details"></a>Ajouter les détails du site de succursale
 
Pour spécifier les détails d’un site :
  
Spécifiez un nom de ville pour le nouveau site de succursale. La valeur Ville est facultative.
  
Spécifiez un nom de département/province pour le nouveau site de succursale. La valeur Département/Province est facultative.
  
Spécifiez un nom de code pays/région pour le nouveau site de succursale. La valeur Code pays/région est facultative.
  
Pour plus d’informations, voir [Sites](http://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx).
