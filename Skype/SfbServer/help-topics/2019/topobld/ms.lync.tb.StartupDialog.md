# <a name="topology-builder-startup-dialog"></a>Boîte de dialogue de démarrage du Générateur de topologie
 
Vous démarrez le Générateur de topologie pour configurer les propriétés et la configuration de votre déploiement. Au démarrage, le Générateur de topologie affiche une boîte de dialogue qui demande la source du document de topologie. Vous sélectionnez l’une des trois options :
  
- **Télécharger la topologie à partir d’un déploiement existant** Sélectionnez cette option si vous avez exécuté le Générateur de topologies et que vous avez correctement publié votre topologie sur le serveur central de gestion. Cette option demande et télécharge une copie locale de la configuration actuelle, ce qui vous permet de modifier et de configurer le déploiement existant.
    
- **Ouvrir la topologie à partir d’un fichier local** Sélectionnez cette option si vous configurez un déploiement mais que vous n’avez pas encore publié la configuration. Cette option vous permet d’enregistrer une copie locale du travail de configuration dans le Générateur de topologie et d’y revenir ultérieurement.
    
- **Nouvelle topologie** Sélectionnez cette option pour commencer la configuration d’un nouveau document de topologie. Cette option ouvre un nouveau document de topologie que vous configurez pour répondre aux exigences du déploiement que vous configurerez.
