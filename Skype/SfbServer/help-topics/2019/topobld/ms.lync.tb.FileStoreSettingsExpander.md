# <a name="file-store-settings-expander"></a>Expandeur des paramètres du magasin de fichiers
 
Pour modifier les propriétés des magasins de fichiers définis :
  
Pour le nom du partage de fichiers, vous pouvez définir le nom d’un nouveau partage. Vous devrez ensuite créer ce partage sur le serveur qui est nommé dans la propriété FQDN du serveur de fichiers.
  
> [!NOTE]
> Vous ne pouvez pas modifier le nom de domaine complet du serveur qui est spécifié dans la propriété FQDN du serveur de fichiers. Si vous devez placer le partage de fichiers du magasin de fichiers sur un autre serveur, créez une nouvelle entrée de magasin de fichiers.
