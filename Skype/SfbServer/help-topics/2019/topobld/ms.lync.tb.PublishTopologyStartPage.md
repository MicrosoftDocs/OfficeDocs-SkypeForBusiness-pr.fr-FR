# <a name="publish-topology"></a>Publier la topologie
 
La page **publier la topologie** s’affiche lorsque vous commencez le processus de publication. Lisez et vérifiez que les tâches sont terminées avant de poursuivre. Lorsque vous êtes prêt à continuer, cliquez sur **suivant**. Si vous devez fermer le processus de publication, cliquez sur **Annuler**.
