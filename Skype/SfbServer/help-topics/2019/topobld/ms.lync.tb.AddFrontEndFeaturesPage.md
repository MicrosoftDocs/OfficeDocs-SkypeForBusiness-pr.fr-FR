# <a name="add-front-end-features"></a>Ajouter des fonctionnalités frontales

Sélectionnez chaque fonctionnalité que vous voulez activer pour le pool frontal. Au minimum, vous devez déployer la messagerie instantanée (IM), qui est toujours activée et ne peut pas être désélectionnée. Vous avez également la possibilité de déployer d’autres fonctionnalités, telles que des conférences, des voix entreprise et des contrôles d’admission des appels (CAC). Certaines fonctionnalités dépendent de votre topologie.

- Conférence

- Voix Entreprise

- Contrôle d’admission des appels (CAC)

- Archivage

- Surveillance

> [!NOTE]
> Si vous voulez activer le CAC dans votre déploiement, vous devez activer le CAC dans exactement un pool par site central. Nous vous recommandons d’utiliser le CAC si vous déployez des fonctions vocales ou des conférences A/V.

Pour plus d’informations sur la spécification de fonctionnalités pour un pool Enterprise Edition, voir [définir et configurer un pool frontal](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) dans la documentation de déploiement.
