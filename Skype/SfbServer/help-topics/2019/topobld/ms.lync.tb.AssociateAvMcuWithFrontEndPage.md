# <a name="associate-av-mcu-with-front-end"></a>Associer le service MCU audio/vidéo au serveur frontal
 
Tous les pools de serveurs frontaux Enterprise Edition dont la fonction de conférence est activée doivent disposer d’un service de conférence A/V colocalisé ou bien doivent utiliser un pool de conférence A/V autonome. Tous les pools de serveurs frontaux Enterprise Edition ayant un service de conférence A/V colocalisé peuvent être sélectionnés pour utiliser ce nouveau pool de conférence A/V autonome.
