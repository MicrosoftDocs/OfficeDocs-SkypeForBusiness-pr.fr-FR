# <a name="add-director-features"></a>Ajouter des fonctionnalités de directeur
 
Si vous souhaitez que le pool directeur pour inclure le serveur de surveillance, sélectionnez Activer la **surveillance (CDR et mesures QoE)**.
  
Pour plus d’informations sur l’ajout d’un pool directeur, voir [Setting Up the Director](http://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)et les [scénarios pour le directeur](http://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx) .
