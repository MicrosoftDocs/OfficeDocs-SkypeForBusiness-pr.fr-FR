# <a name="add-director-features"></a>Ajouter des fonctionnalités de directeur

Si vous voulez que le pool de directeurs inclue la surveillance du serveur, sélectionnez **surveillance (CdR et QoE métriques)**.

Pour plus d’informations sur l’ajout d’un pool de réalisateurs, voir [scénarios pour le directeur](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx) et [configuration du réalisateur](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx).
