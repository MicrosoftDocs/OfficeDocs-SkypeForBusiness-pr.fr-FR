# <a name="legacy-merge-edge-type"></a>Fusion hérité : type de serveur Edge
 
Pour un **Serveur Edge unique** ou un serveur **Edge consolidé à charge équilibrée**, cliquez sur **Suivant** pour indiquer les informations de configuration. Pour un **serveur Edge étendu,** la configuration de chaque serveur doit être modifiée en configuration Edge consolidée avant de pouvoir être fusionnée.
