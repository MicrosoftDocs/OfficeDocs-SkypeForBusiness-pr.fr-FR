# <a name="legacy-merge-edge-type"></a>Fusion hérité : type de serveur Edge
 
Pour un **Serveur de périphérie unique** ou **à charge équilibrée serveur Edge consolidé** du serveur, cliquez sur **suivant** pour spécifier les informations de configuration. Pour un **Bord développé**, configuration de chaque serveur doit être modifiée pour une configuration du serveur Edge consolidée avant qu’il peut être fusionné.
