# <a name="legacy-merge-edge-type"></a>Fusion hérité : type de serveur Edge
 
Dans le cas d’un serveur **Edge unique** ou d’un serveur **de périphérie consolidé équilibré** , cliquez sur **suivant** pour spécifier les informations de configuration. Pour un **bord étendu**, la configuration de chaque serveur doit être modifiée en une configuration de périphérie consolidée avant de pouvoir être fusionnée.
