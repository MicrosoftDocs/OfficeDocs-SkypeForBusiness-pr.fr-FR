# <a name="add-archiving-server"></a>Ajouter un serveur d’archivage
 
Pour déployer un serveur d’archivage dans votre conception de la topologie, vous devez spécifier le nom de domaine complet du serveur sur lequel vous voulez déployer le serveur d’archivage. Avant de publier une topologie qui inclut l’archivage et installation Skype pour Business Server, vous devez avoir effectué toutes les conditions préalables au déploiement d’archivage.
