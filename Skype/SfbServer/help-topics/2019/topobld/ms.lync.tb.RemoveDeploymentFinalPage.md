# <a name="remove-deployment"></a>Suppression du déploiement
 
Cliquez sur **Terminer** pour créer une topologie vide en guise de première étape de mise hors service de l’ensemble de votre déploiement.
  
> [!CAUTION]
> Cette action ne peut pas être annulée.
