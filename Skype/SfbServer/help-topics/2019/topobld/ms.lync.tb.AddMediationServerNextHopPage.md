# <a name="add-mediation-server-nexthop"></a>Ajouter un tronçon suivant pour le serveur de médiation
 
Pour définir le saut suivant pour un serveur de médiation que vous définissez, sélectionnez, dans la liste de listes suivantes du pool du **saut** suivant, le serveur Enterprise Edition ou le serveur Standard Edition Server qui fera le saut suivant pour le serveur de médiation. Cliquez sur **Suivant** pour continuer.
