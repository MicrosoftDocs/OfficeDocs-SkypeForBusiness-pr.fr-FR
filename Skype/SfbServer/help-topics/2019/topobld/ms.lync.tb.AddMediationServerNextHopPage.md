# <a name="add-mediation-server-nexthop"></a>Ajouter un tronçon suivant pour le serveur de médiation
 
Pour définir le tronçon suivant pour un serveur de médiation que vous définissez, sélectionnez dans la liste déroulante **pool du tronçon suivant** du serveur Enterprise Edition server ou un serveur Standard Edition server qui jouera le tronçon suivant du serveur de médiation. Cliquez sur **Suivant** pour continuer.
