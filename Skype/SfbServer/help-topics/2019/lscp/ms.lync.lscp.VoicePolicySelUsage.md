# <a name="select-pstn-usage-record"></a>Sélectionner un enregistrement d’utilisation PSTN
 
Utilisez la boîte de dialogue **Sélectionner un enregistrement d’utilisation RTC** pour rechercher un enregistrement d’utilisation de réseau téléphonique public commuté (RTC) existant que vous pouvez utiliser.
  
> [!NOTE]
> Pour récupérer tous les enregistrements d’utilisation RTC disponibles, n’entrez rien dans la zone de recherche, puis cliquez sur l’icône de recherche.
