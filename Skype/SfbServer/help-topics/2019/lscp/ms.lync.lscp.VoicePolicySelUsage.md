# <a name="select-pstn-usage-record"></a>Sélectionner un enregistrement d’utilisation PSTN
 
Utilisez la boîte de dialogue **Sélectionner un enregistrement d’utilisation PSTN** pour trouver un enregistrement d’utilisation réseau téléphonique commuté public existant que vous pouvez utiliser.
  
> [!NOTE]
> Pour récupérer tous les enregistrements d’utilisation PSTN disponibles, ne tapez rien dans la zone de recherche et puis cliquez sur l’icône de recherche.
