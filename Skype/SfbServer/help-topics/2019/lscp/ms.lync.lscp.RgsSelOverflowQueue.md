# <a name="select-overflow-queue"></a>Sélectionnez la file d’attente de débordement
 
Utilisez la boîte de dialogue **Sélectionner une file d’attente de débordement** pour sélectionner une file d’attente dans laquelle les appels seront transférés lorsqu’une file d’attente de response group est pleine.
  
Dans la zone Rechercher, tapez entièrement ou partiellement le nom de la file d’attente que vous souhaitez utiliser. Pour retourner toutes les files d’attente disponibles, renseignez la zone de recherche, cliquez sur l’icône de recherche.
  
Dans la liste des files d’attente retournées par la recherche, sélectionnez la file d’attente que vous souhaitez utiliser pour transférer les appels lorsqu’une file d’attente de response group est pleine.
