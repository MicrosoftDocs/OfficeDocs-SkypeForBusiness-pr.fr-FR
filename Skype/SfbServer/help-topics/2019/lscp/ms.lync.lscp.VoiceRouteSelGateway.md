# <a name="select-trunk"></a>Sélectionner une jonction
 
Utilisez la boîte de dialogue **Sélectionner un Trunk** pour rechercher un Trunk existant que vous pouvez utiliser.
  
> [!NOTE]
> Pour récupérer tous les circuits disponibles, n’entrez rien dans la zone de recherche, puis cliquez sur l’icône Rechercher.
