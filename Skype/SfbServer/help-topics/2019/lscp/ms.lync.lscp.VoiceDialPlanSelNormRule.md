# <a name="select-normalization-rules"></a>Sélectionner des règles de normalisation
 
Les règles de normalisation utilisent .NET Framework expressions régulières pour traduire les numéros de téléphone composés au format E.164. Une ou plusieurs règles de normalisation doivent être affectées à chaque plan de numérotation.
  
Pour utiliser une règle de normalisation existante, cliquez sur une règle dans la liste, puis sur **OK**.
