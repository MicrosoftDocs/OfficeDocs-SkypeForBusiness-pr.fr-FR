# <a name="select-from-active-directory"></a>Sélectionner à partir d’Active Directory
 
Utilisez la boîte de dialogue Sélectionner à partir **d’Active Directory** pour rechercher les utilisateurs que vous souhaitez activer pour Skype Entreprise Server. Dans la zone de recherche, tapez l’adresse SIP ou le nom d’affichage de l’utilisateur que vous souhaitez trouver. Pour récupérer tous les utilisateurs disponibles, ne tapez rien dans la zone de recherche.
