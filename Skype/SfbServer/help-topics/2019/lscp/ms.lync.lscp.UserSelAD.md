# <a name="select-from-active-directory"></a>Sélection à partir d’Active Directory
 
Utilisez la boîte de dialogue **Sélectionner à partir d’Active Directory** pour rechercher les utilisateurs que vous voulez activer pour Skype entreprise Server. Dans la zone de recherche, tapez l’adresse SIP ou le nom d’affichage de l’utilisateur recherché. Pour extraire tous les utilisateurs disponibles, ne tapez rien dans la zone de recherche.
