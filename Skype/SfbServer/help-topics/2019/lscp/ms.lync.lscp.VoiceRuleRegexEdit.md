# <a name="translation-rule-type-a-regular-expression"></a>Règle de conversion : saisir une expression régulière
 
Dans le champ **suivre ce modèle** , spécifiez le modèle qui sera utilisé pour correspondre aux numéros à traduire. Dans le champ **règle de traduction** , spécifiez un modèle pour le format des numéros traduits. Par exemple, si vous entrez ^\+(\d{9}\d+)$ dans le champ de **correspondance de ce modèle** et 011$ 1 dans le champ **règle de traduction** , la règle traduira le numéro + 441235551010 comme suit, 011441235551010.
