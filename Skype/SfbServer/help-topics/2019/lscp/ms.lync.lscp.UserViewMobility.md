# <a name="users-view-mobility-policy"></a>Utilisateurs : Afficher une stratégie de mobilité
 
La boîte de dialogue **affichage – stratégie de mobilité** affiche les propriétés de la stratégie de mobilité sélectionnée.
  
Pour plus d’informations sur l’attribution de stratégies de mobilité, voir [attribuer une stratégie de mobilité par utilisateur](http://technet.microsoft.com/library/d8bf997f-4bc7-48d3-973b-323505f55e9d.aspx) et la [Configuration de la stratégie de mobilité](http://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx).
