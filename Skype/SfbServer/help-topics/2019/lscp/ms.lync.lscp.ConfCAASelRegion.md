# <a name="select-regions"></a>Sélection des régions
 
Utilisez la boîte de dialogue **Sélectionner des régions** pour rechercher les régions disponibles dans votre environnement. Pour utiliser une région existante, sélectionnez un site dans la liste, puis cliquez sur **OK**.
  
> [!NOTE]
> Pour extraire toutes les régions disponibles, ne tapez rien dans la zone de recherche, puis cliquez sur l’icône de recherche.
