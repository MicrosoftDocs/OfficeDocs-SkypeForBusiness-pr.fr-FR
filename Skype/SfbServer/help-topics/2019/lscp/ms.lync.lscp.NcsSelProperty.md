# <a name="assign-properties"></a>Affecter des propriétés
 
Utilisez la boîte de dialogue **Affecter des propriétés** pour spécifier une description et une ID de site réseau pour plusieurs sous-réseaux à la fois. Dans **Description**, tapez la description que vous souhaitez utiliser pour les sous-réseaux que vous avez sélectionnés. Dans **ID de site réseau**, sélectionnez l’ID de site réseau que vous voulez affecter aux sous-réseaux sélectionnés. Cliquez sur **OK** pour affecter ces propriétés aux sous-réseaux sélectionnés.
