# <a name="move-all-users"></a>Déplacement de tous les utilisateurs

Dans la **boîte de** dialogue Déplacer des utilisateurs, vous pouvez sélectionner les pools source et de destination pour déplacer des comptes d’utilisateurs d’un pool vers un autre dans votre déploiement.

Pour plus d’informations sur le déplacement d’utilisateurs, voir [Move Users to Another Pool](/previous-versions/office/lync-server-2013/lync-server-2013-move-users-to-another-pool).
