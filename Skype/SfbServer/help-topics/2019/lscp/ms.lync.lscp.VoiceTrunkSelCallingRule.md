1. [Règles de traduction dans Skype entreprise Server](../../../plan-your-deployment/enterprise-voice-solution/translation-rules.md)

2. [Trunk Configuration and Translation Rules](https://technet.microsoft.com/library/0c339511-a185-484e-94f0-dbe918b7e48a.aspx)

3. [Création ou modification d’une règle de traduction pour appeler une présentation dans Skype entreprise Server](../../../deploy/deploy-enterprise-voice/called-id-presentation-rules.md)
