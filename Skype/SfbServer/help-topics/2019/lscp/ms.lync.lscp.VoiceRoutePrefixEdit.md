# <a name="prefix-edit"></a>Edition du préfixe
 
Pour définir les préfixes ou à partir des chiffres, ajouter des préfixes des numéros qui doit gérer cet itinéraire, y compris le + si nécessaire. Par exemple, vous pouvez associer tous les numéros commençant par +123.
  
Pour plus d’informations sur les différentes procédures que vous pouvez effectuer à l’aide de la Skype pour le panneau de configuration serveur Business, voir [Gérer les Skype pour Business Server 2015](../../manage/manage.md).
