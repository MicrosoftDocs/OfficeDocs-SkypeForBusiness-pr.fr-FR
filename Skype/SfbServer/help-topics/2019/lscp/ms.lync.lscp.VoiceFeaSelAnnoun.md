# <a name="select-an-announcement-service"></a>Sélectionner un Service d’annonce
 
Utilisez la boîte de dialogue **Sélectionner un Service d’annonce** pour sélectionner un service d’annonce qui gèrera les appels entrants à une plage de numéros non attribués.
  
Dans la zone Rechercher, tapez entièrement ou partiellement le nom du service d’annonce que vous souhaitez utiliser. Pour renvoyer tous les services d’annonce disponibles, renseignez la zone de recherche, cliquez sur l’icône de recherche.
  
Dans la liste des services d’annonce retournés par la recherche, sélectionnez le service que vous souhaitez utiliser pour gérer les appels entrants vers la plage de numéros non attribués.
