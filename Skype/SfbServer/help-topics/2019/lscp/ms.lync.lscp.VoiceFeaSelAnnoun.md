# <a name="select-an-announcement-service"></a>Sélectionner un service d’annonce
 
Utilisez la boîte de dialogue **Sélectionner un service d’annonce** pour sélectionner un service d’annonce qui traitera les appels entrants dans une plage de numéros non attribués.
  
Dans la zone de recherche, tapez tout ou partie du nom du service d’annonce que vous souhaitez utiliser. Pour renvoyer tous les services d’annonce disponibles, laissez vide la zone de recherche et cliquez sur l’icône Rechercher.
  
Dans la liste des services d’annonce renvoyés par la recherche, sélectionnez le service que vous souhaitez utiliser pour gérer les appels entrants vers la plage de numéros non attribués.
