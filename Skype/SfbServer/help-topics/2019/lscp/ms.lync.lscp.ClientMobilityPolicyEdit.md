# <a name="edit-client-mobility-policy"></a>Modifier la stratégie de mobilité client

Dans la boîte de dialogue **Modifier la stratégie de mobilité**, vous pouvez modifier les détails d’une stratégie de mobilité.

## <a name="ui-reference"></a>Référence d’interface utilisateur

Les listes suivantes décrivent les menus, commandes, champs et propriétés de la page.


- **Commit** Cliquez pour enregistrer les modifications apportées aux paramètres de stratégie.

- **Annuler** Cliquez pour arrêter la modification de la stratégie et ignorer les modifications apportées aux paramètres de stratégie.

- **Nom** Nom de la stratégie de mobilité.

- **Description** Description de la stratégie de mobilité.

Pour plus d’informations sur les stratégies de mobilité, voir [Mobility Policies](/previous-versions/office/lync-server-2013/lync-server-2013-mobility-policies).
