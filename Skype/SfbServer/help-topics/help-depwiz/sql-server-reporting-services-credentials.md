# <a name="sql-server-reporting-services-credentials"></a>SQL Server Reporting Services (informations d’identification)
 
Pour spécifier les informations d’identification pour SQL Server Reporting Services (SSRS) pour accéder à la base de données de surveillance, procédez comme suit:
  
Dans la **page spécifier les informations d’identification**, spécifiez les informations d’identification de l’utilisateur que vous voulez utiliser pour accéder à la base de données de surveillance (QoEMetrics et LcsCDR). 
  
> [!NOTE]
> Pour plus d’informations, consultez la section «Spécification des informations d’identification et de connexion pour les [https://go.microsoft.com/fwlink/p/?LinkId=263431](https://go.microsoft.com/fwlink/p/?LinkId=263431)sources de données de rapport (SSRS)». 
  
Pour plus d’informations sur le déploiement des rapports Reporting Services pour le serveur qui exécute la surveillance, voir [installer des rapports de surveillance dans Skype entreprise server 2015](../../deploy/deploy-monitoring/install-monitoring-reports.md).
