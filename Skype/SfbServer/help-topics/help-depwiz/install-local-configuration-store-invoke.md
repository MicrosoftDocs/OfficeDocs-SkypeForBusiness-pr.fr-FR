# <a name="install-local-configuration-store-invoke"></a>Magasin de Configuration Local Installation (appeler)
 
Le volet de **Commandes d’exécution** affiche l’état des tâches pour l’installation du réplica local en lecture seule du magasin Central de gestion. Pendant l’exécutant des tâches, le volet Office est mis à jour lorsque les commandes sont exécutées. Vous devez consulter le statut de la tâche et notez les erreurs ou les avertissements qui sont affichent. Lorsque les tâches sont terminées, le message **état de la tâche : terminé** s’affiche, et la liste déroulante pour les fichiers journaux affiche les noms des fichiers journaux. Pour afficher les fichiers journaux qui ont été créés, cliquez sur un fichier journal à afficher, puis cliquez sur **Afficher le journal**. Passez en revue tous les fichiers journaux pour la réussite, les erreurs et les messages d’avertissement.
  
Après avoir lu les fichiers journaux, cliquez sur **Terminer** pour terminer l’installation de la configuration locale.
