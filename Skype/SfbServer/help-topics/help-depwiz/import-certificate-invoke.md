# <a name="import-certificate-invoke"></a>Importer le certificat (appeler)
 
Sur la page **En cours d’exécution de commandes** , passez en revue les informations concernant le certificat importé. Dans la liste déroulante, sélectionnez le fichier de journal d’importation de certificat. Assurez-vous que le fichier journal de certificat d’importation est visible dans la liste déroulante. Cliquez sur **Afficher le journal** pour consulter les détails de l’importation du certificat. Lorsque vous avez fini de consulter le résumé et le fichier journal, cliquez sur **Terminer**.
