# <a name="certificate-request-geography"></a>Demande de certificat (géographique)
 
La page **Informations géographiques** demande des informations supplémentaires pour la demande de certificat. Sur cette page, vous sélectionnez les **pays**et vous tapez Département **** et **Ville/localité** dans une zone de texte. Notez que vous n’employez pas d’abréviations et devez spécifier le nom complet du département ou de province et Ville/localité.
