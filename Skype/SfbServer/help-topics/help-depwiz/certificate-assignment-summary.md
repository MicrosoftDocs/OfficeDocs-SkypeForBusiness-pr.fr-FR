# <a name="certificate-assignment-summary"></a>Assignation de certificat (résumé)
 
Avant la fin de l’assignation du certificat, une liste **Résumé de l’affectation du certificat** comprenant les détails du certificat à assigner vous est présentée. Passez ces informations en revue afin de vous assurer que le nom du sujet et les autres noms du sujet sont corrects. Lorsque vous avez terminé de passer en revue le résumé, cliquez sur **Suivant**.
