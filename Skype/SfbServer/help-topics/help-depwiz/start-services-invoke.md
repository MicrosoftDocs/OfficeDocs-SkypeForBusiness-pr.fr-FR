# <a name="start-services-invoke"></a>Démarrer les services (Invoke)
 
Le volet Résumé dans le volet de **L’exécution des commandes** affiche l’état des tâches émis pour démarrer les services pour le Skype pour serveur de rôle Business Server 2015 que vous déployez. Le résumé dans le volet des tâches ne représente pas une indication en temps réel du démarrage du service. Parmi le Skype pour services Business Server peut prendre longtemps pour commencer le processus de démarrage initial. Les tâches exécutent la commande de démarrage, mais n’attendent pas de savoir si le service est lancé avec succès. Si vous devez surveiller le démarrage et l’état du service, utilisez la console MMC (Microsoft Management Console) Services Windows.
  
Sous le volet des tâches, la liste déroulante affiche le fichier journal **Démarrer les services**. Pour afficher le fichier journal, cliquez sur **Afficher le journal**. Cliquez sur **Terminer** pour terminer la tâche.
