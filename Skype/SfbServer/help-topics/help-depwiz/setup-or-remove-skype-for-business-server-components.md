# <a name="setup-or-remove-skype-for-business-server-components"></a>Installer ou supprimer des composants Skype Entreprise Server
 
Pour installer et activer, désactiver ou désinstaller Skype pour les composants Business Server 2015, vous utilisez **étape 2 : installer ou supprimer des composants serveur Skype**. Vous devez être connecté dans en tant qu’administrateur local sur l’ordinateur que vous êtes l’installation ou la modification et doit être capable de lire les utilisateurs des Services de domaine Active Directory et les groupes dans le domaine actuel. Pour commencer, cliquez sur **Exécuter**. Cette opération provoque la lecture la définition de la topologie basée sur le magasin central de gestion. Les composants logiciels nécessaires sont installés et configurés en fonction du rôle défini dans le magasin central de gestion. Lorsque l’installation est terminée, consultez le Résumé, puis cliquez sur **Terminer**.
  
> [!TIP]
> Si vous avez besoin passer en revue les fichiers journaux qui sont créés par l’Assistant déploiement, vous pouvez trouver les sur l’ordinateur où a été exécuté l’Assistant déploiement, dans le répertoire des utilisateurs de l’utilisateur Active Directory qui a exécuté l’étape. Par exemple, si l’utilisateur est connecté en tant qu’administrateur de domaine dans le domaine Contoso.net, les fichiers journaux se trouvent dans : > C:\Users\Administrator.Contoso\AppData\Local\Temp 
  
> [!NOTE]
> Si vous avez déjà installé Skype pour les composants Business Server 2015 sur cet ordinateur, l’Assistant Déploiement reconnaît cela, et le bouton à l’étape 2 s’affichera comme **Réexécuter**. Cela vous permet d’exécuter l’étape autant de fois que nécessaire afin de configurer ou modifier correctement le serveur.
