# <a name="setup-or-remove-skype-for-business-server-components"></a>Installer ou supprimer des composants Skype Entreprise Server
 
Pour installer et activer, désactiver ou désinstaller Skype pour les composants Business Server 2015, vous utilisez **étape 2 : installation ou suppression des composants serveur Skype**. Vous devez être connecté dans en tant qu’administrateur local sur l’ordinateur que vous utilisez l’installation ou la modification et doit être capable de lire des utilisateurs de Services de domaine Active Directory et des groupes dans le domaine actuel. Pour commencer, cliquez sur **Exécuter**. Cette opération provoque la lecture la définition de la topologie basée sur le magasin central de gestion. Les composants logiciels nécessaires sont installés et configurés en fonction du rôle défini dans le magasin central de gestion. Lorsque l’installation est terminée, consultez le Résumé, puis cliquez sur **Terminer**.
  
> [!TIP]
> Si vous devez passer en revue les fichiers journaux qui sont créés par l’Assistant de déploiement, vous les trouverez sur l’ordinateur où l’Assistant de déploiement a été exécuté, dans le répertoire des utilisateurs de l’utilisateur Active Directory qui a exécuté l’étape. Par exemple, si l’utilisateur connecté en tant qu’administrateur de domaine dans le domaine Contoso.net, les fichiers journaux se trouvent dans : > C:\Users\Administrator.Contoso\AppData\Local\Temp 
  
> [!NOTE]
> Si vous avez déjà installé Skype pour les composants Business Server 2015 sur cet ordinateur, l’Assistant de déploiement reconnaît cela, et s’affiche sous la forme **d’Exécuter à nouveau**le bouton à l’étape 2. Cela vous permet d’exécuter l’étape autant de fois que nécessaire afin de configurer ou modifier correctement le serveur.
