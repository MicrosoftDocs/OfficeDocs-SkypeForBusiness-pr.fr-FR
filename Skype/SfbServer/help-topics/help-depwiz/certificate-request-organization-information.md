# <a name="certificate-request-organization-information"></a>Demande de certificat (informations sur l’organisation)
 
Le volet **d’Informations de l’organisation** vous permet de spécifier le nom de votre organisation et une unité d’organisation (UO) (généralement la division ou département) qui demande le certificat. Les informations que vous tapez dans les zones de texte **d’Unité d’organisation** et de l' **organisation** apparaissent sur le certificat et permettant d’identifier qui a demandé le certificat, le certificat est affiché par d’autres utilisateurs.
  
> [!NOTE]
> Le nom de l’unité d’organisation sur un certificat n’est pas nécessairement identique à celui d’une unité d’organisation dans les Services de domaine Active Directory. En fonction de la façon dont vous avez configuré Active Directory, le nom que vous utilisez peut ou ne peut pas être le même.
