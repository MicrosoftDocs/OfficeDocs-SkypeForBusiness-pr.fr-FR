# <a name="certificate-request-subject-alternate-names"></a>Demande de certificat (noms alternatifs du sujet)
 
Le **nom de l’objet / noms d’objet alternatifs** page présente un récapitulatif du nom du sujet et de tous les noms d’objet alternatifs qui rempliront le certificat. Les noms présentés pour le nom du sujet et les noms d’objet alternatifs sont collectées à partir du document de définition, telle qu’elle a été créée lorsque vous avez défini le déploiement dans le Générateur de topologies.
  
Si vous devez modifier le nom du sujet ou des noms d’objet alternatifs, cliquez sur **Annuler** pour quitter l’Assistant demande de certificat et exécutez à nouveau le Générateur de topologies. Puis le republier la nouvelle configuration avec le sujet correctement défini et les noms d’objet alternatifs.
