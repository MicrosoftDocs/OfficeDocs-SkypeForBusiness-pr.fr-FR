# <a name="certificate-request-invoke"></a>Demande de certificat (appeler)
 
La page de **L’exécution de commandes** affiche un volet d’état que les commandes à exécuter pour la génération de la demande de certificat. Une fois les tâches terminées, la liste déroulante située sous le volet de résumé contienne une liste des fichiers journaux. Utilisez la liste déroulante pour sélectionner un fichier journal à afficher, puis cliquez sur **Afficher le journal**. Internet Explorer démarre et affiche le fichier journal. Dans le cas d’une demande ayant échoué, vous devez examiner les fichiers journaux pour déterminer la cause de l’échec. Vous pouvez exécuter à nouveau la demande de certificat pour résoudre le problème, corriger le problème indiqué dans le fichier journal. Continuer bien que les pages de demande de certificat et de demande de certificat à nouveau.
