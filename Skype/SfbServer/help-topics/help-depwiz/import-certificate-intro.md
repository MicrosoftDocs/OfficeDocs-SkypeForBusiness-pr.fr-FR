# <a name="import-certificate-intro"></a>Certificat d’importation (introduction)
 
Pour importer un certificat, vous devez fournir un chemin d’accès du fichier de certificat. Dans la zone de texte **fichier de sélectionner un certificat** , vous pouvez taper le chemin d’accès complet et le nom de fichier, ou cliquez sur le bouton **Parcourir** et naviguez jusqu'à l’emplacement du chemin d’accès et le nom du fichier (en général, un fichier .p7b, .pfx ou .cer).
  
Si le certificat contient une clé privée, activez la case à cocher **fichier de certificat contient la clé privée du certificat**. Lorsque cette case à cocher est sélectionnée, la saisie de texte de **mot de passe** est activée. Si vous possédez un certificat avec une clé privée associée, un mot de passe est généralement placée sur la clé privée lors de la création du certificat. Vous entrez le mot de passe pour la clé privée pour permettre le certificat et la clé privée à importer dans le magasin de certificats. Lorsque vous avez fourni les informations pour le chemin d’accès du fichier de certificat et, éventuellement, le mot de passe clé privé, si nécessaire, cliquez sur **suivant**.
  
> [!IMPORTANT]
> Si vous ne connaissez pas le mot de passe pour la clé privée, l’importation échouera.
