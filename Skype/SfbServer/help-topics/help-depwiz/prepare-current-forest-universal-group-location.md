# <a name="prepare-current-forest-universal-group-location"></a>Préparer la forêt de courant (emplacement du groupe universel)
 
Vous devez indiquer où vous devez créer les groupes universels de la forêt. La page **Emplacement du groupe universel** utilise le domaine dans lequel vous avez ouvert une session sur l’ordinateur est un membre par défaut. Si vous préférez créer les groupes universels dans un domaine autre que le **domaine Local**par défaut, sélectionnez **Domaine** et tapez le nom de domaine pleinement qualifié (FQDN) dans la zone de texte. Fourniture de qu'un domaine de remplacement est utile lorsque vous exécutez la tâche de préparation de la forêt à partir d’un domaine différent où vous souhaitez que les groupes universels d’être créé. Cliquez sur **Suivant** pour continuer.
