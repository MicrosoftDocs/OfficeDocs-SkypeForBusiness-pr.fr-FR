# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Installer ou supprimer des composants Skype Entreprise Server (Introduction)
 
Vous vous préparez à installer et activer ou désactiver et désinstaller des composants Skype Entreprise Server 2015. Les processus qui s’exécutent sont définis par la configuration actuelle de la définition publiée dans le magasin de gestion centrale. Pour effectuer correctement les tâches, vous devez être connecté en tant que membre du groupe Administrateurs local et, au minimum, être en mesure de lire les services de domaine Active Directory.
  
Cliquez sur **Suivant** pour passer à l’écran **Exécution de commandes**.
