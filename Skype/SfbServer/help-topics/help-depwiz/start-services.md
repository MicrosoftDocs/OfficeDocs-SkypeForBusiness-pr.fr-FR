# <a name="start-services"></a>Démarrer les Services
 
Pour démarrer les services sur le rôle de serveur de déploiement, cliquez sur **exécuter** dans **étape 4 : démarrez les Services**. Vous devez être connecté en tant que membre du groupe Administrateurs local ou équivalent. 
  
> [!NOTE]
> Si vous **Démarrez les Services** a été exécuté, vous pouvez réexécuter la commande en cliquant sur **Exécuter à nouveau**.
