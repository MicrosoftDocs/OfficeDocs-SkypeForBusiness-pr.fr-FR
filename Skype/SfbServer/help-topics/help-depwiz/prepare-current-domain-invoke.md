# <a name="prepare-current-domain-invoke"></a>Préparer le domaine actuel (appeler)
 
La page de **L’exécution de commandes** affiche un résumé des tâches en cours d’exécution. Consultez l’écran récapitulatif et assurez-vous que les tâches ont réussi, et qu’il n’y a pas d’erreurs. Examinez les avertissements pour vous assurer qu’elles sont attendues. Sous le volet Résumé, vous devriez voir **état de la tâche : fin**. Dans la liste déroulante, le fichier journal de **Préparer le domaine** doit être visible. S’il n’est pas le cas, sélectionnez le fichier journal et puis cliquez sur **Afficher le journal** pour consulter les actions détaillées et les résultats des tâches. Cliquez sur **Terminer** pour terminer la préparation du domaine.
