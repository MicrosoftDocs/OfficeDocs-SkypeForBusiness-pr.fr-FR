# <a name="prepare-current-forrest-intro"></a>Préparer la forêt actuelle (introduction)
 
Pour commencer la tâche **Préparer la forêt**, cliquez sur **Suivant**. L’exécution **de la** tâche Préparer la forêt crée des paramètres globaux et des groupes universels à utiliser Skype Entreprise Server 2015.
