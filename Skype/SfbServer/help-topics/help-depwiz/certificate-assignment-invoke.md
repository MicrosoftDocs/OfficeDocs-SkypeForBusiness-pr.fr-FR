# <a name="certificate-assignment-invoke"></a>Assignation de certificat (appel)
 
La page **Exécution de commandes** affiche les actions effectuées par les tâches pour assigner le certificat au serveur. Lorsque les tâches sont terminées, vous pouvez afficher le journal qui décrit en détail le processus d’affectation de certificat. Pour afficher le fichier journal, cliquez sur **Afficher le journal**.
