# <a name="certificate-assignment-invoke"></a>Assignation du certificat (appeler)
 
La page de **L’exécution de commandes** affiche les mesures prises par les tâches pour l’attribution du certificat au serveur. Lorsque les tâches sont terminées, vous pouvez afficher le journal qui détaille le processus d’affectation de certificat. Pour afficher le fichier journal, cliquez sur **Afficher le journal**.
