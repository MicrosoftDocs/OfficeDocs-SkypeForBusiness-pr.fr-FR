# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services (groupes)

Dans le champ **Spécifiez le groupe en lecture seule**, spécifiez le groupe auquel vous souhaitez accorder l’accès en lecture seule aux rapports d’analyse. Utilisez le format &lt;domaine&gt;\\&lt;groupe accès&gt;. Par exemple : contoso\RTCUniversalReadOnlyAdmins.
  
Affectation d’un groupe d’accès en lecture seule est facultatif. Si vous ne souhaitez pas autoriser l’accès en lecture seule, vous pouvez le configurer ultérieurement directement dans SQL Server Reporting Services.
