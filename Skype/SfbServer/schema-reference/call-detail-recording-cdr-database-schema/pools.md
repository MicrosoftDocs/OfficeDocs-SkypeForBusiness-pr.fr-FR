# <a name="pools-table"></a>Table Pools
 
La table de Pools est prise en charge une table qui stocke des informations sur le regroupement de divers. Chaque enregistrement de la table représente un pool.
  
|**Colonne**|**Type de données**|**Index de la clé**|**Détails**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce pool.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |Nom de domaine complet du pool.  <br/> |
