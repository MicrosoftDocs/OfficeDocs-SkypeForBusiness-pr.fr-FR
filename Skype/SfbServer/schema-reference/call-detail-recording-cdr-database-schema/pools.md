# <a name="pools-table"></a>Table Pools
 
La table pools est une table de prise en charge qui stocke des informations sur les divers pools. Chaque enregistrement de la table représente un pool.
  
|**Colonne**|**Type de données**|**Clé/Index**|**Détails**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce pool.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |Nom de domaine complet du pool.  <br/> |
