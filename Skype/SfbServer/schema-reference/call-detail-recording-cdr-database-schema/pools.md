# <a name="pools-table"></a>Table Pools
 
Le tableau de Pools est une table de prise en charge qui stocke des informations sur le pool différents. Chaque enregistrement de la table représente un pool.
  
|**Colonne**|**Type de données**|**Clé/Index**|**Détails**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce pool.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |Nom de domaine complet de pool.  <br/> |
