# <a name="user-view"></a>Affichage utilisateur
 
L’affichage utilisateur stocke les informations sur les utilisateurs qui ont participé à des appels ou des sessions ayant des enregistrements dans la base de données. Cet affichage a été présenté dans Microsoft Lync Server 2013.
  
|**Colonne**|**Type de données**|**Détails**|
|:-----|:-----|:-----|
|UserId  <br/> |int  <br/> |Numéro unique identifiant cet utilisateur.  <br/> |
|UserUri  <br/> |nvarchar (450)  <br/> |URI de l’utilisateur.  <br/> |
|TenantKey  <br/> |identificateur  <br/> |Client de l’utilisateur. Pour plus d’informations, voir la [table locataires](tenants.md) . <br/> |
|UriType  <br/> |nvarchar(256)  <br/> |Type d’URI de l’utilisateur. Pour plus d’informations, voir la [table UriTypes](uritypes.md) . <br/> |
