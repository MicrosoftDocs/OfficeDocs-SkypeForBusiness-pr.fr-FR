# <a name="servers-table"></a>Table Servers
 
La table serveurs est une table de prise en charge qui stocke des informations sur les différents serveurs. Chaque enregistrement de la table représente un serveur.
  
|**Colonne**|**Type de données**|**Clé/Index**|**Détails**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce serveur.  <br/> |
|**ServerFQDN** <br/> |nvarchar(256)  <br/> | <br/> |Nom de domaine complet du serveur.  <br/> |
