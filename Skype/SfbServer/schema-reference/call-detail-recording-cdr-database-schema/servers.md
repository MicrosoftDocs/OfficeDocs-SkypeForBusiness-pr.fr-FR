# <a name="servers-table"></a>Table Servers
 
Le tableau de serveurs est une table de prise en charge qui stocke des informations sur les différents serveurs. Chaque enregistrement de la table représente un seul serveur.
  
|**Colonne**|**Type de données**|**Clé/Index**|**Détails**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce serveur.  <br/> |
|**Nom_complet_serveur** <br/> |nvarchar(256)  <br/> | <br/> |Nom de domaine complet de serveur.  <br/> |
