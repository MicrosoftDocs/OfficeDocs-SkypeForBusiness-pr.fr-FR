# <a name="servers-table"></a>Table Servers
 
La table de serveurs est une table de support qui stocke des informations sur les différents serveurs. Chaque enregistrement de la table représente un serveur.
  
|**Colonne**|**Type de données**|**Index de la clé**|**Détails**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Principal  <br/> |Numéro unique qui identifie ce serveur.  <br/> |
|**Nom_complet_serveur** <br/> |nvarchar(256)  <br/> | <br/> |Nom de domaine complet du serveur.  <br/> |
