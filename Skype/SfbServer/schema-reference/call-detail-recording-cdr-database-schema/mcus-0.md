# <a name="mcus-view"></a>Affichage des MCU
 
L’affichage des MCU stocke les informations sur les MCU ayant participé à des sessions de conférence. Cet affichage a été présenté dans Microsoft Lync Server 2013.
  
|**Colonne**|**Type de données**|**Détails**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |Numéro unique identifiant le MCU.  <br/> |
|**McuUri** <br/> |nvarchar (450)  <br/> |URI de la MCU.  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |Type d’URI MCU. Pour plus d’informations, voir la [table UriTypes](uritypes.md) . <br/> |
