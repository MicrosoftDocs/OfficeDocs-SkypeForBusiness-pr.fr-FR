# <a name="uritypes-table"></a>Table UriTypes
 
La UriTypes Table contient les différents types d’URI (Uniform resource identifier) surveillés dans Skype pour Business Server 2015.
  
|**Colonne**|**Type de données**|**Index de la clé**|**Détails**|
|:-----|:-----|:-----|:-----|
|**UriTypeId** <br/> |tinyint  <br/> |Principal  <br/> |Identificateur unique assigné à un type d’URI.  <br/> |
|**UriType** <br/> |nvarchar(256)  <br/> || Descriptions des différents types d’URI. Les valeurs autorisées sont les suivantes : <br/>  0 - Uri de téléphone <br/>  1 : Uri de l’utilisateur <br/> |
