# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Table de fabricants dans Skype pour Business Server 2015
 
La table de fabricants est une table de prise en charge. Chaque enregistrement contient des informations sur le fabricant d’un périphérique (téléphone de bureau).
  
|**Colonne**|**Type de données**|**Index de la clé**|**Détails**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce fabricant.  <br/> |
|**Fabricant** <br/> |nvarchar(256)  <br/> | <br/> |Nom du fabricant.  <br/> |
