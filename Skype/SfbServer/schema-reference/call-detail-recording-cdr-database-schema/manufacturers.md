# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Tableau constructeurs dans Skype entreprise Server 2015
 
La table Manufacturers est une table de prise en charge. Chaque enregistrement stocke les informations relatives à un fabricant de périphériques (téléphone de bureau).
  
|**Colonne**|**Type de données**|**Clé/Index**|**Détails**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce fabricant.  <br/> |
|**Fabricant** <br/> |nvarchar(256)  <br/> | <br/> |Nom du fabricant.  <br/> |
