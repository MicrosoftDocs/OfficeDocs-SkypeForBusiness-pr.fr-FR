# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Table Manufacturers dans Skype pour Business Server 2015
 
Le tableau des fabricants est une table de prise en charge. Chaque enregistrement stocke des informations sur un fabricant (téléphone de bureau).
  
|**Colonne**|**Type de données**|**Clé/Index**|**Détails**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce fabricant.  <br/> |
|**Fabricant** <br/> |nvarchar(256)  <br/> | <br/> |Nom du fabricant.  <br/> |
