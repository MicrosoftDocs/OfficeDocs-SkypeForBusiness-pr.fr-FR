# <a name="macaddress-table"></a>Table MacAddress
 
La table de l’adresse MAC est une table de prise en charge. Chaque enregistrement représente une source.
  
|**Colonne**|**Type de données**|**Index de la clé**|**Détails**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant l’adresse Mac.  <br/> |
|**Adresse MAC** <br/> |varchar(256)  <br/> |Unique  <br/> |Chaîne de l’adresse MAC.  <br/> |
