|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant l’adresse Mac.  <br/> |
|**MacAddress** <br/> |varchar (256)  <br/> |Différent  <br/> |Chaîne d’adresses Mac.  <br/> |
