# <a name="appliedbandwidthsource-table"></a>Table AppliedBandwidthSource
 
La table AppliedBandwidthSource est un tableau de prise en charge. Chaque enregistrement représente une source.
  
|**Colonne**|**Type de données**|**Index de la clé**|**Détails**|
|:-----|:-----|:-----|:-----|
|**AppliedBandwidthSourceKey** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant la source.  <br/> |
|**AppliedBandwidthSource** <br/> |varchar(256)  <br/> |Unique  <br/> |Il s’agit de la source de l’embout de la bande passante est imposée. Il décrit la limite de bande passante provenance (par exemple, « Policy Server », « Arrêter le serveur » ou « Modalité »).  <br/> |
