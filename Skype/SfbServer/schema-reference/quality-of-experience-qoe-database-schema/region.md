# <a name="region-table"></a>Table Region
 
La table région est une table de prise en charge. Chaque enregistrement représente un pays/région définie dans le paramètre de configuration réseau.
  
|**Colonne**|**Type de données**|**Index de la clé**|**Détails**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant le pays ou la région.  <br/> |
|**RegionName** <br/> |nvarchar (128)  <br/> |Unique  <br/> |Le nom du pays/de la région.  <br/> |
