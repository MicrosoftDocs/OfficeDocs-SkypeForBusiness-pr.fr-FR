# <a name="region-table"></a>Table Region
 
La table Region est une table de prise en charge. Chaque enregistrement représente un pays/région définie dans les paramètres de configuration réseau.
  
|**Colonne**|**Type de données**|**Clé/Index**|**Détails**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant la pays/région.  <br/> |
|**RegionName** <br/> |nvarchar (128)  <br/> |Unique  <br/> |Nom de pays/région.  <br/> |
