# <a name="payloaddescription-table"></a>Table PayloadDescription
 
La table PayloadDescription est un tableau de prise en charge. Chaque enregistrement représente un Codec, qui est utilisé dans une session audio ou vidéo.
  
|**Colonne**|**Type de données**|**Index de la clé**|**Détails**|
|:-----|:-----|:-----|:-----|
|**PayloadDescriptionKey** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant le Codec.  <br/> |
|**PayloadDescription** <br/> |nvarchar(256)  <br/> |Unique  <br/> |Nom du codec.  <br/> |
