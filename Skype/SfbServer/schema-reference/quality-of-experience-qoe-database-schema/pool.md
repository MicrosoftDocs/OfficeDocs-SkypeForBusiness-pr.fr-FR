# <a name="pool-table"></a>Table Pool
 
Le tableau de réserve est une table de prise en charge qui stocke des informations sur les divers pools de front-end. Chaque enregistrement de la table représente un pool.
  
|**Colonne**|**Type de données**|**Clé/Index**|**Détails**|
|:-----|:-----|:-----|:-----|
|**PoolKey** <br/> |int  <br/> |Principal  <br/> |Numéro unique identifiant ce pool.  <br/> |
|**PoolName** <br/> |nvarchar(256)  <br/> |Différent  <br/> |Nom de domaine complet du pool.  <br/> |
