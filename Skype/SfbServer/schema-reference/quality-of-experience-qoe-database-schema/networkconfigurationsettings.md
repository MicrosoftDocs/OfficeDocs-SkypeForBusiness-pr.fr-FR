# <a name="networkconfigurationsettings-view"></a>Affichage NetworkConfigurationSettings
 
Le mode NetworkConfigurationSettings stocke les informations relatives à la configuration du réseau. Cet affichage a été présenté dans Microsoft Lync Server 2013.
  
|**Colonne**|**Type de données**|**Détails**|
|:-----|:-----|:-----|
|SubnetIP  <br/> |int  <br/> |Adresse IP du sous-réseau (seules les adresses IPv4 sont prises en charge).  <br/> |
|UserSiteName  <br/> |nvarchar(128  <br/> |Nom du site de l’utilisateur.  <br/> |
|RegionName  <br/> |nvarchar(128  <br/> |Nom du pays/de la région de l’utilisateur.  <br/> |
