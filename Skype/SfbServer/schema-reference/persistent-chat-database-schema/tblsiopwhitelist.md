# <a name="tblsiopwhitelist"></a>tblSiopWhiteList
 
tblSiopWhiteList est la liste des compléments enregistrés qui peuvent être associés à des nœuds.
  
**Celles**

|**Colonne**|**Type**|**Description**|
|:-----|:-----|:-----|
|siopID  <br/> |GUID, pas null  <br/> |GUID du complément.  <br/> |
|siopName  <br/> |nvarchar (50), pas null  <br/> |Nom d’affichage du complément.  <br/> |
|siopUrl  <br/> |nvarchar (255), pas null  <br/> |URL du complément.  <br/> |
   
**Clé**

|**Colonne**|**Description**|
|:-----|:-----|
|siopID  <br/> |Clé primaire.  <br/> |
