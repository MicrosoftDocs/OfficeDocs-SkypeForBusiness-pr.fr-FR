# <a name="tblsiopwhitelist"></a>tblSiopWhiteList
 
tblSiopWhiteList est la liste des compléments enregistrés qui peuvent être associées à des nœuds.
  
**Colonnes**

|**Colonne**|**Type de**|**Description**|
|:-----|:-----|:-----|
|siopID  <br/> |GUID, pas null  <br/> |GUID de la macro complémentaire.  <br/> |
|siopName  <br/> |nvarchar (50), non null  <br/> |Nom complet de la macro complémentaire.  <br/> |
|siopUrl  <br/> |nvarchar (255), non null  <br/> |URL de la macro complémentaire.  <br/> |
   
**Clé**

|**Colonne**|**Description**|
|:-----|:-----|
|siopID  <br/> |Clé primaire.  <br/> |
