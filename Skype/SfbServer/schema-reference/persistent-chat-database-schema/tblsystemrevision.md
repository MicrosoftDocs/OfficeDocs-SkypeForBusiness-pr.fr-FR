# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision contient le numéro de révision utilisé avec la table tblAdminLock pour assurer la cohérence entre plusieurs clients administrateur.
  
**Colonnes**

|**Colonne**|**Type**|**Description**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint, non null  <br/> |Nombre de révisions.  <br/> |
