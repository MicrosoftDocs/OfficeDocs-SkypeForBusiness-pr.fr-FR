## <a name="configure-a-quality-of-service-policy-in-skype-for-business-server-for-your-conferencing-application-and-mediation-servers"></a><span data-ttu-id="ee136-159">Configurer une stratégie de qualité de service dans Skype Entreprise Server pour vos serveurs de conférence, d’applications et de médiation</span><span class="sxs-lookup"><span data-stu-id="ee136-159">Configure a Quality of Service policy in Skype for Business Server for your Conferencing, Application, and Mediation servers</span></span>

<span data-ttu-id="ee136-p108">La configuration de plages de ports facilite l’utilisation de la qualité de service en s’assurant que tout le trafic d’un type donné (par exemple, tout le trafic audio) passe par le même ensemble de ports. Cela permet au système d’identifier et de marquer facilement un paquet donné : si le port 49152 est réservé au trafic audio, les paquets qui passent par le port 49152 peuvent être marqués avec un code DSCP qui indique qu’il s’agit d’un paquet audio. Cela permet aux routeurs d’identifier le paquet comme étant un paquet audio, et de lui donner une priorité supérieure aux paquets non marqués (par exemple les paquets utilisés pour copier un fichier d’un serveur sur un autre).</span><span class="sxs-lookup"><span data-stu-id="ee136-p108">Configuring port ranges facilitates the use of Quality of Service by ensuring that all traffic of a specified type (for example, all audio traffic) travels through the same set of ports. In turn, this enables routers to identify the packet as an audio packet, and give it higher priority than unmarked packets (such as packets used to copy a file from one server to another).</span></span>

<span data-ttu-id="ee136-163">Cependant, la limitation d’un ensemble de ports à un type de trafic spécifique ne permet pas que ces paquets soient automatiquement marqués avec le code DSCP approprié.</span><span class="sxs-lookup"><span data-stu-id="ee136-163">However, simply restricting a set of ports to a specific type of traffic does not result in packets traveling through those ports being marked with the appropriate DSCP code.</span></span> <span data-ttu-id="ee136-164">En plus de définir des plages de ports, vous devez également créer des stratégies de qualité de service qui spécifient le code DSCP à associer à chaque plage de ports.</span><span class="sxs-lookup"><span data-stu-id="ee136-164">In addition to defining port ranges, you must also create Quality of Service policies that specify the DSCP code to be associated with each port range.</span></span> <span data-ttu-id="ee136-165">Pour Skype Entreprise Server, cela signifie généralement la création de deux stratégies : une pour l’audio et une pour la vidéo.</span><span class="sxs-lookup"><span data-stu-id="ee136-165">For Skype for Business Server, that typically means creating two policies: one for audio and one for video.</span></span>
