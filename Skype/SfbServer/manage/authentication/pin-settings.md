# <a name="manage-pin-settings-in-skype-for-business-server"></a>Gérer les paramètres de code confidentiel dans Skype pour Business Server
 
**Résumé :** Gérer les paramètres de code confidentiel dans Skype pour Business Server.
  
Utilisez les procédures décrites dans les sections suivantes pour gérer les codes confidentiels dans Skype pour Business Server.
  
Vous pouvez gérer Skype pour les stratégies de code confidentiel de Business Server à partir de Skype pour le panneau de configuration serveur Business ou Skype pour Business Server Management Shell. Appliquez les procédures ci-dessous pour configurer des stratégies de code confidentiel pour votre organisation.
  
- [Afficher les informations de stratégie de code confidentiel dans Skype pour Business Server](view-pin-policy-information.md)
    
- [Créer une nouvelle stratégie de code confidentiel dans Skype pour Business Server](create-a-new-pin-policy.md)
    
- [Modifier une stratégie de code confidentiel existante dans Skype pour Business Server](modify-an-existing-pin-policy.md)
    
- [Supprimer une stratégie de code confidentiel dans Skype pour Business Server](delete-a-pin-policy.md)
    
- [Affecter une stratégie de code confidentiel par utilisateur dans Skype pour Business Server](assign-a-per-user-pin-policy.md)
    
Utilisez les procédures suivantes pour gérer les codes confidentiels de conférence rendez-vous des utilisateurs à partir de la section **utilisateurs** de Skype pour le panneau de configuration serveur Business.
  
- [Définir des conférences rendez-vous d’un utilisateur code confidentiel dans Skype pour Business Server](set-a-user-s-dial-in-conferencing-pin.md)
    
- [Afficher les informations de code confidentiel dans Skype pour Business Server utilisateur](view-user-pin-information.md)
    
- [Verrouiller ou déverrouiller un code confidentiel dans Skype pour Business Server](lock-or-unlock-a-user-pin.md)
