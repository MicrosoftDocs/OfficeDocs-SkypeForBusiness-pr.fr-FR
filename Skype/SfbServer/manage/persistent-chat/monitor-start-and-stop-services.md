# <a name="monitor-start-and-stop-the-persistent-chat-services-in-skype-for-business-server-2015"></a>Surveillance, lancement et arrête des services de conversation permanente dans Skype Entreprise Server 2015
 
**Résumé :** Découvrez comment démarrer, arrêter et contrôler les services de conversation permanents dans Skype pour Business Server 2015.
  
Le Chat permanente des services et les services de conformité de conversation permanents font partie de la Skype pour la topologie de serveur d’entreprise peuvent par conséquent être surveillés, arrêtés et démarrés à l’aide des applets de commande suivantes :
  
|||
|:-----|:-----|
|Get-CsWindowsService  <br/> |Renvoie des informations détaillées sur Skype pour les composants Business Server 2015 qui s’exécutent en tant que services Windows.  <br/> |
|Démarrer-CsWindowsService  <br/> |Démarre le service.  <br/> |
|Stop-CsWindowsService  <br/> |Arrête le service.  <br/> |
   
Pour plus d’informations sur la façon d’utiliser les applets de commande, reportez-vous à la section [Skype pour Business Server 2015 Management Shell](../management-shell.md).
