# <a name="call-diagnostic-reports-per-user-in-skype-for-business-server"></a>Appeler des rapports de Diagnostic (par utilisateur) dans Skype pour Business Server
 
**Résumé :** Découvrez les utilisateur appel rapports de Diagnostic utilisés dans Skype pour Business Server.
  
Les rapports de diagnostic des appels fournissent des informations par utilisateur concernant les sessions de conférence et P2P ayant échoué. À l’heure actuelle, il n’existe qu’un seul rapport.
  
## <a name="in-this-section"></a>Contenu de cette section

 [User Activity Report dans Skype pour Business Server](user-activity-report.md) Fournit des informations sur les activités de conférence et d’égal à égal pour chacun de vos utilisateurs.
