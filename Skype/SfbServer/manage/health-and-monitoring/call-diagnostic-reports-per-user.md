# <a name="call-diagnostic-reports-per-user-in-skype-for-business-server-2015"></a>Rapports de diagnostic des appels (par utilisateur) Skype Entreprise Server 2015
 
**Résumé :** Obtenir des informations sur l’utilisateur appeler rapports de Diagnostic utilisé dans Skype pour Business Server 2015.
  
Les rapports de diagnostic des appels fournissent des informations par utilisateur concernant les sessions de conférence et P2P ayant échoué. À l’heure actuelle, il n’existe qu’un seul rapport.
  
## <a name="in-this-section"></a>Contenu de cette section

 [Rapport d’activité de l’utilisateur dans Skype pour Business Server 2015](user-activity-report.md) Fournit des informations sur les activités de peer-to-peer et de conférence pour chacun de vos utilisateurs.
