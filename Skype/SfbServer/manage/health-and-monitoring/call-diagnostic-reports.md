# <a name="call-diagnostic-reports-in-skype-for-business-server-2015"></a>Rapport de diagnostic des appels dans Skype Entreprise Server 2015
 
**Summary:** Learn about the multi-user Call Diagnostic Reports used in Skype for Business Server 2015.
  
Les rapports de diagnostic des appels fournissent des informations récapitulatives et des données de diagnostic relatives aux sessions P2P et de conférence ayant échoué.
  
## <a name="in-this-section"></a>Contenu de cette section

- [Call Diagnostic Summary Report in Skype for Business Server 2015](summary-report.md) Provides an overall summary of failed peer-to-peer sessions and conference sessions. Les sessions P2P sont des sessions impliquant seulement deux utilisateurs. Les sessions de conférence impliquent trois participants ou plus.
    
- [Peer-to-Peer Activity Diagnostic Report in Skype for Business Server 2015](peer-to-peer-activity-diagnostic-report.md) Provides an overall trend view of failed peer-to-peer sessions. Une session P2P implique uniquement deux participants.
    
- [Conference Diagnostic Report in Skype for Business Server 2015](conference-diagnostic-report.md) Provides an overall trend view of failed conferencing sessions and trend views for each conference modality. Les sessions de conférence impliquent au moins trois participants.
    
- [Top Failures Report in Skype for Business Server 2015](top-failures-report.md) Provides a list of the most frequent failures and their trends over time.
    
- [Failure Distribution Report in Skype for Business Server 2015](failure-distribution-report.md) Provides an analysis of failed sessions.
    
- [Failure List Report in Skype for Business Server 2015](failure-list-report.md) Provides detailed information about the individual participants involved in a failed conference.
    
- [Diagnostic Report in Skype for Business Server 2015](diagnostic-report.md) Provides diagnostic and troubleshooting information (including SIP response codes and diagnostic headers and IDs) for failed sessions.
