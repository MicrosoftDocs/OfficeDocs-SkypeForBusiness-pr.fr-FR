# <a name="call-detail-recording-cdr-in-skype-for-business-server"></a>(DCR) dans Skype pour Business Server des détails des appels
 
**Résumé :** Découvrez les détails des appels (CDR) les enregistrements utilisés dans Skype pour Business Server.
  
L’enregistrement des détails des appels permet d’enregistrer les informations de diagnostic et d’utilisation relatives aux activités P2P, notamment la messagerie d’instance, les appels VoIP (protocole voix sur IP), le partage d’application, le transfert de fichiers et les réunions. Les données d’utilisation permettent de calculer le retour sur investissement alors que les données de diagnostic permettent de résoudre les problèmes spécifiques aux activités et réunions P2P. Lorsque vous installez Skype pour Business Server, vous allez installer également une collection de paramètres de configuration globaux CDR prédéfinie. Utilisez les rubriques de cette section pour configurer l’enregistrement des détails des appels.
