# Configurer votre ordinateur pour Skype pour la gestion de l'entreprise Online à l'aide de Windows PowerShell

> [!IMPORTANT]
> Cet article a été traduit automatiquement, voir l'[avertissement](ebf5adf0-d848-4e99-baf4-387d1711e1d0.md#MT_Footer). Vous pouvez consulter la version en anglais de cet article [ici](https://support.office.com/en-us/article/ebf5adf0-d848-4e99-baf4-387d1711e1d0). 
  
Gestion à distance de Skype Entreprise Online à l'aide de Microsoft PowerShell est pris en charge uniquement sur les ordinateurs 64 bits exécutant un des systèmes d'exploitation suivants :
  
- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
Outre le système d'exploitation pris en charge, l'ordinateur doit également exécuter les éléments suivants :
  
- PowerShell
    
- Module Skype Entreprise Online Connector
    
Vous pouvez trouver des informations sur le téléchargement et installation de ces éléments dans les rubriques suivantes :
  
- [Téléchargez et installez le Skype pour module Business Connector en ligne](download-and-install-the-skype-for-business-online-connector-module.md)
    
- [Télécharger et installer Windows PowerShell 3.0](download-and-install-windows-powershell-3-0.md)
